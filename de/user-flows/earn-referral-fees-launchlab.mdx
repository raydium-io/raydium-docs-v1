## Wie es funktioniert

Wenn Sie einen Token-Seite-Link teilen, der Ihre Wallet-Adresse als Referrer enthält, generiert jeder, der über Ihren Link tradet, eine **0,1%-Provision**, die direkt an Ihre Wallet gesendet wird. Die Provision wird:

- In SOL (oder dem Quote-Token des Pools) denominiert.
- Automatisch auf jeder Transaktion an Ihre Wallet gesendet – kein separater Claim-Schritt erforderlich.
- Auf Käufe und Verkäufe auf der Bonding-Kurve angewendet, solange der Token noch nicht abgestuft ist.

### Sitzungsumfang

Ein Referral-Link zeigt auf einen bestimmten Token, aber wenn der Benutzer, der Ihren Link angeklickt hat, dann *andere* LaunchLab-Tokens in derselben Browser-Sitzung tradet, können Sie auch Provisionen für diese Trades verdienen. Dies erweitert Ihr Verdienstpotenzial über den einzelnen Token hinaus, den Sie ursprünglich geteilt haben.

## Erstellen Sie Ihren Referral-Link

1. Verbinden Sie Ihre Wallet auf [raydium.io/launchlab](https://raydium.io/launchlab).
2. Navigieren Sie zur Detail-Seite eines beliebigen LaunchLab-Tokens.
3. Klicken Sie auf die Schaltfläche **Teilen** auf der Token-Seite.
4. Kopieren Sie den erstellten Link – er enthält Ihre Wallet-Adresse als Referrer-Parameter.
5. Teilen Sie den Link überall, wo Ihr Publikum ist: in sozialen Medien, Communities, Newslettern.

<Note>
  Ihre Wallet muss verbunden sein, wenn Sie auf Teilen klicken. Der Link wird keinen gültigen Referrer-Parameter enthalten, wenn er ohne verbundene Wallet erstellt wird.
</Note>

## Was Sie verdienen

| Szenario | Provision |
|---|---|
| Empfänger kauft den verlinkten Token | 0,1% des Trade-Wertes |
| Empfänger verkauft den verlinkten Token | 0,1% des Trade-Wertes |
| Empfänger tradet andere LaunchLab-Tokens in derselben Sitzung | 0,1% dieser Trades |

Einnahmen werden in Echtzeit gesendet – es gibt kein Depot oder ausstehenden Saldo zum Beanspruchen.

## Vergleich: LaunchLab-Referral vs. Swap-Referral

| | LaunchLab-Referral | Swap-Referral (Blinks) |
|---|---|---|
| Rate | 0,1% | 1% |
| Umfang | LaunchLab-Token-Seiten | Raydium-Swap-Oberfläche |
| Zahlung | Automatisch, pro Trade | Manueller Claim oder automatisch abhängig von Integration |
| Setup | Share-Schaltfläche auf Token-Seite | Erstellen über [referrals-and-blinks](/de/user-flows/referrals-and-blinks) |
