## Unterstützte Sicherheiten

Raydium Perps unterstützt Einzahlungen von großen Blockchains und Collateral-Assets wie SOL, USDC, USDT, ETH, BNB, WBTC, YUSD und USD1, wo verfügbar.

Die genaue Liste, Blockchains und Limits werden von der zugrunde liegenden Börse kontrolliert. Nutzen Sie die App als Informationsquelle, bevor Sie einzahlen.

## Globale Limits

Globale Limits gelten für die gesamte Börse und werden von allen Benutzern geteilt. Die referenzierte Raydium-Seite listet Beispiele auf wie:

| Collateral | Blockchain-Beispiele |
|---|---|
| USDT | Arbitrum, BSC, Ethereum |
| ETH | Arbitrum, Base, Ethereum |
| BNB | BSC |
| USD1 | BSC, Ethereum |
| YUSD | BSC, Ethereum |
| WBTC | Ethereum |
| SOL | Solana |

Wenn ein globales Limit ausgeschöpft ist, können Neueinzahlungen für diesen Asset möglicherweise nicht verfügbar sein, auch wenn Ihr Wallet unter seinem eigenen Limit liegt.

## Benutzerspezifische Limits

Benutzerspezifische Limits gelten für jedes Konto oder Wallet. Die referenzierte Raydium-Seite listet benutzerspezifische Limits für Assets wie USDT, ETH, SOL, BNB, YUSD, WBTC und USD1 auf.

Überprüfen Sie den Einzahlungsbildschirm, bevor Sie Gelder senden. Wenn die App einen niedrigeren verfügbaren Einzahlungsbetrag anzeigt als Ihr Wallet-Guthaben, erreichen Sie möglicherweise ein benutzerspezifisches oder globales Collateral-Limit.

## Einzahlungsablauf

1. Öffnen Sie [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC).
2. Verbinden Sie Ihr Wallet und starten Sie den Einzahlungsablauf.
3. Wählen Sie das Collateral-Asset und die Quellblockchain aus.
4. Überprüfen Sie den Betrag, das Zielkonto und alle Warnungen der Börse.
5. Bestätigen Sie die Einzahlung.
6. Warten Sie, bis der Kontostand und die verfügbare Marge aktualisiert werden.
