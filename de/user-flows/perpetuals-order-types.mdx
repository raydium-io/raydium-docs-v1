## Ordertypen

Raydium Perps unterstützt folgende Ordertypen:

| Ordertyp | Funktion | Wann nutzen |
|---|---|---|
| **Market** | Kauft oder verkauft sofort zum besten verfügbaren Preis im Orderbuch. Nicht erfüllte Restmengen werden gelöscht, wenn das Buch die volle Menge nicht erfüllen kann. | Sie wünschen sofortige Ausführung und akzeptieren mögliche Slippage. |
| **Limit** | Kauft oder verkauft zum angegebenen Limitpreis oder besser. Die Order wird nur erfüllt, wenn der Markt diesen Preis erreicht. | Sie wünschen Preiskontrolle und können auf eine Erfüllung warten. |
| **Stop Market** | Bleibt inaktiv, bis der Trigger-Preis erreicht wird, dann wird sie zur Market-Order. | Sie wünschen einen Ausführungs-Trigger und bevorzugen schnelle Ein- oder Ausstiegsumsetzung nach Trigger. |
| **Stop Limit** | Bleibt inaktiv, bis der Trigger-Preis erreicht wird, dann wird sie zur Limit-Order. | Sie wünschen einen Trigger, möchten aber noch einen Limitpreis festlegen. |
| **Scale** | Teilt eine größere Order in mehrere kleinere Limit-Orders über eine Preisspanne auf. | Sie möchten sich graduell ein- oder aussteigen. |

## Order-Flags

Order-Flags ändern das Verhalten von Limit-ähnlichen Orders.

| Flag | Verhalten |
|---|---|
| **IOC** | Immediate-or-cancel. Erfüllt so viel wie möglich sofort zu Ihrem Limitpreis oder besser, dann wird der Rest gelöscht. |
| **FOK** | Fill-or-kill. Erfüllt die gesamte Order sofort zu Ihrem Limitpreis oder besser, oder bricht mit keiner Teilerfüllung ab. |
| **Post-only** | Fügt Liquidität als Maker hinzu. Wenn sie sofort Matching verursachen würde, wird sie stattdessen gelöscht. |
| **Reduce-only** | Verhindert, dass die Order Ihre Position vergrößert. Sie kann nur eine bestehende Position in diesem Markt reduzieren oder schließen. |

## Überprüfungen vor Order-Submission

Bestätigen Sie vor dem Absenden einer Order:

- Markt und Richtung sind korrekt.
- Größe und Hebel entsprechen Ihrer Absicht.
- Stop-Trigger und Limitpreis sind nicht vertauscht.
- Reduce-only ist aktiviert, wenn Sie nur Exposure schließen oder reduzieren möchten.
- Post-only ist aktiviert, wenn Sie Maker-Verhalten wünschen.
