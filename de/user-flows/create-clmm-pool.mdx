<Info>
  Ein CLMM-Pool erfordert zwei dauerhafte Entscheidungen: einen **Gebührenstaffelpunkt** und einen **Anfangspreis**. Sie müssen auch eine anfängliche Position öffnen, damit der Pool Liquidität hat. Seien Sie besonders vorsichtig, bevor Sie signieren.
</Info>

## Was Sie benötigen

- Ein Solana-Wallet mit:
  - Den zwei Token-Mints, die Sie kombinieren möchten.
  - Etwa 0,3 SOL für Pool- und Positions-Erstellungsmieten sowie Prioritätsgebühren.
- Eine Entscheidung zum Gebührenstaffelpunkt.
- Einen klaren Anfangsmarktpreis und erste Range.

CLMM unterstützt sowohl SPL Token als auch Token-2022 Mints.

## Gebührenstaffelpunkte

| Gebührenstaffelpunkt | Empfohlen für |
|----------|----------|
| 0,01% | Stablecoin-zu-Stablecoin-Paare |
| 0,05% | Korrelierte Paare |
| 0,25% | Die meisten volatilen Hauptpaare |
| 1% | Long-Tail oder exotische Paare |

Niedrigere Gebührenstaffelpunkte erfordern engere Preise und höheres Volumen. Höhere Gebührenstaffelpunkte tolerieren mehr Volatilität, können aber das Volumen reduzieren.

## UI-Anleitung

Auf [raydium.io/clmm/create-pool](https://raydium.io/clmm/create-pool):

1. **Wählen Sie das Tokenpaar.** Fügen Sie beide Mint-Adressen ein.
2. **Wählen Sie den Gebührenstaffelpunkt.** Verwenden Sie die UI-Empfehlung als Ausgangspunkt.
3. **Legen Sie den Anfangspreis fest.** Verwenden Sie einen bestehenden Marktpreis. Falls bereits ein Pool für das Paar vorhanden ist, vergleichen Sie ihn damit.
4. **Wählen Sie die erste Range.** Geben Sie den unteren und oberen Preis für die anfängliche Position ein.
5. **Geben Sie den Einzahlungsbetrag ein.** Die UI berechnet das erforderliche Token-Verhältnis aus Ihrer Range.
6. **Überprüfen Sie und signieren Sie.** Bestätigen Sie das Paar, den Gebührenstaffelpunkt, den Anfangspreis, die Range, die Einzahlungsbeträge und die Miete.

## Den Anfangspreis korrekt setzen

Wenn für das gleiche Paar bereits andere CLMM- oder CPMM-Pools vorhanden sind, verwenden Sie deren aktuellen Preis als Ausgangspunkt. Falls kein Raydium-Pool vorhanden ist, nutzen Sie eine zuverlässige Marktquelle.

Wenn Sie den Anfangspreis falsch setzen, wird der erste Trade den Pool wahrscheinlich arbitragieren und die unterbewertete Seite ausleeren. Es gibt keinen einfachen Reset-Button. Normalerweise müssen Sie die Liquidität abheben und den Pool neu erstellen.

## Erste Position: Range-Auswahl

- **Enge Range:** am besten für korrelierte Paare, erfordert aber aktive Verwaltung.
- **Mittlere Range:** häufig für volatile Hauptpaare wie SOL/USDC.
- **Breite Range:** niedrigere Kapitaleffizienz, aber leichter zu ignorieren.
- **Vollständige Range:** verhält sich eher wie ein CPMM, verzichtet aber auf viele Vorteile der konzentrierten Liquidität.

Wenn der aktuelle Preis außerhalb Ihrer Range liegt, befindet sich die anfängliche Position außerhalb der Range und verdient keine Gebühren, bis sich der Preis in die Range bewegt.

## Checkliste nach der Erstellung

1. **Pool-Seite wird geladen.** Bestätigen Sie die Pool-Adresse, das Paar und den Gebührenstaffelpunkt.
2. **Position erscheint.** Überprüfen Sie Ihr Raydium-Portfolio auf die Position NFT.
3. **Range ist korrekt.** Bestätigen Sie den unteren und oberen Preis.
4. **Liquidität ist nicht null.** Der Pool sollte aktive Liquidität für Ihre ausgewählte Range anzeigen.
5. **Testen Sie einen winzigen Swap.** Verifizieren Sie, dass die Route funktioniert, bevor Sie den Pool öffentlich teilen.

## Fehlerbehebung

### Die Erstellungskosten sind höher als erwartet

Breite Ranges und kleine Gebührenstaffelpunkte können mehr Tick-Array-Mieten erfordern. Verengen Sie die anfängliche Range oder fügen Sie mehr SOL hinzu, bevor Sie es erneut versuchen.

### Falsche Token-Reihenfolge

Die UI normalisiert die Token-Reihenfolge, daher kann die angezeigte Token A/Token B von der eingegebenen Reihenfolge abweichen. Überprüfen Sie das Preisangebot sorgfältig, bevor Sie signieren.

### Falscher Gebührenstaffelpunkt

Sie können den Gebührenstaffelpunkt nach der Pool-Erstellung nicht ändern. Wenn er falsch ist, schließen Sie Ihre Liquidität und erstellen Sie einen neuen Pool mit dem korrekten Staffelpunkt.

### Pool ohne Liquidität

Das Erstellen von Pool-Status ohne eine Position macht den Pool für Swaps unbrauchbar. Öffnen Sie eine anfängliche Position, bevor Sie den Pool teilen.

### Anfängliche Position außerhalb der Range

Wenn Ihre erste Range den Anfangspreis nicht einschließt, verdient die Position keine Gebühren. Öffnen Sie eine Position um den aktuellen Preis neu.
