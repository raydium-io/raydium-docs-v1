## Was Raydium Perps ist

Raydium Perps ist eine Perpetual-Futures-Handelsplattform ohne Gasgebühren unter [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC). Sie können Perpetual Futures aus einer Solana-Wallet heraus handeln, ohne Gasgebühren für jede Orderplatzierung, Anpassung oder Stornierung zu zahlen.

Das Produkt nutzt die Infrastruktur von Orderly Network als zugrunde liegenden Handelsplatz. Raydium stellt die Benutzeroberfläche zur Verfügung, während Orderly das zentrale Orderbuch, das Matching und die Perps-Mechaniken betreibt.

Wichtige Punkte:

- **Perpetual Futures:** Handeln Sie Long- oder Short-Positionen, ohne den Basiswert zu halten.
- **Gaslose Orders:** Das Platzieren und Stornieren von Orders erfordert keine Solana-Transaktion.
- **Hebelwirkung:** Die Märkte unterstützen Leverage mit Limits je nach Vermögensklasse und Venue-Einstellungen.
- **Multi-Collateral-Einzahlungen:** Hinterlegen Sie unterstützte Sicherheiten und nutzen Sie diese als Margin.
- **Selbstverwaltung:** Sie verbinden Ihre Wallet und genehmigen Kontovorgänge.

<Warning>
  Raydium Perps ist möglicherweise in bestimmten Jurisdiktionen nicht verfügbar. Nutzen Sie das Produkt nicht, wenn Ihr Standort oder Ihre lokale Gesetzgebung den Zugang zu Raydium Perps oder dem zugrunde liegenden Handelsplatz untersagt.
</Warning>

## Erste Schritte beim Handel

Sie können ein Konto finanzieren und direkt aus der Perps-App mit dem Handel beginnen.

### 1. Navigieren Sie zu Perpetuals

Öffnen Sie [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC). Überprüfen Sie die URL, bevor Sie Ihre Wallet verbinden.

### 2. Erstellen Sie ein Konto

Neue Nutzer erstellen ein Perps-Konto. Dies ist ein einmaliger Einrichtungsschritt. Überprüfen Sie die Wallet-Eingabeaufforderung, bevor Sie signieren.

### 3. Hinterlegen Sie Sicherheiten

Zahlen Sie unterstützte Sicherheiten ein. Raydium Perps unterstützt Einzahlungen von großen Blockchains und akzeptiert Sicherheiten wie natives SOL und USDC, wo verfügbar.

Bestätigen Sie nach der Einzahlung Ihr Kontoguthaben und die verfügbare Margin, bevor Sie eine Order platzieren.

### 4. Beginnen Sie mit dem Handel

Wählen Sie einen Markt, überprüfen Sie das Orderbuch, entscheiden Sie sich für einen Ordertyp, legen Sie Größe und Leverage fest und senden Sie die Order ab. Beginnen Sie in kleinem Umfang, bis Sie verstanden haben, wie Margin, Liquidationspreis und Funding die Position beeinflussen.

## Prüfung vor dem Eröffnen einer Position

- **Markt:** Bestätigen Sie, dass Sie den gewünschten Perps-Markt ausgewählt haben.
- **Richtung:** Long profitiert von steigenden Preisen; Short profitiert von fallenden Preisen.
- **Leverage:** Höheres Leverage erhöht das Liquidationsrisiko.
- **Liquidationspreis:** Überprüfen Sie, wo die Position liquidiert werden kann.
- **Ordertyp:** Market Orders priorisieren Ausführung; Limit Orders bestimmen den Preis.
- **Margin:** Stellen Sie sicher, dass Ihr Konto nach Gebühren über ausreichende verfügbare Margin verfügt.
