<Info>
  Eine LaunchLab-Plattform ist eine On-Chain-Konfiguration, die von einer Launch-Oberfläche oder einer Community genutzt wird. Plattformeinstellungen können Gebühren, Branding, Creator-Fee-Support und das Verhalten nach Migration beeinflussen.
</Info>

LaunchLab unterstützt Drittanbieter-Launch-Umgebungen über Platform PDAs. Eine Plattform kann ihre eigene Markenidentität für Token-Launches bieten und dabei die LaunchLab-Programminfrastruktur sowie die Raydium-Pool-Migration nutzen.

## Was eine Plattform steuern kann

| Einstellung | Bedeutung |
|---|---|
| Creator-Fee-Support | Legt fest, ob Token-Ersteller vor und nach der Migration Gebühren verdienen können. |
| Plattformgebühren | Bestimmt, wohin die Gebühreneinnahmen auf Plattformebene fließen. |
| Branding-Metadaten | Gibt der Plattform einen Namen, eine Beschreibung und ein Logo, das in App-Oberflächen angezeigt werden kann. |
| Launch-Gruppierung | Ermöglicht es der UI, unter derselben Plattform gestartete Tokens anzuzeigen. |
| Post-Migration-Pool-Einstellungen | Kann die CPMM-Gebührenkonfiguration bei der Migration von Launches zu CPMM definieren. |
| LP-Verteilung | Kann beeinflussen, wie LP-Tokens nach der Graduation verbrannt, Creatorern zugewiesen oder der Plattform zugewiesen werden. |

## Worauf Creator prüfen sollten

Bevor Sie einen Launch unter einer Plattform durchführen, bestätigen Sie:

- Welche Wallet die Plattformgebühren erhält.
- Ob Creator Fees aktiviert sind.
- Ob die Post-Migration-Liquidität zu CPMM oder AMM v4 geht.
- Wie LP-Tokens nach der Graduation aufgeteilt werden.
- Ob ein Fee Key NFT geprägt wird und wer dieses erhält.
- Ob die Plattform eigene Bedingungen, Richtlinien oder Zulassungsregeln hat.

<Warning>
  Plattformeinstellungen können die Creator-Ökonomie und Nutzererwartungen erheblich beeinflussen. Überprüfen Sie diese vor dem Signieren der Launch-Transaktion. Diese Seite erläutert nur Mechaniken und ist keine rechtliche, finanzielle oder Anlageberatung.
</Warning>

## Worauf Trader prüfen sollten

Beim Kauf eines LaunchLab-Tokens kann die Plattform den Gebührenpfad und die Behandlung der Post-Migration-Liquidität beeinflussen. Vor dem Handel:

- Überprüfen Sie die Plattform des Tokens und die Launch-Einstellungen.
- Bestätigen Sie den Post-Migration-Pool-Typ, falls die UI diesen anzeigt.
- Prüfen Sie, ob LP verbrannt, gesperrt oder zugewiesen wird.
- Verifizieren Sie die Token Mint von der Token-Seite, nicht nur von Social-Media-Texten.
