## Voraussetzungen

Führen Sie dies einmal aus:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Sie benötigen:

- **Node 18+** mit ESM-Unterstützung (`"type": "module"` in `package.json` oder `.mjs`-Erweiterung).
- **Ein Solana-Keypair** unter `~/.config/solana/id.json` (oder anderswo – setzen Sie `KEYPAIR`).
- **Ein Wallet mit Guthaben.** ~0,05 SOL reichen für jedes einzelne Skript in diesem Kapitel.
- **Eine RPC-URL.** Kostenlose öffentliche RPC funktioniert zum Testen; für mehr als einen schnellen Test verwenden Sie einen kostenpflichtigen Anbieter.

Setzen Sie dies einmal pro Shell:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Alle drei Skripte verwenden standardmäßig Mainnet. Um auf Devnet auszuführen, setzen Sie `RPC_URL=https://api.devnet.solana.com` und ändern Sie `cluster: "mainnet"` in `cluster: "devnet"` im Skript. Beachten Sie, dass die Likviditätsausstattung neuer Pools auf Devnet begrenzt ist; Sie müssen wahrscheinlich Ihre eigenen Test-Token erstellen.
</Info>
