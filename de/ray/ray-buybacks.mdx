## Flussübersicht

```text
Swap-Gebühr
  -> LP-Anteil
  -> RAY-Rückkauf-Anteil
  -> Treasury-Anteil, sofern zutreffend
```

Für CLMM- und CPMM-Pools ist die Handelsgebühr wie folgt aufgeteilt:

```text
84% an LPs
12% zu RAY-Rückkäufen
4% zum Treasury
```

Für Standard-AMM-v4-Pools ist die Handelsgebühr wie folgt aufgeteilt:

```text
88% an LPs
12% zu RAY-Rückkäufen
```

## Was der Rückkauf-Anteil bedeutet

Der Rückkauf-Prozentsatz bezieht sich auf die **Handelsgebühr**, nicht auf den vollständigen Handelsbetrag.

Beispiel: Ein Pool erhebt eine `0,25%`-Swap-Gebühr. Der Rückkauf-Anteil beträgt `12%` dieser Gebühr.

```text
Handelsbetrag:       1.000 USDC
Swap-Gebühr:         1.000 * 0,25% = 2,50 USDC
RAY-Rückkauf-Anteil: 2,50 * 12% = 0,30 USDC
```

Der effektive Rückkauf-Betrag beträgt in diesem Beispiel `0,03%` des Handelsbetrags.

## Einzugsadressen

Protokoll-seitige Gebühren werden von Programm-spezifischen Adressen eingezogen, bevor sie konvertiert oder weitergeleitet werden.

| Quelle | Einzugsadresse |
|---|---|
| CLMM | `projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416` |
| CPMM | `ProCXqRcXJjoUd1RNoo28bSizAA6EEqt9wURZYPDc5u` |
| Standard AMM v4 | `PNLCQcVCD26aC7ZWgRyr5ptfaR7bBrWdTFgRWwu2tvF` |

Einzugsadressen können viele verschiedene Token halten. Da jeder Pool Gebühren in den Token zahlt, die er handelt, kann eine Einzugsadresse USDC, wSOL, RAY oder andere SPL-Token halten, bevor sie weitergeleitet werden.

## Rückkauf-Verwahradresse

Zurückgekaufter RAY wird unter folgender Adresse verwahrt:

```text
DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

Sie können diese Adresse direkt auf Solscan oder mit SPL-Token-Tools inspizieren:

```bash
spl-token accounts --owner DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

## Warum Rückkäufe On-Chain sind

Der Rückkauf-Fluss von Raydium ist von öffentlichen Adressen und Transaktionsverlauf nachprüfbar.

- **Guthaben sind sichtbar:** Einzugs- und Verwahradressen können von jedermann inspiziert werden.
- **Transfers sind nachverfolgbar:** Eingangs-Gebührentransfers und ausgehende Weiterleitungstransaktionen sind öffentlich.
- **Keine private Buchhaltung erforderlich:** Die Blockchain ist das Ledger für den Fluss.

## Wie Sie einen Rückkauf-Fluss verifizieren

1. Öffnen Sie eine Raydium-Swap-Transaktion in Solscan oder Solana Explorer.
2. Identifizieren Sie das Pool-Programm und die Gebührentransfers.
3. Ordnen Sie die Bewegung der Protokoll-seitigen Gebühren der relevanten Einzugsadresse zu.
4. Inspizieren Sie ausgehende Transaktionen von der Einzugsadresse.
5. Bestätigen Sie die RAY-Ansammlung unter `DdHDoz...VEZaz`.

Für die vollständige Adressübersicht siehe [`ray/treasury`](/de/ray/treasury).
