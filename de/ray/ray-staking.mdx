  Sie können RAY in der Raydium App staken, um zusätzliche RAY-Belohnungen zu verdienen.
</Info>

## So staken Sie RAY

1. Öffnen Sie [raydium.io](https://raydium.io) und verbinden Sie Ihr Wallet.
2. Gehen Sie zur Seite **Staking**.
3. Geben Sie die Menge an RAY ein, die Sie staken möchten.
4. Klicken Sie auf **Stake**.
5. Überprüfen Sie die Transaktion in Ihrem Wallet und genehmigen Sie sie.

Nach Bestätigung der Transaktion zeigt die Staking-Seite Ihren gestakten Saldo und ausstehende Belohnungen an.

## So unstaken Sie

1. Öffnen Sie die Seite **Staking** oder **Portfolio**.
2. Klicken Sie auf die Minus-Schaltfläche für Ihre gestakte RAY-Position.
3. Geben Sie die zu unstakende Menge ein.
4. Bestätigen und signieren Sie die Transaktion in Ihrem Wallet.

## Vor dem Staking

- Halten Sie SOL in Ihrem Wallet für Transaktionsgebühren bereit.
- Überprüfen Sie den RAY Mint: `4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R`.
- Überprüfen Sie ausstehende Belohnungen und Transaktionsdetails, bevor Sie signieren.
