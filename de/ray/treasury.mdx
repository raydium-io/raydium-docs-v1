```text

  -> Programmspezifische Sammeladresse
      -> RAY-Buyback-Route
      -> USDC-Treasury-Route, falls zutreffend

Pool-Erstellungsgebühren
  -> Programmspezifische SOL-Gebührenadresse
```

Jede Adresse in diesem Ablauf ist öffentlich und überprüfbar.

## Gebührensammel-Adressen

Wenn ein Swap eine protokollseitige Gebühr zahlt, sammelt das relevante Programm diese Gebühr bei einer Sammeladresse.

| Quelle | Sammeladresse |
|---|---|
| CLMM | `projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416` |
| CPMM | `ProCXqRcXJjoUd1RNoo28bSizAA6EEqt9wURZYPDc5u` |
| Standard AMM v4 | `PNLCQcVCD26aC7ZWgRyr5ptfaR7bBrWdTFgRWwu2tvF` |

Sammeladressen können eine gemischte Menge von SPL-Token halten. Beispielsweise können Gebühren aus SOL/USDC-Pools wSOL- und USDC-Guthaben erzeugen, während SOL/RAY-Pools wSOL- und RAY-Guthaben erzeugen können.

Überprüfen Sie eine Sammeladresse mit:

```bash
spl-token accounts --owner projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416
```

## Pool-Erstellungsgebühren

Die Erstellung eines CPMM- oder Standard-AMM-v4-Pools erfordert eine SOL-denominierte Pool-Erstellungsgebühr. Diese Gebühren sammeln sich bei programmspezifischen Adressen.

| Quelle | Adresse |
|---|---|
| CPMM | `DNXgeM9EiiaAbaWvwjHj9fQQLAX5ZsfHyvmYUNRAdNC8` |
| Standard AMM v4 | `7YttLkHDoNj9wyDur5pM1ejNaAvT9X4eqaYcHQqtj2G5` |

Die CLMM-Pool-Erstellung erfordert zwar immer noch Kontogebühren, verwendet jedoch nicht den gleichen separaten SOL-Pool-Erstellungsgebühren-Pfad.

## So überprüfen Sie einen Gebührenfluss

Jeder kann Raydiums Gebührenfluss aus öffentlichen Transaktionsdaten überprüfen:

1. Ziehen Sie eine Swap-Transaktion aus einem Raydium-Pool.
2. Identifizieren Sie das Pool-Programm: CLMM, CPMM oder Standard AMM v4.
3. Überprüfen Sie die Gebührenbuchhaltung und Token-Transfers in der Transaktion.
4. Gleichen Sie die protokollseitige Bewegung mit der relevanten Sammeladresse ab.
5. Verfolgen Sie ausgehende Transaktionen von der Sammeladresse.
6. Gleichen Sie die RAY-Akkumulation mit `DdHDoz...VEZaz` oder USDC-Guthaben mit den oben aufgeführten Treasury-Adressen ab.
