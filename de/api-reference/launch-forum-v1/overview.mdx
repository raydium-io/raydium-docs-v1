Die Forum API ermöglicht Diskussionen und Community-Engagement rund um LaunchLab Pools. Posten Sie Kommentare mit optionalen Bildanhängen und rufen Sie Pool-Diskussionsstränge ab. Alle Schreiboperationen erfordern eine Authentifizierung über die Auth API.

## API Endpoints

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Authentifizierung

**Lesevorgänge (GET):** Keine Authentifizierung erforderlich.

**Schreibvorgänge (POST):** Erfordert Header `ray-token` – ein JWT, das von der Auth API nach Wallet-Signatur-Verifikation ausgestellt wird.

[Token von der Auth API abrufen](/de/api-reference/launch-auth-v1/overview)

## Request Headers

Wenn Sie Kommentare posten, fügen Sie den Authentifizierungs-Header ein:

```
ray-token: <jwt_token_from_auth_api>
```

## Funktionen

- **Kommentare posten:** Textkommentare mit optionalen Bildanhängen einreichen (max. 5 MB)
- **Diskussionen durchsuchen:** Alle Kommentare zu einem Pool mit Pagination anzeigen
- **Inhaltsmoderation:** NSFW-Erkennung und Filterung versteckter Kommentare
- **Pagination:** Abfrage mit `lastId` und `fetchType` (old/new) für effizientes Durchsuchen
- **Bild-Upload:** Kommentare unterstützen Pinata IPFS-Bild-Hosting

## Anwendungsfälle

- **Community-Diskussionen** rund um Token-Projekte
- **Projekt-Ankündigungen** und Updates über Kommentare
- **Nutzerfeedback** und Sentiment-Verfolgung
- **Sozialer Nachweis** und Community-Engagement-Metriken
- **Inhaltsmoderation** mit NSFW- und Spam-Erkennung
