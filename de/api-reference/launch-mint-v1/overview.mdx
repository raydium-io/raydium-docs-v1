Die Mint API bietet umfassenden Zugriff auf LaunchLab-Token-Erstellung, -Ermittlung und -Verwaltung. Fragen Sie Mints nach Name, Symbol oder Ersteller ab, rufen Sie Leaderboards ab und reichen Sie neue Mint-Erstellungsanfragen ein.

## API-Endpunkte

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Authentifizierung

Lesevorgänge erfordern keine Authentifizierung. Schreibvorgänge (Mint-Erstellung) erfordern einen `ray-token`-Header, der von der Auth API nach Wallet-Signatur-Verifizierung ausgestellt wird.

[Auth API Dokumentation](/de/api-reference/launch-auth-v1/overview)

## Anwendungsfälle

- **Suchen und entdecken** Sie Token nach Name oder Symbol
- **Zeigen Sie Leaderboards** an (Top-Mints nach Marktkapitalisierung, Neueste, Trends)
- **Rufen Sie Benutzerstatistiken** ab (Erstellungs- und Handelsvolumen im Zeitverlauf)
- **Fragen Sie Vesting-Zeitpläne** für Token-Unlock-Informationen ab
- **Reichen Sie Mint-Erstellungsanfragen** mit Metadaten und Bildern ein
- **Verfolgen Sie Referrer-Belohnungen** aus Plattform-Referralprogrammen
