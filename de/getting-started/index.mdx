## Die 30-Sekunden-Übersicht

**Raydium ist eine DEX auf Solana.** Sie ermöglicht es Ihnen, einen Token gegen einen anderen auszutauschen, ohne eine zentralisierte Börse zu nutzen. Dazu benötigen Sie:

1. Eine **Solana-Wallet** (Browser-Erweiterung oder mobile App) — speichert Ihren privaten Schlüssel und signiert Transaktionen.
2. Etwas **SOL** in dieser Wallet — bezahlt Transaktionsgebühren (~\$0,0005 pro Swap) und alle Token, die Sie kaufen möchten.
3. Ein Browser auf [`raydium.io`](https://raydium.io) — die offizielle Benutzeroberfläche.

Alles Weitere — Auswahl einer Wallet, deren Finanzierung, Wahl des ersten Trades — wird auf den obigen Seiten behandelt.

## Warnsignale vom ersten Tag an

Bevor Sie Geld einzahlen, prägen Sie sich diese Regeln ein:

- **Teilen Sie niemals Ihre Seed-Phrase.** Nicht mit Support, nicht mit einer „Wallet verifizieren"-Seite, nicht in DMs. Der einzige Ort, an dem eine Seed-Phrase eingegeben wird, ist während der initialen Wallet-Einrichtung.
- **Verwenden Sie nur `raydium.io`.** Phishing-Kopien imitieren die Benutzeroberfläche pixelgenau. Speichern Sie die echte URL als Lesezeichen.
- **Vertrauen Sie der Token-Adresse, nicht dem Token-Namen.** Jeder kann einen Token „USDC" erstellen. Die echte USDC-Mint ist `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`. Die Benutzeroberfläche zeigt ein **verified**-Badge neben kuratierten Mints.
- **Eine dApp sollte Sie niemals um die „Genehmigung" für Ihre gesamte Wallet bitten.** Normal Transaktionen signieren spezifische Anweisungen. Eine pauschale Genehmigung ist ein Drain.

Weitere Details unter [`getting-started/trust-and-safety`](/de/getting-started/trust-and-safety).
