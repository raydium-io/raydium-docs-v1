## Was eine mit Raydium kompatible Wallet können muss

Eine Wallet ist mit Raydium kompatibel, wenn sie alle der folgenden Dinge kann:

- Ein Solana-Keypair lokal halten und **niemals** den privaten Schlüssel vom Gerät übertragen.
- **Versionierte (V0) Transaktionen signieren.** Raydium-Swaps verwenden Address Lookup Tables und benötigen V0-Unterstützung. Wallets, die nur Legacy-Transaktionen signieren, können einen Raydium-Swap nicht abschließen.
- **Sich mit dApps über den Wallet Adapter Standard verbinden.** Dies ist das, was den „Connect Wallet"-Button auf raydium.io funktioniert.
- **Eine klare Transaktionsvorschau anzeigen**, bevor Sie genehmigen, einschließlich der aufgerufenen Programme und der Token-Bewegungen.
- **Associated Token Accounts unterstützen** (eine ATA erstellen, wenn Sie zum ersten Mal einen neuen Mint erhalten).

Dies sind grundlegende Anforderungen. Jede moderne Solana-Wallet erfüllt sie. Alles andere ist unsicher, um Raydium-Transaktionen zu signieren – Punkt.

## Features, die über die Grundanforderungen hinaus wichtig sind

Sobald eine Wallet die Grundanforderungen erfüllt, hängen die Unterschiede von Ihrem Workflow und Ihrem Risikoprofil ab. Entscheiden Sie, welche der folgenden Punkte für **Sie** wichtig sind, und wählen Sie eine Wallet, die diese handhabt:

| Feature | Wann es wichtig ist |
|---|---|
| **Hardware-Wallet-Kopplung** | Sie halten mehr als eine gelegentliche Menge. Die Software-Wallet wird zum Transaction Builder; das Hardware-Gerät hält den Schlüssel und signiert. Die meisten seriösen Solana-Wallets koppeln mindestens mit Ledger; einige koppeln mit Keystone, SafePal oder anderen Geräten. |
| **Mobile-dApp-Browser** | Sie erwarten, Raydium hauptsächlich vom Telefon aus zu nutzen. Der In-App-Browser der Wallet ist das, was raydium.io lädt und Transaktionen auf Mobile signiert. |
| **Verwaltung mehrerer Konten** | Sie trennen Gelder auf Hot/Cold/Experimental-Wallets und möchten schnell wechseln, ohne Extensions zu jonglieren. |
| **Token-2022-Erweiterungsbewusstsein** | Sie interagieren mit Pools, die Token-2022-Mints verwenden (z.B. Transfer-Fee-Tokens). Eine Wallet, die diese Erweiterungen in der Signatur-UI anzeigt, hilft Ihnen, unerwartete Gebühren vor dem Signieren zu erkennen. |
| **Priority-Fee-Anzeige** | Sie handeln während Engpässen und möchten sehen, was die Wallet für den Compute-Budget-Preis gesetzt hat, bevor Sie genehmigen. |
| **Solana Actions / Blinks-Unterstützung** | Sie möchten die One-Click-Swap-Links nutzen, die Raydium in sozialen Medien veröffentlicht. Siehe [`user-flows/referrals-and-blinks`](/de/user-flows/referrals-and-blinks). |
| **Open-Source-Codebase** | Sie möchten den Wallet-Client überprüfen können (oder sich darauf verlassen, dass die Community dies getan hat). |
| **Multi-Chain-Konten** | Sie haben Vermögen auf anderen Chains als Solana und bevorzugen eine Wallet für alle. Überprüfen Sie, dass die Multi-Chain-Unterstützung für die Chains, die Sie tatsächlich nutzen, gereift ist; sie variiert. |

### Hardware-Wallets

Eine Hardware-Wallet koppelt mit einer der oben genannten Software-Wallets (die Software-Seite wird zum Transaction Builder; das Hardware-Gerät hält den Schlüssel und signiert).

| Gerät | Typ | Offizielle Website |
|---|---|---|
| Ledger | USB / Bluetooth | [ledger.com](https://www.ledger.com) |
| Keystone | Air-gapped (QR-Code-Signierung) | [keyst.one](https://keyst.one) |

Jede Software-Wallet dokumentiert, welche Hardware-Geräte sie unterstützt; überprüfen Sie vor dem Kauf.

## Hot/Cold-Wallet-Muster

Eine gängige Praxis, die unabhängig davon funktioniert, welche Wallet Sie wählen:

- Eine **Hot Wallet** hält einen kleinen Betrag, bei dem Sie akzeptieren, dass er verloren gehen könnte – zum täglichen Swapping und zum Ausprobieren.
- Eine **Cold Wallet** (gekoppelt mit einem Hardware-Gerät) hält alles, das Sie verlieren würde, bedauern würde. Verschieben Sie Gelder nur, wenn Sie sie aktiv in der Hot Wallet benötigen.

Raydium behandelt jede Wallet als vollständig unabhängig: separate Positionen, separate LP-Tokens, separate Farm-Rewards, separate Historie. Das Wechseln von Wallets ist ein Klick entfernt.

## Wo man Wallet-Optionen findet

Das Modal „Connect Wallet" der Raydium-UI listet jede Wallet auf, die sich mit Raydium über den Wallet Adapter integriert hat. Durchsuchen Sie es, um zu sehen, was derzeit unterstützt wird. Sonanas offizielles Wallet-Verzeichnis ([`solana.com/ecosystem/explore?categories=wallet`](https://solana.com/ecosystem/explore?categories=wallet)) ist ein weiterer neutraler Ausgangspunkt.

Wenn Sie eine Wallet bewerten, arbeiten Sie diese Checkliste durch:

- [ ] Gewartet: aktuelle Releases, ein aktiver Issue-Tracker, ein Security-Disclosure-Kanal.
- [ ] Überprüft: ein veröffentlichtes Audit durch eine seriöse Firma oder Open-Source-Code mit einer aktiven Community.
- [ ] V0-Transaktionsunterstützung bestätigt (vor der Übertragung echter Gelder auf Devnet testen).
- [ ] Falls Sie ein Hardware-Gerät verwenden planen: Die Wallet dokumentiert, welche Geräte sie unterstützt und wie die Kopplung funktioniert.
- [ ] Sie haben den offiziellen Download über die verifizierte Domain des Projekts gefunden – **nicht** über eine gesponserte Anzeige oder einen weitergeleitet Link.

## Grundlagen der Wallet-Sicherheit

Unabhängig davon, welche Wallet Sie wählen:

- **Schreiben Sie die Seed-Phrase auf Papier.** Speichern Sie das Papier an zwei physisch getrennten Orten. **Geben Sie die Seed-Phrase nie irgendwo außer während einer Wallet-Wiederherstellung bei einer Neuinstallation ein.**
- **Nutzen Sie einen Password-Manager** für das Unlock-Passwort der Wallet – und halten Sie dieses Passwort völlig getrennt vom Seed.
- **Verwenden Sie ein dediziertes Browser-Profil** für Krypto-Aktivitäten mit der Mindestanzahl an installierten Erweiterungen. Phishing-Erweiterungen sind der einzige größte Ableitwert.
- **Behandeln Sie jede Wallet-Connect-Aufforderung mit Misstrauen.** Lesen Sie, was Sie signieren. Wenn eine Transaktion die Gesamtmenge eines Tokens statt eines bestimmten Betrags übertragen würde, lehnen Sie sie ab.
- **Lesezeichen `raydium.io`.** Navigieren Sie immer vom Lesezeichen aus, niemals von einer Suchmaschinenwerbung.

Weitere Sicherheitsrichtlinien finden Sie unter [`getting-started/trust-and-safety`](/de/getting-started/trust-and-safety).
