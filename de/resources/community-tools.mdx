## Was hier aufgelistet ist

Ein Tool erhält einen Platz auf dieser Seite, wenn es (a) öffentlich zugänglich ist, (b) tatsächlich gegen Raydiums Mainnet-Programme zum Zeitpunkt der Auflistung funktioniert, und (c) gepflegt wird — das heißt, das Projekt hatte einen Commit, Release oder sichtbare Aktivität in den letzten 90 Tagen.

Die folgenden Kategorien sind eine erste Taxonomie; reichen Sie ein Tool ein, das in keine davon passt, und wir fügen eine neue Kategorie hinzu.

## Ein Tool einreichen

Um einen Eintrag auf dieser Seite hinzuzufügen oder zu aktualisieren:

1. Forken Sie das [Docs-Repository](https://github.com/raydium-io) (die Docs-Quelle lebt im öffentlichen Raydium-Docs-Repository — der genaue Pfad ist in der README).
2. Bearbeiten Sie `resources/community-tools.mdx`.
3. Reichen Sie einen PR ein mit:
   - Tool-Name und Link.
   - Einzeiliger Beschreibung (was es ist, welche Kategorie).
   - GitHub- oder X-Handle des Verwalters (zur Verifizierung).
   - Eine Notiz zur Aktivität (letzter Release / letzter aussagekräftiger Commit).

Auflistungen werden überprüft auf die „öffentlich zugänglich / funktioniert tatsächlich / gepflegt"-Messlatte. Selbstbewerbungs-Auflistungen ohne funktionierende Software oder Tools, deren primärer Zweck mit der Benutzersicherheit in Konflikt steht (Drainer, Fake-Pool-Generatoren), werden nicht zusammengeführt.

## Entfernungsrichtlinie

Ein Tool wird von dieser Seite entfernt, wenn:

- Die gehostete Instanz länger als 30 Tage offline war.
- Das Repository archiviert oder gelöscht wurde.
- Der Verwalter länger als 180 Tage schweigt und das Tool aufhört zu funktionieren.
- Das Tool beginnt, sich an benutzerfeindlichem Verhalten zu beteiligen (Drainer-Scripts, irreführende UX, usw.).

Die Entfernung liegt im Ermessen der Docs-Verwalter; eine Wiederaufnahme ist willkommen, sobald das Tool wieder online und gepflegt ist.

## Sicherheitshinweis

Ein Tool, das hier aufgelistet ist und Ihre Wallet-Seed-Phrase verlangt, das Sie verpflichtet, Gelder auf eine Verwahradresse zu senden, oder das garantierte Renditen verspricht, ist **kein** Tool, das Sie verwenden sollten. Die Auflistung hier ist eine niedrige Hürde — sie bürgt nicht für Sicherheit. Überprüfen Sie immer, was Sie autorisieren, bevor Sie eine Transaktion signieren.
