## Dalam satu paragraf

Raydium Protocol adalah kumpulan smart contract permissionless dan non-custodial yang menggerakkan swap, pool, posisi likuiditas, dan peluncuran token di Solana. [Raydium.io](https://raydium.io) adalah aplikasi web resmi dan satu titik akses untuk menggunakan pasar tersebut: Anda dapat menukar token, menyediakan likuiditas, meluncurkan token melalui LaunchLab, dan mengakses Raydium Perps dari antarmuka yang sama. Raydium adalah DEX dengan TVL terbesar di Solana dengan sekitar \$1,8B per April 2026, mengarahkan mayoritas volume Solana melalui poolnya, dan menggerakkan likuiditas yang muncul di fitur swap dalam aplikasi di sebagian besar wallet Solana.

## Permukaan produk inti

Permukaan produk inti Raydium mencakup swap, likuiditas, peluncuran token, insentif, dan perpetuals. Beberapa adalah program on-chain Raydium native; Raydium Perps adalah permukaan UI yang ditenagai oleh Orderly Network.

### AMM v4 — Constant-product

Produk Raydium asli (2021). Matematika constant-product (`x × y = k`) untuk penetapan harga pool. Awalnya juga menghubungkan ke orderbook limit sentral OpenBook sehingga perdagangan bisa terisi di AMM atau orderbook, tetapi **integrasi OpenBook telah dinonaktifkan sejak saat itu** — pool tidak lagi berbagi likuiditas ke OpenBook, dan semua traffic swap saat ini mengalir hanya melalui kurva AMM. Masih berjalan dengan ~\$300M TVL dan sepenuhnya operasional, tetapi **tidak lagi direkomendasikan sebagai default untuk pool baru**: Token-2022 tidak didukung, dan CPMM lebih murah dan fleksibel untuk pasangan baru yang khas.

Lihat [`products/amm-v4/overview`](/id/products/amm-v4/overview).

### CPMM — AMM constant-product modern

Penerus AMM v4 (2024). Matematika `x × y = k` yang sama, tetapi lebih sederhana dan murah digunakan: tanpa ketergantungan OpenBook, dukungan Token-2022 native, ~4× biaya pembuatan lebih rendah, semua biaya swap masuk ke LP secara default. Direkomendasikan sebagai default untuk pool baru.

Lihat [`products/cpmm/overview`](/id/products/cpmm/overview).

### CLMM — Concentrated liquidity

Concentrated liquidity gaya Uniswap V3 (2022). LP memilih rentang harga; likuiditas mereka hanya aktif saat harga berada dalam rentang tersebut. Modal yang sama → quote jauh lebih baik untuk trader → APR biaya lebih tinggi untuk LP, dengan biaya perlu mengelola rentang secara aktif. Produk dominan untuk pool likuiditas dalam untuk tahun 2026.

Lihat [`products/clmm/overview`](/id/products/clmm/overview).

### LaunchLab — Peluncuran token

**LaunchLab** adalah tempat peluncuran token Raydium (2025). Proyek meluncurkan token di bonding curve; pembeli berdagang langsung terhadap kurva; setelah threshold graduasi tercapai, likuiditas yang terakumulasi kurva bermigrasi otomatis ke pool CPMM yang sepenuhnya berkembang. Pasar primer dan pasar sekunder bersatu.

Lihat [`products/launchlab/overview`](/id/products/launchlab/overview).

## Siapa yang menggunakan Raydium

- **Retail swapper**: menukar token melalui [raydium.io](https://raydium.io) atau melalui swap dalam aplikasi wallet apa pun yang mengarahkan melalui layer agregator Solana.
- **LP**: menyetor ke pool untuk mendapatkan biaya swap; stake di farm untuk reward tambahan.
- **Trader perpetuals**: berdagang pasar perpetual melalui Raydium Perps, ditenagai oleh Orderly Network.
- **Tim token**: meluncurkan token baru melalui LaunchLab, seed CPMM pool untuk likuiditas pasca-peluncuran, menjalankan farm untuk mendorong retensi LP.
- **Agregator**: Jupiter dan sebagian besar router pihak ketiga menjadikan Raydium sebagai tempat likuiditas utama.
- **Bot dan protokol**: arbitrase, market-making, auto-compounder, likuidasi protokol lending — semua bersusun dengan Raydium melalui CPI atau RPC langsung.

## Di mana likuiditas Raydium muncul

Anda sudah menggunakan Raydium jika Anda:

- Menggunakan swap built-in wallet Solana apa pun — sebagian besar mengarahkan melalui pool Raydium (sering kali melalui Jupiter).
- Berdagang di Jupiter — ~60% volume yang diarahkan Jupiter mengenai pool Raydium.
- Menggunakan Drift, Kamino, MarginFi, atau protokol DeFi lainnya — beberapa di antaranya mendapatkan feed harga dan rute likuidasi melalui Raydium CLMM.
- Memegang memecoin Solana — banyak diluncurkan melalui LaunchLab dan hidup di pool CPMM yang lulus.

Agregator dan wallet semua membangun di API publik Raydium (`api-v3.raydium.io`) dan kondisi pool on-chain langsung. Raydium tidak memerlukan kemitraan khusus apa pun untuk integrasi — semuanya permissionless.

## Diferensiator kunci

Dibandingkan dengan DEX Solana lainnya:

| | Raydium | Orca | Phoenix | Jupiter |
|--|---------|------|---------|---------|
| AMM | CPMM, AMM v4 | N/A | N/A | N/A (agregator) |
| CLMM | CLMM | Whirlpools | N/A | N/A |
| Orderbook | Tidak ada saat ini (integrasi OpenBook AMM v4 dinonaktifkan) | N/A | Orderbook native | N/A |
| Peluncuran token | LaunchLab | WaveBreak | N/A | N/A |
| Farm | Ya | Ya | N/A | N/A |
| TVL (Apr 2026) | ~\$1,8B | ~\$800M | ~\$200M | Agregator; mengarahkan beberapa B/bulan |

Dibandingkan dengan DEX Ethereum seperti Uniswap:

- **Finalitas transaksi**: ~1 detik di Solana vs ~12 detik di Ethereum.
- **Biaya transaksi**: \$0,001–0,01 khas vs \$5–50 khas di mainnet Ethereum.
- **Desain mempool**: Solana tidak memiliki mempool publik; Ethereum ada. Ini memengaruhi MEV, pola front-running, dan desain integrasi.
- **Token-2022**: Raydium mendukung ekstensi Token-2022 native (biaya transfer, hook, dll.); Uniswap V3 tidak memiliki yang setara di ERC-20.
