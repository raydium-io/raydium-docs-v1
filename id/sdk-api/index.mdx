## Pilih jalur integrasi Anda

Lima jalur untuk dipilih. Jalur yang tepat tergantung pada siapa yang memanggil Raydium dan seberapa banyak kontrol yang Anda butuhkan.

| Jalur | Terbaik untuk | Latensi | Logika kustom | Bahasa |
|---|---|---|---|---|
| [Trade API](/id/sdk-api/trade-api) | Backend, bot, Blink — di mana pun Anda ingin tx dibangun server | Terendah | Terbatas (rute preset) | Apa pun |
| [TypeScript SDK](/id/sdk-api/typescript-sdk) | Aplikasi Node/TS dan frontend yang memerlukan akses builder lengkap | Sedang | Penuh | TS / JS |
| [Integrasi Python](/id/sdk-api/python-integration) | Backend Python dan pipeline ML | Sedang | Penuh (pembangunan instr manual) | Python |
| [Rust CPI](/id/sdk-api/rust-cpi) | Menyusun program yang memanggil Raydium dari on-chain | N/A (on-chain) | Penuh + composable | Rust |
| [Anchor IDL](/id/sdk-api/anchor-idl) | Codegen untuk bahasa tanpa dukungan first-party (Go, Kotlin, Swift, ...) | Variabel | Penuh | Apa pun dengan alat Anchor codegen |

Jika Anda tidak yakin: backend dan bot → **Trade API**; frontend dan aplikasi dengan kontrol penuh → **TypeScript SDK**; komposisi on-chain → **Rust CPI**.
