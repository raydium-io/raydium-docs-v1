## Prasyarat

Jalankan ini sekali:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Anda memerlukan:

- **Node 18+** dengan dukungan ESM (`"type": "module"` di `package.json` atau ekstensi `.mjs`).
- **Keypair Solana** di `~/.config/solana/id.json` (atau di mana saja — atur `KEYPAIR`).
- **Dompet dengan dana.** ~0,05 SOL mencakup skrip tunggal apa pun di bab ini.
- **URL RPC.** RPC publik gratis berfungsi untuk pengujian; untuk apa pun lebih dari uji smoke, gunakan penyedia berbayar.

Atur ini sekali per shell:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Ketiga skrip secara default menggunakan mainnet. Untuk menjalankan di devnet, atur `RPC_URL=https://api.devnet.solana.com` dan ubah `cluster: "mainnet"` menjadi `cluster: "devnet"` di dalam skrip. Perhatikan bahwa likuiditas seed pool di devnet sangat jarang; harapkan untuk membuat token tes Anda sendiri.
</Info>

## Setelah bab ini

Setiap skrip berakhir dengan output minimal (ID pool, signature tx) dan satu baris yang menunjuk ke halaman berikutnya di bab [SDK & API](/id/sdk-api) untuk operasi yang dilakukannya. Ketika Anda siap menulis integrasi Anda sendiri:

- **Backend dan bot** → [Trade API](/id/sdk-api/trade-api).
- **Frontend dan aplikasi kontrol penuh** → [TypeScript SDK](/id/sdk-api/typescript-sdk).
- **Komposisi on-chain** → [Rust CPI](/id/sdk-api/rust-cpi).
- **Memilih di antara mereka** → tabel perbandingan di bagian atas [SDK & API](/id/sdk-api).
