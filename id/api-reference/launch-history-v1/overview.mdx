History API menyediakan data perdagangan historis untuk pool LaunchLab. Ambil data candlestick (OHLC) pada beberapa interval (1 menit, 5 menit, 15 menit) dan akses catatan perdagangan individual dengan paginasi.

## Endpoint API

**Mainnet:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## Autentikasi

Tidak ada autentikasi yang diperlukan. Semua endpoint dapat diakses oleh publik.

## Fitur Data

- **Data Kline:** Candlestick OHLC pada interval 1m, 5m, dan 15m
- **History Perdagangan:** Perdagangan individual dengan trader, jumlah, dan cap waktu
- **Paginasi:** Gunakan `nextPageKey` untuk kueri dataset besar yang efisien
- **Batasan Limit:** Permintaan kline menerima 1-500 lilin per kueri (standar 300); perdagangan menerima 1-100 (standar 50)
- **Penyaringan:** Saring history perdagangan berdasarkan dompet trader dan rentang jumlah

## Use Case

- **Tampilan chart** dengan visualisasi candlestick
- **Analisis harga** dan indikator analisis teknis
- **Statistik perdagangan** dan pelacakan volume
- **Jejak audit perdagangan pengguna** yang disaring berdasarkan dompet dan rentang tanggal
- **Penemuan harga historis** untuk mint
