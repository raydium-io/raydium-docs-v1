## Pemberitahuan migrasi

**Untuk integrasi baru, gunakan [API v3](/id/api-reference/api-v3/overview) sebagai gantinya.** Endpoint v3 menawarkan:

- Performa dan caching yang lebih baik
- Skema respons yang lebih sederhana
- Organisasi endpoint yang lebih jelas
- Pengembangan fitur yang berkelanjutan
