Owner API menyediakan tampilan per-dompet dari posisi dan saldo Raydium. API ini mengambil data terkache untuk posisi, likuiditas terkunci, pool yang dibuat, dan partisipasi IDO, sehingga ideal untuk UI dompet dan dasbor portofolio.

## Host

| Lingkungan | Host                       |
|-------------|---------------------------|
| Mainnet     | `owner-v1.raydium.io`     |
| Devnet      | `owner-v1-devnet.raydium.io` |

## Ikhtisar

API mengekspos lima endpoint utama yang diorganisir berdasarkan tipe sumber daya:

- **`/position/stake/{owner}`** — Posisi stake (AMM standar / CPMM LP)
- **`/position/clmm-lock/{owner}`** — Posisi CLMM terkunci
- **`/create-pool/{owner}`** — Pool yang dibuat oleh dompet ini
- **`/main/ido/{owner}`** — Data partisipasi IDO

## Kesegaran data

Layanan ini menyimpan data posisi dalam cache dan memperbaruinya secara berkala. Perubahan posisi yang sangat baru (dalam hitungan detik) mungkin tidak langsung muncul dalam respons.

## Pola umum

### Ambil semua posisi stake untuk sebuah dompet

```bash
curl -s "https://owner-v1.raydium.io/position/stake/{owner}"
```

Ganti `{owner}` dengan kunci publik dompet dalam format base58.

### Ambil posisi terkunci

```bash
curl -s "https://owner-v1.raydium.io/position/clmm-lock/{owner}"
```

### Periksa partisipasi IDO

```bash
curl -s "https://owner-v1.raydium.io/main/ido/{owner}"
```

## Hasil kosong

Jika dompet tidak memiliki data untuk endpoint tertentu, layanan mengembalikan kesalahan 404 dengan metadata permintaan:

```json
{
  "id": "...",
  "success": false
}
```
