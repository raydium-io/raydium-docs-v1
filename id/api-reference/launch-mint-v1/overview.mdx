API Mint menyediakan akses komprehensif ke pembuatan token LaunchLab, penemuan, dan manajemen. Cari mint berdasarkan nama, simbol, atau pembuat, ambil papan peringkat, dan kirimkan permintaan pembuatan mint baru.

## API Endpoints

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Autentikasi

Operasi baca tidak memerlukan autentikasi. Operasi tulis (pembuatan mint) memerlukan header `ray-token` yang dikeluarkan oleh API Autentikasi setelah verifikasi tanda tangan dompet.

[Dokumentasi API Autentikasi](/id/api-reference/launch-auth-v1/overview)

## Kasus Penggunaan

- **Cari dan temukan** token berdasarkan nama atau simbol
- **Lihat papan peringkat** (mint teratas menurut kapitalisasi pasar, terbaru, terpopuler)
- **Ambil statistik pengguna** (volume pembuatan dan perdagangan seiring waktu)
- **Kueri jadwal vesting** untuk informasi pembukaan token
- **Kirim permintaan pembuatan mint** dengan metadata dan gambar
- **Lacak reward referrer** dari program rujukan platform
