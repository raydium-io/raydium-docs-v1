## Apa itu Raydium API v3?

Raydium API v3 menyediakan antarmuka REST untuk mengkueri pool likuiditas, metadata token, farm, dan statistik rantai di seluruh venue perdagangan Raydium pada Solana. API ini digunakan oleh dompet, agregator, router DEX, dan pedagang untuk mengambil data pool kanonik, feed harga, dan analitik historis.

API dirancang untuk throughput tinggi — respons biasanya berjarak 1–5 menit tergantung sensitivitas endpoint. Semua respons dibungkus dalam amplop konsisten dengan metadata.

## Amplop respons

Semua endpoint mengembalikan amplop JSON dengan metadata permintaan dan muatan respons:

### Respons sukses

```json
{
  "id": "abc123def456",
  "success": true,
  "data": {
    // Data spesifik endpoint
  }
}
```

### Respons error

```json
{
  "id": "abc123def456",
  "success": false,
  "msg": "query ids type error"
}
```

Bidang `id` adalah pengenal permintaan unik (UUID). Selalu periksa `success` sebelum memproses `data`.

## Gotcha umum

### Urutan pasangan mint di `/pools/info/mint`

Ketika Anda mengkueri `/pools/info/mint?mint1=ABC&mint2=XYZ`, endpoint secara internal menormalkan pasangan dengan mengurutkan mint secara alfabetis. Jika Anda perlu mencari satu mint, lewati `mint2` sepenuhnya — API akan menemukan semua pool yang berisi mint tersebut.

### Nilai filter jenis pool

Parameter kueri `poolType` menerima nilai enum spesifik:
- `all` — semua jenis pool
- `concentrated` — hanya pool CLMM
- `standard` — hanya pool AMM (constant-product)
- `allFarm` — pool dengan setidaknya satu farm aktif
- `concentratedFarm` — pool terkonsentrasi dengan farm
- `standardFarm` — pool standar dengan farm

Kesalahan ketik atau ketidaksesuaian kepekaan huruf akan ditolak.

### Nilai bidang sortir pool

Nilai `poolSortField` yang valid adalah:
- `default` (mengurutkan berdasarkan volume 24j)
- `liquidity` (TVL)
- `volume24h`, `volume7d`, `volume30d`
- `fee24h`, `fee7d`, `fee30d`
- `apr24h`, `apr7d`, `apr30d`

### Paginasi dengan `/pools/info/list-v2`

Endpoint v2 menggunakan paginasi berbasis kursor melalui `nextPageId`. Token buram dikembalikan dalam respons dan harus dilewatkan kembali apa adanya. Jangan coba membuat kursor sendiri — kursor ini mengenkode status Elasticsearch.

### Default dan batas ukuran halaman

- `/pools/info/list` dan `/pools/info/mint` membatasi `pageSize` pada **1000**.
- `/farms/info/lp` membatasi `pageSize` pada **100**.
- `/pools/info/list-v2` membatasi `size` pada **1000**.

Permintaan yang melampaui batas ditolak dengan error.

### Konfigurasi dynamic-fee dan limit-order CLMM

Dua endpoint baru mengekspos data kalibrasi untuk alur `CreateCustomizablePool` CLMM:

- `GET /main/clmm-dynamic-config` — daftar tier `DynamicFeeConfig` (periode filter, periode decay, faktor reduksi, numerator maksimum, kontrol dynamic-fee). Lewatkan `id` salah satunya ke `createCustomizablePool` ketika `enableDynamicFee=true`.
- `GET /main/clmm-limit-order-config` — konfigurasi limit-order per-pool: apakah limit order diaktifkan, jumlah input minimum, dan penjaga `limit_order_admin` off-chain yang berwenang untuk menyelesaikan pesanan terisi.

Data limit-order per-dompet (terbuka / terisi / ditutup) berada di Temp API; lihat [`api-reference/temp-api-v1/overview`](/id/api-reference/temp-api-v1/overview).

## Contoh penggunaan

**Dapatkan statistik utama:**

```bash
curl -s https://api-v3.raydium.io/main/info | jq .
```

**Daftar 10 pool teratas berdasarkan TVL:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/list?poolType=all&poolSortField=liquidity&sortType=desc&pageSize=10&page=1' | jq .
```

**Dapatkan pool SOL-USDC:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/mint?mint1=So11111111111111111111111111111111111111112&mint2=EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v&poolType=all&poolSortField=liquidity&sortType=desc&pageSize=5&page=1' | jq .
```

**Dapatkan harga mint:**

```bash
curl -s 'https://api-v3.raydium.io/mint/price?mints=So11111111111111111111111111111111111111112,EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v,4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R' | jq .
```

**Dapatkan pool farm untuk token LP:**

```bash
curl -s 'https://api-v3.raydium.io/farms/info/lp?lp=<lp_mint>&pageSize=10&page=1' | jq .
```
