API IPFS Dinamis menyediakan metadata dan gambar yang diregenerasi untuk posisi likuiditas CLMM dan CPMM di Solana. Layanan ini dirancang untuk dApp yang merender NFT dinamis—seperti kartu posisi likuiditas terkonsentrasi—dengan menyediakan informasi pool real-time, valuasi posisi, dan biaya yang belum diklaim tanpa memerlukan lookup on-chain.

## Host

| Lingkungan | Host                               |
|------------|-----------------------------------|
| Mainnet    | `dynamic-ipfs.raydium.io`         |
| Devnet     | `dynamic-ipfs-devnet.raydium.io`  |

## Ikhtisar

API mengekspos tiga endpoint, masing-masing mengembalikan objek antarmuka posisi lengkap:

- **`/clmm/position`** — Ambil metadata untuk posisi CLMM yang aktif
- **`/lock/clmm/position`** — Ambil metadata untuk posisi CLMM yang terkunci  
- **`/lock/cpmm/position`** — Ambil metadata untuk posisi CPMM yang terkunci

Setiap respons mencakup detail pool lengkap (token, desimal, TVL), jumlah posisi, valuasi USD, persentase TVL dalam pool, serta biaya dan reward yang terkumpul (tetapi belum diklaim).

## Kasus penggunaan

Hasilkan NFT visualisasi posisi dinamis dengan meminta metadata untuk posisi tertentu. API menghitung status posisi real-time dari data on-chain dan menyimpan hasil dalam cache, sehingga ideal untuk:

- Merender kartu posisi dengan nilai USD langsung dan akumulasi biaya
- Menampilkan persentase TVL dan jumlah token dalam pool
- Menunjukkan biaya trading yang belum diklaim dan token reward
- Membangun dashboard portfolio yang melacak kinerja posisi

## Pola umum

### Ambil posisi CLMM

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

Parameter `id` harus berupa kunci publik on-chain dari akun posisi (alamat Solana dalam format base58).

### Ambil posisi yang terkunci

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
