Forum API memungkinkan diskusi dan keterlibatan komunitas di sekitar pool LaunchLab. Posting komentar dengan lampiran gambar opsional dan ambil thread diskusi pool. Semua operasi penulisan memerlukan autentikasi melalui Auth API.

## API Endpoints

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Autentikasi

**Operasi baca (GET):** Tidak memerlukan autentikasi.

**Operasi tulis (POST):** Memerlukan header `ray-token` — JWT yang dikeluarkan oleh Auth API setelah verifikasi tanda tangan dompet.

[Dapatkan token dari Auth API](/id/api-reference/launch-auth-v1/overview)

## Request Headers

Saat memposting komentar, sertakan header autentikasi:

```
ray-token: <jwt_token_from_auth_api>
```

## Fitur

- **Post Komentar:** Kirim komentar teks dengan lampiran gambar opsional (maksimal 5 MB)
- **Browsing Diskusi:** Lihat semua komentar pada pool dengan pagination
- **Content Moderation:** Deteksi NSFW dan penyaringan komentar tersembunyi
- **Pagination:** Kueri dengan `lastId` dan `fetchType` (old/new) untuk browsing yang efisien
- **Image Upload:** Komentar mendukung hosting gambar IPFS Pinata

## Use Cases

- **Diskusi komunitas** di sekitar proyek token
- **Pengumuman proyek** dan pembaruan melalui komentar
- **Umpan balik pengguna** dan pelacakan sentimen
- **Bukti sosial** dan metrik keterlibatan komunitas
- **Content moderation** dengan deteksi NSFW dan spam
