## Ringkasan penulisan

- Setiap panduan dimulai dengan "ini adalah masalahnya" (misalnya, "transaksi swap saya gagal di bawah beban") dan berakhir dengan resep yang telah divalidasi.
- Sertakan benchmark jika bermakna (misalnya, penggunaan CU tipikal untuk swap CPMM vs. swap CLMM).
- Tautkan ke SDK & API untuk primitif yang mendasar; jangan mendefinisikan ulang.
