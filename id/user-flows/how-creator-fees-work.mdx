## Fase 1 — Pra-migrasi (bonding curve)

Saat token Anda diperdagangkan di bonding curve, komponen creator fee diambil dari setiap pembelian dan penjualan serta terakumulasi dalam vault per-token. Ini dikonfigurasi oleh **platform** tempat token Anda diluncurkan.

### Detail

- **Didenominasikan dalam** token quote (biasanya SOL).
- **Tarif** — ditetapkan oleh platform saat peluncuran. Periksa jadwal biaya platform atau status on-chain token.
- **Dapat diklaim kapan saja** — buka halaman **Portfolio** Anda di raydium.io dan klaim biaya yang tertunda kapan saja. Tidak perlu menunggu graduation.
- **Berhenti saat graduation** — tidak ada biaya pra-migrasi baru yang terakumulasi setelah curve graduation. Saldo sisa apa pun di vault tetap dapat diklaim.

### Cara mengklaim

1. Hubungkan dompet kreator ke [raydium.io](https://raydium.io).
2. Navigasi ke **Portfolio**.
3. Temukan token Anda di bawah token yang Anda buat.
4. Klik **Claim** di samping saldo pre-migration fee.

Klaim programatis: lihat [`products/launchlab/creator-fees`](/id/products/launchlab/creator-fees) untuk instruksi SDK.

## Fase 2 — Pasca-migrasi (Fee Key NFT)

Ketika bonding curve token Anda mencapai ambang batas graduation, likuiditas bermigrasi ke pool CPMM Raydium. Token LP dimintai dan didistribusikan sesuai **konfigurasi platform**:

| Penerima | Default Raydium | Deskripsi |
|---|---|---|
| Burned (Burn & Earn) | 90% | Terkunci secara permanen; biaya perdagangan dari bagian ini mengalir ke mekanisme kepercayaan Burn & Earn. |
| Creator (Fee Key NFT) | 10% | Fee Key NFT dimintai ke dompet Anda. NFT ini adalah klaim Anda atas 10% dari biaya perdagangan LP yang sedang berjalan di pool. |
| Platform | 0% (default Raydium) | Beberapa platform pihak ketiga mengambil bagian di sini. |

Default ini berlaku untuk token yang diluncurkan melalui UI LaunchLab Raydium sendiri. Platform pihak ketiga mungkin mengonfigurasi pembagian yang berbeda — selalu periksa halaman platform sebelum meluncurkan.

### Fee Key NFT

Fee Key NFT adalah token non-fungible yang dimintai ke dompet Anda saat graduation yang mewakili hak Anda untuk mengumpulkan LP trading fees dari bagian Anda dari pool pasca-graduation.

<Warning>
  **Jangan bakar atau transfer Fee Key NFT kecuali Anda ingin dengan sengaja melepaskan hak klaim fee.** Jika NFT hilang atau dibakar, hak klaim fee **selamanya hilang** — tidak ada jalur pemulihan.
</Warning>

- Fee Key dapat diperdagangkan atau ditransfer; siapa pun yang memegangnya dapat mengklaim biaya.
- Biaya terakumulasi terus-menerus dari aktivitas perdagangan LP di pool CPMM.
- Klaim dari halaman Portfolio, atau melalui instruksi fee-claim pool CPMM secara programatis.

### Cara mengklaim biaya pasca-migrasi

1. Hubungkan dompet yang memegang Fee Key NFT.
2. Navigasi ke **Portfolio** di raydium.io.
3. Temukan pool token yang telah graduation di bawah posisi Anda.
4. Klik **Claim** untuk mengumpulkan biaya perdagangan yang terakumulasi.

## Jika biaya pasca-migrasi tidak diaktifkan

Jika platform mengatur creator fees sebagai disabled, token Anda graduation ke **pool AMM v4** (jalur migrasi legacy) alih-alih CPMM. Dalam mode ini:
- Tidak ada Fee Key NFT yang dimintai.
- Tidak ada creator fee pasca-migrasi tingkat program.
- Beberapa opsi klaim fee alternatif mungkin ada melalui alat CLI tergantung pada cara peluncuran dikonfigurasi.

## Ringkasan aliran biaya

```
Fase bonding curve:
  Setiap beli/jual → kecil % → creator vault (token quote)
  Klaim kapan saja dari Portfolio

Graduation:
  Token LP dimintai → dibagi per konfigurasi platform
  ├── X% dibakar (Burn & Earn)
  ├── Y% → creator sebagai Fee Key NFT      ← Anda memegang ini
  └── Z% → platform (jika berlaku)

Pasca-graduation:
  Setiap CPMM swap → LP fees → dibagi berdasarkan LP share
  Bagian Fee Key NFT Anda → dapat diklaim dari Portfolio
```
