<Warning>
  LaunchLab adalah infrastruktur peluncuran token tanpa izin. Siapa pun dapat membuat token, memilih nama atau ticker, dan mengonfigurasi parameter peluncuran. Selalu verifikasi alamat mint dan pengaturan peluncuran sebelum membeli atau mempromosikan token. Halaman ini hanya menjelaskan mekanika produk dan bukan saran finansial, hukum, atau investasi.
</Warning>

LaunchLab memungkinkan kreator mengeluarkan token dan memulai likuiditas awal melalui bonding curve. Pembeli berdagang terhadap kurva sebelum graduasi. Ketika peluncuran mencapai ambang graduasinya, likuiditas bermigrasi ke pool AMM Raydium dan perdagangan berlanjut melalui rute Raydium normal.

## Siapa yang menggunakan LaunchLab

| Audiens | Tugas umum |
|---|---|
| Trader | Temukan token tahap awal, beli atau jual di bonding curve, dan lacak kemajuan graduasi. |
| Kreator | Luncurkan token tanpa kode, konfigurasi parameter kurva, dan secara opsional dapatkan biaya kreator. |
| Platform | Bangun lingkungan peluncuran bermerek dengan pengaturan biaya, branding, dan permukaan penemuan peluncuran mereka sendiri. |

## Cara kerja peluncuran

1. Kreator mengonfigurasi metadata token, pasokan, pengaturan kurva, target graduasi, dan pengaturan biaya opsional.
2. Token dibuka di bonding curve. Pembeli dan penjual berdagang langsung terhadap kurva tersebut.
3. Harga kurva berubah seiring pembelian dan penjualan terjadi.
4. Ketika kurva mencapai target graduasinya, likuiditas bermigrasi ke pool Raydium.
5. Setelah graduasi, pengguna berdagang token melalui Raydium Swap dan agregator yang merutekan melalui pool Raydium.

## Mode pembuatan

| Mode | Gunakan saat | Pengaturan kunci |
|---|---|---|
| JustSendit | Anda menginginkan peluncuran tanpa kode tercepat dengan pengaturan default. | Nama, ticker, gambar, tautan media sosial opsional, pembelian pertama opsional. |
| LaunchLab | Anda memerlukan kontrol lebih besar atas pasokan, target pengumpulan dana, alokasi kurva, vesting, atau biaya kreator. | Pasokan, target pengumpulan dana, alokasi kurva, vesting, bagian biaya pasca-migrasi. |

JustSendit menggunakan target graduasi default. Mode LaunchLab memungkinkan kreator menetapkan target graduasi, tunduk pada batasan UI dan program.

## Graduasi dan likuiditas

Graduasi menutup bonding curve dan memindahkan aset ke pool Raydium. Jenis pool pasca-migrasi dan perlakuan LP bergantung pada konfigurasi platform dan pengaturan biaya kreator.

- Jika biaya kreator pasca-migrasi diaktifkan, peluncuran bermigrasi ke CPMM dan NFT Fee Key dapat mewakili hak biaya kreator.
- Jika platform menggunakan jalur warisan tanpa biaya kreator pasca-migrasi, peluncuran dapat bermigrasi ke AMM v4.
- Pengaturan platform dapat mengubah cara token LP dibakar, dikunci, atau ditetapkan setelah migrasi.

## Dasar-dasar keamanan

- Verifikasi mint token, bukan hanya ticker atau logo.
- Tinjau target graduasi dan kemajuan kurva sebelum berdagang.
- Periksa pengaturan biaya kreator dan platform sebelum meluncurkan atau membeli.
- Perlakukan NFT Fee Key sebagai aset hak biaya. Jika ditransfer atau dibakar, hak klaim bergerak atau mungkin hilang.
- Baca [Kepercayaan dan keamanan](/id/getting-started/trust-and-safety) sebelum menghubungkan dompet atau menandatangani transaksi.
