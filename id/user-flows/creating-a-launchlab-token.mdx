## Dua mode pembuatan

LaunchLab menawarkan dua jalur pembuatan yang dapat diakses dari [raydium.io/launchlab/create](https://raydium.io/launchlab/create):

| | **JustSendit** | **LaunchLab** |
|---|---|---|
| Waktu setup | ~1 menit | ~5 menit |
| Pasokan token | Default tetap | Dapat dikonfigurasi sepenuhnya |
| Target SOL | 85 SOL (tetap) | Minimum 24 SOL, kustom |
| Kurva % | Default | 51–80% di kurva |
| Vesting | Tidak ada | Opsional |
| Biaya pasca-migrasi | Default (10% bagian LP kreator) | Dapat dikonfigurasi |
| Terbaik untuk | Peluncuran komunitas cepat | Peluncuran token terstruktur |

## Mode JustSendit

Cara tercepat untuk meluncurkan. Input minimal diperlukan; platform mengisi default yang masuk akal.

### Yang Anda butuhkan

- Dompet terkoneksi dengan SOL yang cukup untuk biaya transaksi (~0,05–0,1 SOL untuk pembuatan).
- Nama token dan simbol ticker.
- Gambar atau GIF yang memenuhi persyaratan UI.

### Langkah-langkah

1. Buka [raydium.io/launchlab/create](https://raydium.io/launchlab/create) dan pilih **JustSendit**.
2. Masukkan **nama** dan **ticker** token.
3. Secara opsional tambahkan **deskripsi** dan **tautan media sosial** (situs web, Twitter/X, Telegram).
4. Unggah **gambar atau GIF logo**.
5. Secara opsional masukkan **jumlah pembelian awal** dalam SOL untuk membeli token saat pembuatan. Ini mengubah status harga kurva awal dan harus ditinjau sebelum menandatangani.
6. Tinjau ringkasan dan klik **Buat**.
7. Setujui transaksi di dompet Anda.

Token Anda diluncurkan segera di kurva bonding. Token ini dipromosikan ke pool AMM Raydium setelah 85 SOL telah dikumpulkan oleh kurva.

## Mode LaunchLab

Pembuatan kontrol penuh dengan pasokan yang dapat dikonfigurasi, target penggalangan dana, alokasi kurva, vesting, dan parameter biaya.

### Yang Anda butuhkan

- Dompet terkoneksi dengan ~0,1–0,5 SOL untuk pembuatan dan sewa vault.
- Semua input di bawah siap sebelum Anda mulai (pengaturan tidak dapat diubah setelah peluncuran).

### Opsi konfigurasi

#### Detail token

- **Nama, ticker, logo** — sama seperti JustSendit.
- **Deskripsi dan tautan media sosial** — direkomendasikan untuk kemudahan penemuan.

#### Alokasi pasokan dan kurva

- **Total pasokan** — jumlah total token yang akan dicetak (misalnya 1.000.000.000).
- **Alokasi kurva (%)** — persentase total pasokan yang ditempatkan di kurva bonding untuk pembelian publik (51–80%). Sisanya menabur pool likuiditas pasca-promosi.

<Warning>
  Alokasi kurva dan total pasokan **tidak dapat diubah** setelah peluncuran dan tetap. Pilih dengan cermat sebelum menandatangani.
</Warning>

#### Target penggalangan dana SOL

- Ambang batas promosi minimum adalah **24 SOL**. Anda dapat menetapkannya lebih tinggi.
- Ini adalah jumlah SOL yang harus dikumpulkan kurva sebelum token dipromosikan ke pool CPMM.
- Menetapkannya terlalu tinggi berisiko kurva tidak akan pernah dipromosikan jika permintaan tidak mencukupi.

#### Vesting (opsional)

Vesting mengunci sebagian dari pasokan token dan mereleasenya secara linear ke dompet yang ditunjuk selama jadwal.

| Bidang | Deskripsi |
|---|---|
| Total jumlah terkunci | Token yang akan dikunci di semua penerima manfaat (sebagai % dari total pasokan, tunduk pada batas program). |
| Periode tebing | Waktu tunggu setelah promosi sebelum token apa pun membuka (misalnya 180 hari). |
| Periode pembukaan | Durasi pembukaan linear setelah tebing (misalnya 365 hari). |

Token yang terkunci tidak memasuki kurva dan bukan bagian dari pool likuiditas promosi. Mereka dirilis ke dompet setiap penerima manfaat selama jadwal di atas.

**Aturan vesting penting:**
- Periode tebing dan pembukaan tetap saat peluncuran — tidak dapat diubah secara retroaktif.
- Jika penerima manfaat kehilangan kunci dompet mereka, token yang terkunci tidak dapat dijangkau selamanya. Gunakan multisig jika pemulihan penting.
- Bagian apa pun dari `total_locked_amount` yang tidak dialokasikan ke penerima manfaat sebelum promosi akan terjebak selamanya. Alokasikan jumlah penuh.

Untuk mekanika on-chain lengkap, lihat [Vesting LaunchLab](/id/products/launchlab/vesting).

#### Berbagi biaya pasca-migrasi

Ketika token dipromosikan:
- Token LP dicetak ke pool CPMM.
- Secara default di Raydium: **90% token LP dibakar** (terkunci selamanya melalui Burn & Earn) dan **10% diberikan kepada kreator** sebagai NFT Fee Key.
- NFT Fee Key memberi Anda hak untuk mengumpulkan 10% dari biaya perdagangan LP dari pool pasca-migrasi — selamanya.

Aktifkan ini jika Anda ingin pendapatan berkelanjutan dari likuiditas token Anda dan menerima tanggung jawab penjagaan NFT Fee Key. Lihat [Bagaimana biaya kreator bekerja](/id/user-flows/how-creator-fees-work) untuk detail.

### Langkah-langkah

1. Pilih mode **LaunchLab** di halaman pembuatan.
2. Isi detail token (nama, ticker, logo, deskripsi, media sosial).
3. Atur **total pasokan** dan **persentase alokasi kurva**.
4. Atur **target penggalangan dana SOL** Anda (≥ 24 SOL).
5. Konfigurasi **vesting** jika diperlukan (atur dompet penerima manfaat dan jadwal).
6. Aktifkan **berbagi biaya pasca-migrasi** jika Anda ingin bagian LP kreator.
7. Secara opsional masukkan jumlah pembelian awal.
8. Tinjau ringkasan lengkap — rentang harga kurva, kondisi promosi, pembagian LP.
9. Klik **Buat** dan setujui transaksi.

## Setelah token Anda diluncurkan

### Pantau kemajuan kurva

Halaman LaunchLab token Anda menunjukkan kemajuan real-time:
- % kurva terjual.
- SOL terkumpul vs. target promosi.
- Harga saat ini dan bagan harga.

LaunchLab tidak mengintervensi peluncuran yang gagal mencapai promosi. Pantau kemajuan kurva dan berkomunikasi dengan jelas dengan komunitas Anda.

### Promosi

Promosi dipicu secara otomatis oleh transaksi pertama yang mendorong kurva melewati ambang batas — ini dapat dipanggil oleh pengguna mana pun (pemanggil mendapatkan bonus kecil). Tidak ada tindakan yang diperlukan dari Anda sebagai kreator.

Setelah promosi:
- Pool CPMM dibuat dengan SOL yang dikumpulkan dan token kurva yang tersisa sebagai likuiditas awal.
- Jika Anda mengaktifkan biaya pasca-migrasi, **NFT Fee Key** akan dicetak ke dompet Anda.
- Klaim biaya kreator pra-migrasi dari halaman Portfolio Anda.

### Klaim biaya kreator pra-migrasi

Jika platform Anda mengaktifkan biaya kreator selama fase kurva bonding, biaya terakumulasi dalam vault. Klaim dari halaman **Portfolio** di raydium.io kapan saja.

## Jebakan yang harus dihindari

| Jebakan | Yang harus dilakukan |
|---|---|
| Menetapkan target SOL terlalu tinggi | Pilih target yang dapat dicapai pasar secara realistis. |
| Over-alokasi vesting sebelum penerima manfaat dikonfirmasi | Pasokan vested yang tidak dialokasikan tidak dapat diakses selamanya setelah promosi. |
| Mentransfer atau membakar NFT Fee Key secara tidak sengaja | Hak klaim biaya hilang selamanya. Simpan NFT di dompet pembuatan atau dompet yang Anda kontrol. |
| Tidak melakukan pembelian awal | Tanpa pembelian awal, bot sniping dapat menangkap semua token awal dengan harga mendekati nol, yang dapat merusak kepercayaan komunitas. |
