## Apa itu token LaunchLab

Token LaunchLab dikeluarkan pada **bonding curve**: kurva harga yang dikelola oleh smart contract, di mana harga naik seiring lebih banyak token dibeli dan turun saat token dijual. Kurva beroperasi hingga ambang batas graduation (biasanya 85 SOL, meskipun pembuat dapat menetapkan jumlah berbeda) tercapai, saat itu token secara otomatis bermigrasi ke pool Raydium CPMM atau AMM v4 dan menjadi dapat diperdagangkan dengan bebas melalui routing DEX normal.

Sebelum graduation, Anda hanya dapat membeli dan menjual melawan kurva itu sendiri. Setelah graduation, token diperdagangkan di Raydium seperti pool lainnya.

## Sebelum Anda membeli

### Lakukan riset sendiri

- Verifikasi bahwa alamat mint token cocok dengan yang diumumkan oleh proyek bersangkutan — penipu secara rutin membuat token dengan nama yang identik.
- Periksa apakah **mint authority telah dibatalkan** (revoked). Jika belum, pembuat dapat memencetak pasokan baru dan mengencerkan posisi Anda. Dompet seperti Solflare dan penjelajah seperti Solscan menampilkan status ini.
- Tinjau kemajuan bonding curve: berapa banyak token yang telah terjual, berapa banyak yang tersisa, dan harga saat ini. Ini terlihat di halaman LaunchLab token.
- Periksa riwayat alamat pembuat jika memungkinkan.

### Pahami mekanika kurva

Harga pada bonding curve **tidak tetap**. Setiap pembelian menaikkan harga; setiap penjualan menurunkannya. Pembelian besar tepat sebelum Anda dapat secara signifikan menaikkan harga yang Anda bayar (risiko front-running / sandwich). Gunakan batas slippage (lihat di bawah).

## Langkah demi langkah: membeli melalui UI

1. Navigasikan ke [raydium.io/launchlab](https://raydium.io/launchlab) dan temukan token yang ingin Anda beli.
2. Klik halaman token untuk membuka tampilan detail.
3. Hubungkan dompet Anda jika belum.
4. Masukkan jumlah SOL (atau token kutipan) yang ingin Anda keluarkan.
5. Tinjau **token yang diestimasi diterima** dan **price impact** yang ditampilkan di UI.
6. Sesuaikan **toleransi slippage** Anda jika diperlukan (lihat di bawah).
7. Klik **Buy** dan setujui transaksi di dompet Anda.
8. Setelah transaksi terkonfirmasi, token akan muncul di dompet Anda.

<Tip>
  Anda dapat menjual kembali ke kurva kapan saja sebelum graduation menggunakan tombol **Sell** di halaman token yang sama. Setelah graduation, token berada di pool Raydium; gunakan antarmuka Swap standar sebagai gantinya.
</Tip>

## Slippage

<Warning>
  **Tetapkan toleransi slippage yang sesuai.** Token bonding curve dapat volatil. Jika Anda menetapkan slippage terlalu rendah, transaksi Anda mungkin gagal saat harga bergerak antara kutipan dan eksekusi. Jika Anda menetapkannya terlalu tinggi, Anda terpapar oleh bot sandwich. Titik awal yang wajar untuk sebagian besar peluncuran adalah 1–3%; naikkan hanya jika diperlukan pada peluncuran likuiditas sangat rendah yang mendekati graduation.
</Warning>

Toleransi slippage adalah perbedaan harga maksimal yang bersedia Anda terima antara harga yang dikutip dan harga yang dieksekusi. UI LaunchLab memungkinkan Anda menetapkan ini sebelum mengonfirmasi transaksi Anda.

## Setelah graduation

Saat ambang batas graduation kurva tercapai (oleh pedagang apa pun), token bermigrasi ke pool AMM Raydium secara otomatis. Pada saat itu:

- Bonding curve ditutup; tidak ada pembelian atau penjualan lebih lanjut yang melaluinya.
- Token Anda tetap berada di dompet Anda tanpa sentuhan.
- Anda sekarang dapat memperdagangkan token di halaman Swap Raydium atau agregator apa pun yang mengindeks pool Raydium.
- Migrasi pool dapat memerlukan beberapa menit untuk muncul di UI agregator.

## Risiko

| Risiko | Deskripsi |
|---|---|
| Risiko kontrol likuiditas | Penanganan LP tergantung pada platform dan pengaturan peluncuran. Periksa apakah LP dibakar, dikunci, ditetapkan ke pembuat, atau ditetapkan ke platform. |
| Tidak ada graduation | Jika kurva tidak pernah mencapai ambang batasnya, perdagangan tetap terbatas pada kurva saja. Anda masih dapat menjual kembali ke kurva, tetapi berpotensi dengan harga lebih rendah dari yang Anda bayar. |
| Volatilitas harga | Bonding curve secara inheren spekulatif. Harga dapat jatuh mendekati nol. |
| Token palsu | Penipu membuat token mirip. Verifikasi alamat mint secara independen. |
| Risiko smart contract | Tidak ada program on-chain yang bebas risiko. Hanya gunakan dana yang siap Anda pertaruhkan. |
