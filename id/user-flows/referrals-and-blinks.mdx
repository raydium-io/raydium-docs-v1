## Yang Anda dapatkan

- URL yang dapat dibagikan yang mengisi sebelumnya antarmuka swap Raydium dengan token tujuan pilihan Anda **dan dompet Anda sebagai penerima rujukan**.
- Ketika seseorang melakukan swap melalui tautan sambil tautan aktif di sesi mereka, **1% dari volume input dialihkan ke dompet perujuk** sebagai bagian dari transaksi swap.
- Tautan yang dibagikan di X.com dirender sebagai **Blinks** (Solana Actions) jika dompet penampil memiliki Actions / Blinks diaktifkan — widget swap satu klik di dalam postingan.

## Cara membuat tautan

1. Buka [raydium.io/swap](https://raydium.io/swap) dan hubungkan dompet Anda. Alamat rujukan diturunkan dari dompet yang terhubung; tanpa koneksi tautan masih berfungsi tetapi tidak memperoleh komisi.
2. Pilih **token tujuan** yang ingin Anda biarkan orang lain swap masuk. Sisi sumber dibiarkan untuk dipilih oleh penerima.
3. Klik ikon berbagi/tautan di panel swap (kanan atas).
4. Salin URL yang dihasilkan. Bagikan di mana saja.

Format URL stabil. Jika Anda membangun aplikasi yang menghasilkan tautan rujukan secara langsung.

## Cara kerja Blinks

**Blinks (Blockchain Links)** adalah standar Solana Foundation untuk merender URL tindakan on-chain sebagai UI interaktif di dalam platform media sosial. Dompet Solana yang mendukung Blinks membaca metadata tautan, membuat transaksi yang telah dibangun sebelumnya, dan menampilkan prompt penandatanganan inline kepada pengguna.

Persyaratan:

- **Dompet penampil harus memiliki Actions / Blinks diaktifkan**. Ini biasanya toggle di pengaturan dompet.
- **Platform harus menghormati standar tersebut**. X.com melakukannya melalui unfurler-nya; beberapa lainnya tidak, dalam hal ini tautan kembali ke hyperlink biasa.

Dari perspektif perujuk, rendering Blink bersifat otomatis — URL yang sama berfungsi sebagai tautan biasa di dompet yang tidak mendukung Blinks.

## Akuntansi biaya

Biaya rujukan 1% **dikenakan di atas biaya perdagangan AMM**, bukan diambil darinya. Untuk swap di pool dengan biaya total 0,25%:

```
Pengguna mengirim:        1.000 USDC
Biaya perdagangan AMM:     2,50 USDC   (ke LP + pembagian protokol)
Biaya rujukan:           10,00 USDC   (1% → dompet perujuk)
Bersih untuk swap:      987,50 USDC   (kemudian dihargai melalui kurva)
```

Biaya rujukan berlaku untuk rute swap Raydium yang dibuat oleh tautan rujukan. Interaksi pool langsung di luar rute rujukan tidak membawa biaya rujukan.

## Menonaktifkan / keluar dari rujukan

Tidak ada instruksi "hapus rujukan". Untuk melakukan swap tanpa biaya rujukan, cukup muat [raydium.io/swap](https://raydium.io/swap) secara langsung alih-alih mengklik tautan rujukan. UI tidak menggunakan perujuk ketika URL tidak memiliki parameter perujuk.

## Tanya Jawab

**Apakah perujuk perlu menandatangani apa pun?** Tidak. Alamat dompet perujuk adalah satu-satunya data yang bepergian dengan tautan; pembayaran kepada perujuk adalah transfer token di dalam transaksi penukar.

**Bisakah saya merujuk diri sendiri?** Ya, tetapi tidak berguna — Anda membayar sendiri 1%, dikurangi biaya tx yang baru saja Anda bayar pada transaksi yang lebih besar.

**Apa yang terjadi jika token tujuan adalah SPL dengan hook transfer?** Transfer rujukan terjadi di sisi input (dalam USDC atau token sumber), bukan tujuan, sehingga hook transfer pada mint tujuan tidak mempengaruhi pembayaran rujukan. Hook transfer pada token input dapat memblokir swap sepenuhnya; Raydium menolak ini di awal.

**Apakah biaya rujukan masih berlaku jika swap merutekan melalui beberapa pool?** Ya — 1% dihitung pada jumlah input asli dan independen dari rute.
