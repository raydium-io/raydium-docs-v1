## Cara kerjanya

Ketika Anda membagikan tautan halaman token yang menyertakan alamat dompet Anda sebagai referrer, siapa pun yang bertransaksi melalui tautan Anda menghasilkan **komisi 0,1%** yang dibayarkan langsung ke dompet Anda. Komisi tersebut:

- Didenominasikan dalam SOL (atau token quote dari pool).
- Dikirim secara otomatis ke dompet Anda pada setiap transaksi — tanpa langkah klaim terpisah.
- Berlaku untuk pembelian dan penjualan di bonding curve selama token masih dalam tahap pre-graduation.

### Cakupan sesi

Tautan referral menunjuk ke token spesifik, tetapi jika pengguna yang mengklik tautan Anda kemudian bertransaksi dengan token LaunchLab *lainnya* dalam sesi browser yang sama, Anda mungkin juga mendapatkan komisi dari transaksi tersebut. Ini memperluas potensi penghasilan Anda di luar token tunggal yang awalnya Anda bagikan.

## Membuat tautan referral Anda

1. Hubungkan dompet Anda di [raydium.io/launchlab](https://raydium.io/launchlab).
2. Arahkan ke halaman detail token LaunchLab apa pun.
3. Klik tombol **Share** di halaman token.
4. Salin tautan yang dihasilkan — tautan tersebut menyertakan alamat dompet Anda sebagai parameter referrer.
5. Bagikan tautan di mana pun audiens Anda berada: media sosial, komunitas, newsletter.

<Note>
  Dompet Anda harus terhubung saat Anda mengklik Share. Tautan tidak akan berisi parameter referrer yang valid jika dibuat tanpa dompet yang terhubung.
</Note>

## Apa yang Anda dapatkan

| Skenario | Komisi |
|---|---|
| Penerima membeli token yang ditautkan | 0,1% dari nilai transaksi |
| Penerima menjual token yang ditautkan | 0,1% dari nilai transaksi |
| Penerima bertransaksi dengan token LaunchLab lain dalam sesi yang sama | 0,1% dari transaksi tersebut |

Penghasilan dikirim secara real-time — tidak ada vault atau saldo tertunda untuk diklaim.

## Perbandingan: referral LaunchLab vs. referral Swap

| | Referral LaunchLab | Referral Swap (Blinks) |
|---|---|---|
| Tarif | 0,1% | 1% |
| Cakupan | Halaman token LaunchLab | Antarmuka Raydium Swap |
| Pembayaran | Otomatis, per transaksi | Klaim manual atau otomatis tergantung integrasi |
| Pengaturan | Tombol bagikan di halaman token | Buat melalui [referrals-and-blinks](/id/user-flows/referrals-and-blinks) |
