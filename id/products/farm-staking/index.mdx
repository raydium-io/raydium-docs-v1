## Apa itu

Program Farm / Staking mendistribusikan token reward kepada akun yang melakukan staking token input (umumnya token LP dari AMM v4 / CPMM / CLMM, namun juga mendukung staking aset tunggal). Setiap farm dapat mengeluarkan hingga beberapa reward mint pada jadwal independen.

**ID Program:** lihat [reference/program-addresses](/id/reference/program-addresses). Beberapa versi farm sedang dijalankan (v3, v5, v6) dan berbeda dalam reward slots dan admin primitives — halaman [versions-and-migration](/id/protocol-overview/versions-and-migration) mendokumentasikan mana yang harus digunakan untuk deployment mana.
