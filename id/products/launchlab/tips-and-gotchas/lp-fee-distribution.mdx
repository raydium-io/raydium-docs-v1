Ketika token LaunchLab dimigrasikan ke CPMM, pool yang dimigrasikan mencetak token LP. Konfigurasi platform mengontrol apa yang terjadi dengan token LP tersebut.

## Pembagian LP

`migrateCpLockNftScale` memiliki tiga bidang. Nilai-nilai tersebut harus berjumlah `1_000_000`, yang mewakili 100%.

| Bidang | Arti |
|---|---|
| `platformScale` | Token LP dikunci melalui Burn & Earn. Platform menerima NFT Fee Key. |
| `creatorScale` | Token LP dikunci melalui Burn & Earn. Pencipta token menerima NFT Fee Key. |
| `burnScale` | Token LP dibakar secara permanen. Tidak ada NFT Fee Key yang dibuat untuk bagian ini. |

Ini adalah pengaturan tingkat platform. Token yang diluncurkan melalui platform yang sama mewarisi distribusi LP yang sama sampai platform memperbarui pengaturan.

## Contoh

```ts
// 40% platform, 50% creator, 10% burned.
migrateCpLockNftScale: {
  platformScale: new BN(400_000),
  creatorScale: new BN(500_000),
  burnScale: new BN(100_000),
}

// 100% creator Fee Key.
migrateCpLockNftScale: {
  platformScale: new BN(0),
  creatorScale: new BN(1_000_000),
  burnScale: new BN(0),
}

// 100% burned.
migrateCpLockNftScale: {
  platformScale: new BN(0),
  creatorScale: new BN(0),
  burnScale: new BN(1_000_000),
}
```

## Hubungan dengan fee CPMM

`migrateCpLockNftScale` mengontrol **siapa yang dapat mengklaim hak fee LP**. Pengaturan ini tidak menentukan tingkat swap fee. Perilaku swap fee pool yang dimigrasikan berasal dari CPMM `AmmConfig` yang dipilih oleh `cpConfigId`.

Misalnya, jika pool CPMM yang dimigrasikan menggunakan konfigurasi trade-fee 0,25% dan `creatorScale` adalah 500.000, NFT Fee Key pencipta mewakili 50% dari hak fee LP yang dikunci. Jika `platformScale` adalah 400.000, NFT Fee Key platform mewakili 40%.

## Memperbarui pembagian

Pembaruan konfigurasi platform dibatasi tingkat berdasarkan epoch. Anggap distribusi LP sebagai kebijakan platform yang penting, bukan pengaturan UI biasa.

Sebelum memperbarui:

- Konfirmasikan bahwa ketiga nilai scale berjumlah `1_000_000`.
- Konfirmasikan bahwa platform memahami siapa yang akan menerima NFT Fee Key setelah migrasi di masa depan.
- Dokumentasikan perubahan untuk para pencipta sebelum peluncuran baru menggunakannya.
