## Apa itu LaunchLab

LaunchLab adalah program peluncuran token Raydium. Sebuah proyek meluncurkan token terhadap bonding curve; pembeli bertransaksi terhadap curve sampai ambang pendanaan tercapai, pada titik tersebut sisa pasokan curve dan token quote yang terkumpul dimigrasikan ke dalam pool AMM Raydium (CPMM atau CLMM) dan perdagangan berlanjut di sana.

**Program ID:** lihat [reference/program-addresses](/id/reference/program-addresses).
