## Ikhtisar 30 detik

**Raydium adalah DEX di Solana.** Ini memungkinkan Anda menukar satu token dengan token lain tanpa menggunakan exchange terpusat. Untuk menggunakannya Anda membutuhkan:

1. **Wallet Solana** (browser extension atau aplikasi mobile) — menyimpan kunci privat Anda dan menandatangani transaksi.
2. Beberapa **SOL** di wallet itu — membayar biaya transaksi (~\$0.0005 per swap) dan token apa pun yang ingin Anda beli.
3. Browser di [`raydium.io`](https://raydium.io) — UI resmi.

Selebihnya — memilih wallet, mendanakannya, memilih trade pertama — tercakup di halaman di atas.

## Bendera merah untuk diperhatikan sejak hari pertama

Sebelum Anda menyetor uang apa pun, internalisasikan aturan ini:

- **Jangan pernah bagikan seed phrase Anda.** Bukan kepada dukungan, bukan ke situs "verifikasi wallet Anda", bukan di DM. Satu-satunya tempat seed phrase diketik adalah selama setup wallet awal.
- **Hanya gunakan `raydium.io`.** Phishing menyalin UI dengan sempurna pixel. Tandai URL asli.
- **Percayai alamat token, bukan nama token.** Siapa pun dapat membuat token bernama "USDC". USDC asli mint adalah `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`. UI menampilkan lencana **verified** di sebelah mint yang dikurasi.
- **DApp tidak boleh meminta Anda untuk "approve" seluruh wallet.** Transaksi normal menandatangani instruksi spesifik. Approval menyeluruh adalah drain.

Detail lebih lanjut di [`getting-started/trust-and-safety`](/id/getting-started/trust-and-safety).
