## A

**AcceleRaytor** — Token launchpad Raydium asli tahun 2021-2023. Dihentikan saat LaunchLab menggantinya. Lihat [`introduction/history-and-milestones`](/id/introduction/history-and-milestones).

**Account** — Baris data di Solana, diidentifikasi oleh pubkey, dimiliki oleh program, menyimpan byte data dan lamport. Lihat [`solana-fundamentals/account-model`](/id/solana-fundamentals/account-model).

**Address Lookup Table (ALT)** — Account yang dipublikasikan yang memungkinkan transaksi mereferensikan account dengan indeks 1 byte, menghemat byte. Raydium SDK menggunakan ALT untuk rute multi-hop. Lihat [`solana-fundamentals/transactions-and-fees`](/id/solana-fundamentals/transactions-and-fees).

**AmmConfig** — Account konfigurasi untuk CPMM/CLMM yang mendefinisikan tier fee dan bagian protocol fee. Berbagai konfigurasi dapat coexist; pool mereferensikan satu saat dibuat. Lihat [`ray/protocol-fees`](/id/ray/protocol-fees).

**AMM v4** — AMM constant-product asli Raydium (2021). Awalnya desain hybrid yang mencerminkan kurva ke OpenBook; integrasi OpenBook telah dinonaktifkan dan pool sekarang beroperasi sebagai AMM pure constant-product. Masih sepenuhnya operasional; tidak lagi default yang direkomendasikan untuk pool baru. Lihat [`products/amm-v4/overview`](/id/products/amm-v4/overview).

**Anchor** — Framework Rust untuk program Solana yang menangani validasi account, error handling, dan menghasilkan IDL. Digunakan oleh CPMM, CLMM, Farm v6, LaunchLab. Lihat [`solana-fundamentals/programs-and-anchor`](/id/solana-fundamentals/programs-and-anchor).

**Anchor discriminator** — Prefix 8 byte pada account dan instruksi yang dikelola Anchor, diturunkan dari nama account/instruksi. Mengidentifikasi tipe account. Lihat [`solana-fundamentals/account-model`](/id/solana-fundamentals/account-model).

**Arbitrage** — Memanfaatkan perbedaan harga antar venue (Raydium ↔ Orca ↔ CEX) untuk menjaga harga tetap sejalan. Fundamental untuk price discovery AMM.

**Associated Token Account (ATA)** — Token account konvensional yang diturunkan dari PDA untuk pasangan (user, mint). Lihat [`solana-fundamentals/spl-token-and-token-2022`](/id/solana-fundamentals/spl-token-and-token-2022).

## B

**Bonding curve** — Fungsi pricing untuk token yang secara deterministik menetapkan harga sebagai fungsi dari supply. Digunakan dalam pre-graduation LaunchLab. Lihat [`products/launchlab/bonding-curve`](/id/products/launchlab/bonding-curve).

**BPF Loader (Upgradable)** — Program loader Solana yang mendeploy dan upgrade program. Lihat [`solana-fundamentals/programs-and-anchor`](/id/solana-fundamentals/programs-and-anchor).

**Bump** — Byte tunggal yang ditambahkan ke seed untuk menurunkan PDA off-curve. Canonical bump adalah nilai tertinggi yang menghasilkan alamat off-curve. Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

## C

**Canonical bump** — Bump yang ditemukan dengan menurun dari 255. Program Raydium selalu menggunakan canonical bump. Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

**CLMM** — Concentrated Liquidity Market Maker, produk Raydium 2022 gaya Uniswap-V3. Lihat [`products/clmm/overview`](/id/products/clmm/overview).

**Compute Budget** — Limit per-transaksi pada compute unit. Default 200k, max 1.4M. Diset via `ComputeBudget::SetComputeUnitLimit`. Lihat [`solana-fundamentals/transactions-and-fees`](/id/solana-fundamentals/transactions-and-fees).

**Compute Unit (CU)** — Unit biaya komputasi Solana. Priority fee dikenakan per CU. Lihat [`integration-guides/priority-fee-tuning`](/id/integration-guides/priority-fee-tuning).

**Constant product** — Invarian AMM `x × y = k`. Lihat [`algorithms/constant-product`](/id/algorithms/constant-product).

**CPI (Cross-Program Invocation)** — Satu program Solana memanggil program lain dalam satu transaksi. Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

**CPMM** — AMM constant-product modern Raydium (2024). Menggantikan AMM v4 untuk pool baru. Lihat [`products/cpmm/overview`](/id/products/cpmm/overview).

**Creator fee (LaunchLab)** — Persentase opsional dari swap fee yang diarahkan ke token creator selama fase bonding-curve. Lihat [`products/launchlab/overview`](/id/products/launchlab/overview).

## D

**DLMM** — Dynamic Liquidity Market Maker. Varian CLMM Meteora. Bukan Raydium.

**Discriminator** — Lihat "Anchor discriminator".

## E

**Emissions (Farm)** — Laju distribusi reward farm, dalam token-per-second. Lihat [`products/farm-staking/overview`](/id/products/farm-staking/overview).

**Escrow** — PDA-held token account menyimpan dana hingga kondisi terpenuhi. Pola integrasi umum. Lihat [`integration-guides/cpi-integration`](/id/integration-guides/cpi-integration).

**Executable account** — Account Solana dengan `executable = true`, menyimpan bytecode program. Lihat [`solana-fundamentals/account-model`](/id/solana-fundamentals/account-model).

## F

**Farm v3 / v5 / v6** — Generasi berturut-turut dari program reward-farming Raydium. v6 adalah yang saat ini; v3/v5 legacy. Lihat [`products/farm-staking/overview`](/id/products/farm-staking/overview).

**Fee growth (CLMM)** — Counter fee-per-unit-liquidity global, dilacak di `fee_growth_global_X` dan `fee_growth_global_Y`. Lihat [`products/clmm/math`](/id/products/clmm/math).

**Fee tier** — Persentase swap-fee spesifik yang ditawarkan oleh pool. Mis., CLMM tier 0.05%. Lihat [`reference/fee-comparison`](/id/reference/fee-comparison).

**Finalization** — Tingkat komitmen terkuat Solana, ~13 detik setelah tx landing. Lihat [`solana-fundamentals/transactions-and-fees`](/id/solana-fundamentals/transactions-and-fees).

**Freeze authority** — Kekuatan untuk freeze (lock) token account untuk mint tertentu. Risiko bagi user yang LP ke token dengan active freeze authority.

## G

**Graduation (LaunchLab)** — Event di mana bonding-curve launch mencapai threshold dan auto-migrate ke pool CPMM. Lihat [`products/launchlab/overview`](/id/products/launchlab/overview).

## H

**Hybrid AMM** — AMM yang mengintegrasikan dengan orderbook eksternal untuk merutekan trade melalui whichever yang memiliki harga lebih baik. AMM v4 awalnya hybrid (terhadap OpenBook), tetapi sisi orderbook telah dinonaktifkan, jadi AMM v4 sekarang beroperasi sebagai AMM pure. CPMM bukan hybrid.

## I

**IDL** — Interface Description Language. Deskripsi JSON Anchor dari instruksi, account, event, error program. Lihat [`sdk-api/anchor-idl`](/id/sdk-api/anchor-idl).

**Impermanent loss (IL)** — Perbedaan dalam nilai token LP versus memegang token underlying, karena price divergence. Fundamental untuk AMM. Lihat [`algorithms/constant-product`](/id/algorithms/constant-product).

**Invoke / invoke_signed** — Syscall Solana untuk CPI. `invoke_signed` digunakan untuk CPI yang diotorisasi PDA. Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

## J

**Jito** — Solana block-engine yang mendukung bundled transaction untuk proteksi MEV. Lihat [`integration-guides/routing-and-mev`](/id/integration-guides/routing-and-mev).

**Jupiter** — DEX aggregator Solana dominan. Merutekan ~60% volume-nya melalui Raydium. Lihat [`introduction/ecosystem-position`](/id/introduction/ecosystem-position).

## K

**Kudelski** — Firma audit yang telah mengaudit program Raydium. Lihat [`security/audits`](/id/security/audits).

## L

**Lamport** — Unit SOL terkecil. 1 SOL = 1.000.000.000 lamport.

**LaunchLab** — Platform token-launch Raydium (2024). Bonding-curve launch yang graduate ke CPMM. Lihat [`products/launchlab/overview`](/id/products/launchlab/overview).

**Liquidity L (CLMM)** — Kontribusi posisi terhadap pool pricing, dalam unit yang didefinisikan oleh math Uniswap V3. Lihat [`algorithms/clmm-math`](/id/algorithms/clmm-math).

**LP (Liquidity Provider)** — User yang menyetor token ke pool sebagai imbalan LP token (atau NFT untuk CLMM) dan klaim pada future fee.

**LP token** — SPL token yang mewakili share LP dalam pool CPMM atau AMM v4. CLMM menggunakan NFT sebagai gantinya.

## M

**MadShield** — Firma audit yang telah mengaudit program Raydium. Lihat [`security/audits`](/id/security/audits).

**Master-chef pattern** — Desain reward-tracking berdasarkan kontrak MasterChef SushiSwap. Gaya akumulator `reward_per_share`. Digunakan oleh Farm v6.

**Memecoin** — Istilah informal untuk token yang didorong komunitas/spekulasi. Banyak yang diluncurkan via LaunchLab.

**MEV (Maximal Extractable Value)** — Nilai yang dapat diekstraksi oleh transaction ordering (front-running, sandwich, back-running). Lihat [`integration-guides/routing-and-mev`](/id/integration-guides/routing-and-mev).

**Mint** — Account Token Mint yang mendefinisikan token. Jangan bingung dengan "minting" verba (mengeluarkan supply baru).

**Multisig** — Account multi-signature memerlukan N-of-M signer. Raydium menggunakan Squads multisig untuk authority. Lihat [`security/admin-and-multisig`](/id/security/admin-and-multisig).

## N

**Non-transferable (Token-2022)** — Ekstensi yang mencegah transfer token. Diblokir oleh penciptaan pool Raydium.

**NFT** — Non-Fungible Token. Posisi CLMM adalah NFT; begitu juga beberapa artifact LaunchLab.

## O

**Observation / ObservationState** — Account price-history on-chain CLMM. Menyimpan cumulative price weighted by time, mengaktifkan TWAP oracle. Lihat [`products/clmm/accounts`](/id/products/clmm/accounts).

**OpenBook** — Dulu Serum. Limit orderbook sentral Solana. AMM v4 *awalnya* terintegrasi dengan OpenBook, tetapi integrasi itu telah dinonaktifkan; CPMM dan CLMM tidak pernah terintegrasi dengannya. Lihat [`products/amm-v4/overview`](/id/products/amm-v4/overview).

**Oracle** — External price feed. Pyth, Jupiter, dan (untuk internal use) CLMM's ObservationState TWAP. Lihat [`security/oracle-and-token-risks`](/id/security/oracle-and-token-risks).

**Orca** — Rival Solana DEX. Whirlpools adalah produk CLMM Orca.

## P

**PDA (Program-Derived Address)** — Alamat yang diturunkan secara deterministik dari program + seed. Tidak ada private key; program pemilik menandatangani via `invoke_signed`. Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

**Permanent delegate (Token-2022)** — Ekstensi yang memberikan wallet designated kekuatan transfer/burn tanpa syarat. Diblokir oleh penciptaan pool Raydium.

**Phoenix** — Native Solana orderbook DEX. Peer, bukan pesaing langsung.

**PnL (AMM v4)** — Historis: Profit and Loss dari OpenBook fill, terakumulasi ke pool AMM v4 dan diselesaikan via crank `MonitorStep`. Dengan integrasi OpenBook dinonaktifkan, tidak ada OpenBook PnL baru yang terakumulasi — counter `state_data.total_pnl_*` membawa nilai historis saja. Path protocol-fee 0.03% yang menggunakan `need_take_pnl_*` tidak terpengaruh dan masih aktif.

**Pool authority** — PDA yang menandatangani transfer vault pool. Satu per program (bukan per pool). Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

**Pool state** — On-chain account menyimpan reserve, fee, konfigurasi pool. Central data structure per produk.

**Priority fee** — Per-CU fee dibayarkan ke validator untuk prioritize transaksi dalam blok. Lihat [`integration-guides/priority-fee-tuning`](/id/integration-guides/priority-fee-tuning).

**Program account** — Account menyimpan metadata/bytecode program. Lihat [`solana-fundamentals/programs-and-anchor`](/id/solana-fundamentals/programs-and-anchor).

**Program Derived Address** — Lihat PDA.

**Protocol fee** — Porsi swap fee yang diarahkan ke RAY buyback atau treasury daripada LP. Lihat [`ray/protocol-fees`](/id/ray/protocol-fees).

**Pyth** — Jaringan oracle Solana. Digunakan oleh frontend untuk USD display; bukan oleh math pool Raydium.

## Q

**Q64.64 (sqrt_price_x64)** — Representasi fixed-point: 64 integer + 64 fractional bit. Digunakan oleh CLMM untuk sqrt-price menghindari floating-point drift. Lihat [`algorithms/clmm-math`](/id/algorithms/clmm-math).

## R

**\$RAY** — SPL token native Raydium. Lihat [`ray/index`](/id/ray).

**Realized fee (CLMM)** — Fee terakumulasi ke posisi sejak `collectFee` terakhir. Diklaim via `collectFee`. Lihat [`products/clmm/math`](/id/products/clmm/math).

**Remaining accounts** — Account tambahan ditambahkan ke daftar account instruksi, diinterpretasi berdasarkan posisi. Digunakan untuk daftar account variable-length (tick array, transfer-hook extra). Lihat [`solana-fundamentals/pdas-and-cpis`](/id/solana-fundamentals/pdas-and-cpis).

**Rent** — Solana mengenakan biaya storage, tetapi semua account modern harus rent-exempt (menyimpan cukup lamport untuk pre-pay rent selamanya). Lihat [`solana-fundamentals/account-model`](/id/solana-fundamentals/account-model).

## S

**Sandwich attack** — Taktik MEV: front-run swap user, kemudian back-run untuk ekstrak price impact. Lihat [`integration-guides/routing-and-mev`](/id/integration-guides/routing-and-mev).

**Serum** — Pendahulu OpenBook. Saat ini tidak aktif; lihat "OpenBook".

**Slippage** — Perbedaan antara quoted dan actual output, dari price impact + price movement. Dikontrol via parameter `min_out` pada swap.

**SOL** — Native token Solana.

**SPL Token** — Program token Solana legacy. Alamat `Tokenkeg...`. Lihat [`solana-fundamentals/spl-token-and-token-2022`](/id/solana-fundamentals/spl-token-and-token-2022).

**sqrt_price_x64** — Lihat Q64.64.

**Squads** — Platform multisig Raydium gunakan untuk operational dan upgrade authority. Lihat [`security/admin-and-multisig`](/id/security/admin-and-multisig).

**Swap / SwapV2** — Varian instruksi Raydium. `SwapV2` mendukung Token-2022; legacy `Swap` tidak. Lihat [`solana-fundamentals/spl-token-and-token-2022`](/id/solana-fundamentals/spl-token-and-token-2022).

## T

**Tick (CLMM)** — Discrete price level. Posisi merentang range tick. Lihat [`algorithms/clmm-math`](/id/algorithms/clmm-math).

**Tick array** — PDA menyimpan state 60 tick berturut-turut. Mengaktifkan chunked storage dari tick lattice. Lihat [`products/clmm/ticks-and-positions`](/id/products/clmm/ticks-and-positions).

**Tick spacing** — Integer gap antara usable tick dalam pool. Terikat ke fee tier (0.01% → 1, 0.25% → 60, dll.). Lihat [`algorithms/clmm-math`](/id/algorithms/clmm-math).

**Timelock** — Delay antara approval dan execution dari action sensitif. Upgrade Raydium memiliki 24-hour timelock. Lihat [`security/admin-and-multisig`](/id/security/admin-and-multisig).

**Token-2022** — Program token yang lebih baru dengan ekstensi. Alamat `TokenzQ...`. Lihat [`solana-fundamentals/spl-token-and-token-2022`](/id/solana-fundamentals/spl-token-and-token-2022).

**Transfer fee (Token-2022)** — Ekstensi mengenakan persentase pada setiap transfer token. Diperhitungkan dalam CPMM/CLMM Raydium via `SwapV2`.

**Transfer hook (Token-2022)** — Ekstensi memanggil program pada setiap transfer. Didukung oleh Raydium dengan caveat.

**TVL (Total Value Locked)** — Sum balance pool/farm dalam USD. TVL Raydium per April 2026: ~\$1,8B.

**TWAP (Time-Weighted Average Price)** — Harga rata-rata weighted by time, diturunkan dari CLMM ObservationState. Lihat [`products/clmm/accounts`](/id/products/clmm/accounts).

## U

**UserLedger (Farm v6)** — PDA menyimpan stake balance user dan reward debt dalam farm.

## V

**Vault** — Token Account dimiliki oleh PDA authority program pool, menyimpan pool reserve. Setiap pool memiliki dua vault (satu per sisi).

**Versioned transaction (v0)** — Format transaksi Solana saat ini mendukung ALT. Legacy transaction adalah version 0xff. Lihat [`solana-fundamentals/transactions-and-fees`](/id/solana-fundamentals/transactions-and-fees).

## W

**Whirlpools** — Produk CLMM Orca. Desain serupa dengan CLMM Raydium; codebase berbeda.

**Wrapped SOL (wSOL)** — SOL dibungkus sebagai SPL Token untuk digunakan dalam konteks token-account-only. Dibuat dan ditutup per-transaksi. Lihat [`user-flows/swap`](/id/user-flows/swap).

## X

**x × y = k** — Invarian constant-product. Lihat [`algorithms/constant-product`](/id/algorithms/constant-product).
