论坛 API 支持围绕 LaunchLab 流动性池的讨论和社区互动。你可以发布带可选图像附件的评论，并获取流动性池讨论线程。所有写操作需要通过 Auth API 进行身份验证。

## API 端点

**主网：** https://launch-forum-v1.raydium.io  
**Devnet：** https://launch-forum-v1-devnet.raydium.io

## 身份验证

**读操作 (GET)：** 不需要身份验证。

**写操作 (POST)：** 需要 `ray-token` 请求头 — 这是钱包签名验证后由 Auth API 签发的 JWT。

[从 Auth API 获取 token](/zh/api-reference/launch-auth-v1/overview)

## 请求头

发布评论时，请包含身份验证请求头：

```
ray-token: <jwt_token_from_auth_api>
```

## 功能特性

- **发布评论：** 提交文本评论，支持可选的图像附件（最大 5 MB）
- **浏览讨论：** 查看流动性池上的所有评论，支持分页
- **内容审核：** NSFW 检测和隐藏评论过滤
- **分页：** 使用 `lastId` 和 `fetchType`（old/new）查询，实现高效浏览
- **图像上传：** 评论支持 Pinata IPFS 图像托管

## 使用场景

- **社区讨论** — 围绕代币项目的讨论
- **项目公告** — 通过评论发布更新和公告
- **用户反馈** — 反馈收集和情感跟踪
- **社交证明** — 社区参与度指标
- **内容审核** — NSFW 和垃圾信息检测
