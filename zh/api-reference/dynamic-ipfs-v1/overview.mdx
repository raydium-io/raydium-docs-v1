Dynamic IPFS API 为 Solana 上的 CLMM 和 CPMM 流动性头寸提供重新生成的元数据与图像。该服务专为需要渲染动态 NFT（例如集中流动性头寸卡片）的 dApp 设计，可实时提供流动性池信息、头寸估值以及未领取的手续费，无需直接查询链上数据。

## 访问地址

| 环境 | Host |
|------|------|
| 主网 | `dynamic-ipfs.raydium.io` |
| 开发网 | `dynamic-ipfs-devnet.raydium.io` |

## 概览

该 API 提供三个端点，每个端点均返回完整的头寸信息对象：

- **`/clmm/position`** — 获取活跃 CLMM 头寸的元数据
- **`/lock/clmm/position`** — 获取已锁定 CLMM 头寸的元数据
- **`/lock/cpmm/position`** — 获取已锁定 CPMM 头寸的元数据

每个响应均包含完整的流动性池详情（代币、精度、TVL）、头寸数量、USD 估值、头寸在池中的 TVL 占比，以及已累计但尚未领取的手续费和奖励。

## 使用场景

通过请求特定头寸的元数据，生成动态头寸可视化 NFT。该 API 从链上数据实时计算头寸状态并缓存结果，非常适合以下场景：

- 渲染显示实时 USD 价值和手续费累计情况的头寸卡片
- 展示 TVL 占比和池中代币数量
- 显示未领取的交易手续费和奖励代币
- 构建用于追踪头寸表现的投资组合仪表盘

## 鉴权

无需鉴权，所有端点均公开访问。

## 速率限制

暂无强制限制，按尽力服务原则提供。

## 常见用法

### 获取 CLMM 头寸

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

`id` 参数应为头寸账户的链上公钥（base58 格式的 Solana 地址）。

### 获取已锁定头寸

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
