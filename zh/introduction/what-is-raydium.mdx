  Raydium 是 Solana 上使用最广泛的去中心化交易所和流动性协议。它始于 2021 年的一个 AMM + OpenBook 整合 DEX，已发展成完整产品套件：经典 AMM 池、常数乘积做市商、集中流动性、代币发行和永续合约界面。本页介绍 Raydium 的功能、存在的理由以及后续阅读方向。

## 一句话总结

Raydium 协议是一套无需许可、非托管的智能合约集合，为 Solana 上的代币交换、流动性池、流动性头寸和代币发行提供支持。[Raydium.io](https://raydium.io) 是官方网页应用和一站式使用入口：你可以在同一界面中交换代币、提供流动性、通过 LaunchLab 发行代币以及使用 Raydium 永续合约。截至 2026 年 4 月，Raydium 是 Solana 上最大的 TVL DEX，约 \$1.8B，路由了 Solana 大部分交易量，并为大多数 Solana 钱包内置的代币交换功能提供流动性支持。

## 核心产品

Raydium 的核心产品涵盖交换、流动性、代币发行、激励和永续合约。其中一些是 Raydium 原生链上程序；Raydium 永续合约是由 Orderly Network 提供支持的界面产品。

### AMM v4 — 常数乘积

原始 Raydium 产品（2021 年）。使用常数乘积数学模型（`x × y = k`）进行池定价。它最初还集成了 OpenBook 的中央限价订单簿，以便交易可以针对 AMM 或订单簿成交，但 **OpenBook 集成已被停用**——池不再与 OpenBook 共享流动性，所有当前交换流量仅通过 AMM 曲线流动。仍运行约 \$300M TVL 并完全正常运作，但 **不再推荐用于新池**：不支持 Token-2022，且 CPMM 对于典型新交易对更便宜、更灵活。

参见 [`products/amm-v4/overview`](/zh/products/amm-v4/overview)。

### CPMM — 现代常数乘积 AMM

AMM v4 的继任者（2024 年）。相同的 `x × y = k` 数学，但使用更简洁、更便宜：无 OpenBook 依赖、原生 Token-2022 支持、创建成本低约 4 倍，所有交换费用默认流向 LP。推荐用于新池的默认选项。

参见 [`products/cpmm/overview`](/zh/products/cpmm/overview)。

### CLMM — 集中流动性

Uniswap V3 风格的集中流动性（2022 年）。LP 选择价格范围；其流动性仅在价格在该范围内时才活跃。相同资本 → 为交易者提供更好的报价 → LP 获得更高的费用 APR，代价是需要主动管理范围。2026 年深度流动性池的主导产品。

参见 [`products/clmm/overview`](/zh/products/clmm/overview)。

### LaunchLab — 代币发行

**LaunchLab** 是 Raydium 的代币发行平台（2025 年）。项目在绑定曲线上发行代币；买家直接与曲线交易；一旦达到毕业阈值，曲线累积的流动性自动迁移到成熟的 CPMM 池。一级市场和二级市场统一。

参见 [`products/launchlab/overview`](/zh/products/launchlab/overview)。

## 谁在使用 Raydium

- **零售交换者**：通过 [raydium.io](https://raydium.io) 或任何钱包的内置交换（通过 Solana 聚合器层路由）交换代币。
- **流动性提供者**：存入池中赚取交换费；质押到农场获得额外奖励。
- **永续合约交易者**：通过由 Orderly Network 提供支持的 Raydium 永续合约交易永续市场。
- **代币团队**：通过 LaunchLab 发行新代币、为发行后流动性创建 CPMM 池、运营农场以激励 LP 保留。
- **聚合器**：Jupiter 和大多数第三方路由器将 Raydium 视为主要流动性场所。
- **机器人和协议**：套利、做市、自动复合、借贷协议清算——所有这些都可以通过 CPI 或直接 RPC 与 Raydium 组合。

## Raydium 流动性的出现位置

如果你符合以下任何情况，说明你已在使用 Raydium：

- 使用任何 Solana 钱包的内置交换——大多数通过 Raydium 池路由（通常通过 Jupiter）。
- 在 Jupiter 上交易——约 60% 的 Jupiter 路由交易量流向 Raydium 池。
- 使用 Drift、Kamino、MarginFi 或其他 DeFi 协议——其中许多通过 Raydium CLMM 获取价格馈送和清算路由。
- 持有 Solana memecoin——许多通过 LaunchLab 发行并存在于毕业后的 CPMM 池中。

聚合器和钱包都基于 Raydium 的公开 API（`api-v3.raydium.io`）和直接链上池状态构建。Raydium 不需要任何特殊合作伙伴关系进行集成——它是无需许可的。

## 关键优势

与其他 Solana DEX 相比：

| | Raydium | Orca | Phoenix | Jupiter |
|--|---------|------|---------|---------|
| AMM | CPMM, AMM v4 | 无 | 无 | 无（聚合器） |
| CLMM | CLMM | Whirlpools | 无 | 无 |
| 订单簿 | 目前无（AMM v4 的 OpenBook 集成已停用） | 无 | 原生订单簿 | 无 |
| 代币发行 | LaunchLab | WaveBreak | 无 | 无 |
| 农场 | 有 | 有 | 无 | 无 |
| TVL（2026 年 4 月） | ~\$1.8B | ~\$800M | ~\$200M | 聚合器；路由每月数十亿 |

与 Uniswap 等以太坊 DEX 相比：

- **交易最终性**：Solana 约 1 秒 vs 以太坊约 12 秒。
- **交易成本**：典型 \$0.001–0.01 vs 以太坊主网典型 \$5–50。
- **内存池设计**：Solana 无公开内存池；以太坊有。这影响 MEV、抢先交易模式和集成设计。
- **Token-2022**：Raydium 支持原生 Token-2022 扩展（转账费用、钩子等）；Uniswap V3 在 ERC-20 上没有等价物。
