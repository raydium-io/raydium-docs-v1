Raydium 是围绕独立 Solana 程序和集成构建的一套产品功能。从这里开始选择合适的产品，然后使用产品页面查看账户、数学原理、指令、费用和代码示例。

## 产品对比

| 产品 | 市场类型 | Token-2022 | OpenBook 依赖 | 使用场景 |
|---------|-------------|------------|---------------------|-------------|
| **AMM v4** | 恒定乘积 (xy=k) | 否 | 非活跃（原本是；OpenBook 集成已停用） | 依赖 v4 市场布局的现有流动性池和集成。新池默认使用 CPMM。 |
| **CPMM** | 恒定乘积 (xy=k) | 是 | 否 | 新的恒定乘积流动性池。支持转账手续费代币。Gas 成本低于 v4。 |
| **CLMM** | 集中流动性（sqrt 价格、ticks） | 是 | 否 | 稳定交易对、紧密范围流动性提供、资本高效的 LP 头寸。 |
| **Farm / Staking** | 奖励分配 | 部分支持 | 否 | 激励 LP 头寸或单资产质押。 |
| **Perps** | 永续期货 | 不适用 | 否 | 通过 Raydium Perps（由 Orderly Network 提供支持）进行杠杆交易。 |
| **Stable AMM** | 稳定币关联 AMM | 否 | 非活跃 AMM v4 样式的接线 | 现有稳定币池和集成；不是新池的默认选项。 |
| **AMM Routing** | 多池路由 | 取决于路由池 | 取决于路由池 | 在一条路由中组合多个 Raydium AMM 池。 |

LaunchLab 有自己的 `Launch tools` 标签页，因为代币发行需要专用的用户和构建者流程。
