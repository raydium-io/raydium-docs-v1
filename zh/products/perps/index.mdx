## 产品概览

**Raydium Perpetual** 是一个无 Gas 费的中央限价订单簿（CLOB）永续合约协议。头寸由 Solana 上的 SPL 钱包注资，但撮合引擎和订单簿由 [Orderly Network](https://orderly.network/) 运营，该网络在多条区块链上（Solana、Ethereum、Base、Arbitrum、BSC）汇集流动性。

主要特性：

- **无 Gas 费交易**。订单创建、修改和取消无需在 Solana 上进行链上交易——签名在本地进行且完全免费。只有充提才涉及链上操作。
- **最高 100 倍杠杆**，取决于具体市场。
- **多链抵押资产**——你可以从各自的原生链充入 USDC / SOL / USDT / ETH / BNB / WBTC / YUSD / USD1；所有盈亏以 USDC 结算。
- **仅限跨币种保证金**。账户中的所有抵押品在所有开仓头寸间共享。
- **单向模式**。你不能同时在同一市场上持有多头和空头头寸；反向订单会减少或翻转现有头寸。
- **自我保管**。资金保留在你的钱包的永续账户中；没有 Raydium 持有的托管钱包。
- **交易所级订单类型**。市价、限价、止损（市价 / 限价）、阶梯式下单，以及 IOC / FOK / Post-Only / Reduce-Only 标志。

## 为什么是独立产品

Raydium 的现货 AMM 和 Raydium Perps 共享品牌和 UI 入口，但在协议层面几乎没有相似之处：

| | Raydium 现货 AMM | Raydium Perps |
|---|---|---|
| 交易场所 | Solana 上的链上流动性池 | 由 Orderly 运营的链外 CLOB |
| 撮合引擎 | AMM 曲线（CPMM / CLMM / AMM v4） | 中央限价订单簿 |
| 结算方式 | 链上实时结算 | 链外结算；充提在链上进行 |
| 价格发现 | AMM 曲线 + 套利 | 订单簿 + 外部价格 |
| 代币范围 | 任何 SPL / Token-2022 铸币 | 仅上线的永续市场（BTC、ETH、SOL 等主流币种） |
| [`reference/program-addresses`](/zh/reference/program-addresses) 中的程序 ID | 有 | 无——详见 Orderly 的合约 |
| Raydium SDK v2 覆盖范围 | 完整 | 不涵盖 |
| Raydium 漏洞赏金 | 有 | 通过 Orderly 处理 |

本文档的其余部分几乎完全关注现货 AMM 和 Raydium 组织 GitHub 中的程序。Perps 在产品层面有文档——订单类型、费用、抵押品、限制——以便读者理解它的用途和使用时机，但深入的协议参考文档位于 Orderly 一侧。

## 集成注意事项

- 现货 Raydium SDK（`@raydium-io/raydium-sdk-v2`）**不**涵盖 Perps。如需编程访问，请使用 [Orderly 的 SDK](https://orderly.network/docs/build-on-omnichain/evm-api/introduction)（TypeScript、REST、WebSocket）。
- 没有"通过永续头寸路由现货互换"的原语。需要同时获得现货和永续敞口的客户端需要独立构建这两个交易。
- Orderly 的风险参数（杠杆上限、清算阈值、抵押品限额）会随时间变化。在向用户报价风险前，始终从 Orderly API 获取最新参数。
