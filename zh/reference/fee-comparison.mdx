## 各产品的交换费

费用值是输入金额的百分比，在池数学计算前应用。

| 产品 | 可用档位 | 默认/常用 | 分成 |
|---------|-----------------|---------------------|-------|
| **AMM v4** | 0.25%（仅此） | 0.25% | 88% LP，12% 协议（25 中的 22/3） |
| **CPMM** | 0.01%、0.25%、1% | 0.25% | 100% LP（协议分成按 AmmConfig 可配置，当前 0%） |
| **CLMM** | 0.01%、0.05%、0.25%、1% | 0.25% | 88% LP，12% 协议 |
| **LaunchLab（毕业前）** | 1%（默认） | 1% | 50% 到池储备，50% 到 LaunchLab 金库 |
| **LaunchLab（毕业后 CPMM）** | 0.25% | 0.25% | 继承 CPMM 分成 |

### 费用分母约定

- **AMM v4**：费用编码为 x/10,000。"25" 表示 0.25%。
- **CPMM/CLMM**：费用编码为 x/1,000,000。"2500" 表示 0.25%。

直接读取链上数据时，请根据池的程序使用正确的分母。

## Token-2022 费用互动

当池的一侧或两侧使用带转账费的 Token-2022 时，有效费用会复合：

```
net input to pool   = amount_in × (1 − transfer_fee_in)
pool charges swap   = net × swap_fee_rate
net output to user  = out × (1 − transfer_fee_out)
```

例：CPMM 0.25% 池，输入端有 1% 转账费，输出端有 0.5% 转账费。

```
amount_in        = 1000
net_to_pool      = 1000 × 0.99    = 990
pool swap fee    = 990 × 0.0025   = 2.475 (kept as LP fee)
amount_after_fee = 987.525
amount_out (AMM) = 987.525 × ratio, say 987.525 × 0.98 = 967.77
net_out_to_user  = 967.77 × 0.995 = 962.94
```

有效费用：**~3.7%** 尽管名义上池费仅为 0.25%。

SDK 通过 `getComputeAmountOut` 计算有效费率 — 引用 Token-2022 池时始终使用它。

## LaunchLab 细节

| 阶段 | 费用 |
|-------|-----|
| 绑定曲线交易（买或卖） | 1%（分成 50/50 池/金库） |
| 毕业（一次性） | 从曲线储备 ~0.04 SOL |
| 毕业后（CPMM 池） | 0.25%（标准 CPMM 费用） |

启动代币的项目通常包括「创建者费」选项（0–0.3%），在绑定曲线阶段从 LP 分成中扣除；在启动时可配置。创建者费**仅限绑定曲线** — 一旦启动毕业为 CPMM 池，结果池按标准 CPMM 档位运行，无创建者附加费。见 [`products/launchlab/creator-fees`](/zh/products/launchlab/creator-fees) 了解完整生命周期以及创建者如何通过 LaunchLab NFT 机制继续从毕业池中获利。

## Farm v6 奖励成本模型

农场本身不收费 — 它们是奖励分配。但运营需要 SOL：

| 成本 | 典型值 |
|------|---------|
| 农场创建 | 0.1 SOL 防止滥用 + ~0.02 SOL 租赁 |
| 补充奖励金库 | 仅代币 + ~0.0005 SOL tx |
| 回收未使用奖励 | ~0.0005 SOL tx |

农场还需要奖励金库中有足够的 lamports 来覆盖农场生命周期的租赁；关闭时返还。

## 如何在链上读取当前值

### CPMM/CLMM AmmConfig

```ts
const config = await raydium.cpmm.getAmmConfigs();
console.log(config[0]);
// {
//   id: "...",
//   tradeFeeRate: 2500,         // 2500 / 1_000_000 = 0.25%
//   protocolFeeRate: 0,         // 0 / 1_000_000 = 0%
//   fundFeeRate: 0,
//   ...
// }
```

### AMM v4 池

```ts
const poolInfo = await raydium.liquidity.getPoolInfoFromRpc({ poolId });
console.log(poolInfo.ammFeesNumerator, poolInfo.ammFeesDenominator);
// 25, 10000 → 0.25%
```

### Farm v6 奖励速率

```ts
const farm = await raydium.farm.getFarmInfoFromRpc({ id: farmId });
console.log(farm.farmData.rewardInfos);
// Per-reward: emissionsPerSecond, startTime, endTime
```
