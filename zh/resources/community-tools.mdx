## 列表内容

工具能够出现在此页面上需要满足以下条件：(a) 公开可用，(b) 在列表发布时确实能与 Raydium 主网程序正常交互，(c) 处于维护状态——即项目在过去 90 天内有提交、发布或明显的社区参与。

下面的分类是初步的分类方式；如果你提交的工具不属于任何现有分类，我们将添加新分类。

## 提交工具

要在此页面上添加或更新条目：

1. Fork [文档仓库](https://github.com/raydium-io)（文档源代码在公开的 Raydium 文档仓库中——确切路径在 README 中）。
2. 编辑 `resources/community-tools.mdx`。
3. 提交包含以下内容的 PR：
   - 工具名称和链接。
   - 一行描述（它是什么，属于哪个分类）。
   - 维护者的 GitHub 或 X 账号（用于验证）。
   - 关于活跃度的说明（最后一次发布 / 最后一次重要提交）。

列表会根据"公开可用 / 确实可行 / 处于维护状态"的标准进行审核。没有实际软件的自我宣传列表，或主要目的与用户安全冲突的工具（资金抽取脚本、虚假池生成器），将不会被合并。

## 删除政策

在以下情况下，工具将从此页面删除：

- 托管实例已离线超过 30 天。
- 仓库已存档或删除。
- 维护者超过 180 天保持沉默，工具停止工作。
- 工具开始从事伤害用户的行为（资金抽取脚本、欺骗性 UI 等）。

删除由文档维护者自行决定；工具恢复在线并得到维护后，欢迎恢复上架。

## 安全提示

此处列出的要求你提供钱包助记词、需要你将资金发送到托管地址或承诺保证收益率的工具，**不是**你应该使用的工具。此处的列表门槛很低——它不能保证安全。在签署交易之前，请始终检查你正在授权的内容。
