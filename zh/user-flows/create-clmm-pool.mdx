  CLMM 池需要两个永久性选择：**费用等级**和**初始价格**。你还需要开设初始头寸，这样池中才有流动性。签署前请仔细检查。

## 所需条件

- 包含以下内容的 Solana 钱包：
  - 你想配对的两个代币 mint。
  - 大约 0.3 SOL，用于池和头寸创建的租赁费用以及优先费用。
- 已确定的费用等级。
- 明确的初始市场价格和首个范围。

CLMM 同时支持 SPL Token 和 Token-2022 mint。

## 费用等级

| 费用等级 | 适用场景 |
|----------|----------|
| 0.01% | 稳定币对稳定币 |
| 0.05% | 相关联的代币对 |
| 0.25% | 大多数波动率高的主流交易对 |
| 1% | 长尾或异域代币对 |

更低的费用等级需要更紧密的价格和更高的成交量。更高的费用等级可以容忍更多波动，但可能降低成交量。

## UI 操作步骤

在 [raydium.io/clmm/create-pool](https://raydium.io/clmm/create-pool)：

1. **选择代币对**。粘贴两个 mint 地址。
2. **选择费用等级**。使用 UI 推荐作为起点。
3. **设置初始价格**。使用现有的市场价格。如果该代币对已有池存在，与之进行比较。
4. **选择首个范围**。输入初始头寸的下限价格和上限价格。
5. **输入存入数量**。UI 会根据你的范围计算所需的代币比例。
6. **检查并签署**。确认代币对、费用等级、初始价格、范围、存入数量和租赁费用。

## 正确设置初始价格

如果同一代币对已存在其他 CLMM 或 CPMM 池，使用其当前价格作为起点。如果 Raydium 上不存在该池，使用可靠的市场数据源。

如果设置的初始价格错误，首次交易可能会套利该池并耗尽低价一侧的流动性。没有简单的重置按钮。通常需要提取流动性并重新创建池。

## 首个头寸：范围选择

- **紧密范围**：最适合相关联的代币对，但需要主动管理。
- **中等范围**：常见于波动率高的主流交易对，如 SOL/USDC。
- **宽范围**：资本效率较低，但更容易放任不管。
- **全范围**：表现更像 CPMM，但失去了集中流动性的大部分优势。

如果当前价格在你的范围之外，初始头寸将超出范围，不会赚取费用，直到价格移入范围。

## 创建后检查清单

1. **池页面加载**。确认池地址、代币对和费用等级。
2. **头寸出现**。在 Raydium 投资组合中检查头寸 NFT。
3. **范围正确**。确认下限价格和上限价格。
4. **流动性非零**。池应该为你选定范围显示活跃流动性。
5. **测试小额交换**。在公开分享池之前，验证路线是否有效。

## 故障排除

### 创建成本高于预期

宽范围和小费用等级可能需要更多 tick 数组租赁。缩小起始范围或在重试前添加更多 SOL。

### 代币顺序错误

UI 会将代币顺序标准化，因此显示的代币 A/代币 B 可能与你输入的顺序不同。签署前仔细审查价格报价。

### 费用等级错误

池创建后无法更改费用等级。如果错误，请关闭流动性并在正确的等级创建新池。

### 零流动性池

创建池状态而不创建头寸会使池无法用于交换。在共享池之前开设初始头寸。

### 初始头寸超出范围

如果你的首个范围不包括初始价格，头寸不会赚取费用。围绕当前价格重新开设头寸。
