## 阶段 1——迁移前（联动曲线）

在代币于联动曲线上交易时，每次买入和卖出都会从中扣除创意者费用部分，并在每个代币专用的保险库中累积。这由你的代币启动所在的**平台**配置。

### 详情

- **计价方式**：以报价代币（通常是 SOL）计价。
- **费率**——由平台在启动时设置。查看平台的费用表或代币的链上状态。
- **可随时领取**——进入 raydium.io 上的**投资组合**页面，随时领取待领费用。无需等待毕业。
- **毕业时停止**——曲线毕业后不再产生新的迁移前费用。保险库中的任何余额仍可领取。

### 如何领取

1. 将创意者钱包连接到 [raydium.io](https://raydium.io)。
2. 导航到**投资组合**。
3. 在你创建的代币下找到该代币。
4. 点击迁移前费用余额旁的**领取**。

程序化领取：详见 [`products/launchlab/creator-fees`](/zh/products/launchlab/creator-fees) 了解 SDK 指令。

## 阶段 2——迁移后（费用密钥 NFT）

当代币的联动曲线达到毕业阈值时，流动性迁移到 Raydium CPMM 池。LP 代币根据**平台配置**铸造并分配：

| 接收方 | Raydium 默认值 | 说明 |
|---|---|---|
| 销毁（销毁与收益） | 90% | 永久锁定；该份额产生的交易费用流向销毁与收益信任机制。 |
| 创意者（费用密钥 NFT） | 10% | 费用密钥 NFT 被铸造到你的钱包。该 NFT 代表你对池中 10% 持续 LP 交易费用的权利。 |
| 平台 | 0%（Raydium 默认值） | 某些第三方平台在此处获得一份额。 |

这些默认值适用于通过 Raydium 自有 LaunchLab UI 启动的代币。第三方平台可能配置不同的分配比例——启动前务必查看平台页面。

### 费用密钥 NFT

费用密钥 NFT 是在毕业时铸造到你钱包的非同质化代币，代表你从毕业后池的份额中收集 LP 交易费的权利。

<Warning>
  **除非你有意放弃费用领取权，否则不要销毁或转移费用密钥 NFT。** 如果 NFT 丢失或被销毁，费用领取权**永久丧失**——没有恢复方案。
</Warning>

- 费用密钥可以交易或转移；持有人可以领取费用。
- 费用从 CPMM 池中的 LP 交易活动持续累积。
- 从投资组合页面领取，或通过 CPMM 池的费用领取指令以程序方式领取。

### 如何领取迁移后费用

1. 连接持有费用密钥 NFT 的钱包。
2. 在 raydium.io 上导航到**投资组合**。
3. 在你的头寸下找到已毕业代币的池。
4. 点击**领取**以收集已累积的交易费用。

---

## 如果迁移后费用被禁用

如果平台禁用创意者费用，你的代币会毕业到 **AMM v4 池**（遗留迁移路径）而非 CPMM。在此模式下：
- 不会铸造费用密钥 NFT。
- 没有程序级的迁移后创意者费用。
- 根据启动配置方式，可能存在一些通过 CLI 工具的替代费用领取选项。

## 费用流汇总

```
联动曲线阶段：
  每次买入/卖出 → 小额百分比 → 创意者保险库（报价代币）
  从投资组合随时领取

毕业：
  LP 代币铸造 → 根据平台配置分配
  ├── X% 销毁（销毁与收益）
  ├── Y% → 创意者作为费用密钥 NFT      ← 你持有这个
  └── Z% → 平台（如适用）

毕业后：
  每次 CPMM 交换 → LP 费用 → 按 LP 份额分配
  你的费用密钥 NFT 份额 → 从投资组合可领取
```
