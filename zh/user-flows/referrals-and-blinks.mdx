## 你会获得什么

- 一个可分享的 URL，可预填充 Raydium 交换 UI 并选择你想要的目标代币**以及你的钱包作为推荐人的接收地址**。
- 当任何人通过该链接进行交换时（在其会话中链接仍然活跃），**输入金额的 1% 会作为交换交易的一部分路由到推荐人的钱包**。
- 在 X.com 上分享的链接会呈现为 **Blinks**（Solana Actions），如果查看者的钱包启用了 Actions / Blinks —— 帖子内部的一键交换小部件。

## 如何创建链接

1. 打开 [raydium.io/swap](https://raydium.io/swap) 并连接你的钱包。推荐地址来自已连接的钱包；没有连接的情况下链接仍然可用，但无法获得费用。
2. 选择你想让他人交换成的**目标代币**。来源端由接收者选择。
3. 点击交换面板中的分享/链接图标（右上角）。
4. 复制生成的 URL。在任何地方分享它。

URL 格式是稳定的。如果你正在构建一个直接生成推荐链接的应用。

## Blinks 如何工作

**Blinks（区块链链接）** 是 Solana 基金会的标准，用于在社交平台内将链上操作 URL 呈现为交互式 UI。支持 Blinks 的 Solana 钱包会读取链接元数据，构建预构建的交易，并向用户显示内联签名提示。

要求：

- 查看者的**钱包必须启用 Actions / Blinks**。这通常是钱包设置中的一个开关。
- **平台必须遵守该标准**。X.com 通过其链接展开器支持；有些平台则不支持，在这种情况下链接会降级为普通超链接。

从推荐人的角度来看，Blink 渲染是自动进行的 —— 同一 URL 在不支持 Blinks 的钱包中也可以作为普通链接工作。

## 费用计算

1% 的推荐费是**在 AMM 交易费之上收取的**，而不是从其中扣除。对于费用总额为 0.25% 的池上的交换：

```
用户发送:        1,000 USDC
AMM 交易费:     2.50 USDC   (给 LP + 协议分成)
推荐费:         10.00 USDC  (1% → 推荐人的钱包)
交换净额:       987.50 USDC (然后通过曲线定价)
```

推荐费适用于由推荐链接创建的 Raydium 交换路由。推荐路由之外的直接池交互不涉及推荐费。

## 禁用 / 退出推荐

没有「移除推荐」指令。要进行没有推荐费的交换，只需直接加载 [raydium.io/swap](https://raydium.io/swap) 而不是点击推荐链接。当 URL 没有推荐人参数时，UI 不使用推荐人。

## 常见问题

**推荐人需要签署任何内容吗？** 不需要。推荐人的钱包地址是唯一随链接传输的数据；对推荐人的支付是在交换人的交易内部进行的代币转账。

**我可以推荐自己吗？** 可以，但没有意义 —— 你要支付自己 1%，还要减去你在一笔更大交易中刚刚支付的交易费。

**如果目标代币是带有转账钩子的 SPL 会怎样？** 推荐转账发生在输入端（以 USDC 或源代币计），而不是目标端，所以目标铸币的转账钩子不会影响推荐付款。输入代币上的转账钩子可能会完全阻止交换；Raydium 会提前拒绝这些。

**如果交换通过多个池路由，推荐费仍然适用吗？** 是的 —— 1% 是根据原始输入金额计算的，与路由无关。
