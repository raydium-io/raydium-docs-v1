## LaunchLab 代币是什么

LaunchLab 代币发行在**粘合曲线**上：一条由智能合约管理的价格曲线，随着更多代币被购买而价格上升，随着代币被售出而价格下降。曲线运行到达毕业阈值（通常为 85 SOL，但创建者可以设置不同的金额）为止，此时代币会自动迁移到 Raydium CPMM 或 AMM v4 池，并可通过正常 DEX 路由自由交易。

毕业前，你只能针对曲线本身进行买卖。毕业后，代币在 Raydium 上像任何其他池一样交易。

## 购买前

### 自己进行研究

- 验证代币的 mint 地址与项目官方宣布的地址相符——骗子经常创建名称相同的代币。
- 检查 **mint 权限是否已被撤销**。如果尚未撤销，创建者可以铸造新供应量并稀释你的头寸。Solflare 等钱包和 Solscan 等浏览器会显示此状态。
- 审查粘合曲线进度：已售出多少代币、剩余多少、当前价格。这在代币的 LaunchLab 页面上可见。
- 如果可能，检查创建者地址的历史。

### 理解曲线机制

粘合曲线上的价格**不是固定的**。每次购买都会推高价格；每次出售都会拉低价格。在你之前的大额购买可能会大幅提高你支付的价格（抢先交易/三明治攻击风险）。使用滑点限制（见下文）。

## 分步指南：通过 UI 购买

1. 导航至 [raydium.io/launchlab](https://raydium.io/launchlab) 并找到你想购买的代币。
2. 点击代币页面以打开其详情视图。
3. 如果还没连接钱包，现在连接。
4. 输入你想花费的 SOL（或报价代币）金额。
5. 查看 UI 中显示的**预期收到的代币**和**价格影响**。
6. 根据需要调整你的**滑点容限**（见下文）。
7. 点击**购买**并在钱包中批准交易。
8. 交易确认后，代币将出现在你的钱包中。

<Tip>
  毕业前，你可以随时使用同一代币页面上的**卖出**按钮将代币卖回给曲线。毕业后，代币在 Raydium 池中；改用标准交换界面。
</Tip>

## 滑点

<Warning>
  **设置合适的滑点容限。** 粘合曲线代币可能波动很大。如果你设置的滑点太低，价格在报价和执行之间移动时交易可能失败。如果设置得太高，你会面临三明治机器人的风险。对大多数启动，1–3% 是一个合理的起点；仅在接近毕业的流动性极低的启动上才根据需要提高。
</Warning>

滑点容限是你愿意接受的报价价格和执行价格之间的最大价格差。LaunchLab UI 允许你在确认交易前设置它。

## 毕业后

当曲线的毕业阈值被达到（由任何交易者）时，代币会自动迁移到 Raydium AMM 池。此时：

- 粘合曲线关闭；不再有买卖通过它。
- 你的代币保持在你的钱包中不变。
- 你现在可以在 Raydium 的交换页面或任何索引 Raydium 池的聚合器上交易代币。
- 池迁移可能需要几分钟才能出现在聚合器 UI 中。

## 风险

| 风险 | 描述 |
|---|---|
| 流动性控制风险 | LP 处理取决于平台和启动设置。检查 LP 是否被销毁、锁定、分配给创建者或分配给平台。 |
| 无毕业 | 如果曲线从未达到其阈值，交易仍然仅限于曲线。你仍然可以卖回给曲线，但价格可能低于你支付的价格。 |
| 价格波动 | 粘合曲线本质上是投机性的。价格可能跌至接近零。 |
| 假代币 | 骗子创建外观相似的代币。独立验证 mint 地址。 |
| 智能合约风险 | 没有链上程序是零风险的。仅使用你准备冒风险的资金。 |
