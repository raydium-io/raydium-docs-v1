## 你需要准备

- 一个 Solana 钱包，其中包含：
  - 足够的各个代币用于初始流动性提供。
  - 约 0.2 SOL 用于池创建租赁、代币账户创建和优先费用。
- 两个代币的 Mint 地址。
- 一个费用等级。默认的 0.25% 等级适合大多数波动性交易对。

## UI 操作步骤

在 [raydium.io/liquidity/create-pool](https://raydium.io/liquidity/create-pool)：

1. **选择池类型。** 选择 **Standard AMM (CPMM)**。仅在你理解范围管理时才使用 CLMM。
2. **输入两个代币。** 粘贴 Mint 地址或按符号搜索。对于 Token-2022 Mint，请查看任何警告信息。
3. **设置费用等级。** 常见选项是 0.01%、0.25% 和 1%。更高的费用适合波动性更大或长尾交易对。
4. **设置初始价格和存入金额。** 输入一侧的金额和初始价格。UI 将计算另一侧的金额。
5. **审查。** 检查 Mint 地址、费用等级、初始价格、LP 代币、池份额和一次性租赁费用。
6. **确认并签署。** 你的钱包签署池创建交易。
7. **等待索引。** 池立即在链上上线，但 Raydium UI 可能需要几分钟才能在搜索中显示它。

## 选择初始价格

池的首个价格来自你的初始存入比例。如果你存入 10 SOL 和 1000 USDC，则初始 SOL 价格为 100 USDC。

选择接近市场价格的价格。如果价格过高或过低，套利交易者将重新平衡池，你将为这个差异支付初始库存成本。

## 创建后检查清单

1. **打开池页面。** 确认交易对、费用等级和池地址。
2. **检查 LP 代币。** 你的钱包或 Raydium 投资组合应该显示新的 LP 代币。
3. **进行小额测试交换。** 用少量代币确认池的路由符合预期。
4. **等待发现。** 如果池没有立即出现在搜索中，几分钟后刷新并按 Mint 或池 ID 搜索。

## Token-2022 注意事项

如果任一 Mint 是带有转账费用的 Token-2022，则有效用户成本是池费用加上 Mint 的转账费用。UI 会在你签署前显示 Token-2022 警告。

如果任一 Mint 有转账挂钩，交换取决于该挂钩程序。如果挂钩程序更改或阻止转账，池可能会变得难以使用。将未知的转账挂钩 Mint 视为更高风险。

不可转账的 Mint 不能在 CPMM 池中使用。

## 故障排除

### 初始价格错误

如果你在签署前发现错误，返回并更正存入比例。如果你已经创建了池，通常需要以更正的价格添加流动性，让套利解决它，或创建新池。

### 优先费用过低

在网络拥堵期间，池创建可能过期。使用钱包或 Raydium UI 中的更高优先费用设置重试。

### 初始流动性太小

很小的池会产生不良的价格影响和糟糕的用户体验。在公开分享池之前添加更多流动性。

### 重复的池

同一代币对可以在不同费用等级存在多个 CPMM 池。创建前，请在 Raydium 中按两个代币 Mint 搜索，并检查项目是否已认可某个池。

### 有权限的 Mint

具有冻结权限的 Mint 可以冻结代币账户。对于未知 Mint，在提供大量流动性之前验证冻结权限和项目控制。
