## 两种创建模式

LaunchLab 在 [raydium.io/launchlab/create](https://raydium.io/launchlab/create) 上提供两条创建路径：

| | **JustSendit** | **LaunchLab** |
|---|---|---|
| 设置时间 | 约 1 分钟 | 约 5 分钟 |
| 代币供应 | 固定默认值 | 完全可配置 |
| SOL 目标 | 85 SOL（固定） | 最少 24 SOL，可自定义 |
| 曲线百分比 | 默认值 | 曲线上 51–80% |
| 解锁 | 无 | 可选 |
| 迁移后费用 | 默认值（创建者 LP 份额 10%） | 可配置 |
| 最适合 | 快速社区启动 | 结构化代币启动 |

## JustSendit 模式

最快的启动方式。需要最少输入；平台填入合理的默认值。

### 你需要准备的内容

- 一个已连接的钱包，持有足够的 SOL 用于交易费用（创建时约需 0.05–0.1 SOL）。
- 代币名称和代码符号。
- 符合 UI 要求的图像或 GIF。

### 步骤

1. 前往 [raydium.io/launchlab/create](https://raydium.io/launchlab/create) 并选择 **JustSendit**。
2. 输入代币 **名称** 和 **代码**。
3. 可选：添加 **描述** 和 **社交链接**（网站、Twitter/X、Telegram）。
4. 上传 **徽标图像或 GIF**。
5. 可选：输入初始购买金额（以 SOL 为单位）在创建时购买代币。这会改变曲线上第一个价格状态，应在签名前审查。
6. 审查摘要并点击 **创建**。
7. 在你的钱包中批准交易。

你的代币立即在粘合曲线上启动。一旦曲线收集了 85 SOL，它就会升级到 Raydium AMM 池。

## LaunchLab 模式

完全控制创建，具有可配置的供应量、融资目标、曲线分配、解锁和费用参数。

### 你需要准备的内容

- 一个已连接的钱包，持有约 0.1–0.5 SOL 用于创建和金库租赁。
- 在开始前准备好下面的所有输入（设置在启动后无法更改）。

### 配置选项

#### 代币详情

- **名称、代码、徽标** ——与 JustSendit 相同。
- **描述和社交链接** ——建议用于提高发现度。

#### 供应量和曲线分配

- **总供应量** ——要铸造的代币总数（例如 1,000,000,000）。
- **曲线分配（%）** ——放在粘合曲线上供公众购买的总供应量百分比（51–80%）。其余部分用于升级后的流动性池。

<Warning>
  曲线分配和总供应量在启动时固定，**无法更改**。签名前请仔细选择。
</Warning>

#### SOL 融资目标

- 最小升级阈值为 **24 SOL**。你可以设置更高的目标。
- 这是曲线必须收集的 SOL 数量，之后代币升级到 CPMM 池。
- 设置过高会冒险导致曲线永远无法升级（如果需求不足）。

#### 解锁（可选）

解锁会锁定部分代币供应，并按时间表线性释放给指定的钱包。

| 字段 | 说明 |
|---|---|
| 总锁定数量 | 跨所有受益人锁定的代币（占总供应量的百分比，受程序上限限制）。 |
| 悬崖期 | 升级后到任何代币解锁之前的等待时间（例如 180 天）。 |
| 解锁期 | 悬崖之后线性解锁的持续时间（例如 365 天）。 |

锁定的代币不进入曲线，也不是升级流动性池的一部分。它们按上述时间表逐步释放到每个受益人的钱包中。

**重要解锁规则：**
- 悬崖期和解锁期在启动时固定——无法事后更改。
- 如果受益人丢失钱包密钥，锁定的代币将永久无法访问。如果恢复很重要，请使用多签。
- 升级前未分配给受益人的 `total_locked_amount` 的任何部分都将永久卡住。请分配全部金额。

有关完整的链上机制，请参阅 [LaunchLab 解锁](/zh/products/launchlab/vesting)。

#### 迁移后费用分享

当代币升级时：
- LP 代币被铸造到 CPMM 池中。
- Raydium 默认设置：**90% 的 LP 代币被销毁**（通过 Burn & Earn 永久锁定）和 **10% 给予创建者** 作为费用密钥 NFT。
- 费用密钥 NFT 让你有权从迁移后的池中永远收取 10% 的 LP 交易费用。

如果你想从代币流动性获得持续收入并接受费用密钥 NFT 保管责任，请启用此选项。详情请参阅 [创建者费用如何运作](/zh/user-flows/how-creator-fees-work)。

### 步骤

1. 在创建页面选择 **LaunchLab** 模式。
2. 填入代币详情（名称、代码、徽标、描述、社交媒体）。
3. 设置 **总供应量** 和 **曲线分配百分比**。
4. 设置你的 **SOL 融资目标**（≥ 24 SOL）。
5. 配置 **解锁**（如需要，设置受益人钱包和时间表）。
6. 启用 **迁移后费用分享**（如果你想要创建者 LP 份额）。
7. 可选：输入初始购买金额。
8. 审查完整摘要——曲线价格范围、升级条件、LP 分配。
9. 点击 **创建** 并批准交易。

## 代币启动后

### 监控曲线进展

你的代币的 LaunchLab 页面显示实时进展：
- 曲线销售百分比。
- 筹集的 SOL 与升级目标的对比。
- 当前价格和价格图表。

LaunchLab 不会干预未能达到升级的启动。监控曲线进展并与社区清楚地沟通。

### 升级

升级由将曲线推过阈值的第一笔交易自动触发——任何用户都可以调用（调用者获得小额奖励）。作为创建者，你无需采取任何行动。

升级后：
- 使用收集的 SOL 和剩余曲线代币作为初始流动性创建 CPMM 池。
- 如果你启用了迁移后费用，**费用密钥 NFT** 被铸造到你的钱包。
- 从你的投资组合页面认领迁移前创建者费用。

### 认领迁移前创建者费用

如果你的平台在粘合曲线阶段启用了创建者费用，它们会在金库中累积。在任何时间从 raydium.io 的 **投资组合** 页面认领它们。

## 要避免的陷阱

| 陷阱 | 怎么做 |
|---|---|
| 设置 SOL 目标过高 | 选择市场能切实达到的目标。 |
| 在确认受益人前过度分配解锁 | 未分配的锁定供应在升级后永久无法访问。 |
| 无意中转移或销毁费用密钥 NFT | 费用索赔权将永久丢失。将 NFT 保留在创建钱包或你控制的钱包中。 |
| 不做初始购买 | 没有早期购买，狙击机器人可能以接近零的成本抢占所有早期代币，这可能会削弱社区信任。 |
