## 开始前的准备

确保你拥有：

- 要存入的代币对。
- 足够的 SOL 用于交易费和优先费。
- Raydium.io 上的正确池子。
- 清楚地理解头寸是被动 CPMM 流动性还是主动管理的 CLMM 流动性。

如果你要创建新池子，请从 [池子创建](/zh/user-flows/choosing-a-pool-type) 开始。

## 向 CPMM 池子添加流动性

1. 打开 Raydium 上的池子页面。
2. 点击 **Add liquidity**（添加流动性）。
3. 输入一个代币的数量。另一个数量将按照池子当前比例自动填充。
4. 查看你将收到的 LP 代币以及预期的池子份额。
5. 在 Raydium 中确认，然后在钱包中批准。

UI 会创建缺失的代币账户、支持的 Token-2022 转账费铸币的账户，并在需要时包装 SOL。

## 从 CPMM 池子移除流动性

1. 打开池子页面或你的投资组合头寸。
2. 点击 **Remove liquidity**（移除流动性）。
3. 输入 LP 代币数量，或使用百分比滑块。
4. 查看你将收到的两个代币的数量。
5. 确认并在钱包中批准。

提取后可能会留下少量舍入尘埃。这对恒定乘积池来说是正常的。

## 打开或管理 CLMM 头寸

CLMM 头寸是 NFT。它们有四个常见操作：

- **打开头寸**：铸造一个新头寸 NFT，并设置初始范围和流动性。
- **增加流动性**：向现有头寸添加更多流动性。
- **减少流动性**：移除部分流动性，同时保留 NFT。
- **关闭头寸**：移除所有流动性、收集待处理的费用和奖励，然后销毁 NFT。

## 打开 CLMM 头寸

1. 打开 CLMM 池子页面。
2. 选择下限价格和上限价格。
3. 输入存入的金额。
4. 查看头寸是在范围内、部分在范围内还是超出范围。
5. 确认并批准。

你选择的范围决定了代币比例。如果当前价格接近下限边，头寸可能主要需要一种代币。如果接近上限边，可能主要需要另一种代币。

## 增加、减少或关闭 CLMM 头寸

- **增加**：打开头寸卡，点击 **Increase**，输入数量，查看所需的代币对，然后签署。
- **减少**：点击 **Decrease**，选择百分比或数量，查看估计的代币返回，然后签署。
- **关闭**：将流动性减少至零、收集未领取的费用和奖励，然后关闭头寸。关闭会销毁头寸 NFT 并在适用时返还租金。

如果当前价格在你选择的范围之外，存入可能是单向的。在你签署单向或超出范围的头寸前，UI 会发出警告。

## 收获农场奖励

当你的 LP 代币质押在活跃农场时，农场奖励会累积。收获会将待处理奖励转移到你的钱包。它不会提取你的质押。

1. 打开 [raydium.io/portfolio](https://raydium.io/portfolio)。
2. 找到你的农场头寸。
3. 点击 **Harvest**（收获）。
4. 查看每条奖励流的待处理金额。
5. 确认并在钱包中批准。

农场的存入和提取通常会在同一操作中清算待处理奖励。仔细查看确认，因为它也可能收获待处理奖励。

## 收集 CLMM 费用和奖励

CLMM 头寸在两个池铸币中赚取交换费。费用在头寸上累积，不会自动复合。

1. 从 [raydium.io/portfolio](https://raydium.io/portfolio) 打开你的 CLMM 头寸。
2. 点击 **Collect fees**（收集费用）或 **Collect rewards**（收集奖励）。
3. 查看代币数量。
4. 确认并批准。

如果 UI 提供 **Collect all**（全部收集），使用它可以一起收集费用和奖励。

要手动复合，请收集费用、如需要则重新平衡两个代币，然后增加头寸的流动性。

## 时间和费用经济学

每次领取都需要支付 Solana 交易费，并在拥堵期间可能包括优先费。

- 对于少量待处理金额，请等待奖励相对于交易费有意义时再领取。
- 对于较大的头寸，每周或每月领取通常足够。
- 除非你在主动复合，否则通常不需要每天领取。

## 验证

在 CPMM 存入后：

- 检查你的钱包显示 LP 代币余额。
- 检查 Raydium 的投资组合页面显示该头寸。
- 打开交易签名并确认池子存入成功。

打开或增加 CLMM 头寸后：

- 检查你的投资组合显示头寸 NFT。
- 确认下限和上限价格范围与你选择的相匹配。
- 检查当前价格是否在范围内。

收获或收集后：

- 检查交易状态是否为 **Success**（成功）。
- 确认奖励或费用代币余额在你的钱包中增加。
- 刷新 Raydium 的投资组合页面，确认待处理金额重置或减少。

## 故障排除

### 存入比例的滑点

存入比例是在报价时计算的。如果你签署前池子价格移动，交易可能失败。重新报价并重试。仅在易波动的交易对需要时提高滑点。

### 存入时的 Token-2022 转账费

如果 Token-2022 铸币收取转账费，池子收到的金额会少于你发送的金额。Raydium 会在你签署前显示转账费警告。

### 提取时的尘埃

舍入可能在池子或你的钱包中留下微小金额。这是正常的，不表示提取失败。

### CLMM 头寸不是 LP 代币

CLMM 头寸是 NFT。检查 Raydium 的投资组合页面或你的钱包的 NFT 视图，而不是寻找可替换的 LP 代币。

### 奖励结束前收获

如果奖励流在你有待处理奖励时结束，待处理金额停止增长。除非农场以不寻常的方式被清算，否则你仍然可以在结束时间后领取它。

### 优先费过低

如果交易待处理或过期，请用更高的优先费设置重试。

### 跨多个农场的大量领取

如果包含太多头寸或奖励流，大批量领取可能失败。一次领取几个头寸。
