## 迁移的原因

| 维度 | AMM v4 | CPMM |
|-----------|--------|------|
| 费用流向 | 协议 + LP 分成 | LP 获得默认交换费分成 |
| OpenBook 依赖 | 是 | 无 |
| 创建成本 | 较高 | 较低 |
| Token-2022 支持 | 否 | 是 |
| 新上市池路由 | 稀少 | 默认 |

如果你的交易对流量很大，LP 费用的增加可能值得迁移。如果旧池仍有充足流动性且项目未指定目标 CPMM 池，迁移可能不值得付出协调成本。

## 迁移前检查清单

针对池运营者和项目金库：

- [ ] 向 LP 提供方宣布迁移时间表。
- [ ] 尽可能给 LP 至少 48 小时的准备时间。
- [ ] 确认目标 CPMM 池和费用等级。
- [ ] 准备迁移后公告，包含新池 ID 和 LP 代币 mint。
- [ ] 确定是否需要新的流动性挖矿。

针对个人 LP：

- [ ] 确认你跟随的是项目官方背书的迁移。
- [ ] 保持足够的 SOL 来支付多个交易。
- [ ] 记录你的 AMM v4 LP 余额和预期的底层代币数量。
- [ ] 检查你的 AMM v4 LP 代币是否在流动性挖矿中质押。

## UI 迁移：个人 LP

迁移分两个步骤：从 AMM v4 移除流动性，然后添加流动性到 CPMM。

### 步骤 1：从 AMM v4 取出

1. 打开 AMM v4 池或你的投资组合头寸。
2. 如果你的 LP 代币在流动性挖矿中质押，先取消质押。
3. 点击**移除流动性**。
4. 选择要提取的百分比。
5. 检查你将收到的代币数量。
6. 确认并批准。

### 步骤 2：存入 CPMM

1. 打开目标 CPMM 池。
2. 点击**添加流动性**。
3. 输入一种代币的数量。
4. 检查配对代币的数量和任何剩余余额。
5. 确认并批准。

在取出和存入之间，池价格可能会波动。如果两个池的比率不同，一些代币可能会作为灰尘留在你的钱包中。

## UI 迁移：项目金库

对于项目金库，流程相同但需要协调：

1. **提前公告**迁移。
2. **在要求用户迁移前为 CPMM 池提供初始流动性**。
3. **先迁移金库 LP** 并发布交易链接。
4. **创建新的流动性挖矿**（如果激励应该继续在 CPMM LP mint 上）。
5. **监控个人 LP 迁移**，时间跨度为几天或几周。
6. **仅在用户有足够时间转移后才弃用旧池**。

## 验证

### 取出后

- AMM v4 LP 代币余额应该减少或变为零。
- 两种底层代币应该出现在你的钱包中。
- 交易在 Solana 浏览器上应显示**成功**。

### 存入后

- CPMM LP 代币余额应该出现在你的钱包中。
- Raydium 的投资组合应显示新头寸。
- CPMM 池页面应显示你的流动性份额。

### 迁移证明

为了透明起见，发布两个交易链接：

- 从 AMM v4 取出。
- 存入 CPMM。

## 回滚考虑

从 CPMM 回滚到 AMM v4 意味着从 CPMM 取出并存入 AMM v4。除非有特定的产品原因，否则通常不值得这样做。AMM v4 池创建成本更高，需要 OpenBook 市场设置。

## 常见问题

### 我的 AMM v4 LP 在流动性挖矿中质押。我能迁移吗？

可以。先取消质押，从 AMM v4 提取 LP，存入 CPMM，如果新的流动性挖矿存在，则质押新的 CPMM LP 代币。

### CPMM 池还不存在

大多数 LP 应该等待项目创建并背书一个目标池。自己创建一个意味着你承担初始价格和流动性风险。

### 我有大额头寸。滑点怎么办？

如果你的头寸相对于池 TVL 很大，分次取出和存入。检查每个报价，如果收到的金额明显少于预期，避免签名。

### 我忘记迁移的 LP 代币会怎样？

它们留在旧池中，继续根据 AMM v4 规则赚取收益。没有任何因素强制迁移。

## 故障排除

### 目标 LP 代币账户是新的

第一次 CPMM 存入可能需要创建新的 LP 代币账户。在你的钱包中保留额外的 SOL 以支付租赁和费用。

### 第二个交易过期

如果取出成功但存入过期，你的代币保留在钱包中。重新打开 CPMM 池，使用更高的优先费设置重试。

### CPMM 池错误

同一交易对可能存在多个 CPMM 池。使用项目背书的池或具有预期费用等级和流动性的池。

### 税务影响

在某些管辖区，LP 取出可能是应税事件。在迁移大额头寸前咨询当地规则。
