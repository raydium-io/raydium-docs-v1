<Warning>
  LaunchLab 是无许可代币发行基础设施。任何人都可以创建代币、选择名称或代码，并配置发行参数。购买或推广代币前，请务必验证铸造地址和发行设置。本页仅解释产品机制，不构成财务、法律或投资建议。
</Warning>

LaunchLab 让创建者通过联合曲线发行代币并引导初始流动性。买方在毕业前与曲线进行交易。当发行达到毕业阈值时，流动性迁移到 Raydium AMM 池，交易继续通过正常的 Raydium 路由进行。

## LaunchLab 的使用者

| 用户 | 常见任务 |
|---|---|
| 交易者 | 发现早期代币，在联合曲线上买卖，并追踪毕业进度。 |
| 创建者 | 无需代码即可发行代币，配置曲线参数，可选择获得创建者费用。 |
| 平台 | 构建带有自定义费用设置、品牌和发行发现界面的品牌化发行环境。 |

## 发行流程

1. 创建者配置代币元数据、供应量、曲线设置、毕业目标和可选费用设置。
2. 代币在联合曲线上开放。买方和卖方直接与曲线进行交易。
3. 随着买卖发生，曲线价格变化。
4. 当曲线达到毕业目标时，流动性迁移到 Raydium 池。
5. 毕业后，用户通过 Raydium Swap 和通过 Raydium 池路由的聚合器交易该代币。

## 创建模式

| 模式 | 何时使用 | 关键设置 |
|---|---|---|
| JustSendit | 希望以默认设置进行最快的无代码发行。 | 名称、代码、图像、可选社交链接、可选首次购买。 |
| LaunchLab | 需要更好地控制供应量、毕业目标、曲线配置、归属或创建者费用。 | 供应量、融资目标、曲线配置、归属、迁移后费用分享。 |

JustSendit 使用默认毕业目标。LaunchLab 模式让创建者设置毕业目标，受 UI 和程序限制。

## 毕业和流动性

毕业关闭联合曲线，将资产迁移到 Raydium 池。迁移后池类型和 LP 处理取决于平台配置和创建者费用设置。

- 如果启用迁移后创建者费用，发行迁移到 CPMM，Fee Key NFT 可代表创建者的费用权利。
- 如果平台使用不具有迁移后创建者费用的旧版路径，发行可能迁移到 AMM v4。
- 平台设置可以改变迁移后 LP 代币是否被销毁、锁定或分配的方式。

## 安全基础知识

- 验证代币铸造地址，而非仅验证代码或徽标。
- 交易前查看毕业目标和曲线进度。
- 发行或购买前检查创建者和平台费用设置。
- 将 Fee Key NFT 视为费用权利资产。如果转移或销毁，权利声索可能转移或丢失。
- 在连接钱包或签署交易前阅读[信任和安全](/zh/getting-started/trust-and-safety)。
