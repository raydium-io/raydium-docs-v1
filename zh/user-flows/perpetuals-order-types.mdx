## 订单类型

Raydium 永续合约支持以下订单类型：

| 订单类型 | 功能说明 | 使用场景 |
|---|---|---|
| **市价单** | 以当前最优订单簿价格即时买入或卖出。如果订单簿无法完全成交，未成交部分会被取消。 | 你希望立即成交，能接受可能的滑点。 |
| **限价单** | 按照你设定的限价或更优价格买入或卖出。订单仅在市场达到该价格时才会成交。 | 你希望对价格有控制权，并且可以等待成交。 |
| **止损市价单** | 保持未激活状态，直到达到触发价格后转换为市价单。 | 你需要一个执行触发条件，并优先考虑触发后快速进出。 |
| **止损限价单** | 保持未激活状态，直到达到触发价格后转换为限价单。 | 你需要触发条件，但仍希望保持限价保护。 |
| **规模化订单** | 将一个较大的订单分拆为多个较小的限价单，跨越一个价格区间。 | 你希望渐进式建仓或平仓。 |

## 订单标志

订单标志可以改变限价类型订单的行为方式。

| 标志 | 行为说明 |
|---|---|
| **IOC** | 即时成交或取消。按照你的限价或更优价格尽可能立即成交，然后取消未成交的部分。 |
| **FOK** | 全部成交或全部取消。按照你的限价或更优价格完全成交订单，否则整个订单被取消，不允许部分成交。 |
| **仅做市** | 以做市商身份提供流动性。如果订单会立即成交，则改为取消。 |
| **仅减仓** | 防止订单增加你的头寸。订单只能减少或平掉该市场的现有头寸。 |

## 提交前的检查清单

在提交订单前，请确认：

- 市场和交易方向正确。
- 数量和杠杆倍数符合你的意图。
- 止损触发价和限价没有反向设置。
- 如果你只打算平仓或减少头寸敞口，已启用仅减仓。
- 如果你想要做市商行为，已启用仅做市。
