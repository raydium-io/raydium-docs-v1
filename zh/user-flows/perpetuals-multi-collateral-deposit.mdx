</Warning>

## 支持的抵押品

Raydium Perps 支持从主要区块链存入抵押品资产，包括 SOL、USDC、USDT、ETH、BNB、WBTC、YUSD 和 USD1（在可用地区）。

具体列表、支持的链和限额由底层交易所控制。存入前请以应用显示的信息为准。

## 全局限额

全局限额应用于整个交易所，所有用户共享。相关 Raydium 页面列出的例子包括：

| 抵押品 | 链示例 |
|---|---|
| USDT | Arbitrum、BSC、Ethereum |
| ETH | Arbitrum、Base、Ethereum |
| BNB | BSC |
| USD1 | BSC、Ethereum |
| YUSD | BSC、Ethereum |
| WBTC | Ethereum |
| SOL | Solana |

如果全局限额已满，即使你的钱包余额低于个人限额，该资产的新存入可能仍无法进行。

## 单个用户限额

单个用户限额应用于每个账户或钱包。相关 Raydium 页面列出了 USDT、ETH、SOL、BNB、YUSD、WBTC 和 USD1 等资产的单个用户限额。

发送资金前请查看存入界面。如果应用显示的可用存入金额低于你的钱包余额，说明你可能达到了单个用户或全局抵押品限额。

## 存入流程

1. 打开 [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC)。
2. 连接你的钱包并打开存入流程。
3. 选择抵押品资产和源链。
4. 核实金额、目标账户和任何交易所警告。
5. 确认存入。
6. 等待账户余额和可用保证金更新。
