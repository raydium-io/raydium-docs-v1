<Info>
  LaunchLab 平台是一种链上配置，由启动界面或社区使用。平台设置可以影响费用、品牌、创建者费用支持和迁移后的流动性池行为。
</Info>

LaunchLab 通过平台 PDA 支持第三方启动环境。平台可以提供自己的品牌代币启动体验，同时使用 LaunchLab 程序基础设施和 Raydium 流动性池迁移。

## 平台可以控制的内容

| 设置 | 重要原因 |
|---|---|
| 创建者费用支持 | 决定代币创建者是否可以获得迁移前和迁移后的费用。 |
| 平台费用 | 设置平台级费用收入的接收地址。 |
| 品牌元数据 | 为平台提供名称、描述和徽标，可以在应用界面中显示。 |
| 启动分组 | 让 UI 显示在同一平台下启动的代币。 |
| 迁移后流动性池设置 | 可以定义平台将启动迁移到 CPMM 时的费用配置。 |
| LP 分配 | 可以影响迁移后 LP 代币如何销毁、分配给创建者或分配给平台。 |

## 创建者应该检查的内容

在任何平台下启动前，确认：

- 哪个钱包接收平台费用。
- 创建者费用是否启用。
- 迁移后流动性是流向 CPMM 还是 AMM v4。
- LP 代币在毕业后如何分割。
- 是否会铸造费用密钥 NFT 以及谁接收它。
- 平台是否有自己的条款、政策或资格要求。

<Warning>
  平台设置可能会显著改变创建者的经济模式和用户预期。在签署启动交易前进行审查。本页仅解释机制，不构成法律、财务或投资建议。
</Warning>

## 交易者应该检查的内容

购买 LaunchLab 代币时，平台可能会影响费用路径和迁移后流动性池处理。交易前：

- 检查代币的平台和启动设置。
- 确认迁移后的流动性池类型（如果 UI 显示）。
- 检查 LP 是否被销毁、锁定或分配。
- 从代币页面验证代币地址，而不是仅从社交媒体文本。
