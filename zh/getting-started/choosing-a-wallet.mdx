## Raydium 兼容钱包必须做什么

钱包是 Raydium 兼容钱包，当且仅当它可以做到以下所有事情：

- 本地保存 Solana 密钥对，**绝不**将私钥传输到设备外。
- **签署版本化（V0）交易。** Raydium 交换使用地址查询表并需要 V0 支持。只能签署传统交易的钱包无法完成 Raydium 交换。
- **通过 Wallet Adapter 标准连接到 dApps。** 这就是使 raydium.io 上的"连接钱包"按钮工作的原因。
- **在你批准前显示清晰的交易预览**，包括被调用的程序和代币流向。
- **支持关联代币账户**（在你首次收到新代币时创建 ATA）。

这些是基础要求。任何现代 Solana 钱包都能满足这些要求。不满足要求的钱包不安全，无法用于签署 Raydium 交易——就是这样。

## 超越基础的重要功能

一旦钱包满足基础要求，差异就取决于你的工作流和风险偏好。决定以下哪些功能对**你**很重要，然后选择能够处理这些功能的钱包：

| 功能 | 何时重要 |
|---|---|
| **硬件钱包配对** | 你持有的金额不仅仅是零散的。软件钱包成为交易构建器；硬件设备持有密钥并签署。大多数声誉良好的 Solana 钱包至少可以与 Ledger 配对；一些可以与 Keystone、SafePal 或其他设备配对。 |
| **手机 dApp 浏览器** | 你主要从手机使用 Raydium。钱包的内置浏览器是在手机上加载 raydium.io 和签署交易的方式。 |
| **多账户管理** | 你将资金分离到热/冷/实验钱包中，并想快速切换而不用费力管理扩展。 |
| **Token-2022 扩展感知** | 你将与使用 Token-2022 代币的池互动（例如转账费代币）。在签署 UI 中显示这些扩展的钱包可以帮助你在签署前发现意外费用。 |
| **优先费显示** | 你在拥堵期间交易，想在批准前看到钱包为计算预算价格设置的内容。 |
| **Solana Actions / Blinks 支持** | 你想使用 Raydium 在社交媒体上发布的一键交换链接。请参阅 [`user-flows/referrals-and-blinks`](/zh/user-flows/referrals-and-blinks)。 |
| **开源代码库** | 你想能够审计钱包客户端（或依赖社区已经这样做）。 |
| **多链账户** | 你在 Solana 以外的链上持有资产，并且希望使用一个钱包管理所有资产。确认多链支持对你实际使用的链是成熟的；这因链而异。 |

## 流行的 Solana 钱包

以下是具有活跃 Solana 支持的知名钱包项目。它们按字母顺序列出；**列入仅作为信息参考，不是背书**，该列表不详尽。安装前验证官方 URL——钓鱼克隆经常对真实域名购买赞助广告。

### 软件钱包

| 钱包 | 运行位置 | 官方网站 |
|---|---|---|
| Phantom | 浏览器扩展、iOS、Android | [phantom.app](https://phantom.app) |
| Solflare | 浏览器扩展、iOS、Android、网页 | [solflare.com](https://solflare.com) |
| OKX Wallet | 浏览器扩展、iOS、Android | [okx.com/wallet](https://www.okx.com/wallet) |
| Backpack | 浏览器扩展、iOS、Android、桌面 | [backpack.app](https://backpack.app) |

这四个都是非托管的，支持 Wallet Adapter，签署 V0 交易，并至少与一个主流硬件设备配对。除此之外，差异是特定于工作流的——使用前一部分的标准来决定哪一个适合你。

### 硬件钱包

硬件钱包与上面的软件钱包之一配对（软件端成为交易构建器；硬件设备持有密钥并签署）。

| 设备 | 类型 | 官方网站 |
|---|---|---|
| Ledger | USB / 蓝牙 | [ledger.com](https://www.ledger.com) |
| Keystone | 气隙（二维码签署） | [keyst.one](https://keyst.one) |

每个软件钱包都会记录它支持的硬件设备；购买前请检查。

## 热/冷钱包模式

一个常见的做法，无论你选择哪个钱包都适用：

- **热钱包**持有你可以接受丢失的小额资金——用于日常交换和尝试事物。
- **冷钱包**（与硬件设备配对）持有任何你不希望丢失的资金。仅在你主动需要在热钱包中使用资金时才转移资金。

Raydium 将每个钱包视为完全独立的：单独的头寸、单独的 LP 代币、单独的农业奖励、单独的历史。切换钱包只需一次点击。

## 在哪里找到钱包选项

Raydium UI 的"连接钱包"模态列出了所有已通过 Wallet Adapter 与 Raydium 集成的钱包。浏览它以查看当前支持的内容。Solana 的官方钱包目录（[`solana.com/ecosystem/explore?categories=wallet`](https://solana.com/ecosystem/explore?categories=wallet)）是另一个中立的起点。

当你评估钱包时，运行此清单：

- [ ] 已维护：最近发布、活跃的问题跟踪器、安全披露渠道。
- [ ] 已审查：由声誉良好的公司发布的审计，或具有活跃社区的开源代码。
- [ ] V0 交易支持已确认（在转移真实资金前在 devnet 上测试）。
- [ ] 如果你计划使用硬件设备：钱包记录了它支持的设备以及配对的工作方式。
- [ ] 你通过项目的已验证域名找到了官方下载——**不是**通过赞助广告或转发链接。

## 钱包安全基础

无论你选择哪个钱包：

- **将助记词写在纸上。** 将纸张存储在两个物理位置不同的地方。**绝不**在任何地方输入助记词，除了在新安装上进行钱包恢复时。
- **为钱包的解锁密码使用密码管理器**——并将该密码与助记词完全分开。
- **为加密活动使用专用浏览器配置文件**，仅安装最少数量的扩展。钓鱼扩展是最大的泄露向量。
- **对每个钱包连接提示持怀疑态度。** 阅读你正在签署的内容。如果交易转移所有代币而不是特定金额，拒绝它。
- **为 `raydium.io` 添加书签。** 始终从书签导航，不要从搜索引擎广告导航。

更多安全指导在 [`getting-started/trust-and-safety`](/zh/getting-started/trust-and-safety)。
