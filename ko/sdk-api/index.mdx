## 대상 개발자

오프체인(TS/JS, Python, 백엔드)에서 또는 다른 Solana 프로그램(Rust CPI)에서 Raydium과 통합하는 개발자들입니다. 또한 AI 에이전트 및 코딩 도구 통합자들도 포함됩니다. [AI 통합](/ko/sdk-api/ai-integration)을 참고하세요.

<Info>
  **먼저 실행해 보고 싶으신가요?** 복사해서 바로 실행할 수 있는 세 가지 스크립트(스왑, CPMM 풀 배포, CLMM 풀 배포)가 [빠른 시작](/ko/quick-start)에 있습니다. 5분 안에 실제 트랜잭션을 실행한 후 여기로 돌아와서 전체 레퍼런스를 확인하세요.
</Info>

## 통합 경로 선택하기

5가지 경로 중에서 선택할 수 있습니다. 올바른 경로는 무엇이 Raydium을 호출하고 얼마나 많은 제어 권한이 필요한지에 따라 결정됩니다.

| 경로 | 최적 사용처 | 지연 시간 | 커스텀 로직 | 언어 |
|---|---|---|---|---|
| [Trade API](/ko/sdk-api/trade-api) | 백엔드, 봇, Blinks — 서버 빌드 tx가 필요한 곳 | 최소 | 제한됨 (사전 정의된 경로) | 모든 언어 |
| [TypeScript SDK](/ko/sdk-api/typescript-sdk) | 전체 빌더 액세스가 필요한 Node/TS 앱과 프론트엔드 | 중간 | 전체 | TS / JS |
| [Python 통합](/ko/sdk-api/python-integration) | Python 백엔드 및 ML 파이프라인 | 중간 | 전체 (수동 인스트럭션 빌드) | Python |
| [Rust CPI](/ko/sdk-api/rust-cpi) | Raydium을 온체인에서 호출하는 프로그램 조합 | 해당 없음 (온체인) | 전체 + 조합 가능 | Rust |
| [Anchor IDL](/ko/sdk-api/anchor-idl) | 자체 지원 언어가 없는 언어(Go, Kotlin, Swift, ...)를 위한 코드젠 | 변동 | 전체 | Anchor 코드젠 도구가 있는 모든 언어 |

확실하지 않으신가요? 백엔드 및 봇 → **Trade API**, 프론트엔드 및 전체 제어 앱 → **TypeScript SDK**, 온체인 조합 → **Rust CPI**를 선택하세요.

## 챕터 구성

<CardGroup cols={2}>
  <Card title="TypeScript SDK" icon="js" href="/ko/sdk-api/typescript-sdk">
    `@raydium-io/raydium-sdk-v2`: 설치, 초기화, 핵심 모듈(풀, 팜, 런치), 트랜잭션 빌더, 일반적인 주의사항입니다.
  </Card>
  <Card title="REST API" icon="cloud" href="/ko/sdk-api/rest-api">
    풀 정보, 민트 정보, 가격 및 서버 빌드 트랜잭션을 위한 공개 엔드포인트입니다. 요청 제한, 캐싱 가이드, 버전 관리입니다.
  </Card>
  <Card title="Trade API" icon="right-left" href="/ko/sdk-api/trade-api">
    서버 빌드 스왑 트랜잭션: 견적, 빌드, 서명, 전송입니다. 백엔드, 봇, Blinks를 위한 SDK 없는 통합 패턴입니다.
  </Card>
  <Card title="Anchor IDL" icon="file-code" href="/ko/sdk-api/anchor-idl">
    각 프로그램의 IDL을 찾을 수 있는 위치, IDL에서 클라이언트를 재생성하는 방법, IDL이 업데이트될 때 주의할 사항입니다.
  </Card>
  <Card title="Rust CPI" icon="rust" href="/ko/sdk-api/rust-cpi">
    다른 Solana 프로그램에서 Raydium 프로그램을 호출하기 위한 패턴 — 어카운트 목록 구성, 서명자 시드, 에러 처리입니다.
  </Card>
  <Card title="Python 통합" icon="python" href="/ko/sdk-api/python-integration">
    Python에서 Raydium 스왑/풀 작업을 구축하기 위한 `solders` + `solana-py` 레시피입니다. 백엔드 및 봇에 유용합니다.
  </Card>
  <Card title="AI 통합" icon="robot" href="/ko/sdk-api/ai-integration">
    MCP 서버, `llms.txt` 인덱스, 페이지별 복사 메뉴, Claude Code/Cursor/Windsurf 및 기타 코딩 에이전트용 사전 빌드 컨텍스트 파일입니다.
  </Card>
</CardGroup>

## 버전 관리 규칙

이 챕터의 모든 페이지의 모든 코드 샘플 상단에는 배너가 있습니다.

```
SDK:       @raydium-io/raydium-sdk-v2 vX.Y.Z
Program:   see /reference/program-addresses
Cluster:   mainnet-beta
Verified:  YYYY-MM-DD
```

코드 샘플을 수정하면 배너를 업데이트하세요.
