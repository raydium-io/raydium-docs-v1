## 이름과 대문자 표기

| 형태 | 사용 |
|---|---|
| **Raydium** | 프로토콜 이름입니다. 항상 제목 표기법을 사용합니다. `raydium`, `RAYDIUM`으로 쓰지 않습니다. |
| **Raydium Protocol** | Raydium 이름이 다른 맥락(화학 원소, 비디오 게임 등)과 겹칠 때 구분이 필요한 경우 사용 가능합니다. |
| **\$RAY** 또는 **RAY** | Raydium의 네이티브 SPL 토큰입니다. 마케팅 문구에서는 달러 접두사 포함, 기술 또는 표 문맥에서는 `RAY`로 표기합니다. |
| **Raydium AMM v4** / **Raydium CPMM** / **Raydium CLMM** / **LaunchLab** | 제품 이름입니다. LaunchLab은 한 단어로 L의 대문자로 표기합니다. |
| **Burn & Earn** | 앰퍼샌드로 연결되는 하나의 구조로, 두 명사 모두 대문자입니다. |

`Raydium DEX`, `Ray protocol`, `Raydiums`(복수형 없음)로 쓰지 않습니다. 프로토콜은 단수 명사입니다.

## 로고

Raydium 로고는 두 가지 형태로 제공됩니다:

- **워드마크** — 전체 "Raydium" 타입 록업입니다. 헤더, 보도자료 및 이름을 읽을 수 있어야 하는 모든 상황에서 사용합니다.
- **심볼 마크** — 독립형 모노그램입니다. 앱 아이콘, 파비콘, 워드마크가 가독성 있는 크기로 맞지 않는 제한된 공간(아바타, 소셜 프로필 사진)에서 사용합니다.

### 라이트 및 다크 버전

각 형태당 두 가지 마무리:

| 파일 | 배경 | 사용 시기 |
|---|---|---|
| `logo-light.svg` | 어두운 배경 | 어두운 UI 표면에 주로 배치합니다. |
| `logo-dark.svg` | 밝은 배경 | 밝은 UI 표면에 주로 배치합니다. |
| `mark-light.svg` | 어두운 배경 | 어두운 표면의 심볼 전용입니다. |
| `mark-dark.svg` | 밝은 배경 | 밝은 표면의 심볼 전용입니다. |

모든 자산은 벡터 SVG입니다. 가능한 한 SVG를 사용합니다. 최종 렌더링 크기에서만 PNG로 래스터화합니다(스케일링된 PNG는 피합니다).

### 여백

로고 주위에 Raydium "R" 문자 높이와 동일한 여백을 유지합니다. 이 여백 내에 텍스트, 아이콘 또는 그래픽을 배치하지 않습니다.

### 최소 크기

워드마크: 화면에서 **120 px** 이상, 인쇄물에서 **20 mm** 이상으로 렌더링합니다.
심볼: 화면에서 **24 px** 이상, 인쇄물에서 **6 mm** 이상으로 렌더링합니다.

이 임계값 아래에서는 로고의 가독성이 떨어지므로 파트너는 다른 형태로 전환해야 합니다(예: 공간이 제한된 경우 워드마크를 심볼로 변경).

## 색상 팔레트

Raydium 정체성은 상태, 깊이 및 대비를 위해 더 어둡고 밝은 변형이 있는 보라색 주색을 기반으로 합니다.

| 역할 | 16진수 | RGB | 사용 |
|---|---|---|---|
| **기본색** | `#8C6EFF` | 140 / 110 / 255 | 주 브랜드 색상입니다. CTA, 링크, 주요 강조 요소. |
| **밝은색** | `#A88BFF` | 168 / 139 / 255 | 호버 상태, 보조 강조 요소, 라이트 테마 배경. |
| **어두운색** | `#5E3BE1` | 94 / 59 / 225 | 활성/누른 상태, 다크 테마 강조 요소, 깊이. |

보조 중립 색상:

| 역할 | 16진수 | 사용 |
|---|---|---|
| **전경(라이트)** | `#0A0A0F` | 밝은 표면의 본문 텍스트. |
| **전경(다크)** | `#FAFAFA` | 어두운 표면의 본문 텍스트. |
| **표면(라이트)** | `#FFFFFF` | 페이지 배경, 라이트 테마. |
| **표면(다크)** | `#0F0F1A` | 페이지 배경, 다크 테마. |
| **테두리** | `#E5E5EB`(라이트) / `#1F1F2E`(다크) | 카드 및 분할선 테두리. |

로고를 다시 칠하지 않습니다. 워드마크와 심볼은 항상 라이트 또는 다크 마스터 마무리로 렌더링됩니다(색상 변조, 그래디언트 교체, 브랜드 색상 적용 금지).

## 타이포그래피

문서와 사이트는 UI 및 본문에 **Inter**를 사용합니다. 제목은 굵은 가중치(600–800)의 Inter를 사용합니다. 코드 블록은 **JetBrains Mono** 또는 모든 등폭 폴백을 사용합니다.

| 스타일 | 폰트 | 가중치 | 사용 |
|---|---|---|---|
| 디스플레이 / H1 | Inter | 800 | 페이지 제목. |
| H2–H4 | Inter | 700 | 섹션 헤더. |
| 본문 | Inter | 400 | 단락 텍스트. |
| 강조 | Inter | 600 | 인라인 강조, 강한 텍스트. |
| 코드 | JetBrains Mono | 400–500 | 인라인 `code`와 코드 블록. |

Inter는 오픈 라이선스이며 [rsms.me/inter](https://rsms.me/inter)에서 사용 가능합니다.

## 로고 권장사항 및 금지사항

**권장:**

- 문서 저장소 `/logo/` 디렉토리의 공식 SVG 마스터를 사용합니다.
- 여백과 최소 크기를 유지합니다.
- 워드마크와 심볼을 시각적으로 균형을 이룬 경우에만 함께 배치합니다(크기가 다른 것을 나란히 표시하지 않습니다).
- 충분한 대비도가 있는 배경에 렌더링합니다(어두운 배경에는 라이트 마스터, 밝은 배경에는 다크 마스터를 사용합니다).

**금지:**

- 로고를 다시 칠합니다. 보라색 워드마크, 무지개, 제공된 라이트/다크 이외의 흑백 변형 없음.
- 가로세로 비율을 왜곡합니다(늘이기, 비틀기 금지).
- 드롭 섀도우, 아웃라인, 글로우 또는 기타 효과를 추가합니다.
- 대비를 보존하는 단색 패널 없이 바쁜 배경 이미지에 로고를 배치합니다.
- 로고를 자릅니다(워드마크에서 "R"만 자르지 않습니다. 대신 심볼 마크를 사용합니다).
- Raydium이 공식적으로 파트너십을 맺지 않은 제품의 보증을 암시하는 데 로고를 사용합니다. 중립적 맥락에서 "powered by Raydium" 또는 "integrated with Raydium" 배지를 사용하며, 검증 의미를 암시하지 않습니다.

## "Powered by Raydium" 배지

통합이 라우팅 소스 또는 유동성 거래소로 Raydium을 사용하는 경우 간단한 텍스트 또는 배지 속성이 환영합니다:

```text
Powered by Raydium
Integrated with Raydium
Liquidity via Raydium
```

이 표현은 사전 승인되었습니다. 통합의 가격 책정, 스왑 또는 풀 표면 근처에 배치합니다(마스트헤드 눈에 띄는 위치는 공식 파트너십을 암시하므로 피합니다).

집계자나 지갑이 Raydium을 통해 라우팅하고 더 눈에 띄는 속성을 원하는 경우 발행 전에 Discord(`#partnerships`)를 통해 팀에 연락합니다.
