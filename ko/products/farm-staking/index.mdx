## 팜 / 스테이킹이란

팜 / 스테이킹 프로그램은 입력 토큰 (일반적으로 AMM v4 / CPMM / CLMM의 LP 토큰이지만 단일 자산 스테이킹도 지원합니다) 을 스테이킹하는 계정에 보상 토큰을 배분합니다. 각 팜은 독립적인 일정으로 최대 여러 보상 민트를 발행할 수 있습니다.

**프로그램 ID:** [reference/program-addresses](/ko/reference/program-addresses)를 참고하세요. 여러 팜 버전이 운영 중입니다 (v3, v5, v6) 이며 보상 슬롯과 관리 프리미티브가 다릅니다. [versions-and-migration](/ko/protocol-overview/versions-and-migration) 페이지에서 각 배포판에 어떤 버전을 사용해야 하는지 안내합니다.
