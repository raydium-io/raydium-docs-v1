</Info>

## 스테이킹하는 방법

1. [raydium.io](https://raydium.io)를 열고 지갑을 연결하세요.
2. **Staking** 페이지로 이동하세요.
3. 스테이킹할 RAY 수량을 입력하세요.
4. **Stake**를 클릭하세요.
5. 지갑에서 거래를 확인하고 승인하세요.

거래가 확인되면 스테이킹 페이지에 스테이킹된 잔액과 대기 중인 보상이 표시됩니다.

## 언스테이킹하는 방법

1. **Staking** 페이지 또는 **Portfolio** 페이지를 열세요.
2. 스테이킹된 RAY 포지션의 마이너스 버튼을 클릭하세요.
3. 언스테이킹할 수량을 입력하세요.
4. 지갑에서 거래를 확인하고 서명하세요.

## 스테이킹하기 전에

- 거래 수수료를 위해 지갑에 SOL을 보유하고 있는지 확인하세요.
- RAY 민트를 확인하세요: `4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R`.
- 서명하기 전에 대기 중인 보상과 거래 세부 정보를 확인하세요.
