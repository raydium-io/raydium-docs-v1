## 이 장의 대상 읽자

다른 블록체인(EVM, Cosmos, Move)에서 온 개발자와 프로토콜 연구자, 또는 솔라나를 처음 접하는 개발자들을 대상으로 합니다. 일반 사용자는 이 장을 건너뛸 수 있습니다.

목표는 실용적입니다. 독자가 Products와 SDK 장을 예상 밖의 내용 없이 읽을 수 있도록 필요한 최소한의 내용만 제공하는 것입니다. 이는 솔라나에 대한 일반적인 튜토리얼이 아니므로, 자세한 내용은 공식 솔라나 문서를 참고하세요.

## 작성 지침

- 각 페이지는 나중에 Raydium 맥락에서 참고되는 하나의 완성된 예제를 제공합니다.
- CPMM이 Token-2022를 지원하므로 Token-2022는 최우선으로 다룹니다. transfer-fee와 freeze-authority는 스왑 계산에 영향을 미칩니다.
- 이 페이지의 코드는 짧게 유지합니다(30줄 미만). 전체 데모는 SDK & API 섹션에 있습니다.
- 공식 솔라나 문서로 한 번만 링크하고 반복하지 않습니다.
