## 사전 준비

다음을 한 번 실행하세요:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

필요한 것들:

- **Node 18 이상** ESM 지원 포함 (`package.json`에 `"type": "module"` 또는 `.mjs` 확장자).
- **Solana 키페어** (`~/.config/solana/id.json` 위치 또는 다른 곳 — `KEYPAIR` 설정).
- **자금이 있는 지갑.** 이 장의 모든 스크립트는 약 0.05 SOL이 필요합니다.
- **RPC URL.** 테스트용 무료 공개 RPC를 사용할 수 있지만, 스모크 테스트 이상을 원하면 유료 공급자를 사용하세요.

셸당 한 번 설정하세요:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **클러스터.** 세 스크립트 모두 메인넷을 기본값으로 사용합니다. 데브넷에서 실행하려면 `RPC_URL=https://api.devnet.solana.com`을 설정하고 스크립트 내의 `cluster: "mainnet"`을 `cluster: "devnet"`으로 변경하세요. 데브넷의 풀 시드 유동성이 제한적임을 주의하세요. 테스트 토큰을 직접 생성해야 할 수 있습니다.
</Info>

## 이 장 이후

각 스크립트는 최소한의 출력(풀 ID, 트랜잭션 서명)과 수행한 작업에 대한 [SDK & API](/ko/sdk-api) 장의 다음 페이지를 가리키는 한 줄을 제공합니다. 자신만의 통합을 작성할 준비가 되면:

- **백엔드 및 봇** → [Trade API](/ko/sdk-api/trade-api).
- **프론트엔드 및 완전 제어 앱** → [TypeScript SDK](/ko/sdk-api/typescript-sdk).
- **온체인 구성** → [Rust CPI](/ko/sdk-api/rust-cpi).
- **선택 기준** → [SDK & API](/ko/sdk-api)의 상단 비교 표.
