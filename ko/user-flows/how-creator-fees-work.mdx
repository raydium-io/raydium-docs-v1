</Warning>

## 1단계 — 사전 마이그레이션(본딩 커브)

토큰이 본딩 커브에서 거래되는 동안, 각 매수 및 매도에서 작은 크리에이터 수수료 부분이 제외되어 토큰별 보관 계정에 누적됩니다. 이는 토큰이 런칭되는 **플랫폼**에 의해 설정됩니다.

### 상세 정보

- **기준 통화** — 인용 토큰(일반적으로 SOL)으로 표시됩니다.
- **비율** — 런칭 시 플랫폼에서 설정합니다. 플랫폼의 수수료 일정 또는 토큰의 온체인 상태를 확인하세요.
- **언제든지 청구 가능** — raydium.io의 **포트폴리오** 페이지로 이동하여 언제든지 대기 중인 수수료를 청구하세요. 졸업을 기다릴 필요가 없습니다.
- **졸업 시 중단** — 커브가 졸업한 후에는 사전 마이그레이션 수수료가 새로 누적되지 않습니다. 보관 계정의 남은 잔액은 계속 청구 가능합니다.

### 청구 방법

1. 크리에이터 지갑을 [raydium.io](https://raydium.io)에 연결합니다.
2. **포트폴리오**로 이동합니다.
3. 생성된 토큰 아래에서 토큰을 찾습니다.
4. 사전 마이그레이션 수수료 잔액 옆의 **청구**를 클릭합니다.

프로그래매틱 청구: SDK 지시사항을 보려면 [`products/launchlab/creator-fees`](/ko/products/launchlab/creator-fees)를 참조하세요.

## 2단계 — 사후 마이그레이션(Fee Key NFT)

토큰의 본딩 커브가 졸업 임계값에 도달하면, 유동성은 Raydium CPMM 풀로 마이그레이션됩니다. LP 토큰은 **플랫폼 설정**에 따라 발행 및 배포됩니다:

| 수신자 | Raydium 기본값 | 설명 |
|---|---|---|
| 소각(소각 & 수익) | 90% | 영구적으로 잠금 상태이며, 이 부분의 거래 수수료는 소각 & 수익 신탁 메커니즘으로 흘러갑니다. |
| 크리에이터(Fee Key NFT) | 10% | Fee Key NFT가 지갑에 발행됩니다. 이 NFT는 풀의 지속적인 LP 거래 수수료의 10%에 대한 청구권을 나타냅니다. |
| 플랫폼 | 0%(Raydium 기본값) | 일부 타사 플랫폼은 여기에서 점유율을 가져갑니다. |

이러한 기본값은 Raydium의 자체 LaunchLab UI를 통해 런칭된 토큰에 적용됩니다. 타사 플랫폼은 다른 분할을 설정할 수 있습니다. 런칭하기 전에 항상 플랫폼 페이지를 확인하세요.

### Fee Key NFT

Fee Key NFT는 졸업 시 지갑에 발행되는 대체 불가능한 토큰으로, 사후 마이그레이션 풀의 해당 부분에서 LP 거래 수수료를 수집할 권리를 나타냅니다.

<Warning>
  **의도적으로 수수료 청구 권리를 포기하려는 경우가 아니면 Fee Key NFT를 소각하거나 이전하지 마세요.** NFT가 손실되거나 소각된 경우, 수수료 청구 권리는 **영구적으로 몰수됩니다** — 복구 경로가 없습니다.
</Warning>

- Fee Key는 거래되거나 이전될 수 있으며, 이를 보유한 누구든 수수료를 청구할 수 있습니다.
- CPMM 풀의 LP 거래 활동에서 수수료는 지속적으로 누적됩니다.
- 포트폴리오 페이지에서 청구하거나 CPMM 풀의 수수료 청구 지시사항을 통해 프로그래매틱 방식으로 청구합니다.

### 사후 마이그레이션 수수료 청구 방법

1. Fee Key NFT를 보유한 지갑을 연결합니다.
2. raydium.io의 **포트폴리오**로 이동합니다.
3. 포지션 아래에서 졸업한 토큰의 풀을 찾습니다.
4. **청구**를 클릭하여 누적된 거래 수수료를 수집합니다.

---

## 사후 마이그레이션 수수료가 활성화되지 않은 경우

플랫폼이 크리에이터 수수료를 비활성화로 설정하면, 토큰은 **AMM v4 풀**(기존 마이그레이션 경로)로 졸업됩니다. 이 모드에서:
- Fee Key NFT가 발행되지 않습니다.
- 프로그램 수준의 사후 마이그레이션 크리에이터 수수료가 없습니다.
- 런칭 방식에 따라 CLI 도구를 통해 사용 가능한 몇 가지 대체 수수료 청구 옵션이 있을 수 있습니다.

## 수수료 흐름 요약

```
본딩 커브 단계:
  각 매수/매도 → 작은 % → 크리에이터 보관소(인용 토큰)
  포트폴리오에서 언제든지 청구

졸업:
  LP 토큰 발행 → 플랫폼 설정에 따라 분할
  ├── X% 소각(소각 & 수익)
  ├── Y% → 크리에이터를 Fee Key NFT로      ← 당신이 이것을 보유합니다
  └── Z% → 플랫폼(해당하는 경우)

사후 졸업:
  모든 CPMM 스왑 → LP 수수료 → LP 점유율에 따라 분할
  당신의 Fee Key NFT 점유율 → 포트폴리오에서 청구 가능
```
