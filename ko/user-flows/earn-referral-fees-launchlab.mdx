<Warning>
  레퍼럴 링크는 토큰의 품질이나 안전성을 보장하지 않습니다. 자신이 뒤에서 지원할 수 있는 토큰만 공유하고, 청중이 민트 주소와 런칭 설정을 직접 확인할 수 있도록 해야 합니다.
</Warning>

## 작동 방식

자신의 지갑 주소를 레퍼러로 포함하는 토큰 페이지 링크를 공유하면, 그 링크를 통해 거래하는 모든 사람이 **0.1% 수수료**를 생성하며, 이는 자신의 지갑으로 직접 전송됩니다. 수수료는 다음과 같은 특징을 가집니다:

- SOL(또는 풀의 견적 토큰)로 표시됩니다.
- 각 거래 시 자동으로 지갑에 송금되므로 별도의 청구 단계가 없습니다.
- 토큰이 사전 졸업 상태일 때 본딩 곡선에서의 매수와 매도에 적용됩니다.

### 세션 범위

레퍼럴 링크는 특정 토큰을 가리키지만, 링크를 클릭한 사용자가 동일한 브라우저 세션 내에서 *다른* LaunchLab 토큰을 거래하면, 그 거래에 대해서도 수수료를 받을 수 있습니다. 이는 초기에 공유한 단일 토큰 이상으로 수익 창출 기회를 확장해줍니다.

## 레퍼럴 링크 생성하기

1. [raydium.io/launchlab](https://raydium.io/launchlab)에서 지갑을 연결합니다.
2. LaunchLab 토큰의 상세 페이지로 이동합니다.
3. 토큰 페이지의 **공유** 버튼을 클릭합니다.
4. 생성된 링크를 복사합니다 — 여기에 레퍼러 매개변수로 자신의 지갑 주소가 포함되어 있습니다.
5. 소셜 미디어, 커뮤니티, 뉴스레터 등 청중이 있는 곳에 링크를 공유합니다.

<Note>
  공유를 클릭할 때 지갑이 연결되어 있어야 합니다. 연결된 지갑 없이 링크가 생성되면 유효한 레퍼러 매개변수를 포함하지 않습니다.
</Note>

## 수익 구조

| 상황 | 수수료 |
|---|---|
| 수신자가 연결된 토큰을 매수 | 거래액의 0.1% |
| 수신자가 연결된 토큰을 매도 | 거래액의 0.1% |
| 수신자가 같은 세션 내 다른 LaunchLab 토큰 거래 | 해당 거래의 0.1% |

수익은 실시간으로 송금되며, 청구할 대기 중인 잔액이 없습니다.

## 비교: LaunchLab 레퍼럴 vs. 스왑 레퍼럴

| | LaunchLab 레퍼럴 | 스왑 레퍼럴 (블링크) |
|---|---|---|
| 비율 | 0.1% | 1% |
| 범위 | LaunchLab 토큰 페이지 | Raydium 스왑 인터페이스 |
| 지급 | 거래당 자동 | 통합에 따라 수동 청구 또는 자동 |
| 설정 | 토큰 페이지의 공유 버튼 | [레퍼럴 & 블링크](/ko/user-flows/referrals-and-blinks)를 통해 생성 |
