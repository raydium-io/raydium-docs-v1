## 번 & 언이란

번 & 언은 풀 생성자(또는 모든 LP)가 예치를 온체인에서 **영구적으로** 만들 수 있게 하면서 거래 수수료를 수확할 권리를 유지하게 해주는 레이디움 기능입니다. 포지션은 더 이상 상환 불가능해지지만, 새로 발행된 **레이디움 수수료 키** NFT가 그 잠금된 포지션에서 수수료를 청구(`collectFees`)할 수 있는 지속적인 권리를 나타냅니다.

일반적인 사용 사례:

- **론칭 신뢰성.** 프로젝트는 배포자의 LP를 잠금 처리하여 러그풀을 하지 않을 것임을 보장합니다. 보유자는 유동성이 영구적임을 온체인에서 확인할 수 있습니다.
- **수수료 창출 수집 가능물.** 수익을 생성하는 양도 가능한 NFT입니다. NFT를 판매하면 수수료 흐름을 이전합니다.
- **재무급 LP.** 프로토콜이 브랜드 / 발견 목적으로 유동성을 영구적으로 유지하고 싶으면서도 수수료 측면을 수익화하기를 원할 때 사용합니다.

## 지원되는 풀 유형

| 프로그램 | 번 & 언 지원 |
|---|---|
| CPMM | 예 |
| CLMM | **풀 범위만 지원** (권장됨) |
| AMM v4 (Hybrid) | 지원 안 함 |

CLMM의 경우, 전체 가격 범위를 포함하는 포지션만 잠금 처리해야 합니다. 일단 잠금되면 리밸런싱할 수 없으므로 가격이 선택한 범위를 벗어나면 그 포지션에서 영구적으로 수수료 획득을 중단하게 됩니다. 좁은 범위의 번 & 언 포지션은 거의 항상 실수입니다.

## 프로그램 수준에서의 잠금 작동 방식

1. 원본 LP 토큰(CPMM) 또는 포지션 NFT(CLMM)가 번 & 언 프로그램이 소유한 프로그램 소유 PDA로 이전됩니다.
2. **수수료 키 NFT**가 호출자에게 발행됩니다. 그 메타데이터는 잠금된 포지션의 PDA를 다시 가리킵니다.
3. 번 & 언은 `collectFees`(또는 풀 유형당 동등한 호출)를 노출하는데, 이는:
   - 서명자가 주어진 잠금된 포지션에 대한 수수료 키 NFT를 보유하고 있는지 확인합니다.
   - CPI를 통해 기초 풀 프로그램의 수수료 청구 지시를 호출합니다.
   - 수집된 수수료를 수수료 키 보유자의 ATA로 라우팅합니다.
4. 대응하는 `withdraw`는 없으므로 기초 LP / 포지션은 에스크로를 절대 빠져나가지 않습니다.

수수료 키는 표준 SPL NFT입니다. 모든 솔라나 NFT 마켓플레이스에서 판매하거나, 지갑 간에 이동하거나, NFT를 받는 대출 프로토콜에서 담보로 사용하거나, 다른 계약에 래핑할 수 있습니다. `collectFees` 호출 시점에 그것을 보유한 사람이 수수료를 받게 됩니다.

## UI 단계별 설명

1. 평소대로 CPMM 풀 또는 풀 범위 포지션이 있는 CLMM 풀을 생성합니다.
2. **유동성 → 생성 → 번 & 언**으로 이동합니다.
3. 잠금할 포지션을 선택합니다.
4. 포지션 세부 정보(NFT 주소, 크기, 풀)를 확인하고 UI가 표시하는 확인 문장을 입력합니다. 이 작업은 되돌릴 수 없으므로 의도적으로 마찰을 많이 만들었습니다.
5. **확인, 유동성 영구 잠금** 클릭합니다.
6. 수수료 키 NFT가 지갑에 도착합니다. 포트폴리오 페이지의 잠금된 포지션 아래에 표시됩니다.

포지션의 다른 수수료와 동일하게 수수료를 청구합니다 — 포트폴리오 페이지나 `collectFees`를 통해.

## API 표면

풀 정보 엔드포인트는 그 풀의 유동성 중 번 & 언을 통해 잠금 처리된 부분의 비율을 나타내는 `burnPercent` 필드를 반환합니다:

```http
GET https://api-v3.raydium.io/pools/info/list
  ?poolType=standard
  &poolSortField=default
  &sortType=desc
  &pageSize=1
  &page=1
```

```json
{
  "data": [
    {
      "id": "...",
      "burnPercent": 99.17,
      ...
    }
  ]
}
```

`burnPercent`는 신뢰 신호를 보내기 위한 편리한 수단입니다: LP가 99%+ 소각(잠금)된 토큰은 LP가 0% 소각된 토큰보다 훨씬 덜 러그 가능합니다.

## 주의할 점

- **되돌릴 수 없습니다.** LP 측은 절대 회수할 수 없습니다. 금액이 다시는 건드리지 않을 것으로 기꺼이 하는 것인지 확인하세요.
- **CLMM 좁은 범위는 함정입니다.** 풀 범위 CLMM 포지션만 잠금 처리하세요. 범위 제한 잠금 포지션이 가격이 대역을 벗어나면 수수료 획득을 중단한다는 것을 정말 이해하지 않은 경우는 제외합니다.
- **수수료 키 NFT를 잃는 것은 수수료를 영구적으로 잃는 것입니다.** 그것을 추장권처럼 다루세요. 실수로 소각하지 마세요; 적대적 계약에 스테이킹하지 마세요.
- **이전 수수료가 있는 Token-2022 민트**: 번 & 언이 지원하지만 전송 수수료 할인이 모든 수수료 청구에 적용되므로 계산하세요.
- **부분 잠금이 없습니다.** 50% 잠금 및 50% 이동 가능하기를 원하면, 먼저 포지션을 두 개 포지션으로 분할한 다음 하나만 잠금 처리하세요.
