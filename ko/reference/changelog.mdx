## 릴리스

<Update label="2026-06-22" description="Stable AMM: 사용하지 않는 OpenBook(마켓) 코드 제거">
  Stable AMM이 오랫동안 사용하지 않던 OpenBook 마켓 메이킹 계정과 코드를 제거합니다. `SwapBaseIn` / `SwapBaseOut` (18 → 9), `Deposit` (14 → 12), `Withdraw` (21/22 → 12) 레이아웃이 더 작아졌으며(이전 레이아웃은 여전히 호환됨), **하드 브레이킹** `WithdrawPnl` 변경 (16 → 10, 호환성 없음), 레퍼럴 수수료 폐지, 단순화된 볼트 전용 풀 자산 공식이 포함됩니다. 대부분의 다른 Stable 명령어는 더 이상 호출할 수 없습니다.

  [전체 항목 읽기 →](/ko/reference/changelog/2026-06-22-stable-amm-openbook-cleanup)
</Update>

<Update label="2026-05-18" description="CLMM: 리미트 오더, 단측 수수료, 동적 수수료">
  세 가지 옵트인 방식의 하위 호환 CLMM 기능: 일급 리미트 오더(`limit_order_admin` 정산 키퍼 포함), 단측 수수료 수집(`CollectFeeOn`), 변동성 추적 동적 수수료입니다. `CreateCustomizablePool`, `PoolState` 재구성(**인덱서 브레이킹 변경**), 새로운 `TickState` 필드, 11개의 새로운 오류 코드(숫자 시프트 포함), 그리고 일치하는 SDK / API 추가 사항이 포함됩니다.

  [전체 항목 읽기 →](/ko/reference/changelog/2026-05-18-clmm-limit-orders)
</Update>

<Update label="2026-04-26" description="초기 공개">

## 문서 규칙

- **버전 관리**: 이 문서는 달력 기반 버전 관리(YYYY-MM-DD)를 사용합니다. 각 업데이트는 새로운 항목 페이지를 추가하고 위의 타임라인 맨 위에 새로운 행을 추가합니다.
- **릴리스당 한 페이지**: 모든 릴리스 요약은 `reference/changelog/` 아래의 자체 페이지에 있으므로 이 인덱스는 짧게 유지되고 각 항목은 독립적으로 링크할 수 있습니다.
- **검증 날짜**: 모든 항목은 온체인 / API 상태 및 프로그램 소스에 대해 마지막으로 교차 검증된 시기를 기록합니다. 명시되지 않은 경우 항목의 주요 날짜를 가정하세요.
- **브레이킹 변경**: 영향을 받는 페이지에서 상자 경고로 표시되고 항목에 태그가 지정됩니다.
- **범위**: 이 변경 로그는 문서 세트 자체를 다룹니다. 프로토콜 자체의 역사적 타임라인은 [`introduction/history-and-milestones`](/ko/introduction/history-and-milestones)에 있으며 "Raydium에서 X가 언제 발생했는가"에 대한 신뢰할 수 있는 출처입니다.

이 문서에서 오류를 발견하면 문서 저장소에서 이슈 또는 풀 요청을 열어주세요. 수정 사항은 변경 로그 항목으로 기록됩니다.
