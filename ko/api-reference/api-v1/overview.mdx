Raydium의 원래 REST API로, 풀, 팜, SDK 및 토큰 데이터를 제공합니다. 현재도 프로덕션 트래픽과 SDK 클라이언트를 지원하고 있지만 레거시로 간주됩니다.

## 호스트

| 환경 | 호스트 |
|-----|--------|
| Mainnet | `api.raydium.io` |
| Devnet | 사용 불가능 |

## 마이그레이션 안내

**새로운 통합에는 [API v3](/ko/api-reference/api-v3/overview)을 사용하세요.** v3 엔드포인트는 다음과 같은 이점을 제공합니다:

- 향상된 성능과 캐싱
- 간소화된 응답 스키마
- 명확한 엔드포인트 구성
- 지속적인 기능 개발

## 관련 문서

권장되는 대체 엔드포인트는 [API v3](/ko/api-reference/api-v3/overview)를 참조하세요. 풀 구조 및 용어는 [CLMM](/ko/products/clmm) 및 [CPMM](/ko/products/cpmm)을 참조하세요.
