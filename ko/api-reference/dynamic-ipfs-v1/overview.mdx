Dynamic IPFS API는 Solana의 CLMM 및 CPMM 유동성 포지션에 대한 재생성된 메타데이터와 이미지를 제공합니다. 이 서비스는 동적 NFT를 렌더링하는 dApp(예: 집중형 유동성 포지션 카드)을 위해 설계되었으며, 온체인 조회 없이 실시간 풀 정보, 포지션 가치 평가, 미청구 수수료를 제공합니다.

## 호스트

| 환경     | 호스트                              |
|---------|-----------------------------------|
| Mainnet | `dynamic-ipfs.raydium.io`         |
| Devnet  | `dynamic-ipfs-devnet.raydium.io`  |

## 개요

API는 세 개의 엔드포인트를 제공하며, 각각 완전한 포지션 인터페이스 객체를 반환합니다.

- **`/clmm/position`** — 활성 CLMM 포지션의 메타데이터 조회
- **`/lock/clmm/position`** — 잠금 CLMM 포지션의 메타데이터 조회
- **`/lock/cpmm/position`** — 잠금 CPMM 포지션의 메타데이터 조회

각 응답에는 전체 풀 상세 정보(토큰, 소수 자릿수, TVL), 포지션 수량, USD 가치 평가, 풀 내 TVL 백분율, 누적 미청구 수수료 및 보상이 포함됩니다.

## 사용 사례

특정 포지션의 메타데이터를 요청하여 동적 포지션 시각화 NFT를 생성합니다. API는 온체인 데이터로부터 실시간 포지션 상태를 계산하고 결과를 캐시하므로 다음에 이상적입니다.

- 실시간 USD 가치 및 수수료 누적과 함께 포지션 카드 렌더링
- 풀 내 TVL 백분율 및 토큰 수량 표시
- 미청구 거래 수수료 및 보상 토큰 표시
- 포지션 성과를 추적하는 포트폴리오 대시보드 구축

## 인증

인증이 필요하지 않습니다. 모든 엔드포인트는 공개적으로 접근 가능합니다.

## 요청 속도 제한

별도의 제한이 없습니다. 최선의 노력 서비스입니다.

## 일반적인 패턴

### CLMM 포지션 조회

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

`id` 파라미터는 포지션 계정의 온체인 공개 키(base58 형식의 Solana 주소)여야 합니다.

### 잠금 포지션 조회

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
