히스토리 API는 LaunchLab 풀의 과거 거래 데이터를 제공합니다. 여러 간격(1분, 5분, 15분)의 캔들(OHLC) 데이터를 조회하고 페이지 분할을 통해 개별 거래 기록에 접근할 수 있습니다.

## API 엔드포인트

**Mainnet:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## 인증

인증이 필요하지 않습니다. 모든 엔드포인트는 공개적으로 접근 가능합니다.

## 데이터 기능

- **캔들 데이터:** 1분, 5분, 15분 간격의 OHLC 캔들차트
- **거래 기록:** 거래자, 수량, 타임스탬프를 포함한 개별 거래
- **페이지 분할:** 대규모 데이터셋 조회에 효율적인 `nextPageKey` 사용
- **한도 제약:** 캔들 요청은 쿼리당 1-500개(기본값 300개), 거래는 1-100개(기본값 50개) 수용
- **필터링:** 거래자 지갑 및 금액 범위로 거래 기록 필터링

## 사용 사례

- 캔들차트 시각화를 통한 **차트 표시**
- **가격 분석** 및 기술적 분석 지표
- **거래 통계** 및 거래량 추적
- 지갑 및 날짜 범위로 필터링된 **사용자 거래 감시 로그**
- 민트의 **과거 가격 발견**
