Forum API는 LaunchLab 풀 주변의 논의와 커뮤니티 참여를 가능하게 합니다. 선택적 이미지 첨부 파일과 함께 댓글을 게시하고 풀 논의 스레드를 조회할 수 있습니다. 모든 쓰기 작업은 Auth API를 통한 인증이 필요합니다.

## API 엔드포인트

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## 인증

**읽기 작업 (GET):** 인증 불필요합니다.

**쓰기 작업 (POST):** `ray-token` 헤더 필요 — 지갑 서명 검증 후 Auth API에서 발급한 JWT입니다.

[Auth API에서 토큰 얻기](/ko/api-reference/launch-auth-v1/overview)

## 요청 헤더

댓글을 게시할 때 다음 인증 헤더를 포함하세요:

```
ray-token: <jwt_token_from_auth_api>
```

## 기능

- **댓글 게시:** 선택적 이미지 첨부 파일 (최대 5 MB)이 포함된 텍스트 댓글 제출
- **논의 브라우징:** 페이지네이션과 함께 풀의 모든 댓글 보기
- **콘텐츠 검증:** NSFW 감지 및 숨겨진 댓글 필터링
- **페이지네이션:** `lastId`와 `fetchType` (old/new)로 효율적인 브라우징 쿼리
- **이미지 업로드:** 댓글은 Pinata IPFS 이미지 호스팅 지원

## 사용 사례

- **토큰 프로젝트** 주변의 커뮤니티 논의
- **프로젝트 공지사항** 및 댓글을 통한 업데이트
- **사용자 피드백** 및 감정 추적
- **사회적 증명** 및 커뮤니티 참여 지표
- **NSFW 및 스팸 감지**를 통한 콘텐츠 검증
