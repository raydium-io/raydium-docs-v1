Owner API는 Raydium 포지션과 잔액에 대한 지갑별 뷰를 제공합니다. 포지션, 잠긴 유동성, 생성된 풀, IDO 참여에 대한 캐시된 데이터를 가져오므로 지갑 UI와 포트폴리오 대시보드에 이상적입니다.

## 호스트

| 환경 | 호스트 |
|------|--------|
| Mainnet | `owner-v1.raydium.io` |
| Devnet | `owner-v1-devnet.raydium.io` |

## 개요

API는 리소스 유형별로 구성된 5개의 주요 엔드포인트를 제공합니다:

- **`/position/stake/{owner}`** — 스테이크 포지션 (표준 AMM / CPMM LP)
- **`/position/clmm-lock/{owner}`** — 잠긴 CLMM 포지션
- **`/create-pool/{owner}`** — 이 지갑에서 생성한 풀
- **`/main/ido/{owner}`** — IDO 참여 데이터

## 데이터 신선도

이 서비스는 포지션 데이터를 캐시하고 주기적으로 업데이트합니다. 최근 포지션 변경 사항 (수초 이내)은 응답에 즉시 나타나지 않을 수 있습니다.

## 일반적인 패턴

### 지갑의 모든 스테이크 포지션 조회

```bash
curl -s "https://owner-v1.raydium.io/position/stake/{owner}"
```

`{owner}`을 지갑의 base58 형식 공개 키로 바꾸세요.

### 잠긴 포지션 조회

```bash
curl -s "https://owner-v1.raydium.io/position/clmm-lock/{owner}"
```

### IDO 참여 확인

```bash
curl -s "https://owner-v1.raydium.io/main/ido/{owner}"
```

## 빈 결과

지갑이 특정 엔드포인트에 대한 데이터가 없으면 서비스는 요청 메타데이터와 함께 404 오류를 반환합니다:

```json
{
  "id": "...",
  "success": false
}
```
