## Para quem é este capítulo

Usuários finais e desenvolvedores que desejam uma receita passo a passo em vez de uma referência. As páginas aqui priorizam *completude de um caminho* em vez de *cobertura de todas as opções*; os capítulos de Produtos e SDK são a referência exaustiva.

Cada fluxo tem duas visualizações lado a lado quando aplicável: **walkthrough da UI** (para usuários finais) e **walkthrough programático** (TS SDK, para desenvolvedores).
