## Fase 1 — Pré-migração (curva de ligação)

Enquanto seu token está sendo negociado na curva de ligação, um componente de taxa do criador é retirado de cada compra e venda e acumulado em um vault por token. Isso é configurado pela **plataforma** em que seu token é lançado.

### Detalhes

- **Denominado em** o token de cotação (geralmente SOL).
- **Taxa** — definida pela plataforma no momento do lançamento. Verifique a tabela de taxas da plataforma ou o estado on-chain do token.
- **Reivindicável a qualquer momento** — acesse sua página de **Portfolio** em raydium.io e reclame as taxas pendentes a qualquer momento. Sem necessidade de aguardar a migração.
- **Encerra na migração** — nenhuma taxa pré-migração nova é acumulada após a curva se formar. Qualquer saldo residual no vault permanece reivindicável.

### Como reivindicar

1. Conecte a carteira do criador a [raydium.io](https://raydium.io).
2. Navegue até **Portfolio**.
3. Encontre o token sob seus tokens criados.
4. Clique em **Claim** ao lado do saldo de taxa pré-migração.

Reivindicação programática: consulte [`products/launchlab/creator-fees`](/pt/products/launchlab/creator-fees) para a instrução do SDK.

## Fase 2 — Pós-migração (Fee Key NFT)

Quando a curva de ligação do seu token atinge seu limite de migração, a liquidez é migrada para um pool CPMM do Raydium. Os tokens LP são cunhados e distribuídos de acordo com a **configuração da plataforma**:

| Destinatário | Padrão Raydium | Descrição |
|---|---|---|
| Queimado (Burn & Earn) | 90% | Permanentemente bloqueado; as taxas de negociação dessa parcela fluem para o mecanismo de confiança Burn & Earn. |
| Criador (Fee Key NFT) | 10% | Um NFT Fee Key é cunhado em sua carteira. Este NFT é seu direito sobre 10% das taxas de negociação LP contínuas do pool. |
| Plataforma | 0% (padrão Raydium) | Algumas plataformas de terceiros levam uma parcela aqui. |

Esses padrões se aplicam aos tokens lançados através da interface LaunchLab do próprio Raydium. Plataformas de terceiros podem configurar divisões diferentes — sempre verifique a página da plataforma antes de lançar.

### O NFT Fee Key

O NFT Fee Key é um token não-fungível cunhado em sua carteira na migração que representa seu direito de coletar taxas de negociação LP da sua parcela do pool pós-migração.

<Warning>
  **Não queime ou transfira o NFT Fee Key a menos que você realmente queira renunciar aos direitos de reivindicação de taxas.** Se o NFT for perdido ou queimado, os direitos de reivindicação de taxa são **permanentemente perdidos** — não há caminho de recuperação.
</Warning>

- O Fee Key pode ser negociado ou transferido; quem o detiver poderá reivindicar as taxas.
- As taxas acumulam continuamente da atividade de negociação LP no pool CPMM.
- Reclame a partir da página do Portfolio ou via a instrução de reivindicação de taxa do pool CPMM programaticamente.

### Como reivindicar taxas pós-migração

1. Conecte a carteira que detém o NFT Fee Key.
2. Navegue até **Portfolio** em raydium.io.
3. Encontre o pool do token migrado sob suas posições.
4. Clique em **Claim** para coletar as taxas de negociação acumuladas.

## Se as taxas pós-migração não estiverem habilitadas

Se uma plataforma desabilitar as taxas do criador, seu token migra para um **pool AMM v4** (o caminho de migração legado) em vez de CPMM. Neste modo:
- Nenhum NFT Fee Key é cunhado.
- Não há taxa de criador pós-migração no nível do programa.
- Algumas opções alternativas de reivindicação de taxa podem existir via ferramentas CLI dependendo de como o lançamento foi configurado.

## Resumo do fluxo de taxas

```
Fase da curva de ligação:
  Cada compra/venda → pequeno % → vault do criador (token de cotação)
  Reclame a qualquer momento do Portfolio

Migração:
  Tokens LP cunhados → divididos por configuração da plataforma
  ├── X% queimados (Burn & Earn)
  ├── Y% → criador como Fee Key NFT      ← você detém isto
  └── Z% → plataforma (se aplicável)

Pós-migração:
  Cada swap CPMM → taxas LP → divididas por parcela LP
  Sua parcela Fee Key NFT → reivindicável do Portfolio
```
