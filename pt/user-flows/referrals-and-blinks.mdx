## O que você recebe

- Uma URL compartilhável que pré-popula a UI de swap do Raydium com seu token de destino escolhido **e sua carteira como destinatária da referência**.
- Quando alguém executa um swap através do link enquanto está ativo em sua sessão, **1% do volume de entrada é roteado para a carteira do referenciador** como parte da transação de swap.
- Links compartilhados no X.com renderizam como **Blinks** (Solana Actions) se a carteira do visualizador tiver Actions / Blinks ativados — um widget de swap com um clique dentro da postagem.

## Como criar um link

1. Abra [raydium.io/swap](https://raydium.io/swap) e conecte sua carteira. O endereço de referência é derivado da carteira conectada; sem uma conexão o link ainda funciona, mas não ganha nenhuma taxa.
2. Selecione o **token de destino** para o qual você quer que outros façam swap. O lado da origem fica a cargo do destinatário escolher.
3. Clique no ícone de compartilhamento/link no painel de swap (canto superior direito).
4. Copie a URL gerada. Compartilhe em qualquer lugar.

O formato da URL é estável. Se você está construindo um app que gera links de referência diretamente.

## Como os Blinks funcionam

**Blinks (Blockchain Links)** são um padrão da Solana Foundation para renderizar URLs de ações on-chain como UI interativa dentro de plataformas sociais. Uma carteira Solana que suporta Blinks lê os metadados do link, constrói uma transação pré-construída e mostra ao usuário um prompt de assinatura inline.

Requisitos:

- A carteira do visualizador **deve ter Actions / Blinks ativados**. Normalmente, isso é uma opção nas configurações da carteira.
- A **plataforma deve respeitar o padrão**. O X.com faz isso através de seu unfurler; alguns outros não fazem, e nesse caso o link volta a um hiperlink simples.

Da perspectiva do referenciador, a renderização de Blink é automática — a mesma URL funciona como um link simples em carteiras que não suportam Blinks.

## Contabilização de taxas

A taxa de referência de 1% é **cobrada além da taxa de negociação do AMM**, não é tirada dela. Para um swap em um pool com taxa total de 0,25%:

```
Usuário envia:      1.000 USDC
Taxa de negociação: 2,50 USDC   (para LP + divisão de protocolo)
Taxa de referência: 10,00 USDC  (1% → carteira do referenciador)
Líquido para swap:  987,50 USDC (então precificado através da curva)
```

Taxas de referência se aplicam à rota de swap do Raydium criada pelo link de referência. Interações de pool diretas fora da rota de referência não carregam taxas de referência.

## Desativando / saindo de uma referência

Não existe uma instrução "remover referência". Para fazer um swap sem taxa de referência, simplesmente carregue [raydium.io/swap](https://raydium.io/swap) diretamente em vez de clicar em um link de referência. A UI não usa referenciador quando a URL não tem parâmetro de referenciador.

## Perguntas frequentes

**O referenciador precisa assinar algo?** Não. O endereço da carteira do referenciador é o único dado que viaja com o link; o pagamento ao referenciador é uma transferência de token dentro da transação do swapper.

**Posso me referenciar?** Sim, mas é inútil — você paga a si mesmo 1%, menos a taxa de tx que acabou de pagar em uma transação maior.

**O que acontece se o token de destino é um SPL com um transfer hook?** A transferência de referência acontece no lado da entrada (em USDC ou o token de origem), não no destino, então transfer hooks no mint de destino não afetam o pagamento de referência. Transfer hooks no token de entrada podem bloquear o swap completamente; o Raydium rejeita esses antecipadamente.

**A taxa de referência ainda se aplica se o swap roteia através de múltiplos pools?** Sim — o 1% é calculado sobre o valor de entrada original e é independente da rota.
