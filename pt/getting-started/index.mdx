## Para quem é este capítulo

Usuários finais — pessoas que querem **usar** a Raydium para fazer swaps, fazer farming ou fornecer liquidez, não construir sobre ela. Se você nunca usou um aplicativo Solana antes, comece aqui. Se já usou, pule para [`user-flows`](/pt/user-flows).

Este capítulo é paralelo a [`solana-fundamentals`](/pt/solana-fundamentals), que é direcionado a desenvolvedores. Ambos cobrem pré-requisitos, mas de ângulos diferentes: este é sobre configurar uma carteira e fazer sua primeira negociação; aquele é sobre modelos de conta e invocação de programas.

## A visão geral de 30 segundos

**Raydium é uma DEX no Solana.** Ela permite que você troque um token por outro sem usar uma exchange centralizada. Para usá-la você precisa:

1. Uma **carteira Solana** (extensão de navegador ou aplicativo móvel) — armazena sua chave privada e assina transações.
2. Alguns **SOL** nessa carteira — paga pelas taxas de transação (~\$0.0005 por swap) e qualquer token que você queira comprar.
3. Um navegador em [`raydium.io`](https://raydium.io) — a UI oficial.

Tudo o resto — escolher uma carteira, financiá-la, escolher sua primeira negociação — está coberto nas páginas acima.

## Sinais de alerta para observar desde o primeiro dia

Antes de depositar qualquer dinheiro, internalize estas regras:

- **Nunca compartilhe sua seed phrase.** Não com suporte, não com um site "verifique sua carteira", não por DMs. O único lugar onde uma seed phrase é digitada é durante a configuração inicial da carteira.
- **Use apenas `raydium.io`.** Phishing copia a UI perfeitamente. Adicione a URL real aos favoritos.
- **Confie no endereço do token, não no nome do token.** Qualquer pessoa pode criar um token chamado "USDC". O USDC real tem o mint `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`. A UI mostra um badge **verificado** ao lado de mints curados.
- **Um dApp nunca deveria pedir para você "aprovar" sua carteira inteira.** Transações normais assinam instruções específicas. Uma aprovação em branco é um drain.

Mais detalhes em [`getting-started/trust-and-safety`](/pt/getting-started/trust-and-safety).

## O que este capítulo não é

- **Não é um tutorial de trading.** Escolher tokens e gerenciar risco são sua responsabilidade pessoal; este capítulo só cobre a mecânica Raydium de executar negociações.
- **Não é conselho financeiro.** "Devo fornecer liquidez?" é uma pergunta estratégica; veja [`user-flows/choosing-a-pool-type`](/pt/user-flows/choosing-a-pool-type) para a *mecânica* dessa decisão.
- **Não é um deep-dive em Solana.** Para isso, veja [`solana-fundamentals`](/pt/solana-fundamentals).
