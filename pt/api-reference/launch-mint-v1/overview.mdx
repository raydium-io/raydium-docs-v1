A API Mint oferece acesso abrangente à criação, descoberta e gerenciamento de tokens LaunchLab. Consulte mints por nome, símbolo ou criador, recupere rankings e envie novas solicitações de criação de mint.

## Endpoints da API

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Autenticação

Operações de leitura não requerem autenticação. Operações de escrita (criação de mint) requerem um header `ray-token` emitido pela API Auth após verificação de assinatura de carteira.

[Documentação da API Auth](/pt/api-reference/launch-auth-v1/overview)

## Casos de Uso

- **Buscar e descobrir** tokens por nome ou símbolo
- **Visualizar rankings** (mints principais por market cap, mais recentes, mais populares)
- **Recuperar estatísticas de usuário** (volume de criação e negociação ao longo do tempo)
- **Consultar cronogramas de vesting** para informações de desbloqueio de tokens
- **Enviar solicitações de criação** de mint com metadados e imagens
- **Rastrear recompensas de referência** de programas de referência da plataforma
