A API do Forum permite discussão e engajamento da comunidade em torno de pools LaunchLab. Publique comentários com anexos de imagem opcionais e recupere threads de discussão de pools. Todas as operações de escrita exigem autenticação via API de Autenticação.

## Endpoints da API

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Autenticação

**Operações de leitura (GET):** Nenhuma autenticação necessária.

**Operações de escrita (POST):** Requer header `ray-token` — um JWT emitido pela API de Autenticação após verificação de assinatura da wallet.

[Obter token da API de Autenticação](/pt/api-reference/launch-auth-v1/overview)

## Headers da Requisição

Ao publicar comentários, inclua o header de autenticação:

```
ray-token: <jwt_token_from_auth_api>
```

## Recursos

- **Publicar Comentários:** Envie comentários de texto com anexos de imagem opcionais (máximo 5 MB)
- **Navegar Discussões:** Visualize todos os comentários em um pool com paginação
- **Moderação de Conteúdo:** Detecção de NSFW e filtragem de comentários ocultos
- **Paginação:** Consulte com `lastId` e `fetchType` (old/new) para navegação eficiente
- **Upload de Imagens:** Comentários suportam hospedagem de imagens IPFS Pinata

## Casos de Uso

- **Discussão comunitária** em torno de projetos de tokens
- **Anúncios de projeto** e atualizações via comentários
- **Feedback do usuário** e rastreamento de sentimento
- **Prova social** e métricas de engajamento da comunidade
- **Moderação de conteúdo** com detecção de NSFW e spam
