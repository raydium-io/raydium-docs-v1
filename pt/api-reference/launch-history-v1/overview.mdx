A History API fornece dados históricos de negociações para pools LaunchLab. Recupere dados de velas (OHLC) em múltiplos intervalos (1 minuto, 5 minutos, 15 minutos) e acesse registros de negociações individuais com paginação.

## Endpoints da API

**Mainnet:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## Autenticação

Nenhuma autenticação necessária. Todos os endpoints são acessíveis publicamente.

## Características dos Dados

- **Dados Kline:** Gráficos de velas OHLC em intervalos de 1m, 5m e 15m
- **Histórico de Negociações:** Negociações individuais com negociador, quantidade e timestamp
- **Paginação:** Use `nextPageKey` para consultas eficientes de grandes conjuntos de dados
- **Restrições de Limite:** Solicitações kline aceitam 1-500 velas por consulta (padrão 300); negociações aceitam 1-100 (padrão 50)
- **Filtragem:** Filtre o histórico de negociações por carteira do negociador e intervalos de quantidade

## Casos de Uso

- **Exibição de gráficos** com visualização de velas
- **Análise de preços** e indicadores de análise técnica
- **Estatísticas de negociações** e rastreamento de volume
- **Trilhas de auditoria** de negociações do usuário filtradas por carteira e intervalo de data
- **Descoberta de preço histórico** para mints
