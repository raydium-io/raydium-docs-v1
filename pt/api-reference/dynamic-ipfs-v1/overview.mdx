A API IPFS Dinâmica fornece metadados e imagens regeneradas para posições de liquidez CLMM e CPMM na Solana. Este serviço foi projetado para dApps que renderizam NFTs dinâmicos—como cartões de posição de liquidez concentrada—fornecendo informações de pool em tempo real, avaliação de posição e taxas não coletadas sem exigir consultas on-chain.

## Hosts

| Ambiente | Host                               |
|----------|-----------------------------------|
| Mainnet  | `dynamic-ipfs.raydium.io`         |
| Devnet   | `dynamic-ipfs-devnet.raydium.io`  |

## Visão geral

A API expõe três endpoints, cada um retornando um objeto de interface de posição completo:

- **`/clmm/position`** — Buscar metadados para uma posição CLMM ativa
- **`/lock/clmm/position`** — Buscar metadados para uma posição CLMM bloqueada  
- **`/lock/cpmm/position`** — Buscar metadados para uma posição CPMM bloqueada

Cada resposta inclui os detalhes completos do pool (tokens, decimais, TVL), quantias de posição, avaliação em USD, percentual de TVL dentro do pool e taxas e recompensas acumuladas (mas não coletadas).

## Casos de uso

Gere NFTs de visualização de posição dinâmica solicitando metadados para uma posição específica. A API calcula o estado da posição em tempo real a partir de dados on-chain e armazena resultados em cache, tornando-a ideal para:

- Renderizar cartões de posição com valor em USD e acúmulo de taxas em tempo real
- Exibir percentual de TVL e quantias de tokens no pool
- Mostrar taxas de negociação não coletadas e tokens de recompensa
- Criar dashboards de portfólio que rastreiam o desempenho da posição

## Autenticação

Nenhuma autenticação é necessária. Todos os endpoints são públicos.

## Limitação de taxa

Nenhuma aplicada. Serviço com melhor esforço.

## Padrões comuns

### Buscar uma posição CLMM

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

O parâmetro `id` deve ser a chave pública on-chain da conta de posição (um endereço Solana em formato base58).

### Buscar uma posição bloqueada

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
