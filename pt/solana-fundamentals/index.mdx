## Para quem é este capítulo

Desenvolvedores e pesquisadores de protocolos vindos de outras blockchains (EVM, Cosmos, Move) ou novos na Solana. Usuários finais podem pular este capítulo.

O objetivo é pragmático: fornecer ao leitor exatamente o que ele precisa ler para entender os capítulos de Produtos e SDK sem surpresas. Este não é um tutorial geral sobre Solana — consulte a documentação oficial da Solana para isso.
