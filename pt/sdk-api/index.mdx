## Para quem é este capítulo

Desenvolvedores integrando com Raydium de forma off-chain (TS/JS, Python, backends) ou de outro programa Solana (Rust CPI). Também agentes de IA e integradores de ferramentas de codificação — veja [Integração com IA](/pt/sdk-api/ai-integration).

<Info>
  **Quer executar algo primeiro?** Três scripts prontos para usar — swap, deploy de pool CPMM, deploy de pool CLMM — estão em [Início rápido](/pt/quick-start). Execute uma transação real em cinco minutos e depois volte aqui para a referência completa.
</Info>

## Escolha seu caminho de integração

Cinco caminhos para escolher. O correto depende do que está chamando o Raydium e de quanto controle você precisa.

| Caminho | Melhor para | Latência | Lógica personalizada | Linguagens |
|---|---|---|---|---|
| [Trade API](/pt/sdk-api/trade-api) | Backends, bots, Blinks — em qualquer lugar onde você queira uma transação construída no servidor | Menor | Limitada (rotas predefinidas) | Qualquer |
| [TypeScript SDK](/pt/sdk-api/typescript-sdk) | Aplicações Node/TS e frontends que precisam de acesso total ao builder | Média | Completa | TS / JS |
| [Integração Python](/pt/sdk-api/python-integration) | Backends Python e pipelines de ML | Média | Completa (construção manual de instruções) | Python |
| [Rust CPI](/pt/sdk-api/rust-cpi) | Composição de programas que chamam Raydium de forma on-chain | N/A (on-chain) | Completa + componível | Rust |
| [IDL Anchor](/pt/sdk-api/anchor-idl) | Geração de código para linguagens sem suporte de primeira classe (Go, Kotlin, Swift, ...) | Variável | Completa | Qualquer com ferramenta de geração Anchor |

Se você não tem certeza: backends e bots → **Trade API**; frontends e aplicativos com controle total → **TypeScript SDK**; composição on-chain → **Rust CPI**.
