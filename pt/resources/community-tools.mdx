## O que está listado aqui

Uma ferramenta ganha um espaço nesta página se (a) é acessível publicamente, (b) realmente funciona contra os programas mainnet do Raydium no momento da listagem, e (c) é mantida — significando que o projeto teve um commit, release ou engajamento visível nos últimos 90 dias.

As categorias abaixo são uma taxonomia inicial; envie uma ferramenta que não se encaixe em nenhuma delas e adicionaremos uma nova categoria.

## Enviando uma ferramenta

Para adicionar ou atualizar uma entrada nesta página:

1. Faça um fork do [repositório de docs](https://github.com/raydium-io) (a fonte de docs reside no repositório de docs Raydium público — o caminho exato está no README).
2. Edite `resources/community-tools.mdx`.
3. Envie um PR com:
   - Nome e link da ferramenta.
   - Descrição de uma linha (o que é, qual categoria).
   - Handle GitHub ou X do mantenedor (para verificação).
   - Uma nota sobre atividade (último release / último commit significativo).

As listagens são revisadas para o padrão "acessível publicamente / realmente funciona / mantido". Listagens auto-promocionais sem software funcional, ou ferramentas cujo propósito principal conflita com a segurança do usuário (drainers, geradores de pools falsos), não serão mescladas.

## Política de remoção

Uma ferramenta é removida desta página se:

- A instância hospedada ficou offline por >30 dias.
- O repositório foi arquivado ou deletado.
- O mantenedor fica em silêncio por >180 dias e a ferramenta deixa de funcionar.
- A ferramenta começa a se envolver em comportamento prejudicial ao usuário (scripts drainer, UX enganosa, etc.).

A remoção fica a critério dos mantenedores de docs; a reinserção é bem-vinda quando a ferramenta voltar a ficar online e mantida.

## Nota de segurança

Uma ferramenta listada aqui que pede a seed phrase da sua wallet, que exige que você envie fundos para um endereço custodiado, ou que promete rendimentos garantidos, **não** é uma ferramenta que você deveria usar. Estar listada aqui é um padrão baixo — não é um aval de segurança. Sempre inspecione o que você está autorizando antes de assinar uma transação.
