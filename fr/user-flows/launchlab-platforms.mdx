<Info>
  Une plateforme LaunchLab est une configuration on-chain utilisée par une surface de lancement ou une communauté. Les paramètres de plateforme peuvent affecter les frais, la marque, le support des frais de créateur et le comportement du pool post-migration.
</Info>

LaunchLab supporte les environnements de lancement tiers par des Platform PDAs. Une plateforme peut offrir sa propre expérience de lancement de token personnalisée tout en utilisant l'infrastructure du programme LaunchLab et la migration des pools Raydium.

## Ce qu'une plateforme peut contrôler

| Paramètre | Importance |
|---|---|
| Support des frais de créateur | Détermine si les créateurs de token peuvent percevoir des frais avant et après migration. |
| Frais de plateforme | Définit où les revenus de frais au niveau plateforme sont envoyés. |
| Métadonnées de marque | Donne à la plateforme un nom, une description et un logo qui peuvent être affichés dans les surfaces applicatives. |
| Groupement des lancements | Permet à l'interface utilisateur d'afficher les tokens lancés sous la même plateforme. |
| Paramètres du pool post-migration | Peut définir la configuration des frais CPMM lors de la migration des lancements vers CPMM. |
| Distribution des LP | Peut affecter la façon dont les tokens LP post-migration sont brûlés, attribués aux créateurs ou assignés à la plateforme. |

## Ce que les créateurs doivent vérifier

Avant de lancer sous n'importe quelle plateforme, confirmez :

- Quel portefeuille reçoit les frais de plateforme.
- Si les frais de créateur sont activés.
- Si la liquidité post-migration va vers CPMM ou AMM v4.
- Comment les tokens LP sont divisés après la graduation.
- Si une Fee Key NFT sera générée et qui la reçoit.
- Si la plateforme a ses propres conditions, politiques ou règles d'admissibilité.

<Warning>
  Les paramètres de plateforme peuvent modifier matériellement l'économie du créateur et les attentes des utilisateurs. Examinez-les avant de signer la transaction de lancement. Cette page n'explique que la mécanique et n'est pas un conseil juridique, financier ou d'investissement.
</Warning>

## Ce que les traders doivent vérifier

Lors de l'achat d'un token LaunchLab, la plateforme peut affecter le chemin des frais et la gestion de la liquidité post-migration. Avant de trader :

- Vérifiez la plateforme du token et les paramètres de lancement.
- Confirmez le type de pool post-migration si l'interface utilisateur l'affiche.
- Vérifiez si les LP sont brûlés, verrouillés ou assignés.
- Vérifiez le mint du token depuis la page du token, et non uniquement à partir du texte sur les réseaux sociaux.
