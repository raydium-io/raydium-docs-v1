## Ce que vous obtenez

- Une URL partageble qui pré-remplit l'interface de swap Raydium avec le token destination de votre choix **et votre portefeuille comme destinataire du parrainage**.
- Quand quelqu'un exécute un swap via le lien alors qu'il est actif dans sa session, **1 % du volume d'entrée est acheminé vers le portefeuille du parrain** dans le cadre de la transaction de swap.
- Les liens partagés sur X.com s'affichent comme des **Blinks** (Actions Solana) si le portefeuille du lecteur a les Actions / Blinks activés — un widget de swap en un clic dans le message.

## Comment créer un lien

1. Ouvrez [raydium.io/swap](https://raydium.io/swap) et connectez votre portefeuille. L'adresse de parrainage est dérivée du portefeuille connecté ; sans connexion, le lien fonctionne toujours mais ne génère aucun frais.
2. Sélectionnez le **token destination** dans lequel vous voulez que les autres fassent un swap. Le côté source est laissé au choix du destinataire.
3. Cliquez sur l'icône de partage/lien dans le panneau de swap (en haut à droite).
4. Copiez l'URL générée. Partagez-la n'importe où.

Le format d'URL est stable. Si vous construisez une application qui génère directement des liens de parrainage.

## Comment fonctionnent les Blinks

Les **Blinks (Blockchain Links)** sont une norme de la Fondation Solana pour afficher les URLs d'actions on-chain sous forme d'interface interactive à l'intérieur des plateformes de réseaux sociaux. Un portefeuille Solana compatible Blinks lit les métadonnées du lien, construit une transaction pré-intégrée, et affiche à l'utilisateur une invite de signature en ligne.

Conditions requises :

- Le portefeuille du lecteur **doit avoir les Actions / Blinks activés**. C'est généralement une option dans les paramètres du portefeuille.
- La **plateforme doit respecter la norme**. X.com le fait via son unfurler ; certaines autres ne le font pas, auquel cas le lien retombe à un simple hyperlien.

Du point de vue du parrain, le rendu Blink est automatique — la même URL fonctionne comme un lien simple dans les portefeuilles qui ne supportent pas les Blinks.

## Comptabilité des frais

Les frais de parrainage 1 % sont **facturés en sus du frais de trading AMM**, non pas prélevés dessus. Pour un swap sur un pool avec un frais total de 0,25 % :

```
L'utilisateur envoie :  1 000 USDC
Frais de trading AMM :  2,50 USDC   (vers LP + répartition protocole)
Frais de parrainage :  10,00 USDC   (1 % → portefeuille du parrain)
Montant pour le swap : 987,50 USDC  (puis évalué via la courbe)
```

Les frais de parrainage s'appliquent à la route de swap Raydium créée par le lien de parrainage. Les interactions directes de pools en dehors de la route de parrainage ne comportent pas de frais de parrainage.

## Désactiver / quitter un parrainage

Il n'existe pas d'instruction « supprimer le parrainage ». Pour effectuer un swap sans frais de parrainage, il vous suffit de charger [raydium.io/swap](https://raydium.io/swap) directement au lieu de cliquer sur un lien de parrainage. L'interface n'utilise aucun parrain quand l'URL ne contient pas de paramètre de parrain.

## FAQ

**Le parrain doit-il signer quelque chose ?** Non. L'adresse du portefeuille du parrain est la seule donnée qui voyage avec le lien ; le paiement au parrain est un transfert de token dans la transaction du swappeur.

**Puis-je me parrainer moi-même ?** Oui, mais inutile — vous vous payez 1 %, moins le frais tx que vous venez de payer sur une transaction plus importante.

**Que se passe-t-il si le token destination est un SPL avec un hook de transfert ?** Le transfert de parrainage se fait côté entrée (en USDC ou le token source), non pas côté destination, donc les hooks de transfert sur le mint destination n'affectent pas le paiement de parrainage. Les hooks de transfert sur le token d'entrée peuvent bloquer complètement le swap ; Raydium les rejette à l'avance.

**Les frais de parrainage s'appliquent-ils toujours si le swap achemine à travers plusieurs pools ?** Oui — le 1 % est calculé sur le montant d'entrée original et est indépendant de la route.
