## Comment ça fonctionne

Lorsque vous partagez un lien vers une page de token qui inclut votre adresse de portefeuille comme parrain, toute personne effectuant une transaction via votre lien génère une **commission de 0,1%** versée directement à votre portefeuille. La commission est :

- Libellée en SOL (ou en le token de cotation du pool).
- Envoyée automatiquement à votre portefeuille à chaque transaction — aucune étape de réclamation séparée.
- Applicable aux achats et aux ventes sur la courbe de liaison tant que le token est en phase de pré-graduation.

### Étendue de la session

Un lien de parrainage pointe vers un token spécifique, mais si l'utilisateur qui a cliqué sur votre lien effectue ensuite des transactions sur *d'autres* tokens LaunchLab au cours de la même session navigateur, vous pourrez aussi gagner des commissions sur ces transactions. Cela étend votre potentiel de revenus au-delà du simple token que vous aviez initialement partagé.

## Générer votre lien de parrainage

1. Connectez votre portefeuille sur [raydium.io/launchlab](https://raydium.io/launchlab).
2. Accédez à la page de détails de n'importe quel token LaunchLab.
3. Cliquez sur le bouton **Share** sur la page du token.
4. Copiez le lien généré — il inclut votre adresse de portefeuille comme paramètre de parrain.
5. Partagez le lien partout où se trouve votre audience : réseaux sociaux, communautés, bulletins d'information.

<Note>
  Votre portefeuille doit être connecté lorsque vous cliquez sur Share. Le lien ne contiendra pas de paramètre de parrain valide s'il est généré sans portefeuille connecté.
</Note>

## Ce que vous gagnez

| Scénario | Commission |
|---|---|
| Le destinataire achète le token partagé | 0,1% de la valeur de la transaction |
| Le destinataire vend le token partagé | 0,1% de la valeur de la transaction |
| Le destinataire effectue des transactions sur d'autres tokens LaunchLab dans la même session | 0,1% de ces transactions |

Les revenus sont envoyés en temps réel — il n'y a pas de coffre-fort ni de solde en attente à réclamer.

## Comparaison : parrainage LaunchLab vs. parrainage Swap

| | Parrainage LaunchLab | Parrainage Swap (Blinks) |
|---|---|---|
| Taux | 0,1% | 1% |
| Étendue | Pages de tokens LaunchLab | Interface de swap Raydium |
| Paiement | Automatique, par transaction | Manuel ou automatique selon l'intégration |
| Configuration | Bouton Share sur la page du token | Générer via [Parrainages & Blinks](/fr/user-flows/referrals-and-blinks) |
