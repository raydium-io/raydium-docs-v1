## Phase 1 — Avant migration (courbe de liaison)

Pendant que votre token s'échange sur la courbe de liaison, une composante de frais de créateur est prélevée sur chaque achat et vente, puis accumulée dans un coffre spécifique au token. Ceci est configuré par la **plateforme** sur laquelle votre token est lancé.

### Détails

- **Libellés en** token de cotation (généralement SOL).
- **Taux** — défini par la plateforme au moment du lancement. Consultez le barème tarifaire de la plateforme ou l'état on-chain du token.
- **Réclamables à tout moment** — allez sur votre page **Portfolio** sur raydium.io et réclamez les frais en attente à tout moment. Pas d'attente jusqu'à la graduation.
- **S'arrête à la graduation** — aucun nouveau frais de pré-migration n'est accumulé après la graduation de la courbe. Tout solde résiduel dans le coffre reste réclamable.

### Comment réclamer

1. Connectez le portefeuille du créateur à [raydium.io](https://raydium.io).
2. Accédez à **Portfolio**.
3. Trouvez le token parmi vos tokens créés.
4. Cliquez sur **Claim** à côté du solde des frais de pré-migration.

Réclamation programmatique : consultez [`products/launchlab/creator-fees`](/fr/products/launchlab/creator-fees) pour l'instruction SDK.

## Phase 2 — Après migration (Fee Key NFT)

Lorsque la courbe de liaison de votre token atteint son seuil de graduation, la liquidité migre vers un pool CPMM Raydium. Des tokens LP sont frappés et distribués selon la **configuration de la plateforme** :

| Destinataire | Défaut Raydium | Description |
|---|---|---|
| Brûlés (Burn & Earn) | 90 % | Verrouillés de manière permanente ; les frais de trading de cette part vont au mécanisme de fiducie Burn & Earn. |
| Créateur (Fee Key NFT) | 10 % | Une Fee Key NFT est frappée dans votre portefeuille. Cette NFT est votre droit sur 10 % des frais de trading LP continus du pool. |
| Plateforme | 0 % (défaut Raydium) | Certaines plateformes tierces prélèvent une part ici. |

Ces paramètres par défaut s'appliquent aux tokens lancés via l'interface LaunchLab de Raydium. Les plateformes tierces peuvent configurer des répartitions différentes — vérifiez toujours la page de la plateforme avant de lancer.

### La Fee Key NFT

La Fee Key NFT est un token non fongible frappé dans votre portefeuille à la graduation qui représente votre droit de percevoir les frais de trading LP de votre part du pool après graduation.

<Warning>
  **Ne brûlez pas et ne transférez pas la Fee Key NFT à moins que vous ne souhaitiez intentionnellement renoncer aux droits de réclamation de frais.** Si la NFT est perdue ou brûlée, les droits de réclamation de frais sont **définitivement perdus** — il n'existe aucun chemin de récupération.
</Warning>

- La Fee Key peut être échangée ou transférée ; celui qui la détient peut réclamer les frais.
- Les frais s'accumulent continuellement à partir de l'activité de trading LP dans le pool CPMM.
- Réclamez depuis la page Portfolio, ou via l'instruction de réclamation de frais du pool CPMM de manière programmatique.

### Comment réclamer les frais après migration

1. Connectez le portefeuille qui détient la Fee Key NFT.
2. Accédez à **Portfolio** sur raydium.io.
3. Trouvez le pool du token gradué parmi vos positions.
4. Cliquez sur **Claim** pour percevoir les frais de trading accumulés.

## Si les frais post-migration ne sont pas activés

Si une plateforme désactive les frais de créateur, votre token se diplôme vers un **pool AMM v4** (le chemin de migration hérité) au lieu d'un CPMM. Dans ce mode :
- Aucune Fee Key NFT n'est frappée.
- Il n'existe pas de frais de créateur post-migration au niveau du programme.
- Certaines options alternatives de réclamation de frais peuvent exister via des outils CLI en fonction de la configuration du lancement.

## Résumé du flux de frais

```
Phase de courbe de liaison :
  Chaque achat/vente → petit % → coffre du créateur (token de cotation)
  Réclamable à tout moment depuis Portfolio

Graduation :
  Tokens LP frappés → répartition par config de plateforme
  ├── X% brûlés (Burn & Earn)
  ├── Y% → créateur comme Fee Key NFT      ← vous détenez ceci
  └── Z% → plateforme (le cas échéant)

Post-graduation :
  Chaque swap CPMM → frais LP → répartition par part LP
  Votre part Fee Key NFT → réclamable depuis Portfolio
```
