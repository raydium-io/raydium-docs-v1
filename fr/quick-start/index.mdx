## Prérequis

Exécutez ceci une fois :

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Vous avez besoin de :

- **Node 18+** avec support ESM (`"type": "module"` dans `package.json` ou extension `.mjs`).
- **Une paire de clés Solana** à `~/.config/solana/id.json` (ou n'importe où — configurez `KEYPAIR`).
- **Un portefeuille financé.** ~0,05 SOL couvre n'importe quel script seul dans ce chapitre.
- **Une URL RPC.** Les RPC publics gratuits fonctionnent pour les tests ; pour autre chose qu'un simple test, utilisez un fournisseur payant.

Configurez ceux-ci une fois par shell :

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Les trois scripts utilisent par défaut mainnet. Pour exécuter sur devnet, définissez `RPC_URL=https://api.devnet.solana.com` et modifiez `cluster: "mainnet"` en `cluster: "devnet"` à l'intérieur du script. Sachez que la liquidité initiale des pools sur devnet est limitée ; attendez-vous à créer vos propres tokens de test.
</Info>
