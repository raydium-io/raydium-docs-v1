## Vue d'ensemble du flux

```text
Frais de swap
  -> Partage LP
  -> Partage rachat RAY
  -> Partage trésorerie, le cas échéant
```

Pour les pools CLMM et CPMM, la répartition des frais de trading est :

```text
84 % aux LPs
12 % aux rachats RAY
4 % à la trésorerie
```

Pour les pools Standard AMM v4, la répartition des frais de trading est :

```text
88 % aux LPs
12 % aux rachats RAY
```

## Qu'est-ce que le partage rachat signifie

Le pourcentage de rachat s'applique au **frais de trading**, pas au montant total de l'échange.

Exemple : un pool facture des frais de swap de `0,25 %`. Le partage rachat est `12 %` de ces frais.

```text
Montant de l'échange :     1 000 USDC
Frais de swap :            1 000 * 0,25 % = 2,50 USDC
Partage rachat RAY :       2,50 * 12 % = 0,30 USDC
```

Le montant rachat effectif est `0,03 %` du montant de l'échange dans cet exemple.

## Adresses de collecte

Les frais du côté protocole sont collectés par des adresses spécifiques au programme avant d'être convertis ou orientés.

| Source | Adresse de collecte |
|---|---|
| CLMM | `projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416` |
| CPMM | `ProCXqRcXJjoUd1RNoo28bSizAA6EEqt9wURZYPDc5u` |
| Standard AMM v4 | `PNLCQcVCD26aC7ZWgRyr5ptfaR7bBrWdTFgRWwu2tvF` |

Les adresses de collecte peuvent détenir de nombreux tokens différents. Chaque pool paie les frais dans les tokens qu'il échange, donc une adresse de collecte peut contenir USDC, wSOL, RAY ou d'autres tokens SPL avant le routage.

## Adresse de détention du rachat

Le RAY racheté est détenu à :

```text
DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

Vous pouvez inspecter cette adresse directement sur Solscan ou avec les outils SPL Token :

```bash
spl-token accounts --owner DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

## Pourquoi les rachats sont on-chain

Le flux de rachat de Raydium est vérifiable à partir d'adresses publiques et d'un historique de transactions.

- **Les soldes sont visibles :** les adresses de collecte et de détention peuvent être inspectées par quiconque.
- **Les transferts sont traçables :** les transferts de frais entrants et les transactions de routage sortantes sont publics.
- **Aucune comptabilité privée n'est requise :** la chaîne est le grand livre du flux.

## Comment vérifier un flux de rachat

1. Ouvrez une transaction de swap Raydium dans Solscan ou Solana Explorer.
2. Identifiez le programme de pool et les transferts de frais.
3. Associez le mouvement des frais côté protocole à l'adresse de collecte pertinente.
4. Inspectez les transactions sortantes de l'adresse de collecte.
5. Confirmez l'accumulation de RAY à `DdHDoz...VEZaz`.

Pour la carte d'adresse complète, consultez [`ray/treasury`](/fr/ray/treasury).
