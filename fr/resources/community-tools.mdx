## Ce qui est répertorié ici

Un outil mérite une place sur cette page s'il (a) est accessible publiquement, (b) fonctionne réellement contre les programmes mainnet de Raydium au moment de l'inscription, et (c) est maintenu — ce qui signifie que le projet a eu un commit, une release ou un engagement visible au cours des 90 derniers jours.

Les catégories ci-dessous forment une taxonomie de départ ; soumettez un outil qui ne correspond à aucune d'entre elles et nous ajouterons une nouvelle catégorie.

## Soumettre un outil

Pour ajouter ou mettre à jour une entrée sur cette page :

1. Forkez le [dépôt de docs](https://github.com/raydium-io) (la source de docs se trouve dans le dépôt public de docs Raydium — le chemin exact est dans le README).
2. Modifiez `resources/community-tools.mdx`.
3. Soumettez une PR avec :
   - Nom et lien de l'outil.
   - Description d'une ligne (ce que c'est, quelle catégorie).
   - Handle GitHub ou X du mainteneur (pour vérification).
   - Une note sur l'activité (dernière release / dernier commit significatif).

Les inscriptions sont examinées selon le critère « accessible publiquement / fonctionne réellement / maintenu ». Les inscriptions auto-promotionnelles sans logiciel fonctionnel, ou les outils dont l'objectif principal entre en conflit avec la sécurité des utilisateurs (drainers, générateurs de faux pools), ne seront pas fusionnées.

## Politique de suppression

Un outil est supprimé de cette page si :

- L'instance hébergée a été hors ligne pendant >30 jours.
- Le dépôt a été archivé ou supprimé.
- Le mainteneur disparaît pendant >180 jours et l'outil cesse de fonctionner.
- L'outil commence à adopter un comportement nuisible aux utilisateurs (scripts drainers, UX trompeur, etc.).

La suppression relève de la discrétion des mainteneurs de docs ; la réinscription est bienvenue une fois que l'outil est de nouveau en ligne et maintenu.

## Note de sécurité

Un outil répertorié ici qui vous demande la phrase d'amorçage de votre portefeuille, qui vous demande d'envoyer des fonds à une adresse de garde, ou qui promet des rendements garantis, n'est **pas** un outil que vous devriez utiliser. L'inscription ici est une barre basse — elle ne garantit pas la sécurité. Inspectez toujours ce que vous autorisez avant de signer une transaction.
