## Pour qui est ce chapitre

Développeurs et chercheurs en protocole venant d'autres blockchains (EVM, Cosmos, Move) ou nouveaux sur Solana. Les utilisateurs finaux peuvent sauter ce chapitre.

L'objectif est pragmatique : vous donner exactement ce que vous devez lire dans les chapitres Produits et SDK sans surprises. Ce n'est pas un tutoriel général sur Solana — consultez la documentation officielle de Solana pour cela.
