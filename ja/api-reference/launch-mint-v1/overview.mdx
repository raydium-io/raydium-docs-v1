Mint APIは、LaunchLabトークンの作成、発見、管理への包括的なアクセスを提供します。名前、シンボル、またはクリエイターによってミントをクエリし、リーダーボードを取得し、新しいミント作成リクエストを送信できます。

## APIエンドポイント

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## 認証

読み取り操作は認証不要です。書き込み操作（ミント作成）には、ウォレット署名検証後にAuth APIによって発行された `ray-token` ヘッダーが必要です。

[Auth APIドキュメント](/ja/api-reference/launch-auth-v1/overview)

## ユースケース

- **トークンを検索・発見する** 名前またはシンボルで検索
- **リーダーボードを表示する** （時価総額トップのミント、最新、最もホットなもの）
- **ユーザー統計を取得する** （作成と取引量の時系列データ）
- **ベスティングスケジュールをクエリする** トークンアンロック情報を確認
- **ミント作成リクエストを送信する** メタデータと画像付き
- **紹介者報酬を追跡する** プラットフォーム紹介プログラムから
