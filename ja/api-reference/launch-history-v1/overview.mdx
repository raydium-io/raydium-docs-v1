History API は LaunchLab プールの過去のトレードデータを提供します。複数の間隔（1分、5分、15分）でロウソク足（OHLC）データを取得し、ページネーション付きの個別トレード記録にアクセスできます。

## API エンドポイント

**メインネット:** https://launch-history-v1.raydium.io  
**デブネット:** https://launch-history-v1-devnet.raydium.io

## 認証

認証は不要です。すべてのエンドポイントは公開アクセス可能です。

## データ機能

- **キラインデータ：** 1m、5m、15m 間隔の OHLC ロウソク足チャート
- **トレード履歴：** トレーダー、金額、タイムスタンプ付きの個別トレード
- **ページネーション：** `nextPageKey` を使用して大規模データセットクエリを効率化
- **制限制約：** キラインリクエストは 1-500 本のキャンドル（デフォルト 300）、トレードは 1-100 件（デフォルト 50）を受け入れます
- **フィルタリング：** トレーダーウォレットと金額範囲でトレード履歴をフィルタリング

## ユースケース

- **チャート表示** - ロウソク足ビジュアライゼーション付き
- **価格分析** とテクニカル分析インジケーター
- **トレード統計** とボリュームトラッキング
- **ユーザートレード監査** - ウォレットと日付範囲でフィルタリング
- **ミント の過去価格発見**
