Dynamic IPFS API は、Solana 上の CLMM および CPMM リクイディティポジションの再生成メタデータと画像を提供します。このサービスは、濃縮流動性ポジションカードなどの動的 NFT をレンダリングする dApp 向けに設計されており、オンチェーン検索を必要とせずにリアルタイムのプール情報、ポジション評価、および請求されていないフィーを提供します。

## ホスト

| 環境      | ホスト                            |
|-----------|-----------------------------------|
| メインネット | `dynamic-ipfs.raydium.io`         |
| デブネット  | `dynamic-ipfs-devnet.raydium.io`  |

## 概要

API は 3 つのエンドポイントを公開し、それぞれ完全なポジションインターフェースオブジェクトを返します。

- **`/clmm/position`** — アクティブな CLMM ポジションのメタデータを取得
- **`/lock/clmm/position`** — ロック済み CLMM ポジションのメタデータを取得
- **`/lock/cpmm/position`** — ロック済み CPMM ポジションのメタデータを取得

各レスポンスには、完全なプール詳細（トークン、デシマル、TVL）、ポジション額、USD 評価、プール内の TVL パーセンテージ、および蓄積された（ただし請求されていない）フィーと報酬が含まれます。

## ユースケース

特定のポジションのメタデータをリクエストすることで、動的ポジションビジュアライゼーション NFT を生成します。API はオンチェーンデータからリアルタイムのポジション状態を計算し、結果をキャッシュするため、以下に最適です。

- ライブ USD 値とフィー蓄積を含むポジションカードのレンダリング
- プール内の TVL パーセンテージとトークン額の表示
- 請求されていない取引フィーと報酬トークンの表示
- ポジションパフォーマンスを追跡するポートフォリオダッシュボードの構築

## 認証

認証は不要です。すべてのエンドポイントはパブリックです。

## レート制限

実装なし。ベストエフォートサービスです。

## 一般的なパターン

### CLMM ポジションを取得する

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

`id` パラメータは、ポジションアカウントのオンチェーン公開キー（base58 形式の Solana アドレス）である必要があります。

### ロック済みポジションを取得する

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
