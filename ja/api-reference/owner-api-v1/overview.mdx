Owner API は、Raydium ポジションとバランスのウォレット単位のビューを提供します。ポジション、ロックされたリクイディティ、作成されたプール、IDO 参加のキャッシュされたデータを取得でき、ウォレット UI とポートフォリオダッシュボードに最適です。

## ホスト

| 環境 | ホスト |
|-------------|---------------------------|
| Mainnet     | `owner-v1.raydium.io`     |
| Devnet      | `owner-v1-devnet.raydium.io` |

## 概要

API は、リソースタイプ別に編成された 5 つのメインエンドポイントを公開しています：

- **`/position/stake/{owner}`** — ステークポジション（標準 AMM / CPMM LP）
- **`/position/clmm-lock/{owner}`** — ロックされた CLMM ポジション
- **`/create-pool/{owner}`** — このウォレットで作成されたプール
- **`/main/ido/{owner}`** — IDO 参加データ

## データの鮮度

このサービスはポジションデータをキャッシュし、定期的に更新します。非常に最近のポジション変更（数秒以内）は、レスポンスに即座に反映されない場合があります。

## 一般的なパターン

### ウォレットのすべてのステークポジションを取得

```bash
curl -s "https://owner-v1.raydium.io/position/stake/{owner}"
```

`{owner}` をウォレットの公開鍵（base58 形式）に置き換えてください。

### ロックされたポジションを取得

```bash
curl -s "https://owner-v1.raydium.io/position/clmm-lock/{owner}"
```

### IDO 参加を確認

```bash
curl -s "https://owner-v1.raydium.io/main/ido/{owner}"
```

## 空の結果

ウォレットに特定のエンドポイントのデータがない場合、サービスはリクエストメタデータを含む 404 エラーを返します：

```json
{
  "id": "...",
  "success": false
}
```
