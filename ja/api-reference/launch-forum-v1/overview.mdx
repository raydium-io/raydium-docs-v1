Forum API は、LaunchLab プールの周りでのディスカッションとコミュニティ エンゲージメントを実現します。オプションの画像添付を含むコメントを投稿し、プール ディスカッション スレッドを取得できます。すべての書き込み操作は Auth API を通じた認証が必要です。

## API エンドポイント

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## 認証

**読み込み操作（GET）:** 認証は不要です。

**書き込み操作（POST）:** `ray-token` ヘッダーが必要です。これは wallet 署名検証後に Auth API から発行される JWT です。

[Auth API からトークンを取得](/ja/api-reference/launch-auth-v1/overview)

## リクエスト ヘッダー

コメントを投稿する場合、認証ヘッダーを含めてください：

```
ray-token: <jwt_token_from_auth_api>
```

## 機能

- **コメント投稿:** オプションの画像添付付き（最大 5 MB）でテキスト コメントを送信
- **ディスカッション閲覧:** ページネーション付きでプール上のすべてのコメントを表示
- **コンテンツ モデレーション:** NSFW 検出と非表示コメント フィルタリング
- **ページネーション:** 効率的な閲覧のため `lastId` と `fetchType`（old/new）を使用してクエリ
- **画像アップロード:** コメントは Pinata IPFS 画像ホスティングをサポート

## ユースケース

- トークン プロジェクト周辺での**コミュニティ ディスカッション**
- コメント経由での**プロジェクト公開情報**と更新
- **ユーザー フィードバック**とセンチメント トラッキング
- **ソーシャル プルーフ**とコミュニティ エンゲージメント メトリクス
- NSFW とスパム検出を用いた**コンテンツ モデレーション**
