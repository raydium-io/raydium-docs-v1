## 何が得られるか

- 選択した先行トークン**と、リファラーのウォレットをリファーラル受取人として**事前入力する Raydium スワップ UI への共有可能な URL。
- リンク経由でスワップを実行する人がいる場合、**入力額の 1% がリファラーのウォレットにルーティング**されます（スワップトランザクションの一部として）。
- X.com で共有されたリンクは、ビューアーのウォレットが Actions / ブリンク機能を有効にしている場合、**ブリンク**（Solana Actions）としてレンダリングされます — ポスト内のワンクリックスワップウィジェットです。

## リンクの作成方法

1. [raydium.io/swap](https://raydium.io/swap) を開き、ウォレットに接続します。リファーラルアドレスは接続されたウォレットから取得されます。接続がない場合、リンクは機能しますが、手数料は獲得されません。
2. 他のユーザーにスワップさせたい**先行トークン**を選択します。ソース側は受信者が選択します。
3. スワップパネル（右上）の共有/リンクアイコンをクリックします。
4. 生成された URL をコピーします。どこでも共有できます。

URL フォーマットは安定しています。リファーラルリンクを直接生成するアプリを構築している場合です。

## ブリンクの仕組み

**ブリンク（Blockchain Links）** は、オンチェーン アクション URL をソーシャルプラットフォーム内のインタラクティブな UI としてレンダリングするための Solana Foundation 標準です。ブリンク対応の Solana ウォレットはリンクメタデータを読み込み、事前構築されたトランザクションを構築し、ユーザーに署名プロンプトをインラインで表示します。

要件：

- ビューアーの**ウォレットは Actions / ブリンクが有効でなければなりません**。これは通常ウォレット設定のトグルです。
- **プラットフォームは標準に従わなければなりません**。X.com はそのアンフューラーを通じて従っています。他のものはそうではない場合、リンクは通常のハイパーリンクにフォールバックします。

リファラーの観点からは、ブリンクレンダリングは自動的です — 同じ URL は、ブリンクをサポートしていないウォレットではプレーンリンクとして機能します。

## 手数料の計算

1% のリファーラル手数料は、**AMM 取引手数料の上に課金され**、カットアウトされません。0.25% の総手数料を持つプールでのスワップの場合：

```
ユーザーが送信:        1,000 USDC
AMM 取引手数料:     2.50 USDC   (LP + プロトコルスプリットへ)
リファーラル手数料:     10.00 USDC   (1% → リファラーのウォレット)
スワップへの入力:     987.50 USDC   (その後、曲線を通じて価格設定)
```

リファーラル手数料は、リファーラルリンクで作成された Raydium スワップルートに適用されます。リファーラルルート外での直接プール相互作用には、リファーラル手数料は含まれません。

## リファーラルの無効化/終了

「リファーラルを削除」命令はありません。リファーラル手数料なしでスワップを実行するには、リファーラルリンクをクリックする代わりに、[raydium.io/swap](https://raydium.io/swap) に直接アクセスしてください。URL にリファラーパラメーターがない場合、UI はリファラーを使用しません。

## よくある質問

**リファラーは何かに署名する必要がありますか?** いいえ。リファラーのウォレットアドレスはリンクで移動する唯一のデータです。リファラーへの支払いはスワッパーのトランザクション内のトークン転送です。

**自分自身をリファーするできますか?** はい、ただし無意味です — より大きなトランザクションで支払ったトランザクション手数料を差し引いて、自分に 1% を支払います。

**先行トークンが転送フックを持つ SPL の場合はどうなりますか?** リファーラル転送は入力側（USDC またはソーストークン）で発生し、先行側ではないため、先行ミント上の転送フックはリファーラル支払いに影響しません。入力トークン上の転送フックはスワップ全体をブロックできます。Raydium はこれらを事前に拒否します。

**スワップが複数のプールを通じてルーティングされる場合、リファーラル手数料はまだ適用されますか?** はい — 1% は元の入力額で計算され、ルートとは独立しています。
