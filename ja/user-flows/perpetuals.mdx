## Raydium Perps とは

Raydium Perps は [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC) で利用できるガスレスのパーペチュアルトレーディングプラットフォームです。Solana ウォレットからパーペチュアル先物を取引でき、注文の発注、更新、キャンセルのたびにガス代を支払う必要はありません。

本プロダクトは基盤となるベニューのために Orderly Network のインフラストラクチャを使用しています。Raydium はプロダクトサーフェスを提供し、Orderly は中央リミットオーダーブック、マッチング、およびパーペチュアルズベニューの仕組みを提供します。

主な特徴：

- **パーペチュアル先物**：原資産を保有せずに、ロングまたはショートのエクスポージャーを取引します。
- **ガスレス取引**：注文の発注とキャンセルは Solana トランザクションを必要としません。
- **レバレッジ**：マーケットはレバレッジをサポートしており、制限は資産とベニューの設定によって異なります。
- **マルチ担保デポジット**：サポートされている担保資産をデポジットして、マージンに使用します。
- **セルフカストディ**：ウォレットを接続してアカウントアクションを承認します。

<Warning>
  Raydium Perps は制限付き管轄区域では利用できない場合があります。お客様の居在地またはローカルロー が Raydium Perps または基盤となるベニューへのアクセスを禁止している場合は、本プロダクトを使用しないでください。
</Warning>

## 取引を開始する方法

パーペチュアルズアプリからアカウントを資金化して、取引を開始できます。

### 1. パーペチュアルズに移動する

[`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC) を開きます。ウォレットを接続する前に URL を確認してください。

### 2. アカウントを作成する

初回ユーザーはパーペチュアルズアカウントを作成します。これは 1 回限りのセットアップステップです。署名前にウォレットプロンプトを確認してください。

### 3. 資金をデポジットする

サポートされている担保をデポジットします。Raydium Perps は主要なチェーンからのデポジットをサポートしており、利用可能な場合はネイティブ SOL や USDC などの担保をサポートしています。

デポジット後、注文を発注する前にアカウント資産と利用可能なマージンを確認してください。

### 4. 取引を開始する

マーケットを選択し、オーダーブックを確認し、注文タイプを選択し、サイズとレバレッジを設定して、注文を送信します。マージン、清算価格、およびファンディングがポジションにどのように影響するかを理解するまで、小さく始めましょう。

## ポジションを開く前に確認すること

- **マーケット**：意図したパーペチュアルマーケットを選択したことを確認します。
- **方向**：ロングは価格の上昇から利益を得ます。ショートは価格の下落から利益を得ます。
- **レバレッジ**：レバレッジが高いほど清算リスクが増加します。
- **清算価格**：ポジションが清算される可能性のある価格を確認します。
- **注文タイプ**：成行注文は実行を優先します。指値注文は価格を管理します。
- **マージン**：手数料後、アカウントに十分な利用可能マージンがあることを確認します。
