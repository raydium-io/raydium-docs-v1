## フェーズ 1 — マイグレーション前（ボンディングカーブ）

トークンがボンディングカーブで取引されている間、各買いと売りからクリエーター手数料の一部が取得され、トークンごとのボールトに蓄積されます。これは、トークンがローンチされた**プラットフォーム**によって設定されます。

### 詳細

- **単位** — クオートトークン（通常は SOL）。
- **レート** — ローンチ時にプラットフォームによって設定されます。プラットフォームの手数料スケジュールまたはトークンのオンチェーン状態を確認してください。
- **いつでも請求可能** — raydium.io の**ポートフォリオ**ページにアクセスして、保留中の手数料をいつでも請求できます。卒業を待つ必要はありません。
- **卒業時に停止** — カーブが卒業した後、新しいマイグレーション前の手数料は蓄積されません。ボールトに残っている余額は請求可能のままです。

### 請求方法

1. クリエーターウォレットを [raydium.io](https://raydium.io) に接続します。
2. **ポートフォリオ**に移動します。
3. 作成したトークンの下でトークンを見つけます。
4. マイグレーション前の手数料残額の隣にある**請求**をクリックします。

プログラマティック請求：SDK インストラクションについては、[`products/launchlab/creator-fees`](/ja/products/launchlab/creator-fees) を参照してください。

## フェーズ 2 — マイグレーション後（Fee Key NFT）

トークンのボンディングカーブが卒業閾値に達すると、リクイディティは Raydium CPMM プールにマイグレーションされます。LP トークンは**プラットフォーム設定**に従ってミントされ配布されます。

| 受取人 | Raydium デフォルト | 説明 |
|---|---|---|
| バーン済み（Burn & Earn） | 90% | 永久にロック済み。このシェアからのトレーディング手数料は Burn & Earn トラストメカニズムに流れます。 |
| クリエーター（Fee Key NFT） | 10% | Fee Key NFT がウォレットにミントされます。このNFT は、プールの進行中の LP トレーディング手数料の 10% に対するあなたの請求権を表します。 |
| プラットフォーム | 0%（Raydium デフォルト） | サードパーティプラットフォームの中には、ここでシェアを取得するものもあります。 |

これらのデフォルトは、Raydium の LaunchLab UI を通じてローンチされたトークンに適用されます。サードパーティプラットフォームは異なるスプリットを設定する可能性があります。ローンチ前に必ずプラットフォームページを確認してください。

### Fee Key NFT について

Fee Key NFT は、卒業時にウォレットにミントされる非代替トークンで、マイグレーション後のプール内のあなたのシェアから LP トレーディング手数料を回収する権利を表します。

<Warning>
  **Fee Key NFT を意図的に手数料請求権を放棄したい場合を除き、バーンまたは転送しないでください。** NFT が失われたりバーンされた場合、手数料請求権は**永久に没収**されます。回復の道はありません。
</Warning>

- Fee Key は取引または転送できます。それを保有している者は手数料を請求できます。
- 手数料は CPMM プール内の LP トレーディング活動から継続的に蓄積されます。
- ポートフォリオページから請求するか、プログラマティックに CPMM プールの手数料請求インストラクションを通じて請求します。

### マイグレーション後の手数料を請求する方法

1. Fee Key NFT を保有するウォレットを接続します。
2. raydium.io の**ポートフォリオ**に移動します。
3. あなたのポジションの下で卒業済みトークンのプールを見つけます。
4. **請求**をクリックして、蓄積されたトレーディング手数料を回収します。

## マイグレーション後の手数料が有効でない場合

プラットフォームがクリエーター手数料を無効として設定した場合、トークンは**AMM v4 プール**（レガシーマイグレーションパス）にアップグレードされ、CPMM ではなくなります。このモードでは：
- Fee Key NFT はミントされません。
- プログラムレベルのマイグレーション後クリエーター手数料はありません。
- ローンチの設定方法に応じて、CLI ツール経由での代替手数料請求オプションが存在する場合があります。

## 手数料フロー概要

```
ボンディングカーブフェーズ：
  各買い/売り → 小数% → クリエーターボールト（クオートトークン）
  ポートフォリオからいつでも請求可能

卒業：
  LP トークンミント → プラットフォーム設定に従ってスプリット
  ├── X% バーン済み（Burn & Earn）
  ├── Y% → Fee Key NFT としてクリエーターへ      ← あなたがこれを保有
  └── Z% → プラットフォーム（該当する場合）

マイグレーション後：
  すべての CPMM スワップ → LP 手数料 → LP シェアで分割
  あなたの Fee Key NFT シェア → ポートフォリオから請求可能
```
