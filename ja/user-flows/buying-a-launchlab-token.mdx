## LaunchLab トークンとは

LaunchLab トークンは**ボンディングカーブ**で発行されます。これはスマートコントラクトで管理される価格曲線で、トークンを買う人が多いほど価格が上がり、売る人が多いほど価格が下がります。カーブは卒業閾値（通常 85 SOL ですがクリエーターが別の金額を設定できます）に達するまで機能し、その時点でトークンは自動的に Raydium CPMM または AMM v4 プールに移行し、通常の DEX ルーティングで自由に取引可能になります。

卒業前は、カーブ自体に対してのみ売買できます。卒業後は、他のプールと同じように Raydium で取引されます。

## 購入前に

### 自分で調査をしてください

- トークンのミントアドレスが、そのプロジェクトが発表したものと一致していることを確認してください。詐欺師は同じ名前のトークンを定期的に作成します。
- **ミント権が取り消されているか**を確認してください。取り消されていない場合、クリエーターが新しいサプライをミントしてあなたのポジションを希薄化させる可能性があります。Solflare のようなウォレットや Solscan のようなエクスプローラーはこのステータスを表示します。
- ボンディングカーブの進捗を確認してください。販売されたトークン数、残されたトークン数、現在の価格が見えます。これはトークンの LaunchLab ページで確認できます。
- クリエーターアドレスの履歴を確認してください。

### カーブの仕組みを理解する

ボンディングカーブの価格は**固定ではありません**。買い注文があるたびに価格は上がり、売り注文があるたびに価格は下がります。あなたの買い注文の直前に大きな買い注文があると、支払う価格が大幅に上がる可能性があります（フロントラン／サンドイッチリスク）。スリッページリミットを使用してください（下記参照）。

## ステップバイステップ：UI での購入方法

1. [raydium.io/launchlab](https://raydium.io/launchlab) に移動し、買いたいトークンを見つけます。
2. トークンページをクリックして詳細ビューを開きます。
3. ウォレットをまだ接続していない場合は接続します。
4. 使いたい SOL（またはクォートトークン）の金額を入力します。
5. UI に表示されている**推定受取トークン数**と**価格への影響**を確認します。
6. 必要に応じて**スリッページトレランス**を調整します（下記参照）。
7. **Buy（買う）**をクリックしてウォレットでトランザクションを承認します。
8. トランザクションが確認されると、トークンがウォレットに表示されます。

<Tip>
  卒業前はいつでも同じトークンページの **Sell（売る）**ボタンを使ってカーブに売り戻せます。卒業後はトークンが Raydium プールに入っているので、代わりに標準のスワップインターフェースを使用してください。
</Tip>

## スリッページ

<Warning>
  **適切なスリッページトレランスを設定してください。**ボンディングカーブトークンは不安定です。スリッページを低く設定しすぎると、価格が変わったときにトランザクションが失敗する可能性があります。高く設定しすぎると、サンドイッチボットにさらされます。ほとんどのローンチでは 1～3% が合理的な開始点です。必要な場合は、卒業間近の流動性が非常に低いローンチでのみ上げてください。
</Warning>

スリッページトレランスは、提示された価格と実行された価格の間で許容する最大価格差です。LaunchLab UI では、トランザクションを確認する前にこれを設定できます。

## 卒業後

カーブの卒業閾値に（任意のトレーダーによって）達すると、トークンは自動的に Raydium AMM プールに移行します。その時点で：

- ボンディングカーブが閉じられます。さらなる売買はカーブを通りません。
- トークンはウォレットに手付かずで残ります。
- Raydium のスワップページまたは Raydium プールをインデックス化する任意のアグリゲーターでトークンを取引できます。
- プール移行がアグリゲーター UI に表示されるまで数分かかることがあります。

## リスク

| リスク | 説明 |
|---|---|
| リクイディティ管理リスク | LP の取り扱いはプラットフォームとローンチ設定によって異なります。LP がバーン、ロック、クリエーターに割り当てられている、またはプラットフォームに割り当てられているかを確認してください。 |
| 卒業しない | カーブがその閾値に達しない場合、取引はカーブだけに制限されたままです。カーブに売り戻すことはできますが、購入時より低い価格の可能性があります。 |
| 価格のボラティリティ | ボンディングカーブは本質的に投機的です。価格はほぼゼロまで下がる可能性があります。 |
| 偽物トークン | 詐欺師は似たようなトークンを作成します。ミントアドレスを独立して確認してください。 |
| スマートコントラクトリスク | オンチェーンプログラムはリスクフリーではありません。失う準備ができた資金のみを使用してください。 |
