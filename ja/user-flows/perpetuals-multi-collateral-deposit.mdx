## サポートされているコラテラル

Raydium Perpsは、SOL、USDC、USDT、ETH、BNB、WBTC、YUSD、USD1など、主要なチェーンおよびコラテラル資産からのデポジットをサポートしています。

正確なリスト、チェーン、制限は、基盤となるベニューによって制御されます。デポジットを行う前に、アプリを信頼できるソースとして扱ってください。

## グローバル制限

グローバル制限はベニュー全体に適用され、すべてのユーザーによって共有されます。参照されるRaydiumページには、以下のような例が記載されています：

| コラテラル | チェーンの例 |
|---|---|
| USDT | Arbitrum、BSC、Ethereum |
| ETH | Arbitrum、Base、Ethereum |
| BNB | BSC |
| USD1 | BSC、Ethereum |
| YUSD | BSC、Ethereum |
| WBTC | Ethereum |
| SOL | Solana |

グローバル制限に達している場合、ウォレットが自身の制限以下であっても、その資産への新しいデポジットは利用できないことがあります。

## ユーザーごとの制限

ユーザーごとの制限は、各アカウントまたはウォレットに適用されます。参照されるRaydiumページには、USDT、ETH、SOL、BNB、YUSD、WBTC、USD1などの資産のユーザーごとの制限が記載されています。

資金を送信する前に、デポジット画面を確認してください。アプリに表示されている利用可能なデポジット額がウォレット残高より少ない場合は、ユーザーごとまたはグローバルのコラテラル上限に達している可能性があります。

## デポジットフロー

1. [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC)を開きます。
2. ウォレットを接続し、デポジットフローを開きます。
3. コラテラル資産とソースチェーンを選択します。
4. 金額、宛先アカウント、およびベニューの警告を確認します。
5. デポジットを確認します。
6. アカウント残高および利用可能証拠金が更新されるまで待ちます。
