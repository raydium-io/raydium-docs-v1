<Info>
  LaunchLabプラットフォームは、ローンチサーフェスまたはコミュニティで使用されるオンチェーン設定です。プラットフォーム設定は、手数料、ブランディング、クリエイター手数料サポート、ミグレーション後のプール動作に影響を与える可能性があります。
</Info>

LaunchLabは、プラットフォーム PDA を通じてサードパーティのローンチ環境をサポートしています。プラットフォームは、LaunchLab プログラムインフラストラクチャと Raydium プールミグレーションを活用しながら、独自のブランド化されたトークンローンチエクスペリエンスを提供できます。

## プラットフォームが制御できること

| 設定 | 重要な理由 |
|---|---|
| クリエイター手数料サポート | トークンクリエイターがミグレーション前後の手数料を獲得できるかどうかを決定します。 |
| プラットフォーム手数料 | プラットフォームレベルの手数料収益をどこに送るかを設定します。 |
| ブランディングメタデータ | プラットフォームに名前、説明、ロゴを付与し、アプリサーフェスに表示できます。 |
| ローンチグループ化 | UI が同一プラットフォーム下でローンチされたトークンを表示できるようにします。 |
| ミグレーション後のプール設定 | プラットフォームがローンチを CPMM にミグレーションする際の CPMM 手数料設定を定義できます。 |
| LP 分配 | ミグレーション後の LP トークンがバーンされるか、クリエイターに割り当てられるか、プラットフォームに割り当てられるかに影響を与える可能性があります。 |

## クリエイターが確認すべきこと

プラットフォーム下でローンチする前に、以下を確認してください。

- プラットフォーム手数料を受け取るウォレット。
- クリエイター手数料が有効になっているかどうか。
- ミグレーション後のリクイディティが CPMM または AMM v4 のどちらに移行するか。
- ローンチ卒業後に LP トークンがどのように分割されるか。
- Fee Key NFT がミントされるかどうかと、誰がそれを受け取るか。
- プラットフォームが独自の利用規約、ポリシー、または適格性ルールを持っているかどうか。

<Warning>
  プラットフォーム設定はクリエイターの経済性とユーザー期待に重大な影響を与える可能性があります。ローンストランザクションに署名する前にそれらを確認してください。このページは仕組みのみを説明しており、法的、財務的、投資アドバイスではありません。
</Warning>

## トレーダーが確認すべきこと

LaunchLab トークンを購入する際、プラットフォームは手数料経路とミグレーション後のリクイディティ処理に影響を与える可能性があります。トレードする前に、以下を確認してください。

- トークンのプラットフォームとローンチ設定。
- UI に表示される場合、ミグレーション後のプールタイプを確認。
- LP がバーンされたか、ロックされたか、割り当てられたかを確認。
- ソーシャルメディアのテキストだけでなく、トークンページからトークンミントを確認。
