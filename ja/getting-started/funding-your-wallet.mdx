## 3 つの資金供給方法

| 方法 | 一般的なコスト | スピード | 必要なセットアップ |
|---|---|---|---|
| **集中型取引所出金** | SOL の固定出金手数料（しばしば 0.01 SOL） | 出金承認後 1～2 分 | SOL 保有の既存取引所アカウント |
| **ウォレット内フィアット オンランプ（Moonpay / Stripe）** | 購入額の 2～4% | 数分～数時間（KYC に依存） | カード / 銀行詳細を入力済み |
| **クロスチェーン ブリッジ** | 0.1～0.5% のブリッジ手数料 + 起点チェーンのガス代 | 数分～1 時間 | 別チェーン上のウォレット / 資金 |

現在お金がある場所に基づいて選択してください。

## 方法 1：集中型取引所出金（最も安い）

Coinbase、Binance、Kraken、OKX、Bybit、またはその他の大手取引所にアカウントをすでにお持ちの場合、この方法が最も手続きが簡単です。

### 手順

1. **取引所で SOL を購入します。** すでにそこで何か暗号資産（USDC、BTC、ETH）を保有している場合は、内部的に SOL にスワップします — 通常は 0.1% 以下の手数料です。
2. **Solana ウォレットからウォレット アドレスをコピーします。** これは 32～44 文字程度の base58 文字列です。**これが入金アドレスです。**
3. **取引所で「出金」に移動します。** SOL を選択します。アドレスを貼り付けます。**Solana** ネットワークを選択します（BEP20 ではなく、ERC20 でもなく — 取引所が複数のネットワークで SOL を提供している場合は、ネイティブ Solana を選択します）。
4. **貼り付けたアドレスの最初の 4 文字と最後の 4 文字が** ウォレット内のものと一致することを確認してください。クリップボードハイジャック マルウェアは実在します。
5. **初めて大きな金額を出金する場合は、小さなテスト額をまず送信してください。**
6. **出金を確認します。** 通常は 2 分以内に到着します。

### 手数料

- **取引所出金手数料：** 通常 0.01～0.02 SOL（約 \$1～\$2）。1 回限りのコストです。
- Solana 側での転送自体の **追加手数料はありません。**

### 注意点

- **間違ったネットワーク。** 一部の取引所では SOL を ERC20 ラッパーとしてリストしています。Solana ウォレットに出金する場合は常にネイティブ Solana を選択してください。そうしないと資金が間違ったチェーンでスタックします。
- **メモ / タグ フィールド。** Solana はメモを使用しません — 取引所がこのフィールドを表示する場合は無視してください。
- **最小出金額。** 取引所には最小額（しばしば 0.1 SOL）があります。注文が小さい場合、出金は失敗します。

## 方法 2：ウォレット内フィアット オンランプ（初心者向け最速）

3 つの主要な Solana ウォレットすべてに組み込みフィアット購入フローがあり、通常は Moonpay、Transak、または Stripe 経由です。

### 手順

1. Solana ウォレットを開きます。
2. 「買う」または「入金 → 暗号資産を購入」をタップします。
3. SOL とローカル通貨での金額を選択します。
4. カード情報（または一部の地域では銀行送金情報）を提供します。
5. 必要に応じて KYC を完了します — 通常は ID の写真 + 自撮り。初回 KYC は 5～30 分かかります。
6. 支払いが確認されたら、SOL がウォレットにクレジットされます。

### 手数料

- **サービス手数料：** Moonpay は ~3.5～4.5%。Transak は ~2.5～4%。Stripe は ~1.5～2% ですが、一部の国でのみ利用可能です。
- **スプレッド：** 提示価格に隠れた追加 0.5～1.5%。
- **総コスト：** この方法では **~3～5%** をスポット価格上で見込んでください。

### 適切な場合

- どこにも暗号資産をまだ保有していない。
- コストより速度とシンプルさを重視する。
- 手数料がドル価値で許容範囲である小額（\$50～\$500）を購入している。

より大きな金額の場合は、取引所を使用してください。

## 方法 3：クロスチェーン ブリッジ

Ethereum、Arbitrum、Base、BNB Chain、Avalanche、または同様のチェーンに資金がある場合は、Solana にブリッジします。

### メイン ブリッジ

- **Wormhole Portal** — [portalbridge.com](https://portalbridge.com). 最も古く、最も広いチェーン サポート。トークンをラップバージョンとしてブリッジします（例：USDC は USDC.wh になります）。そのアセットに CCTP 統合がない限り。ネイティブ アセット配信を確認してください。
- **deBridge** — [debridge.finance](https://debridge.finance). ブリッジ + DEX が 1 つになっています：Ethereum から ETH を送信し、Solana で USDC を受け取ることができます。変換を自動的に処理します。
- **Allbridge Core** — [allbridge.io](https://allbridge.io). ステーブルコインに焦点を当て、ネイティブ配信。
- **Jupiter のブリッジ アグリゲーター** — [jup.ag/bridge](https://jup.ag/bridge) に組み込まれています。複数のブリッジ全体にルーティングします。
- **Circle の CCTP（USDC の場合）** — サポートされているチェーン間（Ethereum、Base、Avalanche、Arbitrum、Solana）でのネイティブ USDC 転送。ラップなし、流動性リスクなし。ほとんどのブリッジ UI を通じてアクセス可能です。

### 手順（汎用）

1. ブリッジ UI にアクセスします。起点チェーン ウォレット（例：MetaMask）と Solana ウォレットの両方を接続します。
2. 起点チェーン、宛先チェーン（Solana）、入力アセット、出力アセットを選択します。
3. クォートを確認します — **手数料、スリッページ、推定時間。**
4. 起点チェーン上のブリッジ コントラクトを承認します（アセットあたり 1 回限り、起点チェーンのガス代がかかります）。
5. ブリッジを実行します。両チェーン上の確認を待ちます。
6. 資金が Solana ウォレットに表示されます。

### 手数料とタイミング

- **ブリッジ手数料：** 一般的に 0.1～0.5%。
- **起点チェーンのガス：** 変動が大きいです。Ethereum メインネット のガスは \$5～\$50 である可能性があります。Base / Arbitrum は数セントです。
- **時間：** ほとんどのルートで 5～30 分。一部の Wormhole ルートは、起点チェーンのファイナリティが遅い場合、より長くかかる可能性があります。

### 注意点

- **ラップ vs ネイティブ USDC（Solana）。** Solana上の「USDC」は、ネイティブ USDC（推奨 — `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`）または Wormhole ラッパー（`A9mUU4qviSctJVPJdBJWkb28deg915LYJKrzQ19ji3FM`）のいずれかです。Raydium では **相互交換できません** — 両方のミント用のプールが存在しますが、流動性はネイティブ ミント上に集中しています。常にネイティブを優先してください。
- **偽のブリッジ サイト。** フィッシングは一般的です。使用するブリッジをブックマークしてください。
- **最小額。** ブリッジはしばしば \$5～\$20 の最小額を設定して、独自の固定コストをカバーします。

## 資金供給後：ウォレットで確認

ウォレットを開いて、以下を確認します：

1. **SOL 残高** が予想値です。
2. **ネットワークが「Solana Mainnet Beta」である**（デフォルト。時々テストネットが誤って有効になっています）。
3. **トランザクション履歴** に受信転送が表示されます。

出金確認から 5 分以内に SOL が表示されない場合は、ウォレットのトランザクション ログを確認してください — 保留中の転送、間違ったアドレス、または間違ったネットワーク エラーの可能性があります。

## 逆方向フロー：オフランプ

Raydium / Solana ウォレットからフィアットまたは別チェーンに資金を取り出すのは、同じ 3 つのパスを逆にしたものです：

- **取引所入金：** SOL（または取引所がリストしている任意の Solana SPL トークン）を取引所入金アドレスに送信します。それをフィアットで売ります。
- **ウォレット内オフランプ：** 一部のウォレットは Moonpay オフランプ（ウォレット → 銀行振込）を提供しています。利用可能性は国によって異なります。
- **ブリッジ：** SOL またはブリッジされた USDC を起点チェーンに送信します。

## SOL 固有の注意点

### 家賃用に予約される最小残高

アドレスを初めて使用するとき、0.00203928 SOL が「家賃」としてアカウント用に保有されます。送信できません。ウォレットが全残高を送信しようとする場合、トランザクションは失敗します — 少なくとも 0.003 SOL を予約として保持してください。

### ラップ SOL（wSOL）

Raydium および多くのプロトコルは「ラップ SOL」を使用します — SOL の SPL トークン形式（ミント `So11111111111111111111111111111111111111112`）。ウォレットは SOL との スワップ時に自動的にラップとアンラップします。時々スワップが失敗して wSOL がウォレットに残ります。ウォレットの UI 経由またはお次の Raydium スワップ経由でアンラップしてください。

### 「ATA を閉じる」回収

保有するすべてのトークンは、開くのに ~0.002 SOL のコストがかかる ATA（Associated Token Account）を占有します。トークン ポジションを完全に閉じ、ATA がもう不要な場合、ウォレットは家賃を回収する「アカウントを閉じる」ボタンを提供する場合があります。無害です — すぐにトークンが再度必要にならないことが確実な場合はこれを実行してください。
