## 30秒での概要

**Raydium は Solana 上の DEX です。** 中央集約型取引所を使用せずに、一方のトークンをもう一方のトークンにスワップできます。使用するには以下が必要です：

1. **Solana ウォレット**（ブラウザ拡張機能またはモバイルアプリ） — 秘密鍵を保持し、トランザクションに署名します。
2. そのウォレット内の **SOL** — トランザクション手数料（スワップあたり約\$0.0005）および購入したいトークンの支払いに使用。
3. [`raydium.io`](https://raydium.io) のブラウザ — 公式 UI。

その他すべて — ウォレットの選択、資金調達、初回トレードの選択 — は上のページで説明されています。

## 最初からチェックすべき危険信号

お金を預ける前に、これらのルールを理解してください：

- **シードフレーズを共有しないでください。** サポートとの相談、「ウォレットを確認」するサイト、DM での共有は厳禁です。シードフレーズを入力する唯一の場所は、ウォレットの初期セットアップ時です。
- **`raydium.io` のみを使用してください。** フィッシングは UI をピクセルパーフェクトでコピーします。本物の URL をブックマークしてください。
- **トークン名ではなく、トークンアドレスを信頼してください。** 誰でも「USDC」という名前のトークンを作成できます。本物の USDC ミントは `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v` です。UI はキュレーションされたミントの横に**確認済み**バッジを表示します。
- **dApp がウォレット全体を「承認」するよう求めることはありません。** 通常のトランザクションは特定の命令に署名します。一括承認は資金流出の原因となります。

詳細は[`getting-started/trust-and-safety`](/ja/getting-started/trust-and-safety)を参照してください。
