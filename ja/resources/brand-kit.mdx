## 名前と大文字表記

| 形式 | 使用方法 |
|---|---|
| **Raydium** | プロトコル名。常にタイトルケース。`raydium`や`RAYDIUM`は使用しません。 |
| **Raydium Protocol** | Raydiumという名前が他のコンテキスト（化学元素やビデオゲームなど）と曖昧になる場合に使用できます。 |

## ロゴ

Raydiumロゴには2つの形があります：

- **ワードマーク** — 完全な「Raydium」タイプロックアップ。ヘッダー、プレスリリース資料、名前が読める必要があるあらゆるコンテキストで使用します。
- **シンボルマーク** — スタンドアロンのモノグラム。アプリアイコン、ファビコン、限定された空間（アバター、ソーシャルプロフィール画像）でワードマークが読める大きさに収まらない場合に使用します。

### ライトおよびダークバージョン

形ごとに2つの仕上げ：

| ファイル | 背景 | 使用する場合 |
|---|---|---|
| `logo-light.svg` | ダーク背景 | ダーク UI サーフェスへのプライマリ配置。 |
| `logo-dark.svg` | ライト背景 | ライト UI サーフェスへのプライマリ配置。 |
| `mark-light.svg` | ダーク背景 | ダークサーフェスのシンボルのみ。 |
| `mark-dark.svg` | ライト背景 | ライトサーフェスのシンボルのみ。 |

すべてのアセットはベクター SVG です。可能な限り SVG を使用してください。最終レンダリングサイズでのみ PNG にラスタライズしてください（スケーリング PNG は避けてください）。

### クリアスペース

ロゴの周囲にRaydium「R」文字の高さに相当するクリアスペースを保つ。このマージン内でロゴをテキスト、アイコン、グラフィックスで混雑させないでください。

### 最小サイズ

ワードマーク：画面上で**120 px**以下、印刷では**20 mm**以下でレンダリングしないでください。
シンボル：画面上で**24 px**以下、印刷では**6 mm**以下でレンダリングしないでください。

これらのしきい値以下では、ロゴの可読性が低下するため、パートナーは代替形式に切り替える必要があります（例えば、スペースが限定されている場合はワードマークをシンボルで置き換えます）。

## カラーパレット

Raydiumのアイデンティティは、状態、深さ、コントラストのためのより暗く、より明るい変数を持つバイオレットプライマリの周りに構築されています。

| 役割 | 16進数 | RGB | 使用方法 |
|---|---|---|---|
| **Primary** | `#8C6EFF` | 140 / 110 / 255 | メインブランドカラー。CTA、リンク、主要なアクセント。 |
| **Light** | `#A88BFF` | 168 / 139 / 255 | ホバー状態、二次的なアクセント、ライトテーマ背景。 |
| **Dark** | `#5E3BE1` | 94 / 59 / 225 | アクティブ/プレス状態、ダークテーマアクセント、深さ。 |

サポートニュートラル：

| 役割 | 16進数 | 使用方法 |
|---|---|---|
| **Foreground (light)** | `#0A0A0F` | ライトサーフェス上の本文テキスト。 |
| **Foreground (dark)** | `#FAFAFA` | ダークサーフェス上の本文テキスト。 |
| **Surface (light)** | `#FFFFFF` | ページ背景、ライトテーマ。 |
| **Surface (dark)** | `#0F0F1A` | ページ背景、ダークテーマ。 |
| **Border** | `#E5E5EB`（ライト）/ `#1F1F2E`（ダーク） | カード分割線ボーダー。 |

ロゴの色を変更しないでください。ワードマークとシンボルは常に、ライトまたはダークのマスター仕上げでレンダリングされます。テイント、グラデーション置き換え、ブランドカラーを通す処理は行いません。

## タイポグラフィ

ドキュメントとサイトは、UI と本文コピーに**Inter**を使用します。見出しは重いウェイト（600～800）で Inter を使用します。コードブロックは**JetBrains Mono**またはその他の等幅フォールバックを使用します。

| スタイル | ファミリー | ウェイト | 使用方法 |
|---|---|---|---|
| Display / H1 | Inter | 800 | ページタイトル。 |
| H2–H4 | Inter | 700 | セクションヘッダー。 |
| Body | Inter | 400 | 段落テキスト。 |
| Emphasis | Inter | 600 | インライン強調、太字テキスト。 |
| Code | JetBrains Mono | 400–500 | インライン`code`およびコードブロック。 |

Inter はオープンライセンスで、[rsms.me/inter](https://rsms.me/inter)から入手できます。

## ロゴの注意事項

**すること：**

- ドキュメント リポジトリの`/logo/`ディレクトリの公式 SVG マスターを使用する。
- クリアスペースと最小サイズを保つ。
- ワードマークとシンボルを両方とも視覚的にバランスの取れた場合のみペアリングしてください（異なるサイズで並べて表示しないでください）。
- 十分なコントラストを持つ背景でレンダリングしてください。ダーク背景にはライトマスター、ライト背景にはダークマスターを使用します。

**しないこと：**

- ロゴの色を変更しません。パープルワードマーク、レインボー、提供されているライト/ダーク以外のモノクロバリエーションはありません。
- アスペクト比を歪めます（ストレッチやスキューなし）。
- ドロップシャドウ、アウトライン、グロー、またはその他の効果を追加しません。
- 対比を保持するソリッドパネルなしで、ビジーな背景画像にロゴを配置しません。
- ロゴをトリミングします（ワードマークから「R」のみをトリミングして表示しないでください。代わりにシンボルマークを使用してください）。
- ロゴを使用して、Raydiumが正式にパートナーシップを結んでいない製品の推奨を意味します。中立的なコンテキストでは「powered by Raydium」または「integrated with Raydium」バッジを使用し、精査されたものとの意味を含めません。

## 「Powered by Raydium」バッジ

統合がルーティングソースまたはリクイディティベニューとして Raydium を使用する場合、シンプルなテキストまたはバッジアトリビューションは歓迎します：

```text
Powered by Raydium
Integrated with Raydium
Liquidity via Raydium
```

これらの表現は事前に承認されています。統合の価格設定、スワップ、またはプールサーフェスの近く（マストヘッドが目立つ位置ではなく）に配置してください。

アグリゲータまたはウォレットとして Raydium を通じてルーティングし、より目立つアトリビューションが必要な場合は、公開する前にDiscord（`#partnerships`）経由でチームに連絡してください。
