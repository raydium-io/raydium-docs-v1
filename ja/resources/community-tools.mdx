## ツール提出方法

このページのエントリーを追加または更新するには：

1. [docs repo](https://github.com/raydium-io) をフォーク（docs ソースは公開 Raydium docs repo に位置 — 正確なパスは README に記載）。
2. `resources/community-tools.mdx` を編集。
3. 以下の内容でプルリクエストを提出：
   - ツール名とリンク。
   - 1 行の説明（何か、どのカテゴリー）。
   - メンテナーの GitHub または X ハンドル（検証用）。
   - 活動に関する注記（最後のリリース・最後の意味あるコミット）。

リストは「公開アクセス可能・実際に動作・保守されている」のバーを満たしているかレビューされます。動作するソフトウェアなし自己宣伝リスティング、またはユーザーセーフティーと矛盾するツール（ドレイナー、偽プール生成）はマージされません。

## 削除ポリシー

ツールは以下の場合、このページから削除されます：

- ホストされたインスタンスが 30 日以上オフライン。
- リポジトリがアーカイブまたは削除。
- メンテナーが 180 日以上沈黙し、ツールが動作停止。
- ツールがユーザーハーム動作を始める（ドレイナースクリプト、欺瞞的 UX など）。

削除は docs メンテナーの裁量で行われます。ツールがオンラインで保守されている場合、再掲載は歓迎します。
