## 製品ごとのスワップ手数料

手数料値は入力額のパーセンテージで、プール計算前に適用されます。

| 製品 | 利用可能なティア | デフォルト/最頻使用 | 分配 |
|---------|-----------------|---------------------|-------|
| **AMM v4** | 0.25%のみ | 0.25% | LP 88%、プロトコル 12%(25の22/3) |
| **CPMM** | 0.01%、0.25%、1% | 0.25% | LP 100%(プロトコル分配は AmmConfig ごとに設定可能、現在 0%) |
| **CLMM** | 0.01%、0.05%、0.25%、1% | 0.25% | LP 88%、プロトコル 12% |
| **LaunchLab(卒業前)** | 1%(デフォルト) | 1% | プール種 50%、LaunchLab トレジャリー 50% |
| **LaunchLab(卒業後 CPMM)** | 0.25% | 0.25% | CPMM 分配を継承 |

### 手数料の分母規約

- **AMM v4**: 手数料は x/10,000 でエンコードされます。"25" は 0.25% を意味します。
- **CPMM/CLMM**: 手数料は x/1,000,000 でエンコードされます。"2500" は 0.25% を意味します。

チェーン上のデータを直接読み取る場合は、プールのプログラムに対して正しい分母を使用してください。

## プロトコル手数料分配

各手数料の「プロトコル」シェア内で、さらなる分配が適用されます:

| 製品 | スワップ手数料のプロトコルシェア | プロトコル内分配 |
|---------|----------------------------|------------------------|
| AMM v4 | 12%(3/25) | トレジャリー 100% |
| CPMM | AmmConfig ごとに設定可能(現在 0%) | 該当なし |
| CLMM | スワップ手数料の 12% | トレジャリー 100% |
| LaunchLab | 卒業前はスワップ手数料の 50% | LaunchLab トレジャリー 100% |

**LP シェア**は自動的に LP トークン保有者に流れます:

- AMM v4 / CPMM: プール準備金に蓄積され、引き出し時に比例配分で分配されます。
- CLMM: `fee_growth_global_X/Y` に蓄積され、`collectFee` で請求可能です。

## 作成コスト

プールをデプロイするために SOL で支払われる一回限りのコスト。

| 製品 | レント コスト | 作成手数料 | 一般的な合計 |
|---------|-----------|--------------|---------------|
| AMM v4 | ~0.15 SOL(複数アカウント) | 0.2 SOL | ~0.35 SOL |
| CPMM | ~0.04 SOL(プール + ボルト + オブザベーション) | 0.15 SOL | ~0.19 SOL |
| CLMM | ~0.075 SOL(プール + オブザベーション + 初期ティック配列) | 0 SOL | ~0.075 SOL |
| Farm v6 | ~0.02 SOL(ファーム状態 + リワード ボルト) | 0.1 SOL(スパム対策) | ~0.12 SOL |
| LaunchLab ローンチ | ~0.015 SOL(ボンディング カーブ アカウント) | 0.1 SOL(LaunchLab 手数料) | ~0.115 SOL |

すべてのレントは回収可能です — プール / ポジション / ファームを閉鎖するとランポートが返金されます。

| 操作 | CU | ベース手数料 | 優先度手数料(10k µL/CU) | 合計ネットワーク |
|-----------|-----|----------|---------------------------|---------------|

| CLMM スワップ(ティックなし) | ~170,000 | 0.000005 SOL | 0.0017 SOL | ~0.0017 SOL |
| CLMM スワップ(4 ティック交差) | ~320,000 | 0.000005 SOL | 0.0032 SOL | ~0.0032 SOL |
| CLMM ポジション を開く | ~280,000 | 0.000005 SOL | 0.0028 SOL | ~0.0028 SOL |
| Farm v6 ステーク | ~130,000 | 0.000005 SOL | 0.0013 SOL | ~0.0013 SOL |

## Token-2022 手数料相互作用

プールの一方または両方が転送手数料を使用する Token-2022 を使用している場合、有効手数料は複合します:

```
pool への net input   = amount_in × (1 − transfer_fee_in)
pool がスワップを請求 = net × swap_fee_rate
user への net output  = out × (1 − transfer_fee_out)
```

例: CPMM 0.25% プール、入力側に 1% 転送手数料、出力側に 0.5% 転送手数料。

```
amount_in        = 1000
net_to_pool      = 1000 × 0.99    = 990
pool swap fee    = 990 × 0.0025   = 2.475 (LP 手数料として保持)
amount_after_fee = 987.525
amount_out (AMM) = 987.525 × ratio, say 987.525 × 0.98 = 967.77
net_out_to_user  = 967.77 × 0.995 = 962.94
```

有効手数料: **~3.7%** (公称 0.25% プール手数料にもかかわらず)

SDK は `getComputeAmountOut` 経由で有効レートを計算します — Token-2022 プールの見積もり時は常に使用してください。

## LaunchLab 仕様

| フェーズ | 手数料 |
|-------|-----|
| ボンディング カーブ トレード(買いまたは売り) | 1%(プール/トレジャリー 50/50 分配) |
| 卒業(一回限り) | カーブ準備金から ~0.04 SOL |
| 卒業後(CPMM プール) | 0.25%(標準 CPMM 手数料) |

トークンをローンチするプロジェクトは、ボンディング カーブ フェーズ中に LP シェアから取られる「クリエーター手数料」オプション(0～0.3%)を含むことが多くあります。ローンチ時に設定可能です。クリエーター手数料は**ボンディング カーブのみ** — ローンチが CPMM プールに卒業すると、結果のプールは標準 CPMM ティアで実行され、クリエーター上乗せはありません。[`products/launchlab/creator-fees`](/ja/products/launchlab/creator-fees)で完全なライフサイクルと、クリエーターが LaunchLab NFT メカニズムを通じて卒業したプールから引き続き稼ぐ方法を参照してください。

## Farm v6 リワード コスト モデル

ファームは厳密には手数料を請求しません — それらはリワード分配です。ただし、運営に SOL がかかります:

| コスト | 一般的 |
|------|---------|
| ファーム作成 | 0.1 SOL スパム対策 + ~0.02 SOL レント |
| リワード ボルトをトップアップ | トークンのみ + ~0.0005 SOL トランザクション手数料 |
| 未使用リワードを取り戻す | ~0.0005 SOL トランザクション手数料 |

ファームは、ファームの寿命中のレント をカバーするのに十分なランポートをリワード ボルトに必要とします。閉鎖すると返金されます。

## チェーン上で現在の値を読み取る方法

### CPMM/CLMM AmmConfig

```ts
const config = await raydium.cpmm.getAmmConfigs();
console.log(config[0]);
// {
//   id: "...",
//   tradeFeeRate: 2500,         // 2500 / 1_000_000 = 0.25%
//   protocolFeeRate: 0,         // 0 / 1_000_000 = 0%
//   fundFeeRate: 0,
//   ...
// }
```

### AMM v4 プール

```ts
const poolInfo = await raydium.liquidity.getPoolInfoFromRpc({ poolId });
console.log(poolInfo.ammFeesNumerator, poolInfo.ammFeesDenominator);
// 25, 10000 → 0.25%
```

### Farm v6 リワード レート

```ts
const farm = await raydium.farm.getFarmInfoFromRpc({ id: farmId });
console.log(farm.farmData.rewardInfos);
// Per-reward: emissionsPerSecond, startTime, endTime
```
