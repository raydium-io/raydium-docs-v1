## Raydium 相容錢包必須具備的功能

錢包是 Raydium 相容的錢包，必須能夠做到以下所有事項：

- 在本地持有 Solana 密鑰對，**絕不**將私鑰傳輸到設備外。
- **簽署版本化（V0）交易。** Raydium 交換使用地址查詢表，需要 V0 支援。只能簽署舊版交易的錢包無法完成 Raydium 交換。
- **透過錢包轉接器標準連接到 dApp。** 這就是讓 raydium.io 上的「連接錢包」按鈕工作的原因。
- **在你批准之前顯示清晰的交易預覽**，包括所呼叫的程式和代幣轉移。
- **支援關聯代幣帳戶**（第一次接收新鑄幣時建立 ATA）。

這些是基本要求。任何現代 Solana 錢包都滿足這些要求。不符合這些要求的錢包簽署 Raydium 交易是不安全的——句號。

## 超越基本要求的重要功能

一旦錢包符合基本要求，區別就歸結為你的工作流程和風險承受能力。決定以下哪些對**你**很重要，並選擇能夠處理它們的錢包：

| 功能 | 何時重要 |
|---|---|
| **硬體錢包配對** | 你持有的金額超過日常金額。軟體錢包成為交易構建器；硬體設備持有金鑰並簽署。大多數信譽良好的 Solana 錢包至少與 Ledger 配對；有些與 Keystone、SafePal 或其他設備配對。 |
| **行動版 dApp 瀏覽器** | 你期望主要從手機使用 Raydium。錢包的應用內瀏覽器是在行動設備上加載 raydium.io 並簽署交易的工具。 |
| **多帳戶管理** | 你將資金分散在熱錢包 / 冷錢包 / 實驗錢包中，希望快速切換而不用費力進行繁瑣操作。 |
| **Token-2022 擴展感知** | 你將與使用 Token-2022 鑄幣的資金池互動（例如轉帳費代幣）。在簽署 UI 中顯示這些擴展的錢包可以幫助你在簽署前發現意外費用。 |
| **優先級費用顯示** | 你在擁塞期間交易，想在批准之前看到錢包設定的計算預算價格。 |
| **Solana 操作 / Blinks 支援** | 你想使用 Raydium 在社群媒體上發布的一鍵交換連結。請參閱 [`user-flows/referrals-and-blinks`](/zh-Hant/user-flows/referrals-and-blinks)。 |
| **開源代碼庫** | 你想能夠審計錢包客戶端（或依賴社群已經這樣做）。 |
| **多鏈帳戶** | 你在 Solana 以外的鏈上持有資產，更傾向於為所有鏈使用一個錢包。驗證多鏈支援對於你實際使用的鏈是否成熟；這會有所不同。 |

## 受歡迎的 Solana 錢包

這四個都是非託管的，支援錢包轉接器，簽署 V0 交易，並與至少一個主流硬體設備配對。除此之外，區別是特定於工作流程的——使用上一部分中的標準來決定哪一個適合你。

### 硬體錢包

硬體錢包與上述軟體錢包之一配對（軟體端成為交易構建器；硬體設備持有金鑰並簽署）。

| 設備 | 類型 | 官方網站 |
|---|---|---|
| Ledger | USB / 藍牙 | [ledger.com](https://www.ledger.com) |
| Keystone | 氣隙式（QR 碼簽署） | [keyst.one](https://keyst.one) |

每個軟體錢包都記錄了它支援的硬體設備；在購買前檢查。

## 熱 / 冷錢包模式

一個無論你選擇哪個錢包都適用的常見做法：

- **熱錢包**持有一小筆你接受可能會丟失的金額——用於日常交換和嘗試新事物。
- **冷錢包**（與硬體設備配對）持有你不希望丟失的任何金額。只有當你在熱錢包中主動需要這些資金時，才移入資金。

Raydium 將每個錢包視為完全獨立的：單獨的頭寸、單獨的 LP 代幣、單獨的農場獎勵、單獨的歷史。切換錢包只需一次點擊。

## 尋找錢包選項的地方

Raydium UI 的「連接錢包」模態列出了所有已透過錢包轉接器與 Raydium 整合的錢包。瀏覽它以查看目前支援的內容。Solana 的官方錢包目錄（[`solana.com/ecosystem/explore?categories=wallet`](https://solana.com/ecosystem/explore?categories=wallet)）是另一個中立的起點。

在你評估錢包時，檢查以下清單：

- [ ] 維護：最近發布、活躍的問題追蹤器、安全漏洞披露渠道。
- [ ] 已審查：由信譽良好的公司發布的審計，或具有活躍社群的開源代碼。
- [ ] V0 交易支援已確認（在轉移真實資金前在開發網上測試）。
- [ ] 如果你計畫使用硬體設備：錢包記錄其支援的設備及配對方式。
- [ ] 你透過專案經過驗證的域名找到了官方下載——**不是**透過贊助廣告或轉發連結。

## 錢包安全基礎知識

無論你選擇哪個錢包：

- **在紙上寫下助記詞。** 將紙張存儲在兩個物理上分開的位置。**絕不**在除了全新安裝上的錢包恢復期間以外的任何地方輸入助記詞。
- **使用密碼管理器**管理錢包的解鎖密碼——並將該密碼與助記詞完全分開。
- **使用專用瀏覽器設定檔**進行加密活動，安裝最少數量的擴展。網路釣魚擴展是最大的洩露向量。
- **對每個錢包連接提示保持懷疑態度。** 閱讀你正在簽署的內容。如果交易轉移代幣的全部而不是特定金額，拒絕它。
- **為 `raydium.io` 建立書籤。** 始終從書籤導航，絕不從搜尋引擎廣告導航。

更多安全指南位於 [`getting-started/trust-and-safety`](/zh-Hant/getting-started/trust-and-safety)。
