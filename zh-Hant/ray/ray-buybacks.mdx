## 流向概覽

```text
交易費
  -> LP 分享
  -> RAY 回購分享
  -> 金庫分享（如適用）
```

對於 CLMM 和 CPMM 池，交易費分配為：

```text
84% 給 LP
12% 給 RAY 回購
4% 給金庫
```

對於標準 AMM v4 池，交易費分配為：

```text
88% 給 LP
12% 給 RAY 回購
```

## 回購分享的含義

回購百分比適用於**交易費**，而不是整個交易金額。

範例：池的交易費為 `0.25%`。回購分享是該費用的 `12%`。

```text
交易金額：        1,000 USDC
交易費：          1,000 * 0.25% = 2.50 USDC
RAY 回購分享：    2.50 * 12% = 0.30 USDC
```

在此範例中，有效回購金額是交易金額的 `0.03%`。

## 為什麼回購在鏈上

Raydium 的回購流向可從公開地址和交易歷史進行審計。

- **餘額可見：** 收集和持有地址可被任何人檢查。
- **轉帳可追蹤：** 傳入的費用轉帳和傳出的路由交易都是公開的。
- **無需私密計帳：** 區塊鏈是流向的帳本。

## 如何驗證回購流向

1. 在 Solscan 或 Solana Explorer 中打開 Raydium 交換交易。
2. 識別池程式和費用轉帳。
3. 將協議端費用流動匹配到相應的收集地址。
4. 檢查來自收集地址的傳出交易。
5. 確認 RAY 在 `DdHDoz...VEZaz` 的累積。
