## Raydium API v3 是什麼？

Raydium API v3 提供 REST 介面，用於查詢 Solana 上 Raydium 交易場所的流動性池、代幣後設資料、農場和鏈上統計數據。錢包、聚合器、DEX 路由器和交易者使用它來取得規範的池數據、價格推送和歷史分析。

該 API 設計用於高吞吐量——回應通常延遲 1–5 分鐘，具體取決於端點敏感性。所有回應都包裝在具有後設資料的一致信封中。

## 基礎主機

| 環境 | URL |
|-------------|-----|
| **主網** | `https://api-v3.raydium.io` |
| **開發網** | `https://api-v3-devnet.raydium.io` |

## 回應信封

所有端點都返回 JSON 信封，包含請求後設資料和回應負載：

### 成功回應

```json
{
  "id": "abc123def456",
  "success": true,
  "data": {
    // 端點特定數據
  }
}
```

### 錯誤回應

```json
{
  "id": "abc123def456",
  "success": false,
  "msg": "query ids type error"
}
```

`id` 欄位是唯一的請求識別符（UUID）。在處理 `data` 之前，務必檢查 `success`。

## 常見注意事項

### `/pools/info/mint` 上的代幣對排序

當你查詢 `/pools/info/mint?mint1=ABC&mint2=XYZ` 時，端點會透過按字母順序排序代幣來內部規範化該對。如果你需要搜尋單個代幣，請完全省略 `mint2`——API 將找到包含該代幣的所有池。

### 池類型篩選值

`poolType` 查詢參數接受特定的枚舉值：
- `all` — 所有池類型
- `concentrated` — 僅 CLMM 池
- `standard` — 僅 AMM（恆定乘積）池
- `allFarm` — 至少有一個活躍農場的池
- `concentratedFarm` — 有農場的濃縮池
- `standardFarm` — 有農場的標準池

輸入錯誤或大小寫不符將被拒絕。

### 池排序欄位值

有效的 `poolSortField` 值為：
- `default`（按 24 小時成交量排序）
- `liquidity`（總鎖定價值）
- `volume24h`、`volume7d`、`volume30d`
- `fee24h`、`fee7d`、`fee30d`
- `apr24h`、`apr7d`、`apr30d`

### 使用 `/pools/info/list-v2` 的分頁

v2 端點通過 `nextPageId` 使用遊標分頁。不透明的令牌在回應中返回，必須原樣傳回。不要嘗試構造你自己的遊標——它編碼 Elasticsearch 狀態。

### 頁面大小預設值和限制

- `/pools/info/list` 和 `/pools/info/mint` 將 `pageSize` 限制在 **1000**。
- `/farms/info/lp` 將 `pageSize` 限制在 **100**。
- `/pools/info/list-v2` 將 `size` 限制在 **1000**。

超過限制的請求將被拒絕並返回錯誤。

### CLMM 動態費用和限單配置

兩個新端點公開 CLMM `CreateCustomizablePool` 流程的校準數據：

- `GET /main/clmm-dynamic-config` — `DynamicFeeConfig` 層級列表（篩選週期、衰退週期、縮減係數、最大分子、動態費用控制）。當 `enableDynamicFee=true` 時，將其中一個的 `id` 傳入 `createCustomizablePool`。
- `GET /main/clmm-limit-order-config` — 按池的限單配置：是否啟用限單、最小輸入金額以及授權結算已填充訂單的離線 `limit_order_admin` 管理員。

每個錢包的限單數據（未結/已填充/已關閉）位於暫存 API；請參見 [`api-reference/temp-api-v1/overview`](/zh-Hant/api-reference/temp-api-v1/overview)。

## 使用範例

**取得主要統計數據：**

```bash
curl -s https://api-v3.raydium.io/main/info | jq .
```

**按總鎖定價值列出前 10 個池：**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/list?poolType=all&poolSortField=liquidity&sortType=desc&pageSize=10&page=1' | jq .
```

**取得 SOL-USDC 池：**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/mint?mint1=So11111111111111111111111111111111111111112&mint2=EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v&poolType=all&poolSortField=liquidity&sortType=desc&pageSize=5&page=1' | jq .
```

**取得代幣價格：**

```bash
curl -s 'https://api-v3.raydium.io/mint/price?mints=So11111111111111111111111111111111111111112,EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v,4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R' | jq .
```

**取得 LP 代幣的農場池：**

```bash
curl -s 'https://api-v3.raydium.io/farms/info/lp?lp=<lp_mint>&pageSize=10&page=1' | jq .
```
