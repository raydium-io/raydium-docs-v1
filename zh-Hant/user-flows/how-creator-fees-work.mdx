## 階段 1 — 遷移前（聯繫曲線）

代幣在聯繫曲線上交易時，每次買賣都會扣除一小部分費用，並在按代幣的金庫中累積。此設定由啟動代幣的**平台**決定。

### 詳細資訊

- **計價方式** — 使用報價代幣（通常為 SOL）。
- **費率** — 由平台在啟動時設定。檢查平台的費用表或代幣的鏈上狀態。
- **隨時可領取** — 前往 raydium.io 上的**投資組合**頁面，隨時領取待領費用。無需等待畢業。
- **畢業時停止** — 曲線畢業後，不再產生新的遷移前費用。金庫中的任何剩餘餘額仍可領取。

### 如何領取

1. 將建立者錢包連接到 [raydium.io](https://raydium.io)。
2. 導覽至**投資組合**。
3. 在你建立的代幣下找到該代幣。
4. 按一下遷移前費用餘額旁的**領取**。

程式化領取：請參閱 [`products/launchlab/creator-fees`](/zh-Hant/products/launchlab/creator-fees) 取得 SDK 指令。

## 階段 2 — 遷移後（費用金鑰 NFT）

代幣的聯繫曲線達到畢業門檻時，流動性會遷移至 Raydium CPMM 資金池。LP 代幣會根據**平台設定**被鑄造並分配：

| 接收者 | Raydium 預設值 | 說明 |
|---|---|---|
| 銷毀（銷毀與賺取） | 90% | 永久鎖定；此份額的交易費用流向銷毀與賺取信託機制。 |
| 建立者（費用金鑰 NFT） | 10% | 費用金鑰 NFT 被鑄造至你的錢包。此 NFT 代表你對資金池 10% 持續 LP 交易費用的領取權。 |
| 平台 | 0%（Raydium 預設值） | 某些第三方平台可能在此領取份額。 |

這些預設值適用於透過 Raydium 自身 LaunchLab UI 啟動的代幣。第三方平台可能設定不同的分配方式 — 啟動前請務必檢查平台頁面。

### 費用金鑰 NFT

費用金鑰 NFT 是在畢業時鑄造至你錢包的非同質化代幣，代表你有權從遷移後資金池的份額中領取 LP 交易費用。

<Warning>
  **除非你刻意想放棄費用領取權，否則不要銷毀或轉移費用金鑰 NFT。** 如果 NFT 遺失或被銷毀，費用領取權會**永久喪失** — 沒有恢復途徑。
</Warning>

- 費用金鑰可以交易或轉移；持有它的人可以領取費用。
- 費用從 CPMM 資金池中的 LP 交易活動持續累積。
- 從投資組合頁面領取，或透過 CPMM 資金池的費用領取指令以程式化方式領取。

### 如何領取遷移後費用

1. 連接持有費用金鑰 NFT 的錢包。
2. 在 raydium.io 導覽至**投資組合**。
3. 在你的部位下找到已畢業代幣的資金池。
4. 按一下**領取**以收集累積的交易費用。

## 如果未啟用遷移後費用

如果平台將建立者費用設為停用，你的代幣會畢業至 **AMM v4 資金池**（舊版遷移路徑），而不是 CPMM。在此模式下：
- 不會鑄造費用金鑰 NFT。
- 沒有程式級別的遷移後建立者費用。
- 某些替代費用領取選項可能透過 CLI 工具存在，取決於啟動的配置方式。

## 費用流程摘要

```
聯繫曲線階段：
  每次買賣 → 小比例 → 建立者金庫（報價代幣）
  隨時從投資組合領取

畢業：
  LP 代幣鑄造 → 按平台設定分配
  ├── X% 銷毀（銷毀與賺取）
  ├── Y% → 建立者作為費用金鑰 NFT      ← 你持有此項
  └── Z% → 平台（如適用）

畢業後：
  每次 CPMM 交換 → LP 費用 → 按 LP 份額分配
  你的費用金鑰 NFT 份額 → 從投資組合領取
```
