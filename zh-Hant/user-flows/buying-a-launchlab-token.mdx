## LaunchLab 代幣是什麼

LaunchLab 代幣以**聯合曲線**發行：一條由智能合約管理的價格曲線，隨著更多代幣被購買而上升，隨著代幣被出售而下降。曲線會持續運作，直到達到畢業門檻（通常為 85 SOL，但建立者可設定不同金額），此時代幣會自動遷移至 Raydium CPMM 或 AMM v4 池，並可透過常規 DEX 路由自由交易。

畢業前，你只能對聯合曲線進行買賣。畢業後，代幣就像 Raydium 上的其他池一樣交易。

## 購買前

### 進行自主研究

- 驗證代幣的 Mint 地址與項目宣布的地址相符——詐騙者經常建立同名的假代幣。
- 檢查 **Mint 權限是否已撤銷**。如果尚未撤銷，建立者可能會鑄造新供應量並稀釋你的持倉。Solflare 等錢包和 Solscan 等區塊瀏覽器會顯示此狀態。
- 查看聯合曲線進度：已售出多少代幣、還剩多少，以及當前價格。這些資訊在代幣的 LaunchLab 頁面上可見。
- 盡可能檢查建立者地址的歷史記錄。

### 了解曲線機制

聯合曲線上的價格**不是固定的**。每次購買都會推高價格；每次出售都會降低價格。在你購買前的大額購買可能會顯著提高你所支付的價格（搶先交易 / 夾心攻擊風險）。使用滑點限制（見下文）。

## 逐步操作指南：透過 UI 購買

1. 前往 [raydium.io/launchlab](https://raydium.io/launchlab) 並找到你想要購買的代幣。
2. 點擊代幣頁面以打開其詳細檢視。
3. 如果尚未連接錢包，請連接你的錢包。
4. 輸入你想要花費的 SOL（或報價代幣）金額。
5. 檢查 UI 中顯示的**預估代幣接收量**和**價格影響**。
6. 如果需要，調整你的**滑點容限**（見下文）。
7. 點擊**購買**並在你的錢包中批准交易。
8. 交易確認後，代幣將出現在你的錢包中。

<Tip>
  在畢業前，你可以使用同一代幣頁面上的**出售**按鈕隨時將代幣賣回給聯合曲線。畢業後，代幣會進入 Raydium 池；請改用標準的交換介面。
</Tip>

## 滑點

<Warning>
  **設定適當的滑點容限。** 聯合曲線代幣可能具有波動性。如果滑點設定過低，當價格在報價和執行之間變動時，你的交易可能會失敗。如果設定過高，你會暴露於夾心機器人的風險下。對於大多數啟動，合理的起點是 1–3%；僅在流動性非常低且接近畢業的啟動項目上，才在必要時提高滑點。
</Warning>

滑點容限是你願意在報價價格和執行價格之間接受的最大價格差異。LaunchLab UI 允許你在確認交易前設定此值。

## 畢業後

當曲線的畢業門檻被任何交易者達到時，代幣會自動遷移至 Raydium AMM 池。此時：

- 聯合曲線關閉；不再有進一步的買賣透過它進行。
- 你的代幣保留在你的錢包中不變。
- 你現在可以在 Raydium 的交換頁面或任何索引 Raydium 池的聚合器上交易代幣。
- 池遷移可能需要幾分鐘才能在聚合器 UI 中出現。

## 風險

| 風險 | 說明 |
|---|---|
| 流動性控制風險 | LP 處理取決於平台和啟動設定。檢查 LP 是否被銷毀、鎖定、分配給建立者或分配給平台。 |
| 未畢業 | 如果曲線永遠無法達到其門檻，交易將僅限於曲線上。你仍然可以將代幣賣回給曲線，但價格可能低於你所支付的價格。 |
| 價格波動 | 聯合曲線本質上是投機性的。價格可能下跌至接近零。 |
| 假代幣 | 詐騙者建立看起來相似的代幣。獨立驗證 Mint 地址。 |
| 智能合約風險 | 沒有完全無風險的鏈上程序。僅使用你準備好承擔風險的資金。 |
