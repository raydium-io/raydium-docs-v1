<Info>
  CLMM 資金池需要兩項永久性選擇：**費用級別**和**初始價格**。你也需要開設一個初始部位，以便資金池有流動性。在簽署前務必謹慎檢查。
</Info>

## 你需要準備什麼

- 一個 Solana 錢包，其中持有：
  - 你想配對的兩個代幣 mint。
  - 大約 0.3 SOL 用於資金池和部位建立的租金及優先費用。
- 一個費用級別決定。
- 明確的初始市場價格和首個價格範圍。

CLMM 支援 SPL Token 和 Token-2022 mint。

## 費用級別

| 費用級別 | 最適合用途 |
|----------|----------|
| 0.01% | 穩定幣對穩定幣配對 |
| 0.05% | 相關配對 |
| 0.25% | 大多數波動性較高的主要配對 |
| 1% | 長尾或異域配對 |

較低的費用級別需要更緊密的價格和更高的交易量。較高的費用級別容許更多波動性，但可能減少交易量。

## UI 逐步導引

在 [raydium.io/clmm/create-pool](https://raydium.io/clmm/create-pool) 上：

1. **選擇代幣配對。** 貼入兩個 mint 地址。
2. **選擇費用級別。** 將 UI 建議作為起點。
3. **設定初始價格。** 使用現有市場價格。如果該配對已有資金池存在，請與其比較。
4. **選擇首個範圍。** 輸入初始部位的下限和上限價格。
5. **輸入存入金額。** UI 根據你的範圍計算所需的代幣比率。
6. **審查並簽署。** 確認該配對、費用級別、初始價格、範圍、存入金額和租金。

## 正確設定初始價格

如果該配對已有其他 CLMM 或 CPMM 資金池存在，使用其當前價格作為起點。如果 Raydium 還沒有該資金池，使用可靠的市場來源。

如果你設定初始價格錯誤，第一筆交易可能會套利該資金池並耗盡定價過低的一側。沒有簡單的重置按鈕。通常你需要提取流動性並重新建立資金池。

## 首個部位：範圍選擇

- **緊密範圍：** 最適合相關配對，但需要主動管理。
- **中等範圍：** 常用於波動性較高的主要配對，如 SOL/USDC。
- **寬廣範圍：** 資本效率較低，但更容易放任不管。
- **完整範圍：** 行為更像 CPMM，但失去了集中式流動性的大部分優勢。

如果當前價格在你的範圍之外，初始部位將超出範圍，直到價格進入範圍才會獲得費用。

## 建立後檢查清單

1. **資金池頁面載入。** 確認資金池地址、配對和費用級別。
2. **部位出現。** 檢查你的 Raydium 投資組合中是否有部位 NFT。
3. **範圍正確。** 確認下限和上限價格。
4. **流動性非零。** 資金池應顯示你選定範圍的活躍流動性。
5. **測試小額交換。** 在公開分享資金池前，驗證路由是否有效。

## 故障排除

### 建立成本高於預期

寬廣的範圍和較低的費用級別可能需要更多的 tick 數組租金。縮小初始範圍或在重試前增加更多 SOL。

### 代幣順序錯誤

UI 會規範化代幣順序，因此顯示的代幣 A/代幣 B 可能與你輸入的順序不同。在簽署前仔細檢查價格報價。

### 費用級別錯誤

建立後無法變更費用級別。如果錯誤，請關閉你的流動性並在正確的級別建立新資金池。

### 零流動性資金池

建立資金池狀態而不開設部位會使資金池無法用於交換。在分享資金池前開設一個初始部位。

### 超出範圍的初始部位

如果你的首個範圍不包括初始價格，該部位將不獲得費用。在當前價格周圍重新開設一個部位。
