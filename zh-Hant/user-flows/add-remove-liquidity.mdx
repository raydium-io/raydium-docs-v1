## 開始前的準備

確保你已備妥：

- 你要存入的代幣對。
- 足夠的 SOL 來支付交易費用和優先費用。
- Raydium.io 上的正確資金池。
- 清楚了解該部位是被動 CPMM 流動性還是主動管理的 CLMM 流動性。

如果你要建立新的資金池，請從[資金池建立](/zh-Hant/user-flows/choosing-a-pool-type)開始。

## 向 CPMM 資金池新增流動性

1. 在 Raydium 上開啟資金池頁面。
2. 點擊**新增流動性**。
3. 輸入一種代幣的金額。另一種代幣的金額會按資金池的當前比率自動填入。
4. 檢視你將收到的 LP 代幣和你的預期池份額。
5. 在 Raydium 中確認，然後在你的錢包中批准。

UI 會建立缺失的代幣帳戶、支持的 Token-2022 轉帳費用鑄幣的帳戶，並在需要時包裝 SOL。

## 從 CPMM 資金池移除流動性

1. 開啟資金池頁面或你的投資組合部位。
2. 點擊**移除流動性**。
3. 輸入 LP 代幣金額，或使用百分比滑塊。
4. 檢視你將收到的兩種代幣金額。
5. 確認並在你的錢包中批准。

提取後可能會留下微量塵埃。這對恆積資金池來說是正常的。

## 開啟或管理 CLMM 部位

CLMM 部位是 NFT。它們有四種常見操作：

- **開啟部位**：使用初始範圍和流動性鑄造新的部位 NFT。
- **增加流動性**：向現有部位添加更多流動性。
- **減少流動性**：在保留 NFT 的同時移除部分流動性。
- **關閉部位**：移除所有流動性、收集待領費用和獎勵，然後銷毀 NFT。

## 開啟 CLMM 部位

1. 開啟 CLMM 資金池頁面。
2. 選擇下限價格和上限價格。
3. 輸入存入金額。
4. 檢視部位是在範圍內、部分在範圍內還是超出範圍。
5. 確認並批准。

你選定的範圍決定了代幣比率。如果當前價格接近下限邊界，部位可能主要需要一種代幣。如果接近上限邊界，則可能主要需要另一種代幣。

## 增加、減少或關閉 CLMM 部位

- **增加**：開啟部位卡片、點擊**增加**、輸入金額、檢視所需的代幣對，然後簽署。
- **減少**：點擊**減少**、選擇百分比或金額、檢視預計的代幣回報，然後簽署。
- **關閉**：將流動性減少至零、收集未領取的費用和獎勵，然後關閉部位。關閉會銷毀部位 NFT 並在適用時返還租金。

如果當前價格在你的選定範圍之外，存入可能是單邊的。UI 會在你簽署單邊或超出範圍的部位之前發出警告。

## 收取農場獎勵

當你的 LP 代幣質押在活躍的農場中時，農場獎勵會累積。收取獎勵會將待領獎勵轉移到你的錢包。它不會提取你的質押。

1. 開啟 [raydium.io/portfolio](https://raydium.io/portfolio)。
2. 找到你的農場部位。
3. 點擊**收取獎勵**。
4. 檢視每個獎勵流的待領金額。
5. 確認並在你的錢包中批准。

農場存入和提取通常會在同一操作中結算待領獎勵。請仔細檢視確認，因為它也可能會收取待領獎勵。

## 收集 CLMM 費用和獎勵

CLMM 部位在兩個資金池鑄幣中賺取掉期費用。費用在部位上累積但不自動複利。

1. 從 [raydium.io/portfolio](https://raydium.io/portfolio) 開啟你的 CLMM 部位。
2. 點擊**收集費用**或**收集獎勵**。
3. 檢視代幣金額。
4. 確認並批准。

如果 UI 提供**全部收集**，請使用它來同時收集費用和獎勵。

若要手動複利，請收集費用、根據需要重新平衡兩種代幣，然後增加部位上的流動性。

## 時機和費用經濟學

每次領取都會產生 Solana 交易費，並且在網路擁堵時可能包含優先費用。

- 對於小額待領金額，請等到獎勵相對於交易費用有意義時再領取。
- 對於較大的部位，每週或每月領取通常就足夠了。
- 除非你在積極複利，否則通常不需要每天領取。

## 驗證

存入 CPMM 後：

- 檢查你的錢包是否顯示 LP 代幣餘額。
- 檢查 Raydium 的投資組合頁面是否顯示該部位。
- 開啟交易簽名並確認資金池存入成功。

開啟或增加 CLMM 部位後：

- 檢查你的投資組合是否顯示部位 NFT。
- 確認下限和上限價格範圍與你選擇的相符。
- 檢查當前價格是否在範圍內。

收取獎勵或費用後：

- 檢查交易狀態是否為**成功**。
- 確認你錢包中的獎勵或費用代幣餘額已增加。
- 刷新 Raydium 的投資組合頁面並確認待領金額重置或減少。

## 問題排解

### 存入比率上的滑點

存入比率在報價時計算。如果你簽署前資金池價格移動，交易可能失敗。重新報價並重試。只在波動性較大的交易對需要時才提高滑點。

### 存入時 Token-2022 轉帳費用

如果 Token-2022 鑄幣收取轉帳費，資金池收到的金額少於你發送的金額。Raydium 會在你簽署前顯示轉帳費用警告。

### 提取時留下的塵埃

四捨五入可能會在資金池或你的錢包中留下微小金額。這是正常的，不表示提取失敗。

### CLMM 部位不是 LP 代幣

CLMM 部位是 NFT。請檢查 Raydium 的投資組合頁面或你的錢包的 NFT 檢視，而不是尋找可替代的 LP 代幣。

### 在獎勵結束前收取獎勵

如果獎勵流在你有待領獎勵時結束，待領金額停止增長。除非農場以不尋常的方式已結束，否則你仍可在結束時間後領取。

### 優先費用太低

如果交易待處理或過期，請使用更高的優先費用設定重試。

### 跨多個農場領取

大量批次領取可能會失敗，如果包含過多部位或獎勵流。請一次領取幾個部位。
