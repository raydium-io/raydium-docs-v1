## 支援的質押資產

Raydium Perps 支援來自主要區塊鏈的存款，包括 SOL、USDC、USDT、ETH、BNB、WBTC、YUSD 和 USD1 等質押資產（如可用）。

具體支援清單、區塊鏈和限額由底層交易平台控制。存款前請以應用程式顯示的資訊為準。

## 全局限額

全局限額適用於整個交易平台，由所有使用者共享。Raydium 參考頁面列出的範例包括：

| 質押資產 | 區塊鏈範例 |
|---|---|
| USDT | Arbitrum、BSC、Ethereum |
| ETH | Arbitrum、Base、Ethereum |
| BNB | BSC |
| USD1 | BSC、Ethereum |
| YUSD | BSC、Ethereum |
| WBTC | Ethereum |
| SOL | Solana |

如果某項資產的全局限額已滿，即使你的錢包低於自己的限額，該資產的新存款可能也無法進行。

## 每位使用者的限額

每位使用者的限額適用於每個帳戶或錢包。Raydium 參考頁面列出 USDT、ETH、SOL、BNB、YUSD、WBTC 和 USD1 等資產的每位使用者限額。

發送資金前請查看存款畫面。如果應用程式顯示的可用存款金額低於你的錢包餘額，你可能達到了每位使用者或全局質押上限。

## 存款流程

1. 開啟 [`perps.raydium.io`](https://perps.raydium.io/perp/PERP_SOL_USDC)。
2. 連接錢包並開啟存款流程。
3. 選擇質押資產和來源區塊鏈。
4. 檢查金額、目標帳戶和任何交易平台警告。
5. 確認存款。
6. 等待帳戶餘額和可用保證金更新。
