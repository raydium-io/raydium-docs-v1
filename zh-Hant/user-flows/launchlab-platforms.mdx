<Info>
  LaunchLab 平台是一個鏈上設定，用於發行介面或社群。平台設定可以影響費用、品牌、創作者費用支援和遷移後的資金池行為。
</Info>

LaunchLab 透過 Platform PDA 支援第三方發行環境。平台可以提供自有品牌的代幣發行體驗，同時使用 LaunchLab 程式基礎設施和 Raydium 資金池遷移。

## 平台可以控制的設定

| 設定 | 重要性 |
|---|---|
| 創作者費用支援 | 決定代幣創作者是否可以賺取遷移前和遷移後的費用。 |
| 平台費用 | 設定平台級別費用收入的流向。 |
| 品牌元資料 | 為平台提供名稱、描述和標誌，可以在應用介面中顯示。 |
| 發行分組 | 讓 UI 可以顯示在同一平台下發行的代幣。 |
| 遷移後資金池設定 | 可以定義平台遷移發行到 CPMM 時的費用設定。 |
| LP 分配 | 可以影響遷移後 LP 代幣如何被銷毀、分配給創作者或分配給平台。 |

## 創作者應該檢查的項目

在任何平台下發行前，請確認：

- 哪個錢包接收平台費用。
- 創作者費用是否啟用。
- 遷移後的流動性是流向 CPMM 還是 AMM v4。
- 畢業後 LP 代幣如何分配。
- 是否會鑄造 Fee Key NFT 以及誰接收它。
- 平台是否有自己的條款、政策或資格要求。

<Warning>
  平台設定可能會實質影響創作者經濟學和使用者預期。在簽署發行交易前，請檢查它們。本頁僅說明機制，不構成法律、財務或投資建議。
</Warning>

## 交易者應該檢查的項目

購買 LaunchLab 代幣時，平台可能會影響費用路徑和遷移後的流動性處理。交易前：

- 檢查代幣的平台和發行設定。
- 若 UI 顯示，請確認遷移後的資金池類型。
- 檢查 LP 是否被銷毀、鎖定或分配。
- 從代幣頁面驗證代幣 mint，而不只是從社交媒體文本。
