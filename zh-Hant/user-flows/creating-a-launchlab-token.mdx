## 兩種建立模式

LaunchLab 提供兩條建立途徑，可從 [raydium.io/launchlab/create](https://raydium.io/launchlab/create) 存取：

| | **JustSendit** | **LaunchLab** |
|---|---|---|
| 設定時間 | 約 1 分鐘 | 約 5 分鐘 |
| 代幣供應量 | 固定預設值 | 完全可配置 |
| SOL 目標 | 85 SOL（固定） | 最少 24 SOL，自訂 |
| 曲線百分比 | 預設值 | 曲線上 51–80% |
| 鎖倉 | 無 | 選用 |
| 迁移後費用 | 預設（創建者流動性池份額 10%） | 可配置 |
| 最適用於 | 快速社群啟動 | 結構化代幣啟動 |

## JustSendit 模式

最快速的啟動方式。所需輸入最少；平台會填入合理的預設值。

### 所需條件

- 已連接的錢包，具備足夠 SOL 以支付交易費用（建立約需 0.05–0.1 SOL）。
- 代幣名稱與交易代碼。
- 符合 UI 要求的圖片或動圖。

### 步驟

1. 前往 [raydium.io/launchlab/create](https://raydium.io/launchlab/create) 並選擇 **JustSendit**。
2. 輸入代幣**名稱**與**交易代碼**。
3. 可選項：新增**說明**與**社交連結**（網站、Twitter/X、Telegram）。
4. 上傳**商標圖片或動圖**。
5. 可選項：輸入**初始購買金額**（以 SOL 計）以在建立時購買代幣。這會改變初始曲線上的價格狀態，簽署前應檢查。
6. 檢查摘要並點擊**建立**。
7. 在錢包中批准交易。

你的代幣立即在結合曲線上啟動。當曲線收集到 85 SOL 後，代幣會升級至 Raydium AMM 池。

## LaunchLab 模式

完全控制的建立模式，供應量、募資目標、曲線配置、鎖倉及費用參數皆可配置。

### 所需條件

- 已連接的錢包，具備約 0.1–0.5 SOL 用於建立與金庫租賃費用。
- 開始前已準備好以下所有輸入項（啟動後無法變更設定）。

### 設定選項

#### 代幣詳情

- **名稱、交易代碼、商標** — 與 JustSendit 相同。
- **說明與社交連結** — 建議提供以提升可發現性。

#### 供應量與曲線配置

- **總供應量** — 要鑄造的代幣總數（例如 1,000,000,000）。
- **曲線配置（%）** — 放置於結合曲線上供公眾購買的總供應量百分比（51–80%）。其餘部分用於啟動迁移後的流動性池。

<Warning>
  曲線配置與總供應量在啟動時**固定**且**無法變更**。簽署前請仔細選擇。
</Warning>

#### SOL 募資目標

- 最少升級門檻為 **24 SOL**。你可以設定更高的金額。
- 這是曲線必須收集的 SOL 金額，達成後代幣會升級至 CPMM 池。
- 設定過高會帶來風險，若需求不足，曲線可能無法升級。

#### 鎖倉（選用）

鎖倉會鎖定部分代幣供應量，並根據時程表線性釋放至指定的錢包。

| 欄位 | 說明 |
|---|---|
| 總鎖定金額 | 跨所有受益者要鎖定的代幣（以總供應量百分比表示，受程式上限影響）。 |
| cliff 期間 | 升級後代幣開始解鎖的等待時間（例如 180 天）。 |
| 解鎖期間 | cliff 後線性解鎖的持續時間（例如 365 天）。 |

鎖定代幣不進入曲線，也不是升級流動性池的一部分。根據上述時程表分別釋放給每位受益者的錢包。

**重要鎖倉規則：**
- Cliff 與解鎖期間在啟動時**固定** — 無法事後變更。
- 若受益者遺失錢包金鑰，鎖定代幣永久無法接觸。若恢復很重要，請使用多籤。
- 升級前未分配給受益者的 `total_locked_amount` 任何部分永久被卡住。請分配完整金額。

如需完整鏈上機制，請見 [LaunchLab 鎖倉](/zh-Hant/products/launchlab/vesting)。

#### 迁移後費用分享

代幣升級時：
- LP 代幣被鑄造至 CPMM 池。
- Raydium 上的預設設定：**90% 的 LP 代幣被銷毀**（通過「銷毀與收益」永久鎖定），**10% 授予創建者**作為費用金鑰 NFT。
- 費用金鑰 NFT 讓你有權從迁移後的池收集 10% 的 LP 交易費用 — 永久有效。

若你想從代幣流動性獲得持續收益並接受費用金鑰 NFT 的託管責任，請啟用此功能。詳情見 [創建者費用如何運作](/zh-Hant/user-flows/how-creator-fees-work)。

### 步驟

1. 在建立頁面選擇 **LaunchLab** 模式。
2. 填寫代幣詳情（名稱、交易代碼、商標、說明、社交媒體）。
3. 設定**總供應量**與**曲線配置百分比**。
4. 設定你的 **SOL 募資目標**（≥ 24 SOL）。
5. 若需要，設定**鎖倉**（設定受益者錢包與時程表）。
6. 若你想要創建者流動性池份額，啟用**迁移後費用分享**。
7. 可選項：輸入初始購買金額。
8. 檢查完整摘要 — 曲線價格範圍、升級條件、LP 分配。
9. 點擊**建立**並批准交易。

## 代幣啟動後

### 監控曲線進展

你的代幣 LaunchLab 頁面顯示實時進度：
- 曲線已售百分比。
- 募集 SOL 與升級目標的比較。
- 目前價格與價格圖表。

LaunchLab 不會干預無法達成升級的啟動。監控曲線進展並與社群進行明確溝通。

### 升級

升級由首筆將曲線推過門檻的交易自動觸發 — 任何用戶都可以觸發（觸發者獲得小額獎勵）。作為創建者，你無需採取任何行動。

升級後：
- 使用收集的 SOL 與剩餘曲線代幣作為初始流動性建立 CPMM 池。
- 若你啟用迁移後費用，**費用金鑰 NFT** 會被鑄造至你的錢包。
- 從 Portfolio 頁面認領迁移前創建者費用。

### 認領迁移前創建者費用

若你的平台在結合曲線階段啟用創建者費用，它們會累積在金庫中。可隨時從 raydium.io 上的 **Portfolio** 頁面認領。

## 應避免的陷阱

| 陷阱 | 解決方式 |
|---|---|
| 設定 SOL 目標過高 | 選擇市場能實際達成的目標。 |
| 在受益者確認前過度分配鎖倉 | 未分配的鎖倉供應在升級後永久無法接觸。 |
| 不小心轉移或銷毀費用金鑰 NFT | 費用認領權永久遺失。將 NFT 保留在建立錢包或你控制的錢包中。 |
| 不進行初始購買 | 沒有早期購買，狙擊機器人可能以接近零成本的價格捕獲所有早期代幣，這會損害社群信任。 |
