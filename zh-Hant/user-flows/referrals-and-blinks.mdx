## 你會獲得什麼

- 一個可分享的 URL，會預先填入 Raydium 交換 UI，其中包含你選擇的目標代幣**以及你的錢包作為推薦費用接收者**。
- 任何人透過該連結進行交換時（在其會話中保持活躍），**輸入量的 1% 會被路由到推薦人的錢包**，作為交換交易的一部分。
- 在 X.com 上分享的連結，若檢視者的錢包已啟用 Actions/Blinks，將渲染為 **Blinks**（Solana Actions）— 在貼文內實現一鍵交換小工具。

## 如何建立連結

1. 開啟 [raydium.io/swap](https://raydium.io/swap) 並連接你的錢包。推薦地址來自所連接的錢包；未連接錢包的情況下連結仍可使用，但無法獲得費用。
2. 選擇你希望他人交換的**目標代幣**。來源側由接收者自行選擇。
3. 點擊交換面板中的分享/連結圖示（右上角）。
4. 複製生成的 URL。在任何地方分享。

URL 格式是穩定的。若你正在建立一個直接生成推薦連結的應用程式。

## Blinks 如何運作

**Blinks（區塊鏈連結）** 是 Solana 基金會的標準，用於在社交平台內將鏈上操作 URL 渲染為互動式 UI。支援 Blinks 的 Solana 錢包會讀取連結元資料，構建預先建立的交易，並向使用者顯示內聯簽署提示。

要求：

- 檢視者的**錢包必須啟用 Actions/Blinks**。這通常是錢包設定中的一個切換開關。
- **平台必須遵守該標準**。X.com 透過其 unfurler 支援；其他平台則不支援，在這種情況下連結會回退為普通超連結。

從推薦人的角度來看，Blink 渲染是自動的 — 不支援 Blinks 的錢包使用相同 URL 時會顯示為普通連結。

## 費用計算

1% 推薦費用是**在 AMM 交易費用之上收取的**，不是從中扣除的。對於總費用為 0.25% 的池進行的交換：

```
使用者發送:        1,000 USDC
AMM 交易費:       2.50 USDC   (分配給 LP + 協議部分)
推薦費用:        10.00 USDC   (1% → 推薦人的錢包)
淨交換額:        987.50 USDC   (然後通過曲線定價)
```

推薦費用適用於由推薦連結建立的 Raydium 交換路由。在推薦路由外的直接池交互不產生推薦費用。

## 停用/退出推薦

沒有「移除推薦」指令。若要進行無推薦費用的交換，只需直接載入 [raydium.io/swap](https://raydium.io/swap)，而不是點擊推薦連結。當 URL 沒有推薦人參數時，UI 不會使用推薦人。

## 常見問題

**推薦人需要簽署任何東西嗎？** 不需要。推薦人的錢包地址是隨連結傳輸的唯一資料；對推薦人的付款是在交換者交易內進行的代幣轉移。

**我可以推薦自己嗎？** 可以，但沒有意義 — 你會支付自己 1%，再減去剛才在更大交易上支付的交易費用。

**如果目標代幣是具有轉移掛鉤的 SPL 會怎樣？** 推薦轉移發生在輸入端（USDC 或來源代幣），不在目標端，所以目標代幣的轉移掛鉤不會影響推薦付款。輸入代幣上的轉移掛鉤可能會完全阻止交換；Raydium 提前拒絕這些。

**如果交換透過多個池路由，推薦費用仍會適用嗎？** 是的 — 1% 是根據原始輸入量計算的，與路由無關。
