## 農場收獲

當你的 LP 代幣質押在活躍農場中時，農場獎勵會不斷累積。收獲會將待領獎勵轉移到你的錢包。這不會提取你的質押。

### UI 操作流程

1. 開啟 [raydium.io/portfolio](https://raydium.io/portfolio)。
2. 找到你的農場頭寸。
3. 點擊**收獲**。
4. 檢視每個獎勵來源的待領金額。
5. 在錢包中確認並批准。

## 存入或提取時同時收獲

農場存入和提取操作通常會在同一筆交易中結算待領獎勵。如果你增加或減少質押，請仔細檢視確認，因為它也可能會收獲待領獎勵。

## CLMM 收集費用

CLMM 頭寸從兩個池幣種中賺取交換費用。費用會累積在頭寸上，但不會自動複利。

1. 從 [raydium.io/portfolio](https://raydium.io/portfolio) 開啟你的 CLMM 頭寸。
2. 點擊**收集費用**。
3. 檢視代幣金額。
4. 確認並批准。

若要手動複利，請收集費用，根據需要重新平衡兩個代幣，然後增加頭寸的流動性。

## CLMM 收集獎勵

某些 CLMM 池除了交換費用外，還有額外的獎勵流。

1. 開啟 CLMM 頭寸。
2. 點擊**收集獎勵**。
3. 檢視獎勵幣種和金額。
4. 確認並批准。

如果 UI 提供**全部收集**，請使用它來同時收集費用和獎勵。

## 時機和費用經濟學

每次領取都會產生 Solana 交易費，在擁擠時期可能還會包含優先費。

- 對於較小的待領金額，應等到獎勵金額相對於交易費更有意義時再領取。
- 對於較大的頭寸，通常每週或每月領取一次就足夠了。
- 除非你在積極複利，否則通常不需要每天都領取。

## 疑難排解

### 獎勵結束前不久收獲

如果獎勵流在你有待領獎勵時結束，待領金額將停止增長。除非農場已以異常方式關閉，否則你仍可在結束時間後領取。

### 轉帳費獎勵幣種

如果獎勵幣種收取轉帳費，你實際收到的金額可能低於待領顯示。確認後檢查錢包餘額變化。

### 首次收獲失敗

刷新頁面並重試。如果你的頭寸是最近建立或遷移的，UI 可能需要初始化或刷新帳戶資料，領取才能正常運作。

### 優先費過低

如果領取交易待處理或過期，請以更高的優先費設定重試。

### 跨多個農場領取

大批量領取如果包含太多頭寸或獎勵流，可能會失敗。請一次領取幾個頭寸。

## 驗證

收獲或收集後：

- 檢查交易狀態是否為**成功**。
- 確認獎勵或費用代幣餘額在你的錢包中增加。
- 刷新 Raydium 的投資組合頁面，並確認待領金額已重置或減少。
