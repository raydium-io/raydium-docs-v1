## 各產品交換費用

費用值為輸入金額的百分比，在池數學計算之前應用。

| 產品 | 可用等級 | 預設／最常用 | 分配 |
|---------|-----------------|---------------------|-------|
| **AMM v4** | 0.25%（僅此一種） | 0.25% | 88% LP，12% 協議（25 的 22/3） |
| **CPMM** | 0.01%、0.25%、1% | 0.25% | 100% LP（協議份額可按 AmmConfig 設定，目前為 0%） |
| **CLMM** | 0.01%、0.05%、0.25%、1% | 0.25% | 88% LP，12% 協議 |
| **LaunchLab（畢業前）** | 1%（預設） | 1% | 50% 用於池種子，50% 用於 LaunchLab 金庫 |
| **LaunchLab（畢業後 CPMM）** | 0.25% | 0.25% | 沿用 CPMM 分配 |

### 費用分母約定

- **AMM v4**：費用編碼為 x/10,000。「25」表示 0.25%。
- **CPMM/CLMM**：費用編碼為 x/1,000,000。「2500」表示 0.25%。

直接讀取鏈上數據時，請對池的程序使用正確的分母。

## Token-2022 費用互動

當池的一側或兩側使用帶轉帳費用的 Token-2022 時，有效費用會疊加：

```
net input to pool   = amount_in × (1 − transfer_fee_in)
pool charges swap   = net × swap_fee_rate
net output to user  = out × (1 − transfer_fee_out)
```

示例：CPMM 0.25% 池，輸入側有 1% 轉帳費用，輸出側有 0.5% 轉帳費用。

```
amount_in        = 1000
net_to_pool      = 1000 × 0.99    = 990
pool swap fee    = 990 × 0.0025   = 2.475 (kept as LP fee)
amount_after_fee = 987.525
amount_out (AMM) = 987.525 × ratio, say 987.525 × 0.98 = 967.77
net_out_to_user  = 967.77 × 0.995 = 962.94
```

有效費用：儘管名義池費用為 0.25%，但實際約為 **~3.7%**。

SDK 通過 `getComputeAmountOut` 計算有效率 — 引用 Token-2022 池時請務必使用它。

## LaunchLab 特定項目

| 階段 | 費用 |
|-------|-----|
| 結合曲線交易（買或賣） | 1%（50/50 池／金庫分配） |
| 畢業（一次性） | ~0.04 SOL 來自曲線儲備 |
| 畢業後（CPMM 池） | 0.25%（標準 CPMM 費用） |

啟動代幣的項目通常包含「創建者費用」選項（0–0.3%），在結合曲線階段從 LP 份額中提取；在啟動時可設定。創建者費用**僅限結合曲線** — 一旦啟動畢業成為 CPMM 池，所得池以標準 CPMM 層級運行，無創建者額外費用。詳見 [`products/launchlab/creator-fees`](/zh-Hant/products/launchlab/creator-fees) 了解完整生命週期以及創建者如何通過 LaunchLab NFT 機制在畢業後的池中持續獲利。

## Farm v6 獎勵成本模型

農場本身不收費 — 它們是獎勵分配。但它們運行時需要 SOL：

| 成本 | 典型 |
|------|---------|
| 農場建立 | 0.1 SOL 反垃圾郵件 + ~0.02 SOL 租金 |
| 補充獎勵金庫 | 只是代幣 + ~0.0005 SOL tx |
| 收回未使用獎勵 | ~0.0005 SOL tx |

農場還需要獎勵金庫中足夠的 lamports 來覆蓋農場生命週期內的租金；關閉時返回。

const config = await raydium.cpmm.getAmmConfigs();
console.log(config[0]);
// {
//   id: "...",
//   tradeFeeRate: 2500,         // 2500 / 1_000_000 = 0.25%
//   protocolFeeRate: 0,         // 0 / 1_000_000 = 0%

//   ...

```ts
const poolInfo = await raydium.liquidity.getPoolInfoFromRpc({ poolId });
console.log(poolInfo.ammFeesNumerator, poolInfo.ammFeesDenominator);
// 25, 10000 → 0.25%
```

const farm = await raydium.farm.getFarmInfoFromRpc({ id: farmId });
