## 前置要求

執行一次：

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

你需要：

- **Node 18+** 並支援 ESM（`package.json` 中有 `"type": "module"` 或使用 `.mjs` 副檔名）。
- **一個 Solana 金鑰對**，位於 `~/.config/solana/id.json`（或其他任何地方──設定 `KEYPAIR`）。
- **一個有資金的錢包。** 約 0.05 SOL 足以執行本章的任何一個腳本。
- **一個 RPC URL。** 免費的公開 RPC 適合測試；若超過簡單的檢測，請使用付費提供商。

每次開啟新 Shell 時設定一次：

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **集群。** 三個腳本預設均使用主網。若要在開發網執行，請設定 `RPC_URL=https://api.devnet.solana.com` 並在腳本內將 `cluster: "mainnet"` 改為 `cluster: "devnet"`。注意開發網上的資金池種子流動性稀少；你需要自己建立測試代幣。
</Info>

## 完成本章後

每個腳本的結尾都會輸出最小化結果（資金池 ID、交易簽名），以及一行指向 [SDK & API](/zh-Hant/sdk-api) 章節中對應操作的下一個頁面的連結。當你準備好撰寫自己的整合時：

- **後端和機器人** → [Trade API](/zh-Hant/sdk-api/trade-api)。
- **前端和完全控制應用** → [TypeScript SDK](/zh-Hant/sdk-api/typescript-sdk)。
- **鏈上組合** → [Rust CPI](/zh-Hant/sdk-api/rust-cpi)。
- **在它們之間做選擇** → [SDK & API](/zh-Hant/sdk-api) 頂部的比較表。
