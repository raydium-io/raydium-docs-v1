## 本頁內容涵蓋

工具若符合以下條件即可列入本頁：(a) 可公開存取、(b) 在列表時確實能與 Raydium 的主網程式運作、(c) 有人維護 — 即專案在過去 90 天內有提交、發布或可見的參與。

下方分類是初步的分類方式；如果你有不符合任何分類的工具，請提交給我們，我們會新增新分類。

## 提交工具

要在本頁新增或更新條目：

1. Fork [文件儲存庫](https://github.com/raydium-io)（文件源位於公開 Raydium 文件儲存庫 — 確切路徑在 README 中）。
2. 編輯 `resources/community-tools.mdx`。
3. 提交包含以下內容的 PR：
   - 工具名稱和連結。
   - 一行說明（它是什麼、屬於哪個分類）。
   - 維護者的 GitHub 或 X 帳號（用於驗證）。
   - 有關活動的備註（最後發布 / 最後有意義的提交）。

列表會根據「可公開存取 / 確實可運作 / 有人維護」的標準進行審查。沒有運作軟體的自我宣傳列表，或主要目的與使用者安全衝突的工具（資金抽取工具、假資金池生成器）不會被合併。

## 移除政策

工具會在以下情況下從本頁移除：

- 託管實例已離線超過 30 天。
- 儲存庫已被封存或刪除。
- 維護者沉默超過 180 天且工具停止運作。
- 工具開始從事危害使用者的行為（資金抽取腳本、欺騙性 UX 等）。

移除由文件維護者決定；一旦工具重新線上並得到維護，歡迎恢復。

## 安全提示

列在此的工具如果要求提供你的錢包種子短語、要求你將資金發送到託管地址，或承諾保證收益，就**不是**你應該使用的工具。列出在這裡是一個低標準 — 它不保證安全性。在簽署交易之前，始終檢查你授權的內容。
