## 名稱和大小寫

| 形式 | 用途 |
|---|---|
| **Raydium** | 協議名稱。始終首字大寫。不得寫成 `raydium` 或 `RAYDIUM`。 |
| **Raydium Protocol** | 在需要與其他語境中的 Raydium 名稱區分時可使用（Raydium 也是化學物質和電玩遊戲名稱）。 |
| **\$RAY** 或 **RAY** | Raydium 的原生 SPL 代幣。在行銷文案中使用美元符號前綴，在技術或表格語境中寫成 `RAY`。 |
| **Raydium AMM v4** / **Raydium CPMM** / **Raydium CLMM** / **LaunchLab** | 產品名稱。LaunchLab 為一個詞，兩個首字母大寫。 |
| **Burn & Earn** | 一個構念，使用連接號，兩個名詞皆首字大寫。 |

不得寫成 `Raydium DEX`、`Ray protocol` 或 `Raydiums`（無複數形）。該協議是單數名詞。

## 標誌

Raydium 標誌有兩種形式：

- **字標** — 完整的「Raydium」文字組合。在標題、新聞稿和需要名稱清晰可讀的任何語境中使用。
- **符號標記** — 獨立單字。在應用程式圖示、網頁圖示和受限空間（頭像、社群媒體個人資料圖片）中使用，因為字標在這些位置可能太小而難以辨讀。

### 淺色和深色版本

每種形式各有兩個版本：

| 檔案 | 背景 | 使用時機 |
|---|---|---|
| `logo-light.svg` | 深色背景 | 深色 UI 表面的主要放置位置。 |
| `logo-dark.svg` | 淺色背景 | 淺色 UI 表面的主要放置位置。 |
| `mark-light.svg` | 深色背景 | 深色表面上的符號專用版本。 |
| `mark-dark.svg` | 淺色背景 | 淺色表面上的符號專用版本。 |

所有資源皆為向量 SVG 格式。儘可能使用 SVG。僅在最終渲染大小時才光柵化為 PNG（避免縮放 PNG）。

### 清晰空間

在標誌周圍保持等於 Raydium「R」字符高度的清晰空間。不得在此邊界內用文字、圖示或圖形擁擠標誌。

### 最小尺寸

字標：在螢幕上不得低於 **120 像素**寬，印刷品上不得低於 **20 毫米**。
符號：在螢幕上不得低於 **24 像素**寬，印刷品上不得低於 **6 毫米**。

低於這些閾值，標誌會失去易讀性，合作夥伴應改為使用替代形式（例如，空間受限時，用符號代替字標）。

## 色彩調色盤

Raydium 視覺識別以紫羅蘭色為主色調，並搭配更深和更淺的變體來表現狀態、層次和對比。

| 角色 | 十六進位 | RGB | 用途 |
|---|---|---|---|
| **主色** | `#8C6EFF` | 140 / 110 / 255 | 主品牌色。行動呼籲、連結、關鍵重點。 |
| **淺色** | `#A88BFF` | 168 / 139 / 255 | 滑鼠懸停狀態、次要重點、淺主題背景。 |
| **深色** | `#5E3BE1` | 94 / 59 / 225 | 啟用/按下狀態、深主題重點、深度。 |

支援中性色：

| 角色 | 十六進位 | 用途 |
|---|---|---|
| **前景色（淺色）** | `#0A0A0F` | 淺色表面上的正文。 |
| **前景色（深色）** | `#FAFAFA` | 深色表面上的正文。 |
| **表面（淺色）** | `#FFFFFF` | 頁面背景，淺色主題。 |
| **表面（深色）** | `#0F0F1A` | 頁面背景，深色主題。 |
| **邊框** | `#E5E5EB`（淺色）/ `#1F1F2E`（深色） | 卡片和分隔線邊框。 |

不得重新著色標誌。字標和符號始終以淺色或深色主版本呈現 — 絕不著色、絕不使用漸層替代、絕不套用品牌色。

## 排版

文件和網站對 UI 和正文使用 **Inter** 字體。標題使用 Inter 粗體重量（600–800）。代碼區塊使用 **JetBrains Mono** 或任何等寬備選字體。

| 風格 | 字族 | 字重 | 用途 |
|---|---|---|---|
| 顯示 / H1 | Inter | 800 | 頁面標題。 |
| H2–H4 | Inter | 700 | 章節標題。 |
| 正文 | Inter | 400 | 段落文字。 |
| 強調 | Inter | 600 | 行內強調、加粗文字。 |
| 代碼 | JetBrains Mono | 400–500 | 行內 `code` 和代碼區塊。 |

Inter 為開放授權字體，可從 [rsms.me/inter](https://rsms.me/inter) 下載。

## 標誌的注意事項

**應做：**

- 使用文件儲存庫 `/logo/` 資料夾中的官方 SVG 主檔案。
- 保持清晰空間和最小尺寸。
- 僅在字標和符號視覺平衡時才配對使用（不得並排顯示尺寸不一的版本）。
- 在具有足夠對比度的背景上呈現 — 深色背景使用淺色版本，淺色背景使用深色版本。

**禁止：**

- 重新著色標誌。不得使用紫色字標、彩虹色或除提供的淺色/深色外的單色變體。
- 扭曲縱橫比（無拉伸、無傾斜）。
- 添加陰影、邊框、發光或其他效果。
- 在繁忙的背景圖像上放置標誌，而不在下方放置對比度保護的純色面板。
- 裁切標誌（不得僅顯示從字標裁切出的「R」— 應改用符號標記）。
- 使用標誌暗示對 Raydium 未正式合作的產品的認可。在中立語境中使用「由 Raydium 提供」或「與 Raydium 整合」徽章，不得暗示任何審查。

## 「由 Raydium 提供」徽章

如果你的整合使用 Raydium 作為路由來源或流動性場所，簡單的文字或徽章歸屬是歡迎的：

```text
Powered by Raydium
Integrated with Raydium
Liquidity via Raydium
```

這些措辭已預先核准。放置在整合的定價、交換或資金池表面附近 — 不得放在版頭顯著位置，以免暗示正式合作。

如果你是聚合器或錢包透過 Raydium 進行路由，且想要更顯著的歸屬，請在發佈前透過 Discord（`#partnerships`）與團隊聯絡。
