## Tài liệu này dành cho ai

Các nhà phát triển tích hợp với Raydium từ bên ngoài chuỗi (TS/JS, Python, backend) hoặc từ một chương trình Solana khác (Rust CPI). Cũng bao gồm các agent AI và những người tích hợp công cụ lập mã — xem [Tích hợp AI](/vi/sdk-api/ai-integration).

<Info>
  **Muốn chạy thử trước?** Ba đoạn mã sẵn sàng chạy — swap, triển khai pool CPMM, triển khai pool CLMM — nằm trong [Bắt đầu nhanh](/vi/quick-start). Chạy một giao dịch thực tế trong vòng năm phút, sau đó quay lại đây để xem tham chiếu đầy đủ.
</Info>

## Chọn con đường tích hợp của bạn

Năm con đường để lựa chọn. Con đường phù hợp tùy thuộc vào thứ đang gọi Raydium và mức độ kiểm soát bạn cần.

| Con đường | Phù hợp cho | Độ trễ | Logic tùy chỉnh | Ngôn ngữ |
|---|---|---|---|---|
| [Trade API](/vi/sdk-api/trade-api) | Backend, bot, Blink — bất kỳ nơi nào bạn muốn máy chủ xây dựng tx | Thấp nhất | Giới hạn (các route được cài sẵn) | Bất kỳ |
| [TypeScript SDK](/vi/sdk-api/typescript-sdk) | Ứng dụng Node/TS và frontend cần truy cập builder đầy đủ | Trung bình | Đầy đủ | TS / JS |
| [Tích hợp Python](/vi/sdk-api/python-integration) | Backend Python và pipeline ML | Trung bình | Đầy đủ (xây dựng instr thủ công) | Python |
| [Rust CPI](/vi/sdk-api/rust-cpi) | Chương trình tổng hợp gọi Raydium từ on-chain | N/A (on-chain) | Đầy đủ + có thể tổng hợp | Rust |
| [Anchor IDL](/vi/sdk-api/anchor-idl) | Codegen cho các ngôn ngữ không có hỗ trợ chính thức (Go, Kotlin, Swift, ...) | Thay đổi | Đầy đủ | Bất kỳ với công cụ Anchor codegen |

Nếu bạn không chắc chắn: backend và bot → **Trade API**; frontend và ứng dụng toàn quyền → **TypeScript SDK**; tổng hợp on-chain → **Rust CPI**.

## Nội dung chương

  <Card title="TypeScript SDK" icon="js" href="/vi/sdk-api/typescript-sdk">
    `@raydium-io/raydium-sdk-v2`: cài đặt, khởi tạo, các module cốt lõi (pool, farm, launch), transaction builder, những cạm bẫy thường gặp.
  </Card>
  <Card title="REST API" icon="cloud" href="/vi/sdk-api/rest-api">
    Các endpoint công khai cho thông tin pool, thông tin mint, giá cả và giao dịch được xây dựng bởi máy chủ. Giới hạn tốc độ, hướng dẫn bộ nhớ cache, kiểm soát phiên bản.
  </Card>
  <Card title="Trade API" icon="right-left" href="/vi/sdk-api/trade-api">
    Giao dịch swap được xây dựng bởi máy chủ: báo giá, xây dựng, ký, gửi. Mô hình tích hợp không cần SDK cho backend, bot và Blink.
  </Card>
  <Card title="Anchor IDL" icon="file-code" href="/vi/sdk-api/anchor-idl">
    Nơi tìm IDL của mỗi chương trình, cách tái tạo client từ nó, điều cần lưu ý khi IDL được cập nhật.
  </Card>
  <Card title="Rust CPI" icon="rust" href="/vi/sdk-api/rust-cpi">
    Các mô hình gọi các chương trình Raydium từ một chương trình Solana khác — xây dựng danh sách tài khoản, seed người ký, xử lý lỗi.
  </Card>
  <Card title="Tích hợp Python" icon="python" href="/vi/sdk-api/python-integration">
    `solders` + `solana-py` công thức nấu ăn để xây dựng swap/pool op của Raydium từ Python. Hữu ích cho backend và bot.
  </Card>
  <Card title="Tích hợp AI" icon="robot" href="/vi/sdk-api/ai-integration">
    Máy chủ MCP, chỉ mục `llms.txt`, menu sao chép mỗi trang, các tệp ngữ cảnh được xây dựng sẵn cho Claude Code / Cursor / Windsurf và các agent lập mã khác.
  </Card>
</CardGroup>

## Kỷ luật phiên bản

Mỗi mẫu mã trên mỗi trang trong chương này có một banner ở trên cùng:

```
SDK:       @raydium-io/raydium-sdk-v2 vX.Y.Z
Program:   see /reference/program-addresses
Cluster:   mainnet-beta
Verified:  YYYY-MM-DD
```

Nếu bạn chỉnh sửa một mẫu mã, hãy cập nhật banner.
