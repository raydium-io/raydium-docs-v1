## Điều kiện tiên quyết

Chạy điều này một lần:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Bạn cần:

- **Node 18+** với hỗ trợ ESM (`"type": "module"` trong `package.json` hoặc phần mở rộng `.mjs`).
- **Một cặp khóa Solana** tại `~/.config/solana/id.json` (hoặc bất kỳ đâu — đặt `KEYPAIR`).
- **Ví được cấp vốn.** ~0.05 SOL đủ cho bất kỳ script nào trong chương này.
- **Một URL RPC.** RPC công cộng miễn phí hoạt động để kiểm tra; để làm bất kỳ điều gì hơn là kiểm tra cơ bản, sử dụng nhà cung cấp trả phí.

Đặt chúng một lần trên shell:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Cả ba script mặc định sử dụng mainnet. Để chạy trên devnet, đặt `RPC_URL=https://api.devnet.solana.com` và thay đổi `cluster: "mainnet"` thành `cluster: "devnet"` bên trong script. Hãy lưu ý rằng thanh khoản hạt giống pool trên devnet rất ít; hãy mong đợi việc tạo ra các token kiểm tra của riêng bạn.
</Info>

## Sau chương này

Mỗi script kết thúc với một đầu ra tối thiểu (ID pool, chữ ký tx) và một dòng duy nhất trỏ đến trang tiếp theo trong chương [SDK & API](/vi/sdk-api) cho hoạt động mà nó thực hiện. Khi bạn sẵn sàng viết tích hợp riêng của mình:

- **Backends và bots** → [Trade API](/vi/sdk-api/trade-api).
- **Frontends và ứng dụng full-control** → [TypeScript SDK](/vi/sdk-api/typescript-sdk).
- **Thành phần trên chuỗi** → [Rust CPI](/vi/sdk-api/rust-cpi).
- **Chọn giữa chúng** → bảng so sánh ở đầu [SDK & API](/vi/sdk-api).
