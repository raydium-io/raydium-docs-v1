Đây là REST API gốc của Raydium để truy xuất dữ liệu pool, farm, SDK và token. API này vẫn phục vụ lưu lượng production và các SDK client, nhưng hiện được coi là phiên bản cũ.

## Host

| Môi trường | Host              |
|-----------|------------------|
| Mainnet   | `api.raydium.io`  |
| Devnet    | Không có sẵn      |

## Thông báo chuyển đổi

**Đối với các tích hợp mới, hãy sử dụng [API v3](/vi/api-reference/api-v3/overview).** Các endpoint v3 cung cấp:

- Hiệu suất và caching tốt hơn
- Schema phản hồi được đơn giản hóa
- Tổ chức endpoint rõ ràng hơn
- Phát triển tính năng liên tục

## Tài liệu liên quan

Xem [API v3](/vi/api-reference/api-v3/overview) để tìm các endpoint thay thế được khuyến nghị. Xem [CLMM](/vi/products/clmm) và [CPMM](/vi/products/cpmm) để hiểu về cấu trúc pool và thuật ngữ.
