Mint API cung cấp quyền truy cập toàn diện vào tạo token, khám phá và quản lý LaunchLab. Bạn có thể truy vấn các mint theo tên, ký hiệu hoặc người tạo, lấy dữ liệu bảng xếp hạng và gửi yêu cầu tạo mint mới.

## API Endpoints

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Xác thực

Các thao tác đọc không yêu cầu xác thực. Các thao tác ghi (tạo mint) yêu cầu header `ray-token` được cấp bởi Auth API sau khi xác minh chữ ký ví.

[Tài liệu Auth API](/vi/api-reference/launch-auth-v1/overview)

## Các trường hợp sử dụng

- **Tìm kiếm và khám phá** token theo tên hoặc ký hiệu
- **Xem bảng xếp hạng** (các mint hàng đầu theo vốn hóa thị trường, mới nhất, được quan tâm nhất)
- **Lấy thống kê người dùng** (khối lượng tạo và giao dịch theo thời gian)
- **Truy vấn lịch vesting** để biết thông tin mở khóa token
- **Gửi yêu cầu tạo mint** với siêu dữ liệu và hình ảnh
- **Theo dõi phần thưởng referrer** từ các chương trình giới thiệu nền tảng
