## Raydium API v3 là gì?

Raydium API v3 cung cấp giao diện REST để truy vấn các liquidity pool, dữ liệu token, farm và thống kê chain trên các sàn giao dịch của Raydium trên Solana. Nó được sử dụng bởi các ví, aggregator, DEX router và người giao dịch để lấy dữ liệu pool chính thức, nguồn giá và phân tích lịch sử.

API được thiết kế cho thông lượng cao — các phản hồi thường cũ từ 1–5 phút tùy thuộc vào độ nhạy của endpoint. Tất cả các phản hồi được bao bọc trong một phong bì đơn nhất với metadata.

## Các host cơ sở

| Môi trường | URL |
|-------------|-----|
| **Mainnet** | `https://api-v3.raydium.io` |
| **Devnet** | `https://api-v3-devnet.raydium.io` |

## Phong bì phản hồi

Tất cả các endpoint trả về một phong bì JSON với metadata của yêu cầu và dữ liệu phản hồi:

### Phản hồi thành công

```json
{
  "id": "abc123def456",
  "success": true,
  "data": {
    // Dữ liệu cụ thể của endpoint
  }
}
```

### Phản hồi lỗi

```json
{
  "id": "abc123def456",
  "success": false,
  "msg": "query ids type error"
}
```

Trường `id` là một định danh yêu cầu duy nhất (UUID). Luôn kiểm tra `success` trước khi xử lý `data`.

## Các vấn đề thường gặp

### Thứ tự cặp mint trên `/pools/info/mint`

Khi bạn truy vấn `/pools/info/mint?mint1=ABC&mint2=XYZ`, endpoint sẽ chuẩn hóa nội bộ cặp bằng cách sắp xếp mint theo thứ tự chữ cái. Nếu bạn cần tìm kiếm một mint duy nhất, hãy bỏ hoàn toàn `mint2` — API sẽ tìm tất cả các pool chứa mint đó.

### Các giá trị bộ lọc loại pool

Tham số truy vấn `poolType` chấp nhận các giá trị enum cụ thể:
- `all` — tất cả các loại pool
- `concentrated` — chỉ các pool CLMM
- `standard` — chỉ các pool AMM (constant-product)
- `allFarm` — các pool có ít nhất một farm hoạt động
- `concentratedFarm` — các pool concentrated có farm
- `standardFarm` — các pool standard có farm

Lỗi chính tả hoặc không khớp với chữ hoa/thường sẽ bị từ chối.

### Các giá trị trường sắp xếp pool

Các giá trị `poolSortField` hợp lệ là:
- `default` (sắp xếp theo khối lượng 24h)
- `liquidity` (TVL)
- `volume24h`, `volume7d`, `volume30d`
- `fee24h`, `fee7d`, `fee30d`
- `apr24h`, `apr7d`, `apr30d`

### Phân trang với `/pools/info/list-v2`

Endpoint v2 sử dụng phân trang dựa trên con trỏ thông qua `nextPageId`. Token mờ được trả về trong phản hồi và phải được truyền lại nguyên văn. Đừng cố tạo con trỏ của riêng bạn — nó mã hóa trạng thái Elasticsearch.

### Các giới hạn và giá trị mặc định về kích thước trang

- `/pools/info/list` và `/pools/info/mint` giới hạn `pageSize` ở **1000**.
- `/farms/info/lp` giới hạn `pageSize` ở **100**.
- `/pools/info/list-v2` giới hạn `size` ở **1000**.

Các yêu cầu vượt quá giới hạn sẽ bị từ chối với một lỗi.

### Các cấu hình CLMM dynamic-fee và limit-order

Hai endpoint mới cung cấp dữ liệu calibration cho luồng CLMM `CreateCustomizablePool`:

- `GET /main/clmm-dynamic-config` — danh sách các tầng `DynamicFeeConfig` (filter period, decay period, reduction factor, max numerator, dynamic-fee control). Chuyển `id` của một trong những cái này vào `createCustomizablePool` khi `enableDynamicFee=true`.
- `GET /main/clmm-limit-order-config` — cấu hình limit-order cho mỗi pool: cho biết các limit order có được bật hay không, số tiền nhập tối thiểu và `limit_order_admin` off-chain được ủy quyền để settle các đơn hàng đã điền.

Dữ liệu limit-order cho mỗi ví (mở / điền / đóng) nằm trên Temp API; xem [`api-reference/temp-api-v1/overview`](/vi/api-reference/temp-api-v1/overview).

## Ví dụ sử dụng

**Lấy thống kê chính:**

```bash
curl -s https://api-v3.raydium.io/main/info | jq .
```

**Liệt kê 10 pool hàng đầu theo TVL:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/list?poolType=all&poolSortField=liquidity&sortType=desc&pageSize=10&page=1' | jq .
```

**Lấy các pool SOL-USDC:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/mint?mint1=So11111111111111111111111111111111111111112&mint2=EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v&poolType=all&poolSortField=liquidity&sortType=desc&pageSize=5&page=1' | jq .
```

**Lấy giá mint:**

```bash
curl -s 'https://api-v3.raydium.io/mint/price?mints=So11111111111111111111111111111111111111112,EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v,4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R' | jq .
```

**Lấy các pool farm cho một token LP:**

```bash
curl -s 'https://api-v3.raydium.io/farms/info/lp?lp=<lp_mint>&pageSize=10&page=1' | jq .
```
