API Forum cho phép thảo luận và tương tác cộng đồng xung quanh các pool LaunchLab. Bạn có thể đăng bình luận với tệp đính kèm hình ảnh tùy chọn và truy xuất các luồng thảo luận pool. Tất cả các hoạt động ghi yêu cầu xác thực thông qua Auth API.

## API Endpoints

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Xác thực

**Hoạt động đọc (GET):** Không yêu cầu xác thực.

**Hoạt động ghi (POST):** Yêu cầu header `ray-token` — một JWT được cấp bởi Auth API sau khi xác minh chữ ký ví.

[Lấy token từ Auth API](/vi/api-reference/launch-auth-v1/overview)

## Request Headers

Khi đăng bình luận, hãy bao gồm header xác thực:

```
ray-token: <jwt_token_from_auth_api>
```

## Tính năng

- **Đăng bình luận:** Gửi các bình luận văn bản với tệp đính kèm hình ảnh tùy chọn (tối đa 5 MB)
- **Duyệt thảo luận:** Xem tất cả các bình luận trên pool với phân trang
- **Kiểm duyệt nội dung:** Phát hiện NSFW và lọc bình luận ẩn
- **Phân trang:** Truy vấn với `lastId` và `fetchType` (old/new) để duyệt hiệu quả
- **Tải lên hình ảnh:** Bình luận hỗ trợ lưu trữ hình ảnh IPFS Pinata

## Trường hợp sử dụng

- **Thảo luận cộng đồng** xung quanh các dự án token
- **Thông báo dự án** và cập nhật thông qua bình luận
- **Phản hồi người dùng** và theo dõi tâm trạng
- **Bằng chứng xã hội** và số liệu tương tác cộng đồng
- **Kiểm duyệt nội dung** với phát hiện NSFW và spam
