Forum API обеспечивает обсуждения и взаимодействие сообщества вокруг пулов LaunchLab. Публикуйте комментарии с опциональными вложениями-изображениями и получайте ветки обсуждений по пулам. Все операции записи требуют аутентификации через Auth API.

## Конечные точки API

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Аутентификация

**Операции чтения (GET):** Аутентификация не требуется.

**Операции записи (POST):** Требуется заголовок `ray-token` — JWT, выданный Auth API после верификации подписи кошелька.

[Получить токен через Auth API](/ru/api-reference/launch-auth-v1/overview)

## Заголовки запроса

При публикации комментариев добавляйте заголовок аутентификации:

```
ray-token: <jwt_token_from_auth_api>
```

## Возможности

- **Публикация комментариев:** Отправка текстовых комментариев с опциональными изображениями (не более 5 МБ)
- **Просмотр обсуждений:** Получение всех комментариев к пулу с пагинацией
- **Модерация контента:** Обнаружение NSFW-материалов и фильтрация скрытых комментариев
- **Пагинация:** Запросы с параметрами `lastId` и `fetchType` (old/new) для эффективного просмотра
- **Загрузка изображений:** Хранение изображений к комментариям через Pinata IPFS

## Сценарии использования

- **Обсуждение в сообществе** вокруг токен-проектов
- **Анонсы проектов** и обновления через комментарии
- **Обратная связь** от пользователей и отслеживание настроений
- **Социальное доверие** и метрики вовлечённости сообщества
- **Модерация контента** с обнаружением NSFW и спама
