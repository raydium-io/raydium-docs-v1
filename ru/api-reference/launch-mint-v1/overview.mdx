Mint API предоставляет полный доступ к созданию, обнаружению и управлению токенами LaunchLab. Выполняйте запросы по названию монеты, символу или создателю, получайте рейтинги и отправляйте новые запросы на создание монет.

## API Endpoints

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Аутентификация

Операции чтения не требуют аутентификации. Операции записи (создание монеты) требуют заголовка `ray-token`, выданного Auth API после проверки подписи кошелька.

[Документация Auth API](/ru/api-reference/launch-auth-v1/overview)

## Сценарии использования

- **Поиск и обнаружение** токенов по названию или символу
- **Просмотр рейтингов** (топ-монеты по рыночной капитализации, новые, самые активные)
- **Получение статистики пользователя** (объём создания и торговли за время)
- **Запрос расписаний доступности** для информации об отпускании токенов
- **Отправка запросов на создание** монет с метаданными и изображениями
- **Отслеживание вознаграждений рефереров** из программ реферралов платформы
