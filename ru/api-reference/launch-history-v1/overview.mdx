API истории предоставляет исторические данные торговли для пулов LaunchLab. Получайте данные свечей (OHLC) с несколькими интервалами (1 минута, 5 минут, 15 минут) и доступ к отдельным записям сделок с постраничной навигацией.

## Endpoints API

**Mainnet:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## Аутентификация

Аутентификация не требуется. Все endpoints общедоступны.

## Функции данных

- **Данные Kline:** свечи OHLC с интервалами 1m, 5m и 15m
- **История сделок:** отдельные сделки с трейдером, суммой и временной меткой
- **Постраничная навигация:** используйте `nextPageKey` для эффективных запросов больших объёмов данных
- **Ограничения лимитов:** запросы kline принимают 1–500 свечей на запрос (по умолчанию 300); сделки — 1–100 (по умолчанию 50)
- **Фильтрация:** фильтруйте историю сделок по кошельку трейдера и диапазонам сумм

## Примеры использования

- **Отображение графиков** с визуализацией свечей
- **Анализ цен** и индикаторы технического анализа
- **Статистика торговли** и отслеживание объёма
- **Журналы аудита сделок пользователей** с фильтром по кошельку и диапазону дат
- **Историческое обнаружение цен** для монет
