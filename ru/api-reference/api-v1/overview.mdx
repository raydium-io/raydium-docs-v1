Это оригинальный REST API Raydium для пулов, ферм, SDK и данных токенов. Он продолжает обслуживать боевой трафик и SDK-клиентов, но теперь считается устаревшим.

## Host

| Окружение | Host               |
|-----------|------------------|
| Mainnet   | `api.raydium.io`  |
| Devnet    | Недоступно        |

## Уведомление о миграции

**Для новых интеграций используйте [API v3](/ru/api-reference/api-v3/overview).** Эндпоинты v3 предоставляют:

- Лучшую производительность и кеширование
- Упрощённые схемы ответов
- Более понятную организацию эндпоинтов
- Постоянную разработку новых функций

## Связанная документация

Смотрите [API v3](/ru/api-reference/api-v3/overview) для получения рекомендуемых альтернативных эндпоинтов. Смотрите [CLMM](/ru/products/clmm) и [CPMM](/ru/products/cpmm) для изучения структур пулов и терминологии.
