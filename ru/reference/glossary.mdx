Плоский список терминов. Используйте Ctrl-F; каждый термин ссылается на страницу с полным объяснением. Термины расположены по алфавиту в рамках буквенных разделов; концепции, родные для Solana (account, PDA и т. д.), включены, так как на них часто ссылаются.

## A

**AcceleRaytor** — оригинальная платформа для запуска токенов Raydium (2021–2023). Снята с обращения, когда её заменил LaunchLab. См. [`introduction/history-and-milestones`](/ru/introduction/history-and-milestones).

**Account** — строка данных на Solana, идентифицируемая публичным ключом, принадлежащая программе и содержащая данные и лампорты. См. [`solana-fundamentals/account-model`](/ru/solana-fundamentals/account-model).

**Address Lookup Table (ALT)** — опубликованный аккаунт, позволяющий транзакциям ссылаться на аккаунты через 1-байтовый индекс, экономя место. SDK Raydium использует ALT для многопрыжковых маршрутов. См. [`solana-fundamentals/transactions-and-fees`](/ru/solana-fundamentals/transactions-and-fees).

**AmmConfig** — аккаунт конфигурации для CPMM/CLMM, определяющий уровень комиссии и долю протокольного сбора. Несколько конфигураций могут сосуществовать; пулы ссылаются на одну при создании. См. [`ray/protocol-fees`](/ru/ray/protocol-fees).

**AMM v4** — оригинальный AMM с постоянным произведением Raydium (2021). Первоначально это был гибридный дизайн, отражавший кривую на OpenBook; интеграция с OpenBook была деактивирована, и пулы теперь работают как чистые AMM с постоянным произведением. Полностью функционален; больше не рекомендуется по умолчанию для новых пулов. См. [`products/amm-v4/overview`](/ru/products/amm-v4/overview).

**Anchor** — Rust-фреймворк для программ Solana, обрабатывающий валидацию аккаунтов, обработку ошибок и генерирующий IDL. Используется в CPMM, CLMM, Farm v6, LaunchLab. См. [`solana-fundamentals/programs-and-anchor`](/ru/solana-fundamentals/programs-and-anchor).

**Anchor discriminator** — 8-байтовый префикс в аккаунтах и инструкциях, управляемых Anchor, производный от имени аккаунта/инструкции. Идентифицирует типы аккаунтов. См. [`solana-fundamentals/account-model`](/ru/solana-fundamentals/account-model).

**Arbitrage** — использование ценовых различий между площадками (Raydium ↔ Orca ↔ CEX) для выравнивания цен. Фундаментально для обнаружения цены в AMM.

**Associated Token Account (ATA)** — условный аккаунт токена, производный от PDA, для пары (пользователь, монета). См. [`solana-fundamentals/spl-token-and-token-2022`](/ru/solana-fundamentals/spl-token-and-token-2022).

## B

**Bonding curve** — функция ценообразования токена, детерминированно устанавливающая цену как функцию предложения. Используется на этапе до завершения в LaunchLab. См. [`products/launchlab/bonding-curve`](/ru/products/launchlab/bonding-curve).

**BPF Loader (Upgradable)** — программа-загрузчик Solana, развёртывающая и обновляющая программы. См. [`solana-fundamentals/programs-and-anchor`](/ru/solana-fundamentals/programs-and-anchor).

**Bump** — один байт, добавляемый к seeds для получения PDA вне кривой. Канонический bump — это наибольшее значение, дающее адрес вне кривой. См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

## C

**Canonical bump** — bump, найденный путём убывания от 255. Программы Raydium всегда используют канонический bump. См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

**CLMM** — Concentrated Liquidity Market Maker, продукт Raydium в стиле Uniswap V3 (2022). См. [`products/clmm/overview`](/ru/products/clmm/overview).

**Compute Budget** — лимит на вычислительные единицы на транзакцию. По умолчанию 200k, максимум 1.4M. Устанавливается через `ComputeBudget::SetComputeUnitLimit`. См. [`solana-fundamentals/transactions-and-fees`](/ru/solana-fundamentals/transactions-and-fees).

**Compute Unit (CU)** — единица вычислительной стоимости на Solana. За приоритетные комиссии взимается плата за CU. См. [`integration-guides/priority-fee-tuning`](/ru/integration-guides/priority-fee-tuning).

**Constant product** — инвариант AMM `x × y = k`. См. [`algorithms/constant-product`](/ru/algorithms/constant-product).

**CPI (Cross-Program Invocation)** — вызов одной программы Solana из другой в рамках одной транзакции. См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

**CPMM** — современный AMM с постоянным произведением Raydium (2024). Заменяет AMM v4 для новых пулов. См. [`products/cpmm/overview`](/ru/products/cpmm/overview).

**Creator fee (LaunchLab)** — необязательный процент от комиссий за свопы, направляемый создателю токена во время фазы bonding curve. См. [`products/launchlab/overview`](/ru/products/launchlab/overview).

## D

**DLMM** — Dynamic Liquidity Market Maker. Вариант CLMM от Meteora. Не Raydium.

**Discriminator** — см. "Anchor discriminator".

## E

**Emissions (Farm)** — скорость распределения вознаграждений фермы в токенах в секунду. См. [`products/farm-staking/overview`](/ru/products/farm-staking/overview).

**Escrow** — аккаунт токена, находящийся в PDA, содержащий средства до выполнения условия. Общий паттерн интеграции. См. [`integration-guides/cpi-integration`](/ru/integration-guides/cpi-integration).

**Executable account** — аккаунт Solana с `executable = true`, содержащий bytecode программы. См. [`solana-fundamentals/account-model`](/ru/solana-fundamentals/account-model).

## F

**Farm v3 / v5 / v6** — последовательные поколения программы reward-farming Raydium. v6 — текущая; v3/v5 — устаревшие. См. [`products/farm-staking/overview`](/ru/products/farm-staking/overview).

**Fee growth (CLMM)** — глобальный счётчик комиссий на единицу ликвидности, отслеживаемый в `fee_growth_global_X` и `fee_growth_global_Y`. См. [`products/clmm/math`](/ru/products/clmm/math).

**Fee tier** — конкретный процент комиссии за свопы, предлагаемый пулом. Например, уровень CLMM 0.05%. См. [`reference/fee-comparison`](/ru/reference/fee-comparison).

**Finalization** — самый сильный уровень обязательства Solana, примерно через 13 секунд после приземления транзакции. См. [`solana-fundamentals/transactions-and-fees`](/ru/solana-fundamentals/transactions-and-fees).

**Freeze authority** — право заморозить (заблокировать) аккаунты токенов для данной монеты. Риск для пользователей, добавляющих ликвидность в токены с активной freeze authority.

## G

**Graduation (LaunchLab)** — событие, при котором запуск bonding curve достигает порога и автоматически переносится в пул CPMM. См. [`products/launchlab/overview`](/ru/products/launchlab/overview).

## H

**Hybrid AMM** — AMM, интегрирующийся с внешней книгой заказов для маршрутизации сделок через ту, которая предлагает лучшие цены. AMM v4 изначально был гибридным (против OpenBook), но его сторона книги заказов с тех пор была деактивирована, так что AMM v4 теперь работает как чистый AMM. CPMM не является гибридным.

## I

**IDL** — Interface Description Language. JSON-описание Anchor для инструкций, аккаунтов, событий и ошибок программы. См. [`sdk-api/anchor-idl`](/ru/sdk-api/anchor-idl).

**Impermanent loss (IL)** — разница в стоимости токенов LP в сравнении с удержанием базовых токенов из-за расхождения цен. Фундаментально для AMM. См. [`algorithms/constant-product`](/ru/algorithms/constant-product).

**Invoke / invoke_signed** — системные вызовы Solana для CPI. `invoke_signed` используется для CPI, авторизованных PDA. См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

## J

**Jito** — блочный движок Solana, поддерживающий связанные транзакции для защиты от MEV. См. [`integration-guides/routing-and-mev`](/ru/integration-guides/routing-and-mev).

**Jupiter** — доминирующий агрегатор DEX Solana. Маршрутизирует примерно 60% своего объёма через Raydium. См. [`introduction/ecosystem-position`](/ru/introduction/ecosystem-position).

## K

**Kudelski** — аудиторская фирма, проводившая аудит программ Raydium. См. [`security/audits`](/ru/security/audits).

## L

**Lamport** — наименьшая единица SOL. 1 SOL = 1 000 000 000 лампортов.

**LaunchLab** — платформа для запуска токенов Raydium (2024). Запуски bonding curve, переходящие в CPMM. См. [`products/launchlab/overview`](/ru/products/launchlab/overview).

**Liquidity L (CLMM)** — вклад позиции в ценообразование пула в единицах, определённых математикой Uniswap V3. См. [`algorithms/clmm-math`](/ru/algorithms/clmm-math).

**LP (Liquidity Provider)** — пользователь, внёсший токены в пул в обмен на LP токены (или NFT для CLMM) и право на будущие комиссии.

**LP token** — SPL токен, представляющий долю LP в пуле CPMM или AMM v4. CLMM использует NFT вместо этого.

## M

**MadShield** — аудиторская фирма, проводившая аудит программ Raydium. См. [`security/audits`](/ru/security/audits).

**Master-chef pattern** — дизайн отслеживания вознаграждений, основанный на контракте MasterChef SushiSwap. Стиль аккумулятора `reward_per_share`. Используется в Farm v6.

**Memecoin** — неформальный термин для обозначения токена, ориентированного на сообщество/спекуляцию. Многие запускаются через LaunchLab.

**MEV (Maximal Extractable Value)** — стоимость, извлекаемая при заказе транзакций (фронт-ран, сэндвич, бэк-ран). См. [`integration-guides/routing-and-mev`](/ru/integration-guides/routing-and-mev).

**Mint** — аккаунт Token Mint, определяющий токен. Не путайте с глаголом "minting" (выпуск нового предложения).

**Multisig** — аккаунт с несколькими подписями, требующий N-из-M подписантов. Raydium использует мультиподписи Squads для органов власти. См. [`security/admin-and-multisig`](/ru/security/admin-and-multisig).

## N

**Non-transferable (Token-2022)** — расширение, которое предотвращает передачу токенов. Заблокировано при создании пула Raydium.

**NFT** — Non-Fungible Token. Позиции CLMM — это NFT; то же самое касается некоторых артефактов LaunchLab.

## O

**Observation / ObservationState** — аккаунт истории цен CLMM в сети. Хранит кумулятивную цену, взвешенную по времени, позволяя создавать TWAP оракулы. См. [`products/clmm/accounts`](/ru/products/clmm/accounts).

**OpenBook** — ранее Serum. Центральная книга лимитных заказов Solana. AMM v4 *изначально* интегрировался с OpenBook, но эта интеграция с тех пор была деактивирована; CPMM и CLMM никогда с ней не интегрировались. См. [`products/amm-v4/overview`](/ru/products/amm-v4/overview).

**Oracle** — внешний источник цены. Pyth, Jupiter и (для внутреннего использования) CLMM's ObservationState TWAP. См. [`security/oracle-and-token-risks`](/ru/security/oracle-and-token-risks).

**Orca** — конкурирующий DEX Solana. Whirlpools — это продукт CLMM от Orca.

## P

**PDA (Program-Derived Address)** — адрес, детерминированно производный от программы + seeds. Без приватного ключа; подписывающая программа подписывает через `invoke_signed`. См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

**Permanent delegate (Token-2022)** — расширение, предоставляющее обозначенному кошельку безусловную власть над передачей/сжиганием. Заблокировано при создании пула Raydium.

**Phoenix** — встроенная в Solana книга заказов DEX. Партнёр, не прямой конкурент.

**PnL (AMM v4)** — исторически: прибыль и убыток от заполнений OpenBook, накопленные в пулах AMM v4 и урегулированные через `MonitorStep` cranks. С деактивированной интеграцией OpenBook никакая новая PnL книги заказов не накапливается — счётчики `state_data.total_pnl_*` содержат только исторические значения. Путь протокольной комиссии 0.03%, использующий `need_take_pnl_*`, не затронут и остаётся активным.

**Pool authority** — PDA, который подписывает трансферты хранилища пула. По одному на программу (не на пул). См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

**Pool state** — аккаунт в сети, содержащий резервы, комиссии и конфигурацию пула. Центральная структура данных на каждый продукт.

**Priority fee** — комиссия за CU, выплачиваемая валидаторам для приоритизации транзакции в блоке. См. [`integration-guides/priority-fee-tuning`](/ru/integration-guides/priority-fee-tuning).

**Program account** — аккаунт, содержащий метаданные/bytecode программы. См. [`solana-fundamentals/programs-and-anchor`](/ru/solana-fundamentals/programs-and-anchor).

**Program Derived Address** — см. PDA.

**Protocol fee** — доля комиссий за своп, направляемая на выкупы RAY или в казну, а не к LP. См. [`ray/protocol-fees`](/ru/ray/protocol-fees).

**Pyth** — сеть оракулов Solana. Используется фронтендами для отображения USD; не используется математикой пула Raydium.

## Q

**Q64.64 (sqrt_price_x64)** — фиксированное представление точки: 64 целые + 64 дробные бита. Используется CLMM для квадратного корня цены, чтобы избежать дрейфа с плавающей точкой. См. [`algorithms/clmm-math`](/ru/algorithms/clmm-math).

## R

**\$RAY** — встроенный SPL токен Raydium. См. [`ray/index`](/ru/ray).

**Realized fee (CLMM)** — комиссии, накопленные в позиции с момента последнего `collectFee`. Требуется через `collectFee`. См. [`products/clmm/math`](/ru/products/clmm/math).

**Remaining accounts** — дополнительные аккаунты, добавленные к списку аккаунтов инструкции, интерпретируемые по позиции. Используются для списков аккаунтов переменной длины (массивы тиков, дополнительные transfer-hook). См. [`solana-fundamentals/pdas-and-cpis`](/ru/solana-fundamentals/pdas-and-cpis).

**Rent** — Solana взимает плату за хранение данных, но все современные аккаунты должны быть освобождены от платы (содержать достаточно лампортов для предварительной оплаты аренды навечно). См. [`solana-fundamentals/account-model`](/ru/solana-fundamentals/account-model).

## S

**Sandwich attack** — тактика MEV: фронт-ран своп пользователя, затем бэк-ран для извлечения влияния на цену. См. [`integration-guides/routing-and-mev`](/ru/integration-guides/routing-and-mev).

**Serum** — предшественник OpenBook. В настоящее время не активен; см. "OpenBook".

**Slippage** — разница между указанным и фактическим результатом, вследствие влияния на цену + движение цены. Контролируется через параметр `min_out` при свопах.

**SOL** — встроенный токен Solana.

**SPL Token** — устаревшая программа токенов Solana. Адрес `Tokenkeg...`. См. [`solana-fundamentals/spl-token-and-token-2022`](/ru/solana-fundamentals/spl-token-and-token-2022).

**sqrt_price_x64** — см. Q64.64.

**Squads** — платформа мультиподписей, которую Raydium использует для операционных органов власти и upgrade. См. [`security/admin-and-multisig`](/ru/security/admin-and-multisig).

**Swap / SwapV2** — варианты инструкций Raydium. `SwapV2` поддерживает Token-2022; устаревший `Swap` — нет. См. [`solana-fundamentals/spl-token-and-token-2022`](/ru/solana-fundamentals/spl-token-and-token-2022).

## T

**Tick (CLMM)** — дискретный уровень цены. Позиции охватывают диапазон тиков. См. [`algorithms/clmm-math`](/ru/algorithms/clmm-math).

**Tick array** — PDA, содержащий состояние 60 последовательных тиков. Позволяет разбить хранилище решётки тиков на части. См. [`products/clmm/ticks-and-positions`](/ru/products/clmm/ticks-and-positions).

**Tick spacing** — целое число промежутка между используемыми тиками в пуле. Связано с уровнем комиссии (0.01% → 1, 0.25% → 60 и т. д.). См. [`algorithms/clmm-math`](/ru/algorithms/clmm-math).

**Timelock** — задержка между одобрением и выполнением чувствительного действия. Обновления Raydium имеют 24-часовую задержку. См. [`security/admin-and-multisig`](/ru/security/admin-and-multisig).

**Token-2022** — более новая программа токенов с расширениями. Адрес `TokenzQ...`. См. [`solana-fundamentals/spl-token-and-token-2022`](/ru/solana-fundamentals/spl-token-and-token-2022).

**Transfer fee (Token-2022)** — расширение, взимающее процент при каждой передаче токена. Учтено в CPMM/CLMM Raydium через `SwapV2`.

**Transfer hook (Token-2022)** — расширение, вызывающее программу при каждой передаче. Поддерживается Raydium с оговорками.

**TVL (Total Value Locked)** — сумма балансов пулов/ферм в USD. TVL Raydium по состоянию на апрель 2026: примерно \$1.8B.

**TWAP (Time-Weighted Average Price)** — средняя цена, взвешенная по времени, производная от CLMM ObservationState. См. [`products/clmm/accounts`](/ru/products/clmm/accounts).

## U

**UserLedger (Farm v6)** — PDA, хранящий баланс ставки пользователя и задолженность по вознаграждению на ферме.

## V

**Vault** — аккаунт токена, принадлежащий PDA органу власти программы пула, содержащий резервы пула. Каждый пул имеет два хранилища (по одному на каждую сторону).

**Versioned transaction (v0)** — текущий формат транзакции Solana, поддерживающий ALT. Устаревшие транзакции имеют версию 0xff. См. [`solana-fundamentals/transactions-and-fees`](/ru/solana-fundamentals/transactions-and-fees).

## W

**Whirlpools** — продукт CLMM от Orca. Похожий дизайн на CLMM Raydium; другая кодовая база.

**Wrapped SOL (wSOL)** — SOL, обёрнутый как SPL Token для использования в контекстах, требующих только token account. Создаётся и закрывается за транзакцию. См. [`user-flows/swap`](/ru/user-flows/swap).

## X

**x × y = k** — инвариант постоянного произведения. См. [`algorithms/constant-product`](/ru/algorithms/constant-product).
