## Для кого этот раздел

Для всех — как справочный материал. Другие разделы ссылаются *сюда*, когда упоминают ID программы, код ошибки или термин, требующий авторитетного определения.

## Инструкции по написанию

- Этот раздел — *единственное* место, где адреса программ и коды ошибок приводятся дословно. Другие страницы ссылаются отсюда.
- Держите глоссарий кратким — только термины, используемые в двух или более разделах.
- Записи журнала изменений должны быть датированы и ссылаться на затронутую(ые) страницу(ы).
