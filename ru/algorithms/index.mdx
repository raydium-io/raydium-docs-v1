## Для кого эта глава

Для исследователей протоколов, аудиторов и продвинутых интеграторов, которые хотят понять *почему* стоит та или иная математика, а не просто вызывать инструкции. На страницах продуктов даётся краткое резюме со ссылками сюда для полных выводов.

## Редакционное руководство

- Формулы в стиле LaTeX используйте экономно; представляйте формулы как блоки кода с поясняющим текстом.
- За каждой формулой следует полностью решённый числовой пример.
- На каждой странице есть секция «Где это реализовано», которая указывает на соответствующий диапазон строк в исходном коде on-chain программы для аудиторов.
- Не переводите формулы повторно на страницах продуктов — добавляйте ссылки сюда.
