При миграции токена LaunchLab в CPMM пул мигрированного пула минтит LP токены. Конфигурация платформы определяет, что произойдет с этими LP токенами.

## Распределение LP

`migrateCpLockNftScale` содержит три поля. Значения должны суммироваться в `1_000_000`, что представляет 100%.

| Поле | Значение |
|---|---|
| `platformScale` | LP токены блокируются через Burn & Earn. Платформа получает Fee Key NFT. |
| `creatorScale` | LP токены блокируются через Burn & Earn. Создатель токена получает Fee Key NFT. |
| `burnScale` | LP токены сжигаются навсегда. Для этой доли Fee Key NFT не создается. |

Это параметр уровня платформы. Токены, запущенные через одну платформу, наследуют одинаковое распределение LP до обновления этого параметра.

## Примеры

```ts
// 40% платформе, 50% создателю, 10% сжжено.
migrateCpLockNftScale: {
  platformScale: new BN(400_000),
  creatorScale: new BN(500_000),
  burnScale: new BN(100_000),
}

// 100% Fee Key создателю.
migrateCpLockNftScale: {
  platformScale: new BN(0),
  creatorScale: new BN(1_000_000),
  burnScale: new BN(0),
}

// 100% сжжено.
migrateCpLockNftScale: {
  platformScale: new BN(0),
  creatorScale: new BN(0),
  burnScale: new BN(1_000_000),
}
```

## Связь с CPMM комиссиями

`migrateCpLockNftScale` определяет **кто может получать права на LP комиссии**. Он не устанавливает процент комиссии за своп. Поведение комиссии мигрированного пула исходит из конфигурации CPMM `AmmConfig`, выбранной через `cpConfigId`.

Например, если мигрированный CPMM пул использует конфиг с 0.25% комиссией и `creatorScale` равен 500,000, Fee Key создателя представляет 50% прав на заблокированные LP комиссии. Если `platformScale` равен 400,000, Fee Key платформы представляет 40%.

## Обновление распределения

Обновления конфигурации платформы ограничены по частоте эпохой. Рассматривайте распределение LP как важную политику платформы, а не случайный параметр интерфейса.

Перед обновлением:

- Убедитесь, что три значения масштаба суммируются в `1_000_000`.
- Убедитесь, что платформа понимает, кто получит Fee Key NFT после будущих миграций.
- Задокументируйте изменение для создателей перед запуском новых токенов.
