## Что это такое

Программа Farm / Staking распределяет токены вознаграждений между учетными записями, которые ставят на кон входной токен (как правило, LP токен из AMM v4 / CPMM / CLMM, но также поддерживает стейкинг одиночных активов). Каждая ферма может выпускать несколько токенов вознаграждений по независимым графикам.

**ID программы:** см. [/ru/reference/program-addresses](/ru/reference/program-addresses). В продакшене работают несколько версий фермы (v3, v5, v6), которые отличаются слотами вознаграждений и примитивами администратора — страница [/ru/protocol-overview/versions-and-migration](/ru/protocol-overview/versions-and-migration) документирует, какую версию использовать для каких развертываний.
