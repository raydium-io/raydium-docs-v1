Raydium — самая используемая децентрализованная биржа и протокол ликвидности на Solana. Он начал работу в 2021 году как AMM с интеграцией OpenBook и вырос в полный набор услуг: классические AMM пулы, маркет-мейкеры с постоянным произведением, концентрированная ликвидность, запуски токенов и перпетуальные контракты в интерфейсе. На этой странице описано, что делает Raydium, почему он существует и куда двигаться дальше.

## В одном абзаце

Raydium Protocol — это набор открытых для всех, не требующих доверия смарт-контрактов, которые обеспечивают работу свопов, пулов, позиций ликвидности и запусков токенов на Solana. [Raydium.io](https://raydium.io) — официальное веб-приложение и универсальная платформа для работы с этими рынками: здесь можно менять токены, предоставлять ликвидность, запускать токены через LaunchLab и получать доступ к Raydium Perps из одного интерфейса. Raydium — крупнейшая по TVL DEX на Solana (примерно \$1.8B на апрель 2026 года), маршрутизирует большую часть объёма Solana через свои пулы и обеспечивает ликвидность, которая появляется в функции встроенного свопа большинства кошельков Solana.

## Основные компоненты

Основные компоненты Raydium охватывают свопы, ликвидность, запуски токенов, стимулы и перпетуальные контракты. Некоторые — это встроенные программы Raydium; Raydium Perps — интерфейс, работающий благодаря сети Orderly Network.

### AMM v4 — Маркет-мейкер с постоянным произведением

Исходный продукт Raydium (2021 год). Формула постоянного произведения (`x × y = k`) для ценообразования в пуле. Изначально он также интегрировался с центральным лимит-ордербуком OpenBook, что позволяло торговцам торговать либо через AMM, либо через книгу, но **интеграция с OpenBook была отключена** — пулы больше не делятся ликвидностью с OpenBook, и весь текущий трафик свопов проходит только через AMM кривую. Работает с объёмом примерно \$300M TVL и полностью функционален, но **больше не рекомендуется как стандартный выбор для новых пулов**: поддержка Token-2022 отсутствует, а CPMM дешевле и гибче для типичной новой пары.

См. [`products/amm-v4/overview`](/ru/products/amm-v4/overview).

### CPMM — Современный маркет-мейкер с постоянным произведением

Преемник AMM v4 (2024 год). Та же математика `x × y = k`, но проще и дешевле в использовании: нет зависимости от OpenBook, встроенная поддержка Token-2022, стоимость создания примерно в 4 раза ниже, все комиссии по свопам по умолчанию идут поставщикам ликвидности. Рекомендуемый стандартный выбор для новых пулов.

См. [`products/cpmm/overview`](/ru/products/cpmm/overview).

### CLMM — Концентрированная ликвидность

Концентрированная ликвидность в стиле Uniswap V3 (2022 год). Поставщики ликвидности выбирают ценовой диапазон; их ликвидность активна только когда цена находится в этом диапазоне. Тот же капитал → намного лучшие котировки для трейдеров → более высокая APR по комиссиям для поставщиков ликвидности ценой активного управления диапазонами. В 2026 году это доминирующий продукт для пулов с глубокой ликвидностью.

См. [`products/clmm/overview`](/ru/products/clmm/overview).

### LaunchLab — Запуски токенов

**LaunchLab** — площадка Raydium для запуска токенов (2025 год). Проекты запускают токены на bonding curve; покупатели торгуют напрямую против кривой; когда достигается порог выпуска, накопленная ликвидность кривой автоматически переходит в полноценный пул CPMM. Первичный и вторичный рынок объединены.

См. [`products/launchlab/overview`](/ru/products/launchlab/overview).

## Кто использует Raydium

- **Обычные трейдеры**: меняют токены через [raydium.io](https://raydium.io) или через встроенный своп в любом кошельке, маршрутизирующем через слой агрегирования Solana.
- **Поставщики ликвидности**: пополняют пулы для заработка на комиссиях за свопы; стейкируют на фермах для получения дополнительных вознаграждений.
- **Трейдеры перпетуальных контрактов**: торгуют перпетуальными рынками через Raydium Perps, работающий благодаря Orderly Network.
- **Команды проектов**: запускают новые токены через LaunchLab, инициализируют пулы CPMM для посизапускной ликвидности, создают фермы для стимулирования удержания поставщиков ликвидности.
- **Агрегаторы**: Jupiter и большинство сторонних маршрутизаторов используют Raydium как основную площадку ликвидности.
- **Боты и протоколы**: арбитраж, маркет-мейкинг, автокомпаундеры, ликвидации кредитных протоколов — всё это взаимодействует с Raydium через CPI или прямой RPC.

## Где появляется ликвидность Raydium

Вы уже используете Raydium, если вы:

- Используете встроенный своп в любом кошельке Solana — большинство маршрутизирует через пулы Raydium (часто через Jupiter).
- Торгуете на Jupiter — примерно 60% маршрутизированного объёма Jupiter проходит через пулы Raydium.
- Используете Drift, Kamino, MarginFi или другой DeFi протокол — многие из них получают ценовые ленты и маршруты ликвидаций через CLMM Raydium.
- Держите мемкоины Solana — многие запущены через LaunchLab и находятся в выпущенном пуле CPMM.

Агрегаторы и кошельки строят интеграции на основе открытого API Raydium (`api-v3.raydium.io`) и состояния пулов в блокчейне. Raydium не требует какого-либо специального партнёрства для интеграции — он открыт для всех.

## Ключевые преимущества

По сравнению с другими DEX на Solana:

| | Raydium | Orca | Phoenix | Jupiter |
|--|---------|------|---------|---------|
| AMM | CPMM, AMM v4 | N/A | N/A | N/A (агрегатор) |
| CLMM | CLMM | Whirlpools | N/A | N/A |
| Ордербук | Нет сегодня (интеграция OpenBook в AMM v4 отключена) | N/A | Встроенный ордербук | N/A |
| Запуски токенов | LaunchLab | WaveBreak | N/A | N/A |
| Фермы | Да | Да | N/A | N/A |
| TVL (апр 2026) | ~\$1.8B | ~\$800M | ~\$200M | Агрегатор; маршрутизирует несколько B/мес |

По сравнению с DEX на Ethereum типа Uniswap:

- **Финальность транзакций**: ~1 секунда на Solana vs ~12 секунд на Ethereum.
- **Стоимость транзакций**: \$0.001–0.01 обычно vs \$5–50 обычно на основной сети Ethereum.
- **Дизайн mempool**: Solana не имеет публичного mempool; Ethereum имеет. Это влияет на MEV, паттерны фронт-раннинга и дизайн интеграций.
- **Token-2022**: Raydium поддерживает встроенные расширения Token-2022 (комиссии за переводы, хуки и т.д.); Uniswap V3 не имеет эквивалента на ERC-20.
