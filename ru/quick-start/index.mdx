## Предварительные требования

Выполните один раз:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Необходимо иметь:

- **Node 18+** с поддержкой ESM (`"type": "module"` в `package.json` или расширение `.mjs`).
- **Keypair Solana** в `~/.config/solana/id.json` (или в другом месте — установите `KEYPAIR`).
- **Финансируемый кошелёк.** ~0.05 SOL достаточно для любого из этих скриптов.
- **RPC URL.** Для тестирования подойдёт бесплатный публичный RPC; для более серьёзного использования используйте платный провайдер.

Установите переменные окружения один раз в сессию:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Все три скрипта по умолчанию работают на mainnet. Для запуска на devnet установите `RPC_URL=https://api.devnet.solana.com` и измените `cluster: "mainnet"` на `cluster: "devnet"` в скрипте. Имейте в виду, что на devnet может быть мало ликвидности для seed пулов; возможно, вам нужно будет создать свои тестовые токены.
</Info>

## После этого раздела

Каждый скрипт выводит минимальный результат (ID пула, подпись транзакции) и одну ссылку на следующую страницу в разделе [SDK & API](/ru/sdk-api) для операции, которую он выполнил. Когда будете готовы написать собственную интеграцию:

- **Бэкенды и боты** → [Trade API](/ru/sdk-api/trade-api).
- **Фронтенды и приложения с полным контролем** → [TypeScript SDK](/ru/sdk-api/typescript-sdk).
- **On-chain композиция** → [Rust CPI](/ru/sdk-api/rust-cpi).
- **Выбор между ними** → таблица сравнения в начале раздела [SDK & API](/ru/sdk-api).
