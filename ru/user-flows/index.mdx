## Для кого этот раздел

Конечные пользователи и разработчики, которым нужен пошаговый рецепт вместо справочника. Страницы здесь уделяют приоритет *полноте одного пути* над *охватом всех вариантов*; разделы «Продукты» и «SDK» являются исчерпывающим справочником.

Каждый сценарий имеет два представления рядом, где это применимо: **Пошаговое руководство UI** (для конечных пользователей) и **Пошаговое руководство программирования** (TypeScript SDK, для разработчиков).
