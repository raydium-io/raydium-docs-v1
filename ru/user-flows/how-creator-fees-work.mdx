## Этап 1 — До миграции (кривая бондинга)

Пока ваш токен торгуется на кривой бондинга, часть комиссии создателя вычитается из каждой покупки и продажи и накапливается в хранилище для каждого токена. Это настраивается **платформой**, на которой запущен ваш токен.

### Детали

- **Номинирована в** котировочном токене (обычно SOL).
- **Ставка** — устанавливается платформой в момент запуска. Проверьте расписание комиссий платформы или состояние токена в цепи.
- **Можно получить в любой момент** — перейдите на страницу **Portfolio** на raydium.io и получите ожидающие комиссии в любой момент. Нет необходимости ждать выпуска.
- **Прекращается при выпуске** — новые комиссии до миграции не накапливаются после выпуска кривой. Любой остаток в хранилище остается доступным для получения.

### Как получить комиссию

1. Подключите кошелек создателя к [raydium.io](https://raydium.io).
2. Перейдите в **Portfolio**.
3. Найдите токен среди ваших созданных токенов.
4. Нажмите **Claim** рядом с балансом комиссии до миграции.

Программное получение: см. [`products/launchlab/creator-fees`](/ru/products/launchlab/creator-fees) для инструкции SDK.

## Этап 2 — После миграции (Fee Key NFT)

Когда кривая бондинга вашего токена достигает порога выпуска, ликвидность мигрирует в пул CPMM Raydium. LP-токены выпускаются и распределяются в соответствии с **конфигурацией платформы**:

| Получатель | По умолчанию Raydium | Описание |
|---|---|---|
| Сожжены (Burn & Earn) | 90% | Постоянно заблокированы; комиссии за торговлю с этой доли поступают в механизм доверия Burn & Earn. |
| Создатель (Fee Key NFT) | 10% | NFT Fee Key выпускается на ваш кошелек. Это NFT — ваше право на 10% текущих комиссий за торговлю LP в пуле. |
| Платформа | 0% (по умолчанию Raydium) | Некоторые сторонние платформы получают долю здесь. |

Эти значения по умолчанию применяются к токенам, запущенным через собственный интерфейс LaunchLab Raydium. Сторонние платформы могут настроить различные разделения — всегда проверяйте страницу платформы перед запуском.

### NFT Fee Key

NFT Fee Key — это неделимый токен, выпущенный на ваш кошелек при выпуске, который представляет ваше право на получение комиссий за торговлю LP из вашей доли пула после выпуска.

<Warning>
  **Не сжигайте и не передавайте NFT Fee Key, если вы не намерены отказаться от прав на получение комиссии.** Если NFT потеряется или сожжется, права на получение комиссии **безвозвратно теряются** — нет пути восстановления.
</Warning>

- Fee Key можно торговать или передавать; тот, кто его держит, может получать комиссии.
- Комиссии накапливаются непрерывно из торговой активности LP в пуле CPMM.
- Получайте со страницы Portfolio или через инструкцию получения комиссии пула CPMM программно.

### Как получить комиссии после миграции

1. Подключите кошелек, который держит NFT Fee Key.
2. Перейдите в **Portfolio** на raydium.io.
3. Найдите пул выпущенного токена среди ваших позиций.
4. Нажмите **Claim**, чтобы собрать накопленные комиссии за торговлю.

---

## Если комиссии после миграции не включены

Если платформа отключает комиссии создателя, ваш токен выпускается в **пул AMM v4** (устаревший путь миграции) вместо CPMM. В этом режиме:
- NFT Fee Key не выпускается.
- Нет комиссии создателя уровня программы после миграции.
- Некоторые альтернативные опции получения комиссии могут существовать через инструменты CLI в зависимости от того, как был настроен запуск.

## Сводка потока комиссий

```
Этап кривой бондинга:
  Каждая покупка/продажа → небольшой % → хранилище создателя (котировочный токен)
  Получайте в любой момент из Portfolio

Выпуск:
  LP-токены выпускаются → разделение по конфигурации платформы
  ├── X% сожжены (Burn & Earn)
  ├── Y% → создатель как Fee Key NFT      ← вы держите это
  └── Z% → платформа (если применимо)

После выпуска:
  Каждый обмен CPMM → комиссии LP → разделение по доле LP
  Ваша доля Fee Key NFT → получаемо из Portfolio
```
