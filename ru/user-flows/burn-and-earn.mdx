## Что такое Burn & Earn

Burn & Earn — это функция Raydium, которая позволяет создателю пула (или любому LP) сделать депозит **постоянным** в блокчейне, сохраняя право на получение торговых комиссий. Позиция становится невозвратной, но вновь созданный **Raydium Fee Key** NFT представляет постоянное право на `collectFees` из заблокированной позиции.

Основные сценарии использования:

- **Установление доверия при запуске.** Проекты обязуются не исчезать, заблокировав LP создателя. Держатели могут проверить на цепи, что ликвидность постоянна.
- **Доходный коллекционный предмет.** Торгуемый NFT, который генерирует доход. Продайте NFT; передайте поток комиссий.
- **ликвидность уровня казначейства.** Протоколы, которые хотят сохранить ликвидность навсегда для бренда / обнаружения, продолжая монетизировать комиссионную часть.

## Поддерживаемые типы пулов

| Программа | Поддержка Burn & Earn |
|---|---|
| CPMM | Да |
| CLMM | **Только полный диапазон** (рекомендуется) |
| AMM v4 (гибридный) | Не поддерживается |

Для CLMM следует заблокировать только позиции, охватывающие весь диапазон цен. После блокировки вы не сможете перебалансировать — если цена выйдет за пределы выбранного диапазона, вы навсегда перестанете получать комиссии по этой позиции. Заблокированная позиция CLMM с узким диапазоном почти всегда ошибка.

## Как работает блокировка на уровне программы

1. Исходный LP-токен (CPMM) или позиционный NFT (CLMM) переводятся в PDA, принадлежащий программе Burn & Earn.
2. **Fee Key NFT** выпускается вызывающей стороне. Его метаданные указывают на PDA заблокированной позиции.
3. Burn & Earn предоставляет `collectFees` (или эквивалентный вызов для типа пула), который:
   - Проверяет, что подписывающая сторона владеет Fee Key NFT для данной заблокированной позиции.
   - Вызывает инструкцию получения комиссии базовой программы пула через CPI.
   - Направляет собранные комиссии на ATA владельца Fee Key.
4. Нет соответствующего `withdraw` — базовый LP / позиция никогда не выходит из эскроу.

Fee Key является стандартным SPL NFT. Он может быть продан на любом рынке NFT Solana, перемещён между кошельками, использован в качестве залога в кредитных протоколах, принимающих NFT, или обёрнут в другой контракт. Кто бы ни владел им во время вызова `collectFees`, получает комиссии.

## Пошаговое руководство по интерфейсу

1. Создайте пул CPMM или пул CLMM с полноценной позицией обычным способом.
2. Перейдите на **Liquidity → Create → Burn & Earn**.
3. Выберите позицию для блокировки.
4. Проверьте детали позиции (адрес NFT, размер, пул) и введите подтверждающее предложение, которое показывает интерфейс. Это намеренно затруднено, потому что операция необратима.
5. Нажмите **Confirm, lock the liquidity permanently** (Подтвердить, постоянно заблокировать ликвидность).
6. Fee Key NFT поступает в ваш кошелёк. Он отображается на странице Portfolio под заблокированными позициями.

Получайте комиссии так же, как с любой позиции — через страницу Portfolio или через `collectFees`.

## API

Конечная точка информации о пуле возвращает поле `burnPercent`, указывающее долю ликвидности этого пула, которая была заблокирована через Burn & Earn:

```http
GET https://api-v3.raydium.io/pools/info/list
  ?poolType=standard
  &poolSortField=default
  &sortType=desc
  &pageSize=1
  &page=1
```

```json
{
  "data": [
    {
      "id": "...",
      "burnPercent": 99.17,
      ...
    }
  ]
}
```

`burnPercent` — удобный способ сигнализировать о надёжности: токен, LP которого заблокирован на 99%+, намного менее уязвим для rug-pull, чем токен с 0% блокировкой.

## Опасности

- **Необратимо.** Сторона LP никогда не может быть восстановлена. Убедитесь, что сумма — это то, к чему вы готовы больше никогда не прикасаться.
- **Узкие диапазоны CLMM — это ловушка.** Блокируйте только полноценные позиции CLMM, если вы действительно не понимаете, что позиция с ограниченным диапазоном перестаёт приносить доход, когда цена выходит за пределы диапазона.
- **Потеря Fee Key NFT означает потерю комиссий навсегда.** Относитесь к нему как к документу о праве собственности. Не сожгите его случайно; не помещайте его в враждебный контракт.
- **Минты Token-2022 с комиссией передачи**: Burn & Earn их поддерживает, но при каждом получении комиссии применяется скидка на комиссию передачи — учитывайте это в расчётах.
- **Нет частичной блокировки.** Если вы хотите 50% заблокировано и 50% подвижно, сначала разделите позицию на две позиции, затем заблокируйте только одну.
