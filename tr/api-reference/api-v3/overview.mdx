## Raydium API v3 nedir?

Raydium API v3, Solana üzerinde Raydium'un ticaret mekanları arasında likidite havuzlarını, token meta verilerini, farm'ları ve zincir istatistiklerini sorgulamak için bir REST arayüzü sağlar. Cüzdan yazılımları, agregörler, DEX yönlendiricileri ve tüccarlar tarafından kanonik havuz verilerine, fiyat akışlarına ve geçmiş analitiğine erişmek için kullanılır.

API yüksek verim için tasarlanmıştır — yanıtlar uç noktanın duyarlılığına bağlı olarak tipik olarak 1–5 dakika eski verilerdir. Tüm yanıtlar meta veri içeren tutarlı bir zarfla sarılmıştır.

## Ana konaklar

| Ortam | URL |
|-------------|-----|
| **Mainnet** | `https://api-v3.raydium.io` |
| **Devnet** | `https://api-v3-devnet.raydium.io` |

## Yanıt zarfı

Tüm uç noktalar istek meta verisi ve yanıt yükünü içeren bir JSON zarfı döndürür:

### Başarılı yanıt

```json
{
  "id": "abc123def456",
  "success": true,
  "data": {
    // Endpoint-specific data
  }
}
```

### Hata yanıtı

```json
{
  "id": "abc123def456",
  "success": false,
  "msg": "query ids type error"
}
```

`id` alanı benzersiz bir istek tanımlayıcısıdır (UUID). `data`'yı işlemeden önce her zaman `success`'i kontrol edin.

## Yaygın sorunlar

### `/pools/info/mint`'te mint çifti sıralaması

`/pools/info/mint?mint1=ABC&mint2=XYZ` sorgusunu yaptığınızda, uç nokta çifti alfabe sırasına göre sıralayarak dahili olarak normalleştirir. Tek bir mint için arama yapmanız gerekiyorsa, `mint2`'yi tamamen çıkartın — API bu mint'i içeren tüm havuzları bulacaktır.

### Havuz türü filtre değerleri

`poolType` sorgu parametresi belirli enum değerleri kabul eder:
- `all` — tüm havuz türleri
- `concentrated` — yalnızca CLMM havuzları
- `standard` — yalnızca AMM (sabit çarpım) havuzları
- `allFarm` — en az bir aktif farm'ı olan havuzlar
- `concentratedFarm` — farm'ları olan yoğunlaştırılmış havuzlar
- `standardFarm` — farm'ları olan standart havuzlar

Yazım hataları veya harf duyarlılığı uyuşmazlıkları reddedilecektir.

### Havuz sıralama alan değerleri

Geçerli `poolSortField` değerleri:
- `default` (24 saatlik hacme göre sıralar)
- `liquidity` (TVL)
- `volume24h`, `volume7d`, `volume30d`
- `fee24h`, `fee7d`, `fee30d`
- `apr24h`, `apr7d`, `apr30d`

### `/pools/info/list-v2`'de sayfalandırma

v2 uç noktası `nextPageId` aracılığıyla imleç tabanlı sayfalandırma kullanır. Donuk token yanıtta döndürülür ve aynen geri geçirilmelidir. Kendi imlecinizi oluşturmaya çalışmayın — Elasticsearch durumunu kodlar.

### Sayfa boyutu varsayılanları ve sınırları

- `/pools/info/list` ve `/pools/info/mint` `pageSize`'ı **1000**'de kaplar.
- `/farms/info/lp` `pageSize`'ı **100**'de kaplar.
- `/pools/info/list-v2` `size`'ı **1000**'de kaplar.

Sınırları aşan istekler hata ile reddedilir.

### CLMM dinamik ücret ve limit sipariş yapılandırmaları

İki yeni uç nokta, CLMM `CreateCustomizablePool` akışı için kalibrasyondaki veriler ortaya koymaktadır:

- `GET /main/clmm-dynamic-config` — `DynamicFeeConfig` seviyelerinin listesi (filtre süresi, bozunma süresi, indirgeme faktörü, azami pay, dinamik ücret kontrolü). Bu değerlerden birinin `id`'sini, `enableDynamicFee=true` olduğunda `createCustomizablePool`'a geçirin.
- `GET /main/clmm-limit-order-config` — havuz başına limit sipariş yapılandırması: limit siparişlerin etkin olup olmadığı, minimum giriş tutarı ve doldurulmuş siparişleri kapatma yetkisine sahip çevrimdışı `limit_order_admin` yöneticisi.

Cüzdan başına limit sipariş verisi (açık / doldurulmuş / kapatılmış) Temp API'de yaşar; bkz. [`api-reference/temp-api-v1/overview`](/tr/api-reference/temp-api-v1/overview).

## Örnek kullanım

**Ana istatistikleri al:**

```bash
curl -s https://api-v3.raydium.io/main/info | jq .
```

**TVL'ye göre ilk 10 havuzu listele:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/list?poolType=all&poolSortField=liquidity&sortType=desc&pageSize=10&page=1' | jq .
```

**SOL-USDC havuzlarını al:**

```bash
curl -s 'https://api-v3.raydium.io/pools/info/mint?mint1=So11111111111111111111111111111111111111112&mint2=EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v&poolType=all&poolSortField=liquidity&sortType=desc&pageSize=5&page=1' | jq .
```

**Mint fiyatlarını al:**

```bash
curl -s 'https://api-v3.raydium.io/mint/price?mints=So11111111111111111111111111111111111111112,EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v,4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R' | jq .
```

**Bir LP token'ı için farm havuzlarını al:**

```bash
curl -s 'https://api-v3.raydium.io/farms/info/lp?lp=<lp_mint>&pageSize=10&page=1' | jq .
```
