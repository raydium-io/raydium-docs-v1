Bu, pool, farm, SDK ve token verilerine yönelik Raydium'un orijinal REST API'sidir. Üretim trafiğine ve SDK istemcilerine hizmet vermeye devam etmektedir ancak artık eski sayılmaktadır.

## Host

| Ortam       | Host               |
|-------------|------------------|
| Mainnet     | `api.raydium.io`  |
| Devnet      | Kullanılamaz      |

## Geçiş bildirimi

**Yeni entegrasyonlar için [API v3](/tr/api-reference/api-v3/overview) kullanın.** v3 uç noktaları şunları sunar:

- Daha iyi performans ve önbellekleme
- Basitleştirilmiş yanıt şemaları
- Daha net uç nokta organizasyonu
- Devam eden özellik geliştirmesi

## İlgili dokümanlar

Önerilen değiştirme uç noktaları için [API v3](/tr/api-reference/api-v3/overview) bölümüne bakın. Pool yapıları ve terminoloji için [CLMM](/tr/products/clmm) ve [CPMM](/tr/products/cpmm) bölümlerine bakın.
