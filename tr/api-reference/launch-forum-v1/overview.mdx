Forum API, LaunchLab pool'ları etrafında tartışma ve topluluk katılımı sağlar. İsteğe bağlı görüntü ekleriyle açıklamalar gönderin ve pool tartışma başlıklarını alın. Tüm yazma işlemleri Auth API aracılığıyla kimlik doğrulaması gerektirir.

## API Uç Noktaları

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Kimlik Doğrulaması

**Okuma işlemleri (GET):** Kimlik doğrulaması gerekli değil.

**Yazma işlemleri (POST):** `ray-token` başlığı gerektirir — cüzdan imzası doğrulamasından sonra Auth API tarafından verilen JWT.

[Auth API'den token alın](/tr/api-reference/launch-auth-v1/overview)

## İstek Başlıkları

Açıklamalar gönderirken, kimlik doğrulama başlığını ekleyin:

```
ray-token: <jwt_token_from_auth_api>
```

## Özellikler

- **Açıklama Gönder:** İsteğe bağlı görüntü ekleriyle (maksimum 5 MB) metin açıklamaları gönderin
- **Tartışmaları Tarayın:** Bir pool'daki tüm açıklamaları pagination ile görüntüleyin
- **İçerik Denetimi:** NSFW algılaması ve gizli açıklamalar filtreleme
- **Pagination:** Verimli tarama için `lastId` ve `fetchType` (eski/yeni) ile sorgulama yapın
- **Görüntü Yükleme:** Açıklamalar Pinata IPFS görüntü barındırmasını destekler

## Kullanım Senaryoları

- Token projeleri etrafında **topluluk tartışması**
- Açıklamalar aracılığıyla **proje duyuruları** ve güncellemeleri
- **Kullanıcı geri bildirimi** ve duygu analizi
- **Sosyal kanıt** ve topluluk katılım metrikleri
- NSFW ve spam algılaması ile **içerik denetimi**
