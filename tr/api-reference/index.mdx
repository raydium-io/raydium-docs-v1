## Hizmet kataloğu

## Kimlik Doğrulama

Çoğu hizmet salt okunur ve anonim istekleri kabul eder. İki desen görülmektedir:

- **Cüzdan imzalı el sıkışma** — `launch-auth-v1` tarafından `ray-token` basılması için gerekli, ardından `launch-forum-v1` tarafından `ray-token` başlığı olarak taşınır. Cüzdanınızla `time:<unix-seconds>` formundaki bir Solana ed25519 mesajını imzalayın, imzayı + cüzdan adresini `launch-auth-v1` `/request-token` adresine gönderin, JWT'yi geri alın ve bunu sonraki forum çağrılarında `ray-token` istek başlığı olarak aktarın.
- **Kimlik doğrulama yok** — diğer her hizmet. Güvenilir arayanlar (Raydium UI'si, entegrasyonlar) uç noktalara HTTPS üzerinden doğrudan erişir.

Mintlify oyun alanı, forum isteklerini göndermeden önce kimlik doğrulama paneline bir `ray-token` yapıştırmanıza izin verir; değer yalnızca tarayıcınızda tutulur.

## Hız sınırları

Tüm sunucular, kaynak IP başına kademeli hız sınırlaması ile Cloudflare'nin arkasındadır.

Daha yüksek sınırlara ihtiyaç duyan entegrasyonlar Raydium ekibine başvurmalıdır. Yayınlanan sınırları aşan patlamalar `HTTP 429` olarak `Retry-After` başlığıyla döndürülür.

## Yanıt zarfı

Çoğu hizmet, JSON yüklerini tek biçimli bir zarfla sarar:

```json
{
  "id": "uuid-v4-per-request",
  "success": true,
  "data": { ... }
}
```

Başarısızlık durumunda:

```json
{
  "id": "uuid-v4-per-request",
  "success": false,
  "msg": "human-readable error string",
  "data": null
}
```

`api-v3`, zarfı `{ "id", "success", "data" }` olarak döndürür. pump-* hizmetleri, hatalar durumunda yalnızca `msg` doldurulmuş aynı şekli üreten `addBorderSuccess` / `addBorderErr` yardımcıları kullanır. Her hizmetin intro sayfasını kontrol ederek kullandığı tam zarfı öğrenin.

## Bu bölümü nasıl kullanacağınız

Sol gezintide herhangi bir uç noktaya tıklayın. Şunu göreceksiniz:

- HTTP yöntemi, tam URL ve temel sunucu.
- OpenAPI spec'inden çekilen türlerle parametreleri talep edin.
- Mainnet'e gerçek bir istek göndermek için parametreler ayarlamanıza ve devnet sunucusu mevcut olduğunda açılır menüden seçmenize olanak tanıyan bir **Deneyin** paneli.
- Yanıt şekli, temsili bir canlı yanıttan ve kaynak kodun `addBorderSuccess` çağrı sitesinden kaynaklanır.

SDK seviyesi erişim — işlemleri istemci tarafında oluşturma, swapları toplu işleme, TypeScript türleri — için bkz. [`sdk-api/typescript-sdk`](/tr/sdk-api/typescript-sdk).
