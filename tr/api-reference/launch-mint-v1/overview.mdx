Mint API, LaunchLab token oluşturma, keşfetme ve yönetim işlemleri için kapsamlı erişim sağlar. Mint'leri ada, sembol veya oluşturucu adına göre sorgulayın, sıralamalar alın ve yeni mint oluşturma istekleri gönderin.

## API Uç Noktaları

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Kimlik Doğrulama

Okuma işlemleri kimlik doğrulama gerektirmez. Yazma işlemleri (mint oluşturma) Auth API tarafından cüzdan imzası doğrulandıktan sonra verilen `ray-token` başlığı gerektirir.

[Auth API Dokümantasyonu](/tr/api-reference/launch-auth-v1/overview)

## Kullanım Alanları

- **Token'ları ada veya sembol ile arama ve keşfetme**
- **Sıralamalar görüntüleme** (pazar değerine göre en iyi mint'ler, en yeniler, en sıcak olanlar)
- **Kullanıcı istatistiklerini alma** (zaman içinde oluşturma ve işlem hacmi)
- **Vesting takvimlerini sorgulama** (token kilidini açma bilgileri için)
- **Mint oluşturma istekleri gönderme** (metaveri ve resimlerle birlikte)
- **Referrer ödüllerini izleme** (platform referral programlarından)
