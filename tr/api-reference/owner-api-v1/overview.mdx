Owner API, Raydium pozisyonları ve bakiyelerinin cüzdan bazlı görünümünü sağlar. Pozisyonlar, kilitli likidite, oluşturulan havuzlar ve IDO katılım verileri için önbelleğe alınmış verileri getirir ve bu da cüzdan arayüzleri ile portföy gösterge tabloları için ideal hale getirir.

## Ana Sunucular

| Ortam | Ana Sunucu                   |
|-------------|---------------------------|
| Mainnet     | `owner-v1.raydium.io`     |
| Devnet      | `owner-v1-devnet.raydium.io` |

## Genel Bakış

API, kaynak türüne göre organize edilmiş beş ana bitiş noktasını ortaya koymaktadır:

- **`/position/stake/{owner}`** — Stake pozisyonları (standart AMM / CPMM LP)
- **`/position/clmm-lock/{owner}`** — Kilitli CLMM pozisyonları
- **`/create-pool/{owner}`** — Bu cüzdan tarafından oluşturulan havuzlar
- **`/main/ido/{owner}`** — IDO katılım verileri

## Veri tazeliği

Bu hizmet, pozisyon verilerini önbelleğe alır ve periyodik olarak günceller. Son yapılan pozisyon değişiklikleri (saniyeler içinde) yanıtlarda hemen görünmeyebilir.

## Yaygın desenler

### Bir cüzdan için tüm stake pozisyonlarını getir

```bash
curl -s "https://owner-v1.raydium.io/position/stake/{owner}"
```

`{owner}` yerine cüzdanın base58 formatındaki genel anahtarını yazın.

### Kilitli pozisyonları getir

```bash
curl -s "https://owner-v1.raydium.io/position/clmm-lock/{owner}"
```

### IDO katılımını kontrol et

```bash
curl -s "https://owner-v1.raydium.io/main/ido/{owner}"
```

## Boş sonuçlar

Bir cüzdanda belirli bir bitiş noktası için veri yoksa, hizmet istek meta verileriyle birlikte 404 hatası döndürür:

```json
{
  "id": "...",
  "success": false
}
```
