## أنواع الأوامر

تدعم Raydium Perps أنواع الأوامر التالية:

| نوع الأمر | ما الذي يفعله | متى تستخدمه |
|---|---|---|
| **سوق** | يشتري أو يبيع فوراً مقابل أفضل أسعار كتاب الأوامر المتاحة. يتم إلغاء أي جزء متبقٍ غير ممتلئ إذا لم يتمكن الكتاب من ملء الحجم الكامل. | تريد تنفيذاً فورياً وتقبل الانزلاق المحتمل. |
| **حد** | يشتري أو يبيع بسعر الحد الخاص بك أو أفضل. يملأ الأمر فقط إذا وصل السوق إلى هذا السعر. | تريد التحكم في السعر ويمكنك الانتظار للملء. |
| **إيقاف سوق** | يبقى غير نشط حتى يتم الوصول إلى سعر التفعيل، ثم يصبح أمر سوق. | تريد مشغل تنفيذ وتعطي الأولوية للخروج أو الدخول بعد التفعيل. |
| **إيقاف حد** | يبقى غير نشط حتى يتم الوصول إلى سعر التفعيل، ثم يصبح أمر حد. | تريد مشغلاً لكنك تريد أيضاً سعر حد. |
| **تدريج** | يقسم أمراً أكبر إلى عدة أوامر حد أصغر عبر نطاق سعر. | تريد الدخول أو الخروج تدريجياً. |

## علامات الأوامر

تغير علامات الأوامر سلوك الأوامر ذات النمط المحدود.

| العلامة | السلوك |
|---|---|
| **IOC** | Immediate-or-cancel. يملأ أكبر قدر ممكن فوراً بسعر الحد الخاص بك أو أفضل، ثم يلغي الجزء المتبقي غير الممتلئ. |
| **FOK** | Fill-or-kill. يملأ الأمر بالكامل فوراً بسعر الحد الخاص بك أو أفضل، أو يلغي بدون ملء جزئي. |
| **Post-only** | يضيف السيولة كمصنع. إذا كان سيطابق فوراً، فسيتم إلغاؤه بدلاً من ذلك. |
| **Reduce-only** | يمنع الأمر من زيادة مركزك. يمكنه فقط تقليل أو إغلاق مركز موجود في هذا السوق. |

## فحوصات المستخدم

قبل تقديم أمر، أكد على:

- السوق والاتجاه صحيحان.
- الحجم والرافعة تطابق نيتك.
- سعر التفعيل الإيقاف وسعر الحد غير معكوسين.
- تم تفعيل reduce-only عندما تقصد فقط إغلاق أو تقليل التعرض.
- تم تفعيل post-only عندما تريد سلوك المصنع.
