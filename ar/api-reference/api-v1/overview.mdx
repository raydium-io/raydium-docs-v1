هذه هي واجهة REST API الأصلية لـ Raydium للمجموعات والمزارع وبيانات SDK والرموز. وهي تخدم حركة المرور في الإنتاج وعملاء SDK ولكنها تُعتبر الآن قديمة.

## المضيف

| البيئة    | المضيف            |
|----------|------------------|
| Mainnet  | `api.raydium.io`  |
| Devnet   | غير متاحة         |

## إشعار الترحيل

**للتكاملات الجديدة، استخدم [API v3](/ar/api-reference/api-v3/overview) بدلاً من ذلك.** تقدم نقاط نهاية v3:

- أداء أفضل وتخزين مؤقت
- مخططات استجابة مبسطة
- تنظيم نقاط النهاية الأوضح
- تطوير الميزات المستمر

## المستندات الذات الصلة

انظر [API v3](/ar/api-reference/api-v3/overview) للحصول على نقاط النهاية البديلة الموصى بها. انظر [CLMM](/ar/products/clmm) و [CPMM](/ar/products/cpmm) للحصول على هياكل المجموعات والمصطلحات.
