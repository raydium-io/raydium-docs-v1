تمكّن واجهة برمجة تطبيقات Forum النقاش والتفاعل المجتمعي حول أحواض LaunchLab. انشر التعليقات مع مرفقات صور اختيارية واسترجع مواضيع النقاش في الأحواض. تتطلب جميع عمليات الكتابة المصادقة عبر Auth API.

## نقاط النهاية (API Endpoints)

**الشبكة الرئيسية (Mainnet):** https://launch-forum-v1.raydium.io  
**شبكة الاختبار (Devnet):** https://launch-forum-v1-devnet.raydium.io

## المصادقة

**عمليات القراءة (GET):** لا تتطلب مصادقة.

**عمليات الكتابة (POST):** تتطلب رأس `ray-token` — وهو JWT صادر من Auth API بعد التحقق من توقيع المحفظة.

[احصل على الرمز من Auth API](/ar/api-reference/launch-auth-v1/overview)

## رؤوس الطلب (Request Headers)

عند نشر التعليقات، أضف رأس المصادقة:

```
ray-token: <jwt_token_from_auth_api>
```

## الميزات

- **نشر التعليقات:** أرسل تعليقات نصية مع مرفقات صور اختيارية (الحد الأقصى 5 MB)
- **تصفح النقاشات:** اعرض جميع التعليقات على حوض مع تقسيم الصفحات
- **تعديل المحتوى:** الكشف عن المحتوى غير اللائق والتعليقات المخفية
- **تقسيم الصفحات:** ابحث باستخدام `lastId` و`fetchType` (old/new) للتصفح الفعال
- **تحميل الصور:** تدعم التعليقات استضافة صور Pinata IPFS

## حالات الاستخدام

- **النقاش المجتمعي** حول مشاريع الرموز
- **إعلانات المشاريع** والتحديثات عبر التعليقات
- **تعليقات المستخدمين** وتتبع الشعور
- **الإثبات الاجتماعي** ومقاييس التفاعل المجتمعي
- **تعديل المحتوى** مع الكشف عن المحتوى غير اللائق والبريد العشوائي
