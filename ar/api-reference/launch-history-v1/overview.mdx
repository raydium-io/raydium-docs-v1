يوفر History API بيانات التداول التاريخية لمجمعات LaunchLab. يمكنك استرجاع بيانات الشموع (OHLC) بفترات زمنية متعددة (دقيقة واحدة، 5 دقائق، 15 دقيقة) والوصول إلى سجلات التداول الفردية مع تقسيم الصفحات.

## نقاط نهاية API

**الشبكة الرئيسية:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## المصادقة

لا تتطلب مصادقة. جميع نقاط النهاية متاحة للعامة.

## ميزات البيانات

- **بيانات Kline:** شموع OHLC بفترات 1 دقيقة و5 دقائق و15 دقيقة
- **سجل التداول:** التداولات الفردية مع بيانات المتداول والكمية والطابع الزمني
- **تقسيم الصفحات:** استخدم `nextPageKey` للاستعلامات الفعالة عن مجموعات البيانات الكبيرة
- **قيود الحد:** تقبل طلبات Kline من 1 إلى 500 شمعة لكل استعلام (الافتراضي 300)؛ التداولات تقبل من 1 إلى 100 (الافتراضي 50)
- **التصفية:** قم بتصفية سجل التداول حسب محفظة المتداول ونطاقات المبالغ

## حالات الاستخدام

- **عرض الرسوم البيانية** مع تصور الشموع
- **تحليل الأسعار** ومؤشرات التحليل الفني
- **إحصائيات التداول** وتتبع الحجم
- **مسارات تدقيق التداول** المصفاة حسب المحفظة ونطاق التاريخ
- **اكتشاف الأسعار التاريخية** للرموز
