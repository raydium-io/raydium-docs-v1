توفر واجهة برمجية Mint وصولاً شاملاً إلى إنشاء عملات LaunchLab واكتشافها وإدارتها. استعلم عن العملات بالاسم أو الرمز أو منشئها، واسترجع اللوحات الترتيبية، وقدّم طلبات إنشاء عملات جديدة.

## نقاط نهاية واجهة برمجية

**الشبكة الرئيسية:** https://launch-mint-v1.raydium.io  
**شبكة التطوير:** https://launch-mint-v1-devnet.raydium.io

## المصادقة

لا تتطلب عمليات القراءة مصادقة. تتطلب عمليات الكتابة (إنشاء العملات) رأس `ray-token` صادر عن واجهة برمجية Auth بعد التحقق من توقيع المحفظة.

[توثيق واجهة برمجية Auth](/ar/api-reference/launch-auth-v1/overview)

## حالات الاستخدام

- **البحث واكتشاف** العملات حسب الاسم أو الرمز
- **عرض اللوحات الترتيبية** (أفضل العملات حسب رأس المال السوقي والأحدث والأكثر حرارة)
- **استرجاع إحصائيات المستخدم** (حجم الإنشاء والتداول عبر الزمن)
- **الاستعلام عن جداول الاستحقاق** للحصول على معلومات فتح العملات
- **تقديم طلبات إنشاء عملات** مع البيانات الوصفية والصور
- **تتبع مكافآت المُحيل** من برامج الإحالة بالنظام الأساسي
