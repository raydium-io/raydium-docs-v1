## لمن كتب هذا الفصل

الفرق التي تبني *فوق* Raydium — المجمعات، المحافظ، برامج التداول المخصصة، برامج التعدين الآلي. إذا كنت مستخدماً نهائياً لـ Raydium، فانتقل إلى [تدفقات المستخدم](/ar/user-flows) بدلاً من ذلك.

## ملخص الكتابة

- يبدأ كل دليل بـ "هذه هي المشكلة" (على سبيل المثال، "معاملات الـ swap الخاصة بي تفشل تحت الحمل") وينتهي برسفة مثبتة.
- أدرج المقاييس حيث يكون ذا مغزى (على سبيل المثال، استخدام CU النموذجي لـ swap CPMM مقابل swap CLMM).
- ارتبط بـ SDK والـ API للبدائل الأساسية؛ لا تعيد تعريفها.
