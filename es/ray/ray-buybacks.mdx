## Flujo de un vistazo

```text
Comisión de swap
  -> Parte para proveedores de liquidez
  -> Parte para recompra de RAY
  -> Parte para tesorería, cuando corresponda
```

Para los pools CLMM y CPMM, la distribución de comisiones de negociación es:

```text
84% para proveedores de liquidez
12% para recompras de RAY
4% para tesorería
```

Para los pools Standard AMM v4, la distribución de comisiones de negociación es:

```text
88% para proveedores de liquidez
12% para recompras de RAY
```

## Qué significa la parte de recompra

El porcentaje de recompra se aplica a la **comisión de negociación**, no al monto completo del intercambio.

Ejemplo: un pool cobra una comisión de swap de `0,25%`. La parte de recompra es `12%` de esa comisión.

```text
Monto del intercambio:      1.000 USDC
Comisión de swap:           1.000 * 0,25% = 2,50 USDC
Parte para recompra de RAY: 2,50 * 12% = 0,30 USDC
```

El monto efectivo de recompra es `0,03%` del monto del intercambio en este ejemplo.

## Direcciones de recopilación

Las comisiones del protocolo se recopilan mediante direcciones específicas del programa antes de convertirse o distribuirse.

| Fuente | Dirección de recopilación |
|---|---|
| CLMM | `projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416` |
| CPMM | `ProCXqRcXJjoUd1RNoo28bSizAA6EEqt9wURZYPDc5u` |
| Standard AMM v4 | `PNLCQcVCD26aC7ZWgRyr5ptfaR7bBrWdTFgRWwu2tvF` |

Las direcciones de recopilación pueden contener muchos tokens diferentes. Cada pool paga comisiones en los tokens que negocia, por lo que una dirección de recopilación puede contener USDC, wSOL, RAY u otros tokens SPL antes de distribuirse.

## Dirección de depósito de recompras

El RAY recomprado se mantiene en:

```text
DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

Puedes inspeccionar esta dirección directamente en Solscan o con herramientas de tokens SPL:

```bash
spl-token accounts --owner DdHDoz94o2WJmD9myRobHCwtx1bESpHTd4SSPe6VEZaz
```

## Por qué las recompras están en cadena

El flujo de recompra de Raydium es auditable desde direcciones públicas e historial de transacciones.

- **Los saldos son visibles:** cualquiera puede inspeccionar las direcciones de recopilación y depósito.
- **Las transferencias son trazables:** las transferencias de comisiones entrantes y las transacciones de distribución salientes son públicas.
- **No se requiere contabilidad privada:** la cadena es el libro mayor del flujo.

## Cómo verificar un flujo de recompra

1. Abre una transacción de swap de Raydium en Solscan o Solana Explorer.
2. Identifica el programa de pool y las transferencias de comisiones.
3. Relaciona el movimiento de comisiones del protocolo con la dirección de recopilación relevante.
4. Inspecciona las transacciones salientes desde la dirección de recopilación.
5. Confirma la acumulación de RAY en `DdHDoz...VEZaz`.

Para el mapa de direcciones completo, consulta [`ray/treasury`](/es/ray/treasury).
