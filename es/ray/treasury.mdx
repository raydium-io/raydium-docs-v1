Esta página lista las direcciones públicas utilizadas para la recopilación de comisiones del protocolo Raydium, recompras de RAY, saldos de tesorería en USDC y comisiones de creación de pools en SOL. Inspecciona los saldos y transferencias en la cadena antes de usarlas para contabilidad.

## Flujo de un vistazo

```text
Comisiones de pool
  -> Dirección de recopilación específica del programa
      -> Ruta de recompra de RAY
      -> Ruta de tesorería en USDC, cuando corresponda

Comisiones de creación de pool
  -> Dirección de comisión en SOL específica del programa
```

Todas las direcciones en este flujo son públicas e inspeccionables.

## Direcciones de recopilación de comisiones

Cuando un swap paga una comisión del lado del protocolo, el programa relevante acumula esa comisión en una dirección de recopilación.

| Fuente | Dirección de recopilación |
|---|---|
| CLMM | `projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416` |
| CPMM | `ProCXqRcXJjoUd1RNoo28bSizAA6EEqt9wURZYPDc5u` |
| AMM estándar v4 | `PNLCQcVCD26aC7ZWgRyr5ptfaR7bBrWdTFgRWwu2tvF` |

Las direcciones de recopilación pueden contener un conjunto mixto de tokens SPL. Por ejemplo, las comisiones de pools SOL/USDC pueden producir saldos de wSOL y USDC, mientras que los pools SOL/RAY pueden producir saldos de wSOL y RAY.

Inspecciona una dirección de recopilación con:

```bash
spl-token accounts --owner projjosVCPQH49d5em7VYS7fJZzaqKixqKtus7yk416
```

## Cómo auditar un flujo de comisiones

Cualquiera puede verificar el flujo de comisiones de Raydium a partir de datos de transacciones públicas:

1. Extrae una transacción de intercambio de un pool de Raydium.
2. Identifica el programa del pool: CLMM, CPMM o AMM estándar v4.
3. Inspecciona la contabilidad de comisiones y las transferencias de tokens en la transacción.
4. Relaciona el movimiento del lado del protocolo con la dirección de recopilación relevante.
5. Rastrea transacciones salientes desde la dirección de recopilación.
6. Relaciona la acumulación de RAY con `DdHDoz...VEZaz` o los saldos de USDC con las direcciones de tesorería anteriores.
