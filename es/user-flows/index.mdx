## Para quién es este capítulo

Usuarios finales y desarrolladores que buscan una receta paso a paso en lugar de una referencia. Las páginas aquí priorizan la *completitud de una ruta* sobre la *cobertura de todas las opciones*; los capítulos de Productos y SDK son la referencia exhaustiva.

Cada flujo tiene dos vistas lado a lado cuando es aplicable: **recorrido por la UI** (para usuarios finales) y **recorrido programático** (TS SDK, para desarrolladores).
