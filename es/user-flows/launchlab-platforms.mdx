<Info>
  Una plataforma de LaunchLab es una configuración en cadena que utilizan una superficie de lanzamiento o una comunidad. La configuración de la plataforma puede afectar las comisiones, la marca, el soporte de comisiones del creador y el comportamiento del pool después de la migración.
</Info>

LaunchLab permite entornos de lanzamiento de terceros a través de Platform PDAs. Una plataforma puede ofrecer su propia experiencia de lanzamiento de tokens personalizada mientras utiliza la infraestructura del programa LaunchLab y la migración de pools de Raydium.

## Qué puede controlar una plataforma

| Configuración | Por qué importa |
|---|---|
| Soporte de comisiones del creador | Determina si los creadores de tokens pueden ganar comisiones antes y después de la migración. |
| Comisiones de plataforma | Establece dónde se envían los ingresos de comisiones a nivel de plataforma. |
| Metadatos de marca | Le da a la plataforma un nombre, descripción y logotipo que se pueden mostrar en las interfaces. |
| Agrupación de lanzamientos | Permite que la UI muestre tokens lanzados bajo la misma plataforma. |
| Configuración del pool post-migración | Puede definir la configuración de comisiones CPMM cuando la plataforma migra lanzamientos a CPMM. |
| Distribución de LP | Puede afectar cómo los tokens LP post-migración se queman, se asignan a creadores o se asignan a la plataforma. |

## Lo que deben verificar los creadores

Antes de lanzarte bajo cualquier plataforma, confirma:

- Qué wallet recibe las comisiones de plataforma.
- Si las comisiones del creador están habilitadas.
- Si la liquidez post-migración va a CPMM o AMM v4.
- Cómo se dividen los tokens LP después de la graduación.
- Si se acuñará un NFT de Fee Key y quién lo recibe.
- Si la plataforma tiene sus propios términos, políticas o reglas de elegibilidad.

<Warning>
  La configuración de la plataforma puede cambiar significativamente la economía del creador y las expectativas del usuario. Revísalas antes de firmar la transacción de lanzamiento. Esta página solo explica la mecánica y no constituye asesoramiento legal, financiero ni de inversión.
</Warning>

## Lo que deben verificar los traders

Cuando compres un token de LaunchLab, la plataforma puede afectar la ruta de comisiones y el manejo de liquidez post-migración. Antes de operar:

- Verifica la plataforma del token y la configuración de lanzamiento.
- Confirma el tipo de pool post-migración si la UI lo muestra.
- Verifica si el LP se quema, se bloquea o se asigna.
- Verifica el mint del token desde la página del token, no solo desde el texto de redes sociales.
