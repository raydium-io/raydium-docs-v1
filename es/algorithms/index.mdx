## Guía de estilo para la redacción

- Usa notación matemática al estilo LaTeX con moderación; muestra las fórmulas como bloques de código acompañados de prosa en línea.
- Cada fórmula va seguida de un ejemplo numérico resuelto.
- Cada página incluye una sección «Dónde está implementado» que apunta al rango de líneas correspondiente en el código fuente del programa on-chain, para los auditores.
- No vuelvas a derivar las mismas fórmulas en las páginas de producto — enlaza aquí.
