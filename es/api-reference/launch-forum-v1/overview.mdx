El API Forum permite discusiones y participación comunitaria alrededor de los pools de LaunchLab. Publica comentarios con adjuntos de imágenes opcionales y recupera hilos de discusión de pools. Todas las operaciones de escritura requieren autenticación a través del API de autenticación.

## Endpoints de API

**Mainnet:** https://launch-forum-v1.raydium.io  
**Devnet:** https://launch-forum-v1-devnet.raydium.io

## Autenticación

**Operaciones de lectura (GET):** No se requiere autenticación.

**Operaciones de escritura (POST):** Requiere encabezado `ray-token` — un JWT emitido por el API de autenticación tras verificación de firma de billetera.

[Obtén token del API de autenticación](/es/api-reference/launch-auth-v1/overview)

## Encabezados de solicitud

Cuando publiques comentarios, incluye el encabezado de autenticación:

```
ray-token: <jwt_token_from_auth_api>
```

## Características

- **Publicar comentarios:** Envía comentarios de texto con adjuntos de imágenes opcionales (máx. 5 MB)
- **Explorar discusiones:** Ver todos los comentarios en un pool con paginación
- **Moderación de contenido:** Detección de contenido NSFW y filtrado de comentarios ocultos
- **Paginación:** Consulta con `lastId` y `fetchType` (old/new) para exploración eficiente
- **Carga de imágenes:** Los comentarios soportan alojamiento de imágenes IPFS en Pinata

## Casos de uso

- **Discusión comunitaria** alrededor de proyectos de tokens
- **Anuncios de proyectos** y actualizaciones mediante comentarios
- **Retroalimentación de usuarios** y seguimiento de sentimiento
- **Prueba social** y métricas de participación comunitaria
- **Moderación de contenido** con detección de NSFW y spam
