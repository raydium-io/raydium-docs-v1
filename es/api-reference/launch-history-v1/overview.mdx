El API de Historial proporciona datos de operaciones históricas para pools de LaunchLab. Recupera datos de velas (OHLC) en múltiples intervalos (1 minuto, 5 minutos, 15 minutos) y accede a registros de operaciones individuales con paginación.

## Endpoints de la API

**Mainnet:** https://launch-history-v1.raydium.io  
**Devnet:** https://launch-history-v1-devnet.raydium.io

## Autenticación

No se requiere autenticación. Todos los endpoints son públicamente accesibles.

## Características de datos

- **Datos de Kline:** Velas OHLC en intervalos de 1m, 5m y 15m
- **Historial de Operaciones:** Operaciones individuales con información de trader, cantidad e instante
- **Paginación:** Usa `nextPageKey` para consultas eficientes en grandes conjuntos de datos
- **Límites de Restricción:** Las solicitudes de kline aceptan 1-500 velas por consulta (predeterminado 300); las operaciones aceptan 1-100 (predeterminado 50)
- **Filtrado:** Filtra el historial de operaciones por billetera de trader y rangos de cantidad

## Casos de uso

- **Visualización de gráficos** con representación de velas
- **Análisis de precios** e indicadores de análisis técnico
- **Estadísticas de operaciones** y seguimiento de volumen
- **Registros de auditoría** de operaciones del usuario filtrados por billetera e intervalo de fechas
- **Descubrimiento de precios históricos** para mints
