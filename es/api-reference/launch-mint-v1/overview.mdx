El API de Mint proporciona acceso integral a la creación, descubrimiento y gestión de tokens en LaunchLab. Consulta mints por nombre, símbolo o creador, recupera leaderboards y envía solicitudes de creación de nuevos mints.

## Endpoints del API

**Mainnet:** https://launch-mint-v1.raydium.io  
**Devnet:** https://launch-mint-v1-devnet.raydium.io

## Autenticación

Las operaciones de lectura no requieren autenticación. Las operaciones de escritura (creación de mints) requieren un encabezado `ray-token` emitido por el Auth API después de la verificación de firma de la cartera.

[Documentación del Auth API](/es/api-reference/launch-auth-v1/overview)

## Casos de uso

- **Buscar y descubrir** tokens por nombre o símbolo
- **Ver leaderboards** (mints principales por capitalización de mercado, más recientes, más candentes)
- **Recuperar estadísticas del usuario** (volumen de creación y trading a lo largo del tiempo)
- **Consultar cronogramas de vesting** para información de desbloqueo de tokens
- **Enviar solicitudes de creación de mints** con metadatos e imágenes
- **Rastrear recompensas de referidor** desde programas de referencia de la plataforma
