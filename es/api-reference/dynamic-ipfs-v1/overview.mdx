El API IPFS Dinámico sirve metadatos e imágenes regeneradas para posiciones de liquidez CLMM y CPMM en Solana. Este servicio está diseñado para dApps que renderizan NFTs dinámicos —como tarjetas de posiciones de liquidez concentrada— proporcionando información del pool en tiempo real, valuación de posiciones y comisiones sin cobrar, sin necesidad de búsquedas en cadena.

## Hosts

| Entorno    | Host                               |
|------------|-----------------------------------|
| Mainnet    | `dynamic-ipfs.raydium.io`         |
| Devnet     | `dynamic-ipfs-devnet.raydium.io`  |

## Descripción general

El API expone tres puntos de acceso, cada uno devolviendo un objeto de interfaz de posición completa:

- **`/clmm/position`** — Obtener metadatos para una posición CLMM activa
- **`/lock/clmm/position`** — Obtener metadatos para una posición CLMM bloqueada
- **`/lock/cpmm/position`** — Obtener metadatos para una posición CPMM bloqueada

Cada respuesta incluye los detalles completos del pool (tokens, decimales, TVL), montos de la posición, valuación en USD, porcentaje de TVL dentro del pool, y comisiones y recompensas acumuladas (pero sin cobrar).

## Casos de uso

Genera visualizaciones dinámicas de posiciones como NFTs solicitando metadatos para una posición específica. El API calcula el estado de la posición en tiempo real a partir de datos en cadena y almacena en caché los resultados, lo que lo hace ideal para:

- Renderizar tarjetas de posición con valor USD en directo y acumulación de comisiones
- Mostrar el porcentaje de TVL y montos de tokens en el pool
- Mostrar comisiones de negociación sin cobrar y tokens de recompensa
- Construir paneles de cartera que rastrean el rendimiento de las posiciones

## Autenticación

No se requiere autenticación. Todos los puntos de acceso son públicos.

## Límite de velocidad

Ninguno aplicado. Servicio de mejor esfuerzo.

## Patrones comunes

### Obtener una posición CLMM

```bash
curl -s "https://dynamic-ipfs.raydium.io/clmm/position?id=4Ygp92zRvl4kCr97V3Zs5xg4k3qX1tN6m8pR2vL9jA1"
```

El parámetro `id` debe ser la clave pública en cadena de la cuenta de posición (una dirección Solana en formato base58).

### Obtener una posición bloqueada

```bash
curl -s "https://dynamic-ipfs.raydium.io/lock/clmm/position?id=<position_pda>"
```
