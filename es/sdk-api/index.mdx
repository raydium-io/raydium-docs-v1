## Para quién es este capítulo

Desarrolladores que integran Raydium desde fuera de la cadena (TS/JS, Python, backends) o desde otro programa de Solana (Rust CPI). También agentes de IA e integradores de herramientas de codificación — consulta [Integración con IA](/es/sdk-api/ai-integration).

<Info>
  **¿Quieres ejecutar algo primero?** Tres scripts listos para usar — swap, desplegar pool CPMM, desplegar pool CLMM — se encuentran en [Inicio rápido](/es/quick-start). Ejecuta una transacción real en cinco minutos y luego vuelve aquí para la referencia completa.
</Info>

## Elige tu ruta de integración

Cinco rutas para elegir. La correcta depende de qué está llamando a Raydium y cuánto control necesitas.

| Ruta | Mejor para | Latencia | Lógica personalizada | Lenguajes |
|---|---|---|---|---|
| [Trade API](/es/sdk-api/trade-api) | Backends, bots, Blinks — en cualquier lugar donde quieras construir una tx desde el servidor | Mínima | Limitada (rutas preestablecidas) | Cualquiera |
| [TypeScript SDK](/es/sdk-api/typescript-sdk) | Aplicaciones Node/TS y frontends que necesitan acceso completo al constructor | Media | Completa | TS / JS |
| [Integración Python](/es/sdk-api/python-integration) | Backends de Python y canales de ML | Media | Completa (construcción de instrucciones manual) | Python |
| [Rust CPI](/es/sdk-api/rust-cpi) | Programas de composición que llaman a Raydium desde la cadena | N/A (en cadena) | Completa + componible | Rust |
| [IDL de Anchor](/es/sdk-api/anchor-idl) | Generación de código para lenguajes sin soporte de primera parte (Go, Kotlin, Swift, ...) | Variable | Completa | Cualquiera con herramienta de generación de código Anchor |

Si no estás seguro: backends y bots → **Trade API**; frontends y aplicaciones de control total → **TypeScript SDK**; composición en cadena → **Rust CPI**.
