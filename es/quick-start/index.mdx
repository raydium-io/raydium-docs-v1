## Requisitos previos

Ejecuta esto una sola vez:

```bash
mkdir raydium-quickstart && cd raydium-quickstart
npm init -y
npm install @raydium-io/raydium-sdk-v2 @solana/web3.js @solana/spl-token bn.js
```

Necesitas:

- **Node 18+** con soporte ESM (`"type": "module"` en `package.json` o extensión `.mjs`).
- **Un keypair de Solana** en `~/.config/solana/id.json` (o en cualquier lugar — establece `KEYPAIR`).
- **Una wallet financiada.** ~0.05 SOL es suficiente para cualquier script de este capítulo.
- **Una URL RPC.** RPC público gratuito funciona para pruebas; para algo más que una prueba rápida, usa un proveedor de pago.

Establece estas variables una vez por sesión de shell:

```bash
export RPC_URL="https://api.mainnet-beta.solana.com"
export KEYPAIR="$HOME/.config/solana/id.json"
```

<Info>
  **Cluster.** Los tres scripts funcionan por defecto en mainnet. Para ejecutar en devnet, establece `RPC_URL=https://api.devnet.solana.com` y cambia `cluster: "mainnet"` a `cluster: "devnet"` dentro del script. Ten en cuenta que la liquidez inicial de pools en devnet es escasa; espera tener que acuñar tus propios tokens de prueba.
</Info>

## Después de este capítulo

Cada script termina con una salida mínima (ID del pool, firma de transacción) y una sola línea apuntando a la siguiente página del capítulo [SDK & API](/es/sdk-api) para la operación que realizó. Cuando estés listo para escribir tu propia integración:

- **Backends y bots** → [Trade API](/es/sdk-api/trade-api).
- **Frontends y aplicaciones de control total** → [TypeScript SDK](/es/sdk-api/typescript-sdk).
- **Composición en cadena** → [Rust CPI](/es/sdk-api/rust-cpi).
- **Elegir entre ellos** → la tabla de comparación en la parte superior de [SDK & API](/es/sdk-api).
